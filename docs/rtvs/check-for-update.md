# <a name="rtvs-current-version"></a>RTVS aktuální verze

Verzi můžete zkontrolovat tak, že v sadě Visual Studio přejdete na > **nápovědu** a přejdete dolů na položku **Nástroje jazyka R pro visual studio:** **Help**

![Zkoumání rtvs verze v sadě Visual Studio (VS2017)](media/current-version.png)

Pokud chcete aktualizovat verzi RTVS ve Visual Studiu 2017, přejděte na instalační program Visual Studia a vyberte tlačítko **Aktualizovat.**

Pro Visual Studio 2015 nejnovější instalační program je k dispozici od [aka.ms/rtvs-current](https://rtvs.blob.core.windows.net/download/RTVS_2017-12-18.1.exe) (přímé stahování).

Viz [Instalace](installing-r-tools-for-visual-studio.md) pro visual studio a požadavky na Windows.

Aktualizace na Microsoft R Open a Microsoft Machine Learning Server (dříve Microsoft R Server) naleznete v tématu: [Produkty Společnosti Microsoft R.](https://azure.microsoft.com/?ocid=cloudplat_hp)
