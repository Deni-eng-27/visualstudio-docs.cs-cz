# <a name="rtvs-current-version"></a>Aktuální verze RTVS

Od poslední aktualizace na tuto stránku nejnovější verzi R nástrojů pro Visual Studio (RTVS) iss **1.0.31108.1213**. Vaše verze se může lišit v závislosti na vaší verzi sady Visual Studio a zda používáte verzi preview kanál.

Vaši verzi můžete zkontrolovat tak, že přejdete do **pomoci > o** v sadě Visual Studio a posouvání dolů na **R Tools pro Visual Studio** položku:

![Zkoumání RTVS verze v sadě Visual Studio (VS2017)](media/current-version.png)

Chcete-li aktualizujte svou verzi RTVS v Visual Studio 2017, přejděte do instalačního programu sady Visual Studio a vyberte **aktualizace** tlačítko.

Pro Visual Studio 2015, je k dispozici v nejnovější instalační program [aka.ms/rtvs aktuální](https://aka.ms/rtvs-current) (přímé ke stažení).

V tématu [instalace](installing-r-tools-for-visual-studio.md) požadavky sady Visual Studio a systému Windows.

Aktualizace Microsoft R otevřete a Microsoft R Server, najdete v tématu: [Microsoft R produkty.](http://aka.ms/rtvs-msft-r)
