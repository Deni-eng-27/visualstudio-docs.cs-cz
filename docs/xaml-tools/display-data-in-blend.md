# <a name="display-data-in-blend-for-visual-studio"></a>Zobrazit data v Blend pro Visual Studio

Ukázková data můžete zobrazit v návrháři při přizpůsobování rozložení stránek. Můžete vygenerovat ukázková data od začátku nebo pomocí existující třídy. Můžete se také připojit k *živým datům* , která se zobrazí v aplikaci při jejím spuštění.

## <a name="generate-sample-data"></a>Generování ukázkových dat

Chcete-li generovat ukázková data, otevřete dokument XAML. Na panelu **data** zvolte ikonu **Vytvořit ukázková data** ![Create Sample data ](../designers/media/30540d76-7256-43ce-b5d9-4b2edf3d339f.png) a pak zvolte **Nová vzorová data**.

Na panelu **data** definujte strukturu vašich dat a potom ji navažte na prvky uživatelského rozhraní na libovolné stránce.

![Panel data](../designers/media/496d7ebc-fe46-42f6-95a8-57b0e5be5d49.png)

Pokud chcete, aby se vaše ukázková data zobrazovala na stránkách při spuštění aplikace, zvolte **Možnosti zdroje dat** ![Data ikonu možnosti zdroje ](../designers/media/ae1fd260-4f84-420d-b196-45fde357d81d.png) a pak zvolte **Povolit při spuštění aplikace**.

![Povolit při spuštění položky nabídky aplikace](../designers/media/05d5356d-91bb-4e6b-b3f7-29b76852c4b3.png)

**Podívejte se na krátké video:** ikona ![Play ](../designers/media/bldadminconsoleinitialconfigicon.PNG) [Vytvoření ukázkových dat od začátku](http://www.bing.com/videos/search?q=blend%20data&qs=n&form=QBVR&pq=blend%20data&sc=8-7&sp=-1&sk=#view=detail&mid=F8F2449A76956D480FD2F8F2449A76956D480FD2).

## <a name="generate-sample-data-from-a-class"></a>Generování ukázkových dat z třídy

Pokud jste již vytvořili třídy, které popisují strukturu vašich dat, můžete z nich vygenerovat ukázková data.

Chcete-li generovat ukázková data z třídy, otevřete dokument XAML a poté na panelu **data** klikněte na ikonu **Vytvořit ukázková** data ![Create Sample data ](../designers/media/30540d76-7256-43ce-b5d9-4b2edf3d339f.png) a potom klikněte na možnost **Vytvořit Ukázková data z třídy**.

**Podívejte se na krátké video:** ikona ![Play ](../designers/media/bldadminconsoleinitialconfigicon.PNG) [kombinace některých datových vazeb pomocí Blendu](https://www.youtube.com/watch?v=LSwPB6CAvjg).
