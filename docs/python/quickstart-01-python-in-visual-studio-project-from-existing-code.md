# <a name="quickstart-create-a-python-project-from-existing-code"></a>Úvodní příručka: Vytvoření projektu Pythonu z existujícího kódu

Po instalaci [podpory Pythonu v sadě Visual Studio](installing-python-support-in-visual-studio.md)je snadné přenést existující kód Pythonu do projektu Sady Visual Studio.

> [!Important]
> Zde popsaný proces nepřesouvá ani nekopíruje původní zdrojové soubory. Pokud chcete pracovat s kopií, nejprve ji duplikujte.

[!INCLUDE[project-from-existing](includes/project-from-existing.md)]

V případě potřeby spusťte program stisknutím **klávesy Ctrl**+**F5** nebo výběrem **možnosti Ladění** > **start bez ladění**.
