# <a name="how-to-manage-build-configurations-with-visual-basic-developer-settings-applied"></a>Postup: Správa konfigurací sestavení s použitým nastavením vývojáře jazyka Visual Basic

Ve výchozím nastavení jsou všechny rozšířené možnosti konfigurace sestavení skryté při použití nastavení vývojáře jazyka Visual Basic. Tento článek vysvětluje, jak ručně povolit tato nastavení sestavení.

## <a name="enable-advanced-build-configurations"></a>Povolení pokročilých konfigurací sestavení

Ve výchozím nastavení skryje nastavení vývojáře jazyka Visual Basic možnost otevření dialogového okna **Správce konfigurace** a seznamy **Konfigurace** a **platformy** v [Návrháři projektu](../ide/reference/application-page-project-designer-visual-basic.md).

1. V nabídce **Tools** (Nástroje) klikněte na **Options** (Možnosti).

2. Rozbalte **položku Projekty a řešení**a klepněte na tlačítko **Obecné**.

    > [!NOTE]
    > **Obecný** uzel je viditelný i v případě, že není zaškrtnuta možnost **Zobrazit všechna nastavení.** Pokud chcete zobrazit všechny dostupné možnosti, klikněte na **Zobrazit všechna nastavení**.

3. Klepněte na tlačítko **Zobrazit rozšířené konfigurace sestavení**.

4. Klikněte na tlačítko **OK**.

     **Nástroj Configuration Manager** je nyní k dispozici v nabídce **Sestavení** a seznamy **Konfigurace** a **Platformy** jsou viditelné v **Návrháři projektu**.
