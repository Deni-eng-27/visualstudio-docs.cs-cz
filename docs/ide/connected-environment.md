# <a name="internet-resources-used-by-visual-studio"></a>Sada Visual Studio prostředků z Internetu

Visual Studio se připojeném prostředí:

- Můžete [přihlášení](../ide/signing-in-to-visual-studio.md) k sadě Visual Studio pro povolení přístupu ke všem službám, které jsou spojené s vaším uživatelským účtem, jakož i všechny účty, které jsou propojeny k tomuto účtu. Nastavení svého účtu přesouvat s vámi, bez ohledu na použijte sadu Visual Studio.  
- Visual Studio [upozorní vás](../ide/visual-studio-notifications.md) při všech součástí instalace, jako je například sady SDK je k dispozici aktualizace.  
- Můžete si stáhnout [NuGet](/nuget/) balíčky k získání knihovny.  
- Visual Studio si můžete stáhnout [rozšíření](../ide/finding-and-using-visual-studio-extensions.md).  
- Můžete připojit ke službám Azure.  
- Úvodní stránka se aktualizuje nejnovější zprávy vývojáře z různých webů.  
- Atd.  

Témata v této části se týkají různé způsoby Visual Studio použije prostředků z Internetu.
