# <a name="personalize-the-visual-studio-ide"></a>Přizpůsobení prostředí IDE sady Visual Studio

Visual Studio můžete přizpůsobit různými způsoby, abyste co nejlépe podpořili vlastní styl vývoje a požadavky. Mnoho nastavení se s vámi přemisíluje mezi instancemi&mdash;sady Visual Studio, viz [Synchronizované nastavení](../ide/synchronized-settings-in-visual-studio.md). Tento článek stručně popisuje různá individuální nastavení a kde najdete další informace.

> [!NOTE]
> Toto téma platí pro Visual Studio v systému Windows. Visual Studio pro Mac najdete [v tématu Přizpůsobení ide Visual Studia pro Mac](/visualstudio/mac/customizing-the-ide).

## <a name="default-settings"></a>Výchozí nastavení

Můžete zvolit výchozí kolekci nastavení, která optimalizuje Visual Studio pro váš typ vývoje. Další informace naleznete v [tématu Nastavení prostředí](environment-settings.md).

## <a name="general-environment-options"></a>Obecné možnosti prostředí

Mnoho možností přizpůsobení je vystaveno prostřednictvím dialogového okna [Možnosti prostředí.](../ide/reference/general-environment-options-dialog-box.md) K tomuto dialogovému oknu lze získat přístup dvěma způsoby:

- Na řádku nabídek zvolte**Možnosti** **nástrojů** > a pokud ještě nejsou rozbaleny, rozbalte uzel **Prostředí.**

- Stiskněte **kombinaci kláves Ctrl**+**Q**, zadejte do vyhledávacího pole **prostředí** a z výsledků zvolte Prostředí **> Obecné.**

> [!TIP]
> Po zobrazení dialogového okna Možnosti můžete stisknutím **klávesy F1** nápovědu k různým nastavením na této stránce.

## <a name="environment-color-themes"></a>Barevné motivy prostředí

Chcete-li změnit barevný motiv mezi světlým, tmavým a modrým, zadejte do vyhledávacího pole **prostředí** a pak zvolte **Prostředí > obecné**. V dialogovém okně **Volby** změňte volbu **Barevný motiv.**

Chcete-li změnit možnosti vybarvení v editoru, zadejte do vyhledávacího pole **prostředí** a pak zvolte **Prostředí > Písma a barvy**. Viz [Postup: Změna písem a barev](../ide/how-to-change-fonts-and-colors-in-visual-studio.md).

### <a name="main-menu-casing"></a>Pouzdra hlavního menu

Můžete změnit záhlaví hlavní nabídky mezi **případem nadpisu** ("soubor") a **všemi zakončeními** ("SOUBOR"). Do vyhledávacího pole zadejte **prostředí,** vyberte **Prostředí > Obecné**a pak změňte **volbu Použít styl písma nadpisu na řádek nabídek.**

### <a name="customize-menus-and-toolbars"></a>Přizpůsobení nabídek a panelů nástrojů

Pokud chcete přidat nebo odebrat položky nabídek nebo panelů nástrojů, [přečtěte si postup: Přizpůsobení nabídek a panelů nástrojů](../ide/how-to-customize-menus-and-toolbars-in-visual-studio.md).

::: moniker range="vs-2017"

## <a name="start-page"></a>Úvodní stránka

Pokud chcete vytvořit vlastní úvodní stránku pro vás a váš tým, přečtěte si informace [o úpravě úvodní stránky](../ide/customizing-the-start-page-for-visual-studio.md).

## <a name="window-layouts"></a>Rozložení oken

Můžete definovat a uložit více rozložení oken a přepínat mezi nimi. Můžete například definovat jedno rozložení pro kódování a jedno pro ladění. Informace o uspořádání pozic a chování oken a uložení vlastních rozložení naleznete v [tématu Přizpůsobení rozložení oken](../ide/customizing-window-layouts-in-visual-studio.md).

## <a name="external-tools"></a>Externí nástroje

Nabídku **Nástroje** můžete přizpůsobit tak, aby spouštěla externí nástroje. Další informace naleznete v [tématu Správa externích nástrojů](../ide/managing-external-tools.md).
