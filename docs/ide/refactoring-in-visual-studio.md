# <a name="refactor-code"></a>Refaktorování kódu

Refaktoring je proces úpravy kódu, aby bylo snazší udržovat, pochopit a rozšiřovat, ale beze změny jeho chování.

## <a name="programming-languages"></a>Programovací jazyky

Různé refaktoringové operace jsou k dispozici pro různé programovací jazyky v sadě Visual Studio:

- Stránky v této části obsahu pokrývají refaktoringy, které jsou k dispozici pro c# a visual basic. Některé příklady jsou [Extrahovat metodu refaktoring](reference/extract-method.md) a [Přesunout typ na odpovídající soubor refaktoring](reference/move-type-to-matching-file.md).
- Informace o refaktoringu kódu jazyka C++ naleznete v [tématu Zápis a refaktoring kódu (C++).](/cpp/ide/writing-and-refactoring-code-cpp)
- Refaktoring podpora f# je poskytována [Visual F# Power Tools](https://marketplace.visualstudio.com/items?itemName=FSharpSoftwareFoundation.VisualFPowerTools), rozšíření visual studio jiného výrobce.
