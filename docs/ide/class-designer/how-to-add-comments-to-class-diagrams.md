# <a name="how-to-add-comments-to-class-diagrams"></a>Postup: Přidání komentářů k diagramům tříd

Obrazce poznámek můžete použít k oslnění diagramů tříd. Obrazec komentáře má jednu vlastnost **Text**, do které můžete zadat text. Obrazce poznámek existují pouze na povrchu diagramu a nikoli v kódu.

Komentář se nachází v zobrazení diagramu třídy v **Návrháře tříd**. Pokud otevřete diagram druhé třídy do stejného projektu, komentáře, které jste vytvořili v prvním zobrazení, nejsou viditelné. Pokud odstraníte diagram, budou odstraněny také všechny komentáře, které obsahuje.

Můžete změnit velikost tvaru komentáře, ale nemůžete změnit další aspekty jeho vzhledu, například barvu pozadí, písmo nebo velikost písma.

## <a name="to-add-a-comment"></a>Přidání komentáře

1. Přetáhněte komentář z **panelu nástrojů Návrháře tříd** do diagramu třídy.

2. Klikněte do nového obrazce komentáře v diagramu a zadejte požadovaný text.
