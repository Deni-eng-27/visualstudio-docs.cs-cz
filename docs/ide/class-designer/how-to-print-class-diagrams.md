# <a name="how-to-print-class-diagrams"></a>Postup: Tisk diagramů tříd

Diagram třídy můžete vytisknout pomocí funkce tisku sady Visual Studio.

## <a name="to-print-a-class-diagram"></a>Tisk diagramu třídy

1. Otevřete diagram třídy.

2. V nabídce **Soubor** klikněte na **Tisk.**

Vytiskne celý diagram třídy. Možná budete muset upravit nastavení v dialogovém okně **Vzhled stránky,** aby bylo možné tisknout v odpovídající velikosti.
