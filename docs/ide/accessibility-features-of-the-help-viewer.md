# <a name="accessibility-features-of-the-help-viewer"></a>Funkce usnadnění programu Help Viewer
Společnosti Microsoft je poskytovat snadněji použitelné produkty a služby pro každodenní použití. Toto téma obsahuje informace o funkcích, produktech a služby, které pomáhají vytvořit Prohlížeč nápovědy dostupné pro osoby s širokou škálu dalo.  
  
## <a name="keyboard-access"></a>Použití klávesnice  
Pomocí klávesnice, můžete ke všem funkcím nápovědy. Další informace najdete v tématu [klávesové zkratky (Help Viewer)](../ide/shortcut-keys-help-viewer.md).  
  
## <a name="font-size"></a>Velikost písma  
Můžete upravit velikost písma v které tématu text se zobrazí v okně dokumentu. Další informace najdete v tématu [přizpůsobit prohlížeče nápovědy](../ide/customize-the-help-viewer.md).  
  
## <a name="window-size"></a>Velikost okna  
Šířka windows navigační nebo dokumentu můžete změnit tak, že odkazuje na oddělovač mezi dvěma windows. Když kurzoru změní na dvojitou šipku, pomocí primární tlačítko myši přetáhněte rozhraní k doleva či doprava.  
  
## <a name="help-viewer-position"></a>Pozice Prohlížeč nápovědy  
Prohlížeč nápovědy můžete přemístit přetažením jeho záhlaví na jinou pozici.
