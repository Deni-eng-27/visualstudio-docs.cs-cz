# <a name="localizing-applications"></a>Lokalizace aplikací
*Lokalizace* je proces přizpůsobení aplikace pro daný jazykové verze nebo národní prostředí. Lokalizace se primárně skládá z překladu uživatelského rozhraní.  
  
## <a name="in-this-section"></a>V tomto oddílu

 Vysvětluje, jak lokalizované prostředky jsou uloženy a získat přístup v sadě Visual Studio.  
  
 [Zabezpečení a lokalizovaná satelitní sestavení](../ide/security-and-localized-satellite-assemblies.md)  
 Popisuje podpisový satelitní sestavení s páry klíčů veřejný soukromý.  
  
 [Čísla verzí pro hlavní a lokalizované satelitní sestavení](../ide/version-numbers-for-main-and-localized-satellite-assemblies.md)  
 Zavádí <xref:System.Resources.SatelliteContractVersionAttribute> třídy, která určuje, které satelitní sestavení pracují s hlavní sestavení aplikace.  
  
 [Neutrální jazyky zdrojů pro lokalizaci](../ide/neutral-resources-languages-for-localization.md)  
 Zavádí <xref:System.Resources.NeutralResourcesLanguageAttribute> třídy, která určuje jazykovou verzi zdrojů, součástí hlavní sestavení aplikace.
