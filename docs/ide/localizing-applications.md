# <a name="localizing-applications"></a>Lokalizace aplikací

*Lokalizace* je proces přizpůsobení aplikace pro daný jazykové verze nebo národní prostředí. Lokalizace se primárně skládá z překladu uživatelského rozhraní.

## <a name="in-this-section"></a>V tomto oddílu
 [Hierarchická organizace zdrojů pro lokalizaci](../ide/hierarchical-organization-of-resources-for-localization.md) vysvětluje, jak lokalizované prostředky jsou uloženy a získat přístup v sadě Visual Studio.

 [Zabezpečení a lokalizované satelitní sestavení](../ide/security-and-localized-satellite-assemblies.md) popisuje podpisový satelitní sestavení s páry klíčů veřejný soukromý.

 [Čísla verzí pro hlavní a lokalizované satelitní sestavení](../ide/version-numbers-for-main-and-localized-satellite-assemblies.md) Introduces <xref:System.Resources.SatelliteContractVersionAttribute> třídy, která určuje, které satelitní sestavení pracují s hlavní sestavení aplikace.

 [Neutrální jazyky zdrojů pro lokalizaci](../ide/neutral-resources-languages-for-localization.md) Introduces <xref:System.Resources.NeutralResourcesLanguageAttribute> třídy, která určuje jazykovou verzi zdrojů, součástí hlavní sestavení aplikace.
