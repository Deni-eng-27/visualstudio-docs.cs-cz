# <a name="f1-help"></a>F1 nápověda

Vypadá to, že jsme nemohli najít to, co jste hledali, a omlouváme se. Můžete nám pomoci zlepšit tím, že opustí zpětnou vazbu prostřednictvím tlačítek v dolní části této stránky.

## <a name="cursor-position"></a>Pozice kurzoru

Pokud hledáte nápovědu F1 k názvu klíčového slova nebo rozhraní API, ujistěte se, že je kurzor umístěn uvnitř nebo jen vpravo od slova před stisknutím **klávesy F1**. Pokud to stále nefunguje, vyberte ze seznamu [užitečných](#useful-links) https://docs.microsoft.comodkazů nebo proveďte vyhledávání na .
