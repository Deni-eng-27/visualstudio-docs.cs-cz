# <a name="customizing-the-editor"></a>Vlastní nastavení editoru
Změna možností můžete přizpůsobit formátování, karty, písma a barvy, IntelliSense a řadu dalších funkcí editoru obecně nebo editor konkrétní jazyk. Další informace o těchto možnostech najdete v tématu [dialogové okno možností textového editoru](../ide/reference/text-editor-options-dialog-box.md) nebo typ `Editor` do **QuickLaunch**. Můžete také změnit zobrazení oken editoru.
