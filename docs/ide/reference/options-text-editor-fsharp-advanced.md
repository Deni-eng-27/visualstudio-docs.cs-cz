# <a name="options-text-editor-f-advanced"></a>Možnosti, Textový editor, F#, Upřesnit

Pomocí stránky **Upřesnit možnosti** můžete upravit některá nastavení textového editoru pro F#. Chcete-li získat přístup k této stránce možností, zvolte**Možnosti** **nástrojů** > a pak zvolte **Textový editor** > **F#** > **Upřesnit**.

## <a name="block-structure-guides"></a>Vodítka blokové struktury

Když je tato volba vybraná, v editoru se zobrazí svislé čáry, které jsou zaokřovány do strukturovaných bloků kódu, což umožňuje snadno identifikovat jednotlivé bloky kódu.

## <a name="outlining"></a>Sbalování

Zaškrtnutím tohoto políčka automaticky načrtnete soubor kódu, který vytvoří sbalitelné bloky kódu.
