# <a name="autorecover-environment-options-dialog-box"></a>Dialogové okno Automatické obnovení, Prostředí, Možnosti

Na této stránce v dialogovém okně **Možnosti** určete, zda se mají soubory automaticky zálohovat, či nikoli. Můžete také určit, zda chcete obnovit upravené soubory, pokud visual studio neočekávaně vypne.

Do tohoto dialogového okna se dostanete tak, že vyberete nabídku **Nástroje,** vyberete **Možnosti**a potom vyberete**automatické obnovení** **prostředí** > . Pokud se tato stránka v seznamu nezobrazí, vyberte v dialogovém okně **Možnosti** vybrat **Zobrazit vše.**

**Ukládat informace o automatickém obnovení každých [n] minut**

Tuto možnost použijte k přizpůsobení, jak často je soubor automaticky uložen v editoru. U dříve uložených souborů je kopie souboru uložena v *%USERPROFILE%\Documents\Visual Studio\\[verze]\Název projektu\\[název projektu].* Pokud je soubor nový a ještě jste ho neuložili, soubor se automaticky uloží pomocí náhodně generovaného názvu souboru.

**Zachovat informace o automatickém obnovení po dobu [n] dnů**

Tuto možnost použijte k určení, jak dlouho visual studio udržuje soubory vytvořené pro automatické obnovení.
