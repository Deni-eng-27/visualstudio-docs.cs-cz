# <a name="options-dialog-box-projects-and-solutions-web-projects"></a>Dialogové okno Možnosti, Projekty a řešení, Webové projekty

Nastaví webový server, který budou [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]webové projekty používat pro vývoj v rámci aplikace . Chcete-li získat přístup k těmto možnostem, vyberte **možnost Nástroje > Možnosti** rozbalte **položku Projekty a řešení**a vyberte možnost Webové **projekty**.

Ve výchozím nastavení používá spuštění webového projektu v sadě Visual Studio vývojový server sady Visual Studio. Další informace naleznete [v tématu Webové servery v sadě Visual Studio pro ASP.NET webových projektech](/previous-versions/aspnet/58wxa9w5\(v\=vs.120\)).

## <a name="settings"></a>Nastavení

**Použití 64bitové verze služby IIS Express pro webové stránky a projekty**

Tuto možnost vyberte, chcete-li místo vývojového serveru sady Visual Studio použít službu IIS Express. Další informace naleznete [v tématu Introducing IIS Express](https://weblogs.asp.net/scottgu/introducing-iis-express) and [IIS Express Overview](/iis/extensions/introduction-to-iis-express/iis-express-overview).

**Upozornit před spuštěním webových aplikací v případě chyb v seznamu chyb**

Pokud je tato možnost nastavena, budete upozorněni, pokud se pokusíte spustit webovou aplikaci, když se nekompiluje bez chyb.
