# <a name="how-to-change-fonts-and-colors-for-the-editor-in-visual-studio"></a>Postup: Změna písem a barev pro editor v sadě Visual Studio

Můžete změnit výchozí plochu písma, upravit velikost písma a změnit barvy popředí a pozadí pro různé textové **položky zobrazení** v editoru kódu. Při změně nastavení písma mějte na paměti následující informace:

- Nastavení pro **písmo** a **velikost** jsou globální pro všechny textové prvky ve všech editorech sady Visual Studio.

- Názvy písem s pevnou šířkou jsou uvedeny tučně.

- **Popředí položky**, **Pozadí položky**a **Tučné** volby lze nastavit pro každý typ textového prvku. Pokud například změníte barvy a vyberete **tučné** pro **komentář** a **záložky**, ostatní typy textových prvků nebudou ovlivněny.

## <a name="change-the-default-font-face-size-and-colors"></a>Změna výchozí plochy, velikosti a barev písma

1. Z nabídky **Nástroje** vyberte **Možnosti.** V části **Prostředí**vyberte [Písma a barvy](../../ide/reference/fonts-and-colors-environment-options-dialog-box.md).

2. V **části Zobrazit nastavení pro**vyberte **texteditor**.

3. Úpravou voleb **Písmo** a **Velikost** změňte plochu a velikost písma pro všechny textové prvky ve všech editorech.

4. Vyberte příslušnou položku v **části Zobrazit položky**a pak upravte **možnosti popředí položky** a **pozadí položky.**

    > [!TIP]
    > Chcete-li obnovit výchozí nastavení, klepněte na **tlačítko Použít výchozí** nastavení.

5. Klikněte na tlačítko **OK**.
