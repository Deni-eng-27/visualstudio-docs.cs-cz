# <a name="regex-completion-through-intellisense-menu"></a>Dokončení regexu prostřednictvím nabídky IntelliSense

Toto refaktoring se vztahuje na:

- C#

- Visual Basic

**Co:** Dokončení regulárního výrazu (regulárnívýraz) prostřednictvím nabídky IntelliSense.

**Kdy:** Chcete napsat regulární výraz s pomocí intelliSense. Technologie IntelliSense poskytuje základní dokončení a vysvětlení toho, co znamenají jednotlivé znaky regulárního výrazu. 

**Proč:** Psaní regulárního výrazu je těžké a IntelliSense vám může pomoci napsat.

## <a name="how-to"></a>Postupy

1. Umístěte kurzor do řetězce regulárního výrazu.
2. Stisknutím **klávesy Ctrl**+**Space** spusťte nabídku **IntelliSense.**
3. Vyberte znak, který chcete přidat do řetězce regulárního výrazu.
