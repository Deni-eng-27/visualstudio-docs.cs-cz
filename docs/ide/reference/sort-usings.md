# <a name="sort-usings"></a>Řazení direktiv Using

Tento refaktoring platí pro:

- C#

**Co:** Řazení direktiv Using.

**Kdy:** Chcete seřadit `using` direktivy v horní části souboru tak, aby byly v abecedním pořadí. 

**Proč:** To usnadňuje vyhledání použití direktivy.

## <a name="how-to"></a>Postupy

1. Vyberte **upravit** z řádku nabídek.
2. Vyberte **Intellisense** > **seřadit direktivy using**.
