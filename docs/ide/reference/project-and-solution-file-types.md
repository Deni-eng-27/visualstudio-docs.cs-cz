# <a name="project-and-solution-file-types"></a>Typy souborů projektu a řešení
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]podporuje mnoho typů souborů. V konkrétním zařízení nainstalované součásti určí, které typy souborů jsou podporovány. Toto téma uvádí typy souborů řešení a projektu, které jsou podporovány pro některé typické instalace. Informace o dalších typů souborů hledání s použitím přípony názvů souborů pro každý typ.  
  
## <a name="solution-files-sln-and-suo"></a>Soubory řešení (.sln a .suo)  
 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]používá dva typy souborů (.sln a .suo) k uložení nastavení, které jsou specifické pro řešení. Tyto soubory souhrnně označované jako soubory řešení, zadejte Průzkumníku řešení s informacemi potřebnými k zobrazení grafické rozhraní pro správu vašich souborů. Umožňují vám soustředit se jenom na své projekty a konečného cíle ne na prostředí samotné pokaždé, když se vrátíte k vývojářských úloh.  
  
|Rozšíření|Název|Popis|  
|---------------|----------|-----------------|  
|.sln|[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]Řešení|Umožňuje uspořádat projekty, položky projektu a řešení položky do řešení.|  
|.suo|Možnosti řešení uživatele|Uchovává informace o přizpůsobení na úrovni uživatele provedené Visual Studio, jako je například zarážky.|
