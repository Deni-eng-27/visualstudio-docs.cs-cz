# <a name="options-dialog-box-projects-and-solutions--locations"></a>Dialogové okno Možnosti: \> Umístění projektů a řešení

Na této stránce můžete nastavit výchozí umístění pro projekty, šablony projektů a šablony položek. Chcete-li získat přístup k těmto možnostem, vyberte**možnost Možnosti** **nástrojů** > , rozbalte **položku Projekty a řešení**a pak vyberte **Umístění**.

Následující možnosti jsou k dispozici na stránce **Umístění.**

## <a name="projects-location"></a>Umístění projektů

Určuje výchozí umístění, kde visual studio vytváří nové projekty a složky řešení. Několik dialogových oken také používá umístění nastavené v této možnosti pro počáteční body složky. Například dialogové okno **Otevřít projekt** používá toto umístění pro zástupce **Moje projekty.**

## <a name="user-project-templates-location"></a>Umístění šablon uživatelských projektů

Určuje výchozí umístění, které se používá k vyhledávání šablon projektů vytvořených uživatelem. Další informace naleznete v [tématu How to: Locate and Organize Templates](../../ide/how-to-locate-and-organize-project-and-item-templates.md).

## <a name="user-item-templates-location"></a>Umístění šablon uživatelských položek

Určuje výchozí umístění, které se používá k vyhledávání šablon položek vytvořených uživatelem. Další informace naleznete v [tématu How to: Locate and Organize Templates](../../ide/how-to-locate-and-organize-project-and-item-templates.md).
