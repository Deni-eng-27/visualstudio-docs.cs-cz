# <a name="convert-anonymous-type-to-tuple"></a>Převedení anonymního typu na řazenou kolekci členů

Toto refaktoring se vztahuje na:

- C#

- Visual Basic

**Co:** Převeďte anonymní typ na řazenou kolekce členů.

**Kdy:** Máte anonymní typ, který se kvalifikuje jako n-tice.

**Proč:** [N-tic](/dotnet/csharp/tuples) jsou užitečné při udržování syntaxe zjednodušené. Tato rychlá akce usnadňuje využití této funkce C#.

## <a name="how-to"></a>Postupy

1. Umístěte kurzor anonymním typem.
2. Stiskněte **klávesu Ctrl**+**.** spouštět nabídku **Rychlé akce a Refaktorings.**

   ![Převést anonymní typ na n-tuple](media/convert-anon-to-tuple.png)

2. Stisknutím **klávesy Enter** přijměte refaktoring.

   ![Převést anonymní typ na n-tuple](media/convert-anon-to-tuple-complete.png)
