# <a name="options-text-editor-javascript-linting"></a>Options, Text Editor, JavaScript, Linting

Použití **Linting** stránku **možnosti** dialogové okno Nastavení možností pro analýzu kódu v editoru kódu. Pro přístup k této stránce, na panelu nabídek zvolte **nástroje** > **možnosti**a potom rozbalte **textový Editor** > **JavaScript**  >  **Linting**.

## <a name="eslint-settings"></a>Nastavení pro ESLint

Tyto možnosti umožňují povolit statickou analýzu kódu jazyka JavaScript a vyberte soubory, které jsou analyzovány. Další informace o ESLint najdete v tématu [ESLint.org](https://eslint.org/).

### <a name="uielement-list"></a>Seznam prvků uživatelského rozhraní

|Možnost|Popis|
|------------|-----------------|
|**Povolit ESLint**|Pokud je vybraná tato možnost, Editor kódu umožňuje statické analýzy kódu.|
|**LINT zahrnuty všechny soubory v projektu, dokonce i zavření souborů**|Pokud je vybraná tato možnost, jsou analyzovány zavřenými soubory, pokud diagnostiky pouze označené pro otevřené soubory.|

## <a name="global-eslint-config-options"></a>Možnosti globální konfigurace ESLint

Tato možnost umožňuje kopírovat umístění konfiguračního souboru globální ESLint. Také pokud jste dříve změnili umístění, můžete obnovit soubor do výchozího umístění.
