# <a name="options-text-editor-c-formatting"></a>Možnosti, textový editor, C#, formátování
Použití **formátování** dialogové okno stránky vlastností pro nastavení možností pro formátování kódu v editoru kódu. Pro přístup k tohoto dialogového okna, klikněte na tlačítko **možnosti** na **nástroje** nabídky, rozbalte položku **textového editoru**, rozbalte položku **C#**a potom klikněte na  **Formátování**.  
  
> [!NOTE]
>  Dialogová okna a příkazy nabídek, které vidíte, se mohou lišit od těch popsaných v nápovědě v závislosti na aktivních nastaveních nebo edici. Chcete-li změnit nastavení, zvolte **nastavení importu a exportu** na **nástroje** nabídky. Další informace najdete v tématu [přizpůsobení prostředí Visual Studio IDE](../../ide/personalizing-the-visual-studio-ide.md).  
  
## <a name="general-settings"></a>Obecná nastavení  
 Obecné nastavení ovlivnit jak editoru kódu se týká možnosti formátování kódu.  
  
## <a name="uielement-list"></a>Seznam prvků uživatelského rozhraní  
  
|Popisek|Popis|  
|-----------|-----------------|  
|**Automaticky formátu dokončit příkaz na;**|Při výběru naformátuje příkazy při dokončení podle možnosti formátování vybrané pro editoru kódu. Pokud nechcete, aby se v editoru kódu pro úpravu příkazy, zrušte zaškrtnutí tohoto políčka.|  
|**Formát automaticky dokončit bloku na}**|Při výběru formáty kódu bloky podle možnosti formátování vybrané pro editoru kódu, jakmile dokončíte blok kódu. Pokud nechcete, aby se v editoru kódu pro úpravu bloky, zrušte zaškrtnutí tohoto políčka.|  
|**Upravit odsazení na vložení**|Při výběru formáty text vložit do editoru kódu podle možnosti formátování vybrané pro editoru kódu. Vymazat toto políčko, pokud nechcete, aby vložit text, který má být změněna.|  
  
## <a name="preview-window"></a>Okno náhledu  
 **Odsazení**, **nové řádky**, **mezery**, a **zabalení** možnosti podokna zobrazení okno náhledu. Okno náhledu ukazuje účinek jednotlivých možností. Chcete-li použít okno náhledu, vyberte možnost formátování. Okno náhledu ukazuje příklad vybrané možnosti. Když změníte nastavení, například při vyberte nebo zrušte zaškrtnutí políčka okno náhledu aktualizací zobrazíte účinek se nové nastavení.  
  
## <a name="remarks"></a>Poznámky  
 Odsazení možnosti na **karty** stránky pro každý jazyk pouze určit, kde editoru kódu umístí kurzor po stisknutí klávesy ENTER na konci řádku. Odsazení možnosti v části **formátování** použít, když kód formátována automaticky, například při vložení kódu do souboru při **upravit odsazení vkládané** je vybrána a kdy se bloku formátovaný je zadán ručně.
