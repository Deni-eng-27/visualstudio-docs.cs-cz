# <a name="options-dialog-box-environment--web-browser"></a>Dialogové okno Možnosti: Webový prohlížeč prostředí \>

Nastaví možnosti pro interní webový prohlížeč i aplikaci Internet Explorer. Chcete-li získat přístup k tomuto dialogovému oknu, klepněte na příkaz **Možnosti** v nabídce **Nástroje,** rozbalte složku **Prostředí** a vyberte **webový prohlížeč**.

> [!NOTE]
> Dialogová okna a příkazy nabídek, které vidíte, se mohou lišit od těch popsaných v nápovědě v závislosti na aktivních nastaveních nebo edici. Chcete-li změnit nastavení, zvolte **Nastavení importu a exportu** v nabídce **Nástroje.** Další informace naleznete v tématu [Reset settings](../environment-settings.md#reset-settings).

> [!IMPORTANT]
> Otevření určitých souborů nebo součástí z webu může spustit kód v počítači.

## <a name="home-page"></a>Domovská stránka

Nastaví stránku zobrazenou při otevření webového prohlížeče IDE.

## <a name="search-page"></a>Hledat stránku

Umožňuje určit vyhledávací stránku interního webového prohlížeče. Toto umístění se může lišit od vyhledávací stránky používané instancemi aplikace Internet Explorer iniciovaných mimo integrované vývojové prostředí (IDE).

## <a name="view-source-in"></a>Zobrazit zdroj v

Nastaví editor použitý k otevření webové stránky, když zvolíte **Zobrazit zdroj** na stránce z interního webového prohlížeče.

- **Zdrojový editor** Tuto možnost vyberte, chcete-li zobrazit zdroj v [editoru](../../ide/writing-code-in-the-code-and-text-editor.md).

- **Editor HTML** Tuto možnost vyberte, chcete-li zobrazit zdroj v [návrháři HTML](https://msdn.microsoft.com/Library/640043cc-3657-4677-a091-bc315e636477). Tento výběr slouží k úpravám webové stránky v jednom ze dvou zobrazení: Návrhové zobrazení nebo standardnítextové zobrazení zdroj.

- **Externí editor** Výběrem zobrazíte zdroj v jiném editoru. Zadejte cestu libovolného editoru, který zvolíte, například Program Poznámkový blok.exe.

## <a name="internet-explorer-options"></a>Možnosti aplikace Internet Explorer

Klepnutím změníte možnosti aplikace Internet Explorer v dialogovém okně **Vlastnosti Internetu.** Změny provedené v tomto dialogovém okně ovlivní interní webový prohlížeč i instance aplikace Internet Explorer iniciované mimo rozhraní IDE sady Visual Studio (například z nabídky Start).

> [!NOTE]
> Dialogové okno **Procházet** pomocí nahraďte interní webový prohlížeč sady Visual Studio prohlížečem podle vašeho výběru. Dialogové okno Procházet pomocí můžete přistupovat například z nabídky pravým tlačítkem myši nebo v místní nabídce souboru HTML v projektu.
