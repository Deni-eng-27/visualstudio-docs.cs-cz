# <a name="shortcut-tips-for-visual-studio"></a>Tipy ke zvýšení zástupce pro Visual Studio

Můžete přejít v sadě Visual Studio snadněji pomocí zástupců v tomto článku. Tyto klávesové zkratky zahrnují zkratky klávesnice a myši, stejně jako text, který můžete zadat umožňující snadnější provádění různých úloh.

Úplný seznam zástupců příkazů najdete v tématu [výchozí klávesové zkratky](../ide/default-keyboard-shortcuts-in-visual-studio.md).

> [!NOTE]
> Toto téma se vztahuje k sadě Visual Studio ve Windows. Visual Studio pro Mac, najdete v části [běžných klávesových zkratek v sadě Visual Studio pro Mac](/visualstudio/mac/keyboard-shortcuts).

## <a name="window-management"></a>Správa oken

|Úloha|Zástupce|
|-|-|
|Maximalizovat plovoucí okno|**Dvakrát klikněte na panel** v záhlaví|
|Maximalizace nebo minimalizace oken|**Vyhrajte**+**šipka nahoru** / **Win**+**šipka dolů**|
|Zadokovat plovoucího okna|**CTRL**+**dvakrát klikněte na panel** v záhlaví|
|Přesunutí/ukotvení plovoucího windows|**Vyhrajte**+**šipku vlevo** / **Win**+**šipka doprava**|
|Zavřít aktivní dokument|**CTRL**+**F4**|
|Zobrazit seznam otevřených souborů|**CTRL**+**Alt**+**šipka dolů**|
|Zobrazit všechna plovoucí okna|**CTRL**+**Shift**+**M**|
|Zobrazit seznam odkazů|**Vyhrajte**+**Alt**+**N**|
|Spustit novou instanci|**Vyhrajte**+**Shift**+**N**|
|Přepínání mezi okny|**Vyhrajte**+**N**|
