# <a name="shortcut-tips-for-visual-studio"></a>Tipy pro zástupce pro Visual Studio

Pomocí zkratek v tomto článku můžete snadněji procházet v sadě Visual Studio. Mezi tyto klávesové zkratky patří klávesové zkratky a klávesové zkratky a text, který můžete zadat, abyste mohli snadněji provést úkol.

Úplný seznam klávesových zkratek příkazů naleznete [v tématu Výchozí klávesové zkratky](../ide/default-keyboard-shortcuts-in-visual-studio.md).

> [!NOTE]
> Toto téma platí pro Visual Studio v systému Windows. Visual Studio pro Mac najdete [v tématu Běžné klávesové zkratky ve Visual Studiu pro Mac](/visualstudio/mac/keyboard-shortcuts).
