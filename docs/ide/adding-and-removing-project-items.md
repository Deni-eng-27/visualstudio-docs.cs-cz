# <a name="adding-and-removing-project-items"></a>Přidávání a odebírání projektových položek
Můžete přidat nebo odebrat projektů a položek projektu kliknutím pravým tlačítkem na projekt nebo řešení uzel v **Průzkumníku řešení**a výběr **přidat** nebo **odebrat**. Odebrání projektu z řešení nebo položku z projektu, neodstraní se z disk nebo zdroj ovládacího prvku Pokud vyberete možnost trvale odstranit.
