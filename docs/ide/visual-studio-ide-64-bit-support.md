# <a name="visual-studio-ide-64-bit-support"></a>Podpora v sadě Visual Studio IDE 64-Bit

Visual Studio umožňuje nastavení aplikací na různých platformách, včetně 64bitových platforem. Další informace o podpoře platforem 64-bit v sadě Visual Studio najdete v tématu [64bitové aplikace](/dotnet/framework/64-bit-apps).

[Nasazení nezbytných součástí pro 64bitové aplikace](../deployment/deploying-prerequisites-for-64-bit-applications.md) obsahuje distribuovatelné součásti, můžete použít jako požadavky na instalaci 64bitové aplikace.
