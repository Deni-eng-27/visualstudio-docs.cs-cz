# <a name="visual-studio-feedback-options"></a>Možnosti zpětné vazby aplikace Visual Studio

Proč váš názor nám? Protože teď bereme jako zpětné vazby od zákazníků vážně; To vede velkou část co děláme. Tady je postup nás kontaktovat, takže můžeme směrovat vaše zpětná vazba k osobě nebo týmu.

## <a name="report-a-problem"></a>Nahlášení problému

::: moniker range="vs-2019"

Pokud máte potíže s aplikací Visual Studio&mdash;například chyby, výkon pomalá nebo neočekávané chování&mdash;nám dejte vědět pomocí **nahlásit problém** nástroj. V sadě Visual Studio, zvolte ikonu zpětné vazby v pravém horním rohu, nebo zvolte **pomáhají** > **odeslat zpětnou vazbu** > **nahlásit problém** z panel nabídek.

![Odeslat ikonu zpětné vazby v sadě Visual Studio. 2019 IDE](./media/vs-2019/send-feedback-icon.png)

V dalším kroku Visual Studio otevře [komunity vývojářů](https://developercommunity.visualstudio.com) stránku v prohlížeči, kde se můžete přihlásit nahlásit problém. Další informace najdete v tématu [postup ohlášení problému se sadou Visual Studio](how-to-report-a-problem-with-visual-studio.md).

::: moniker-end

::: moniker range="vs-2017"

Pokud máte potíže s aplikací Visual Studio&mdash;například chyby, výkon pomalá nebo neočekávané chování&mdash;nám dejte vědět pomocí **nahlásit problém** nástroj. V sadě Visual Studio, zvolte ikonu zpětné vazby u vyhledávacího pole, nebo zvolte **pomáhají** > **odeslat zpětnou vazbu** > **nahlásit problém** z nabídky panel.

![Odeslat ikonu zpětné vazby v sadě Visual Studio. 2019 IDE](./media/send-feedback-icon.png)

V dalším kroku Visual Studio otevře [komunity vývojářů](https://developercommunity.visualstudio.com) stránku v prohlížeči, kde se můžete přihlásit nahlásit problém. Další informace najdete v tématu [postup ohlášení problému se sadou Visual Studio](how-to-report-a-problem-with-visual-studio.md).

::: moniker-end

## <a name="make-a-suggestion"></a>Podat návrh

::: moniker range="vs-2019"

Pokud existuje něco můžeme přidat nebo udělat líp, dejte nám vědět, s použitím **navrhnout funkci** nástroj. V sadě Visual Studio, zvolte ikonu zpětné vazby v pravém horním rohu, nebo zvolte **pomáhají** > **odeslat zpětnou vazbu** > **navrhnout funkci** z řádku nabídek.

V dalším kroku Visual Studio otevře [komunity vývojářů](https://developercommunity.visualstudio.com) stránku v prohlížeči, které se můžete přihlásit v sdílet své nápady. Další informace najdete v tématu [navrhnout funkci pro sadu Visual Studio](suggest-a-feature.md).

::: moniker-end

::: moniker range="vs-2017"

Pokud existuje něco můžeme udělat lépe, dejte nám vědět, s použitím **poslat návrh** nástroj. V sadě Visual Studio, zvolte ikonu zpětné vazby u vyhledávacího pole, nebo zvolte **pomáhají** > **odeslat zpětnou vazbu** > **poslat návrh** z panel nabídek.

V dalším kroku Visual Studio otevře [komunity vývojářů](https://developercommunity.visualstudio.com) stránku v prohlížeči, které se můžete přihlásit v sdílet své nápady. Další informace najdete v tématu [navrhnout funkci pro sadu Visual Studio](suggest-a-feature.md).

::: moniker-end

### <a name="rate-this-product"></a>Ohodnoťte tento produkt

Předběžné verze sady Visual Studio mají **Ohodnoťte tento produkt** nemají položku nabídky, která byla vydaná verze produktu. Použití **Ohodnoťte tento produkt** položky nabídky odeslat zpětnou vazbu o kvalitě sestavení předběžné verze, kterou používáte.

## <a name="improve-the-documentation"></a>Vylepšit dokumentaci k

Existují dva způsoby, pomůžete nám vylepšit dokumentaci:

* Zvolte **byla tato stránka užitečná?** tlačítka pro odeslání názoru v dolní části stránky. Toto tlačítko najdete na naší stránky dokumentace.
* Poskytnutí zpětné vazby na libovolné stránce sady Visual Studio pomocí účtu GitHub [docs.microsoft.com](https://docs.microsoft.com/visualstudio/). Chcete-li to provést, zvolte **odeslat zpětnou vazbu o** > **na této stránce** tlačítko také v dolní části stránky.
