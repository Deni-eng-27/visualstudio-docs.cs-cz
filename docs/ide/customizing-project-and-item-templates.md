# <a name="customizing-project-and-item-templates"></a>Přizpůsobení šablon projektů a položek
Po vytvoření šablony projektů a položek, máte možnost přizpůsobit, aby vyhovovala vašim potřebám.  
  
 Například můžete provádět následující úlohy:  
  
-   Upravit a exportovat stávající šablonu jako vlastní šablony.  
  
     Další informace najdete v tématu [postup: aktualizace existujících šablon](../ide/how-to-update-existing-templates.md).  
  
-   Předejte vlastní parametry do šablonu, která má nahradit existující hodnoty.  
  
     Další informace najdete v tématu [postup: Substitute parametrů v šabloně](../ide/how-to-substitute-parameters-in-a-template.md).  
  
-   Přizpůsobení průvodců, které vytváření projektů ze šablon.  
  
     Další informace najdete v tématu [postupy: použití průvodců se šablonami projektů](../extensibility/how-to-use-wizards-with-project-templates.md).
