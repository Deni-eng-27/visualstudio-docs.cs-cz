# <a name="application-properties"></a>Vlastnosti aplikace
Obecné vlastnosti aplikace Visual Basic, C# a F # aplikace, jako je například typ aplikace, spouštěcí objekt a informace o sestavení, můžete nastavit v **aplikace** podokně **Návrhář projektu**.  
  
 Následující úkoly lze provést v **aplikace** podokně:  
  
-   Zadejte informace o sestavení  
  
-   Zadejte úvodní obrazovky pro aplikaci  
  
-   Zadejte událostí na úrovni aplikace  
  
-   Změňte název sestavení  
  
-   Změnit obor názvů pro aplikaci  
  
-   Změna typu aplikace  
  
-   Změňte spouštěcí objekt pro aplikaci  
  
-   Určení ikony aplikace  
  
-   Povolit vizuální styly Windows XP  
  
-   Zadejte zřizování instancí chování pro aplikaci  
  
-   Povolení ověřování vlastní uživatele  
  
-   Zadejte chování při vypnutí pro aplikaci  
  
 Úlohy dostupné pro vaši aplikaci závisí na jazyku vývoj používá.
