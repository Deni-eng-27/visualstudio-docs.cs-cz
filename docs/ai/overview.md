# <a name="overview-of-visual-studio-tools-for-ai"></a>Přehled nástrojů Visual Studia pro umělou přípravu a i.

Informace o tom, jak stáhnout a nainstalovat rozšíření, najdete na stránce [Instalace nástrojů Visual Studia pro umělou přípravu.](installation.md)
