# <a name="visual-studio-tools-for-ai"></a>Visual Studio Tools pro AI

Visual Studio Tools for AI je rozšíření pro sestavení, testování a nasazení řešení Deep Learning / AI. Hladce se integruje s Azure Machine Learning pro robustní experimentální funkce, včetně, ale bez omezení na odesílání úloh přípravy dat a modelování školení transparentně na různé výpočetní cíle. Kromě toho poskytuje podporu pro vlastní metriky a sledování historie spuštění, což umožňuje reprodukovatelnost datové vědy a auditování. Spolupráce připravená na podnikání umožňuje bezpečnou práci na projektu s jinými lidmi.

Můžete začít s hloubkovým učením pomocí [sady Microsoft Cognitive Toolk (CNTK),](/cognitive-toolkit/) [Google TensorFlow](https://www.tensorflow.org)nebo jiných rámců pro hluboké učení ještě dnes.

## <a name="develop-debug-and-deploy-deep-learning-models-and-ai-solutions"></a>Vývoj, ladění a nasazení modelů hloubkového učení a řešení ai
Využijte funkce produktivity sady Visual Studio a urychlete inovace v oblasti umělou anasy ještě dnes. Používejte vestavěné funkce editoru kódu, jako je zvýraznění syntaxe, technologie IntelliSense a automatické formátování textu. Můžete interaktivně otestovat aplikaci hlubokého učení v místním prostředí pomocí podrobného ladění na místní proměnné a modely.

![hluboké učení ide](media/about/ide.png)

## <a name="get-started-quickly-with-the-azure-machine-learning-sample-gallery"></a>Rychlé zahájení studia začínáme s ukázkovou galerií Azure Machine Learning
Visual Studio Tools for AI je integrované s Azure Machine Learning, aby bylo snadné procházet galerii ukázkových experimentů pomocí CNTK, TensorFlow, MMLSpark a dalších.

![ukázkový průzkumník](media/about/gallery.png)

[Další informace o vytváření projektů v galerii ukázkových obrázků](create-project-gallery.md)

## <a name="scale-out-deep-learning-model-training-andor-inferencing-to-the-cloud"></a>Horizontální navýšení kapacity pro školení modelu hlubokého učení a/nebo odvození do cloudu
Toto rozšíření usnadňuje trénování modelů v místním počítači nebo můžete odesílat úlohy do cloudu pomocí naší integrace s Azure Machine Learning. Úlohy můžete odesílat do různých výpočetních cílů, jako jsou clustery Spark, virtuální počítače Azure GPU a další.

![odeslat úlohu](media/about/submitjobs.png)

[Další informace o trénovacích modelech v cloudu](tensorflow-vm.md)

## <a name="supported-operating-systems"></a>Podporované operační systémy
V současné době toto rozšíření podporuje 64bitové operační systémy windows.
