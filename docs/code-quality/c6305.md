# <a name="c6305"></a>C6305
upozornění C6305: potenciální neshoda mezi hodnotami sizeof a countof

 Toto upozornění indikuje, že proměnná, která drží výsledek `sizeof`, je přidávána nebo odečtena od ukazatele nebo `countof`ho výrazu. Tato akce způsobí neočekávané škálování v aritmetických ukazatelích.

## <a name="example"></a>Příklad
 Následující kód generuje toto upozornění:

```cpp
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Pro opravu tohoto upozornění použijte následující kód:

```cpp
void f(int *p)
{
  // code...
  p += 1;
}
```
