# <a name="c6305"></a>C6305
Upozornění C6305: potenciální Neshoda mezi množstvími sizeof a CountOf

 Toto upozornění signalizuje, že proměnná, která drží výsledek `sizeof`, je přidávána nebo odečtena z ukazatele nebo výrazu `countof`. Tato akce způsobí neočekávané škálování v aritmetických ukazatelích.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
void f(int *p)
{
  // code...
  p += 1;
}
```
