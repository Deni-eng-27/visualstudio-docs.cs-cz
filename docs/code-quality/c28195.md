# <a name="c28195"></a>C28195
Upozornění C28195: funkce byla deklarována jako nabývající paměť v proměnné a byla ukončena, aniž by tak učinila

 Toto upozornění znamená, že prototyp funkce pro analyzovanou funkci má anotaci `__drv_acquiresMemory`. Poznámka `__drv_acquiresMemory` znamená, že funkce získá paměť v určeném umístění výsledku, ale alespoň v jedné cestě funkce nezískala paměť. Všimněte si, že nástroj pro analýzu kódu nerozpozná skutečnou implementaci přidělování paměti (zahrnující aritmetické operace) a nerozpozná, že je přidělena paměť (i když je mnoho obálek rozpoznáno). V tomto případě nástroj pro analýzu kódu nerozpozná, že byla paměť přidělena, a vydá toto upozornění. Chcete-li potlačit falešně pozitivní, použijte upozornění `#pragma` na řádku, který předchází otevírací závorce `{` těla funkce.
