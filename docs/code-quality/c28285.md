# <a name="c28285"></a>C28285
Upozornění C28285: Pro funkci. syntaktická chyba v parametru

 Nástroj Analýza kódu oznamuje toto upozornění, když se v souboru modelu vyskytne pravděpodobná chyba. Některé chyby ve zdrojovém souboru můžou tyto chyby způsobovat taky.
