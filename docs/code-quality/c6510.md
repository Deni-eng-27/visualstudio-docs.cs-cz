# <a name="c6510"></a>C6510
Upozornění C6510: Neplatná Anotace: vlastnost Nullterminated lze použít lze použít pouze pro vyrovnávací paměti, jejichž prvky jsou integrálního typu nebo typu ukazatele: function ' \<function > ' \<parameter >.

 Toto upozornění indikuje nesprávné použití vlastnosti **nullterminated lze použít** (ty, které končí na ' `_z` '). Tento typ vlastnosti lze použít pouze na typech ukazatelů nebo polí.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
