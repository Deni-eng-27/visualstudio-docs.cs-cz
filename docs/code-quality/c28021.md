# <a name="c28021"></a>C28021
Upozornění C28021: Parametr \<param > s poznámkou > \<anno musí být ukazatelem.

 Toto upozornění je hlášeno, není-li objekt s poznámkami typu ukazatel. Tuto poznámku nelze použít s `void` nebo integrálními typy.
