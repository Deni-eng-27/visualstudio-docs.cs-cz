# <a name="c28021"></a>C28021
upozornění C28021: Parametr \<param > je opatřen poznámkou \<anno > musí být ukazatel  
  
 Toto upozornění je hlášené, když objekt se poznámkou není typu ukazatele. Tato anotace nelze použít s `void` nebo integrální typy.
