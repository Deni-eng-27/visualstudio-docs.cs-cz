# <a name="c26402dont_heap_allocate_movable_result"></a>C26402 DONT_HEAP_ALLOCATE_MOVABLE_RESULT

Aby nedošlo k nejasnostem o tom, zda ukazatel vlastní objekt, funkce, která vrací Pohyblivý objekt, by měla být přizpůsobena v zásobníku a vracet jej podle hodnoty namísto vrácení objektu přiděleného haldě. Pokud jsou vyžadovány Sémantika ukazatele, pak vraťte inteligentní ukazatel namísto nezpracovaného ukazatele. Viz [ C++ základní pokyny R. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rr-ptr): *Warn, pokud funkce vrátí objekt, který byl přidělen v rámci funkce, ale má konstruktor Move. Doporučujeme zvážit, že se místo toho vrátí hodnota.*
