# <a name="c28282"></a>C28282
Upozornění C28282: řetězce formátu musí být v předběžných podmínkách

 Toto upozornění indikuje, že se našla anotace `__drv_formatString`, která není ve výchozím nastavení anotace `_Pre_` (`__drv_in`) (parametry funkce jsou standardně `_Pre_`). Ověřte, zda anotace použitá v explicitním bloku s poznámkou `_Post_` (`__drv_out`). Pokud ano, odeberte anotaci z jakéhokoli nadřazeného bloku, který jej umístil do kontextu `_Post_`.
