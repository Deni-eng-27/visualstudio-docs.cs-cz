# <a name="c28230"></a>C28230
Upozornění C28230: typ parametru nemá žádného člena.

 Toto upozornění signalizuje, že argument poznámky se pokusí o přístup ke členu `struct`, `class`nebo `union`, který neexistuje.  Toto upozornění bude také vygenerováno, pokud se parametr pokusí zavolat členskou funkci objektu.

## <a name="example"></a>Příklad

```cpp
#include <sal.h>

struct MyStruct
{
  //...
  int usefulMember;
};

// Oops, the name of the member is spelled wrong so it will not be found
void f(_Out_writes_(value.usefullMember) int *buffer, MyStruct value)
{
  for(int i = 0 ; i < value.usefulMember; i++)
  {
    buffer[i] = i;
    //...
  }
}
```

V tomto příkladu je třeba opravit pouze pravopis.

```cpp
void f(_Out_writes_(value.usefulMember) int *buffer, MyStruct value)
{
  for(int i = 0 ; i < value.usefulMember; i++)
  {
    buffer[i] = i;
    //...
  }
}
```
