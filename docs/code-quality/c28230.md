# <a name="c28230"></a>C28230
Upozornění C28230: typ parametru nemá žádného člena.

 Toto upozornění indikuje, že argument poznámky se pokusí získat přístup k `struct`, `class` nebo `union` a pojmenovaný člen neexistuje.
