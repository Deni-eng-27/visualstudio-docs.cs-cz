# <a name="c6260"></a>C6260
Upozornění C6260: sizeof * sizeof je téměř vždy nesprávné, chtěli jste použít počet znaků nebo počet bajtů?

 Toto upozornění znamená, že byly vynásobeny výsledky dvou operací `sizeof` dohromady. Operátor C/C++ `sizeof` vrátí počet bajtů úložiště, které objekt používá. Je obvykle nesprávné ho vynásobit jinou operací `sizeof`; obvykle se zajímá o počet bajtů v objektu nebo počet prvků v poli (například počet znaků v poli).

 K operátoru `sizeof` je přidruženo několik neintuitivního chování. Například v jazyce C je `sizeof ('\0') == 4,`, protože znak je integrálního typu. V C++je typ znakového literálu `char`, takže `sizeof ('\0') == 1`. V jazyce C a C++jsou však splněny následující podmínky:

```
sizeof ("\0") == 2.
```

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <windows.h>

void f( )
{
  int i;
  i = sizeof (L"String") * sizeof (WCHAR);
  // code ...
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#include <windows.h>

void f( )
{
  int i;
  i= sizeof (L"String") / sizeof (WCHAR);

  /* or to get bytes */
  i = sizeof (L"String");
  // code ...
}
```
