# <a name="ca0053"></a>CA0053
Pravidla sestavení nelze načíst.  
  
 To znamená, že pravidlo sestavení nelze načíst pomocí reflexe. Tato situace může nastat, pokud uživatel má vlastní pravidla, které nebyly správně vytvořen nebo nástroj pro analýzu nelze načíst sestavení z jiných důvodů, například byl odepřen přístup systému souborů.
