# <a name="c28197"></a>C28197
Upozornění C28197: možná nevrácení paměti

 Toto upozornění je hlášeno pro paměť i nevracení prostředků, pokud je prostředek potenciálně aliasem jiného umístění.

 *Ukazatel* ukazuje na přidělenou paměť nebo na jiný přidělený prostředek, který nebyl explicitně uvolněn. Toto upozornění je obvykle způsobeno nedostatečnými poznámkami k volané funkci, i když neodpovídající anotace volání funkce může také tuto situaci zvýšit.

 Toto upozornění může být oznámeno funkcí ukončit, pokud má vstupní argument anotaci `__drv_freesMem` nebo `__drv_aliasesMem`. Toto upozornění obvykle označuje buď platnou netěsnou, nebo funkci volanou aktuální funkcí potřebuje další anotaci.

 Konkrétně absence základních `_In_` a `_Out_` povede k tomuto upozornění poměrně pravděpodobnému, i když mohou být potřeba i poznámky `__drv_aliasesMem` a `__drv_freesMem`. Falešně pozitivní je pravděpodobný výsledek chybějící anotace `_In_`.

 Funkce, které přijímají ukazatel a aliasu (proto zabraňují úniku) by měly být opatřeny `__drv_aliasesMem`. Vytvoříte-li funkci, která vloží objekt do globální struktury nebo předá jej funkci systému, která to provede, je třeba přidat anotaci `__drv_aliasesMem`.

 Funkce, které mají volnou paměť, by měly být opatřeny poznámkou `__drv_freesMem`. Hlavní funkce, které tuto poznámku již mají, mají volnou paměť.

## <a name="example"></a>Příklad
 Následující příklad kódu vygeneruje toto upozornění:

```
char *p = (char *)malloc(10);
test(p);   // does not save a copy of p
```

 Následující příklad kódu předchází tomuto upozornění:

```
char *p = (char *)malloc(10);
test(p);   // does not save a copy of p
free(p);
```
