# <a name="c6259"></a>C6259
Upozornění C6259: kód označený příznakem je nedosažitelný: (\<expression > & \<constant >) v Switch-expr nemůže být vyhodnocena jako \<case-Label >

 Toto upozornění indikuje nedosažitelný kód způsobený výsledkem bitové a (`&`) porovnání ve výrazu přepínače. Příkaz Case, který odpovídá konstantě ve výrazu Switch, je dosažitelný pouze; všechny ostatní příkazy Case nejsou dosažitelné.

## <a name="example"></a>Příklad
 Následující vzorový kód vygeneruje toto upozornění, protože výraz `switch``(rand() & 3)` nemůže vyhodnotit návěští Case (`case` `4`):

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        case 4:
            /* Not reachable */
            break;
        default:
            break;
    }
}
```

 Chcete-li toto upozornění opravit, odeberte nedosažitelný kód nebo ověřte, zda je konstanta použitá v příkazu case správná. Následující kód odstraní nedosažitelný příkaz Case:

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        default:
            break;
    }
}
```
