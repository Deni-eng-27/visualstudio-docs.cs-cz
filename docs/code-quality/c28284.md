# <a name="c28284"></a>C28284
Upozornění C28284: pro symbol nejsou aktuálně podporovány predikáty pro symboly bez funkcí.

 Toto upozornění signalizuje, že podmíněná anotace (predikát, `__drv_when`) byla nalezena na jinou funkci než funkce.
