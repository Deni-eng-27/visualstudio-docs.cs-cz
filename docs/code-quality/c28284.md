# <a name="c28284"></a>C28284
Upozornění C28284: Pro symbol nejsou v současné době podporovány predikáty u symbolů, které nejsou funkční.

 Toto upozornění indikuje, že podmíněná Poznámka (predikát, `__drv_when`) byla nalezena na jiném než funkci.
