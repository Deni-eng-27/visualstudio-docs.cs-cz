# <a name="c26138"></a>C26138
upozornění C26138: Pozastavení korutina při držení zámku \<Zámek >.

Upozornění C26138 upozorní, když je pozastaveno korutina při držení zámku. Obecně platí, nelze víme, jak dlouho bude korutina zůstane v pozastaveném stavu, takže tento model může vést už kritických oddílů, než se očekávalo.

## <a name="example"></a>Příklad
 Následující kód vygeneruje C26138.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>Příklad
 Následující kód se opraví těchto upozornění.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
