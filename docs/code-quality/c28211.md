# <a name="c28211"></a>C28211
Upozornění C28211: Pro SAL_context se očekává název statického kontextu

 Toto upozornění indikuje, že operand pro anotaci `_Static_context_` musí být název kontextu definovaného nástrojem. Toto upozornění nikdy nesmí nastat při běžném použití.
