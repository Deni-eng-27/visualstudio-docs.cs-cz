# <a name="c28211"></a>C28211
upozornění C28211: byl očekáván pro SAL_context název statickém kontextu  
  
 Toto upozornění znamená, že operand pro `_Static_context_` poznámky musí být název kontextu definované nástroj. Toto upozornění nikdy provedeno v běžném provozu.
