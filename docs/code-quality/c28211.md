# <a name="c28211"></a>C28211
upozornění C28211: Pro SAL_context se očekává název statického kontextu

 Toto upozornění signalizuje, že operand `_Static_context_` poznámky musí být název definovaný nástroj kontextu. Toto upozornění by nikdy neměly vzniknout při běžném používání.
