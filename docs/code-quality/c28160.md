# <a name="c28160"></a>C28160
Upozornění C28160: Poznámka k chybě

 Toto upozornění je oznámeno, když byla zjištěna anotace `__drv_error`. Tato poznámka se používá k označení postupů kódování, které by měly být opraveny, a lze ji použít s poznámkou `__drv_when` k označení specifických kombinací parametrů.
