# <a name="c28215"></a>C28215
Upozornění C28215: typefix nelze použít na parametr, který již má typefix

 Použití anotace `__typefix` na parametr, který již má poznámku, je chyba. Anotace `__typefix` jsou používány pouze ve výjimečných případech a toto upozornění se neočekává při běžném použití.
