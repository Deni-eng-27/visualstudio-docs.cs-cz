# <a name="ca0055"></a>CA0055
Nepodařilo se načíst soubor:\<cesta >  
  
 Nástroj pro analýzu kódu se nepodařilo načíst cílový soubor, který byl zadán pro analýzu. K tomu dojde, pokud soubor nebyl nalezen nebo byl odepřen přístup k souboru. Obvykle to dochází, pokud se uživatel rozhodl přepsat vlastnost vstupní sestavení v [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], nebo vyvolat fxcopcmd.exe ručně.
