# <a name="c28228"></a>C28228
Upozornění C28228: parametr Anotace: v poznámkách nelze použít typ

 Toto upozornění znamená, že parametr je typu, který není podporován. Poznámky můžou používat jenom omezené sady typů jako parametry. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
