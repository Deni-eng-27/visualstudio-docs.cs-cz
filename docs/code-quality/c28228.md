# <a name="c28228"></a>C28228
upozornění C28228: Parametr poznámky: v poznámkách nelze použít typ

 Toto upozornění signalizuje, že je parametr typu, který není podporován. Poznámky můžete použít pouze omezená sada typy jako parametry. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se neočekává se zobrazí v typické použití.
