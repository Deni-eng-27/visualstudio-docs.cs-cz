 Toto upozornění znamená, že parametr typu, který není podporován. Poznámky, můžete použít jenom omezenou sadu typy jako parametry. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v typické použití.
