# <a name="c6277"></a>C6277
Warning C6277: název aplikace s hodnotou NULL s cestou bez uvozovek při volání > \<function: Výsledkem je ohrožení zabezpečení, pokud cesta obsahuje mezery.

 Toto upozornění znamená, že parametr názvu aplikace má hodnotu null a v názvu cesty ke spustitelnému souboru může být mezera. V takovém případě, pokud není název spustitelného souboru "plně kvalifikovaný", pravděpodobně došlo k potížím se zabezpečením. Uživatel se zlými úmysly může vložit neautorizovaný spustitelný soubor se stejným názvem dřív v cestě. Chcete-li toto upozornění opravit, můžete zadat název aplikace namísto předání hodnoty null, nebo pokud pro název aplikace předáte hodnotu null, použijte uvozovky kolem cesty ke spustitelnému souboru.

## <a name="example"></a>Příklad
 Následující vzorový kód vygeneruje toto upozornění, protože parametr názvu aplikace má hodnotu null a název cesty ke spustitelnému souboru obsahuje mezeru. existuje riziko, že lze spustit jiný spustitelný soubor z důvodu způsobu, jakým funkce analyzuje mezery. Další informace najdete v tématu [CreateProcess](https://docs.microsoft.com/windows/desktop/api/processthreadsapi/nf-processthreadsapi-createprocessa).

```
#include <windows.h>
#include <stdio.h>

void f_defective()
{
    STARTUPINFOA si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof( si ) );
    si.cb = sizeof( si );
    ZeroMemory( &pi, sizeof( pi ) );
    if( !CreateProcessA( NULL,
                        "C:\\Program Files\\MyApp",
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,
                        &pi ) )
    {
        puts( "CreateProcess failed." );
        return;
    }
    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );

    // Close process and thread handles.
    CloseHandle( pi.hProcess );
    CloseHandle( pi.hThread );
}
```

 Chcete-li toto upozornění opravit, použijte uvozovky kolem cesty ke spustitelnému souboru, jak je znázorněno v následujícím příkladu:

```
#include <windows.h>
#include <stdio.h>

void f ()
{
    STARTUPINFOA si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof( si ) );
    si.cb = sizeof( si );
    ZeroMemory( &pi, sizeof( pi ) );

    if( !CreateProcessA( NULL,
                        "\"C:\\Program Files\\MyApp.exe\"",
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,
                        &pi ) )
    {
        puts( "CreateProcess failed." );
        return;
    }
    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );

    // Close process and thread handles.
    CloseHandle( pi.hProcess );
    CloseHandle( pi.hThread );
}
```
