# <a name="c28159"></a>C28159
Upozornění C28159: Zvažte místo toho použití jiné funkce.

 Toto upozornění je hlášeno pro ovladače, což naznačuje, že používáte upřednostňované volání funkce, které je sémanticky ekvivalentní funkci, kterou ovladač volá. Toto je obecná zpráva s upozorněním. byla použita anotace `__drv_preferredFunction` (případně s podmínkou `__drv_when` () anotace) k označení špatného postupu kódování.

## <a name="example"></a>Příklad
 Následující příklad kódu vygeneruje toto upozornění:

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 Následující příklad kódu předchází tomuto upozornění:

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
