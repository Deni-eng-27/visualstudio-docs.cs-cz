# <a name="c6318"></a>C6318
Upozornění C6318: nesprávně definovaná __try/\__except: použijte konstantu EXCEPTION_CONTINUE_SEARCH nebo jinou konstantu, která se vyhodnotí jako nula ve výrazu filtru výjimky. Kód v bloku obslužné rutiny výjimek není spuštěn

 Toto upozornění znamená, že pokud dojde k výjimce v chráněném bloku této strukturované obslužné rutiny výjimek, výjimka nebude zpracována, protože ve výrazu filtru výjimky je použit konstantní `EXCECPTION_CONTINUE_SEARCH`.

 Tento kód je ekvivalentní chráněnému bloku bez bloku obslužných rutin výjimek, protože blok obslužné rutiny není spuštěn.

## <a name="example"></a>Příklad
 Následující kód generuje toto upozornění:

```cpp
#include <excpt.h>
#include <stdio.h>

void f (char *pch)
{
  __try
  {
    // assignment might fail
    *pch = 0;
  }
  __except (EXCEPTION_CONTINUE_SEARCH) // warning 6318
  {
    puts("Exception Occurred");
  }
}
```

 Pro opravu tohoto upozornění použijte následující kód:

```cpp
#include <excpt.h>
#include <stdio.h>
#include <windows.h>

void f (char *pch)
{
  __try
  {
    // assignment might fail
    *pch = 0;
  }
  __except( (GetExceptionCode() == EXCEPTION_ACCESS_VIOLATION) ?
                EXCEPTION_EXECUTE_HANDLER : EXCEPTION_CONTINUE_SEARCH )
  {
    puts("Access violation");
  }
}
```
