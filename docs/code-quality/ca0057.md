# <a name="ca0057"></a>CA0057
Porušení nelze mapovat na aktuální sadu cílů a pravidla

 K této chybě dojde, pokud je FxCopCmd.exe je vyvolána ručně pomocí souboru projektu nebo / importovat XML sestavy, který má porušení zásad, která nelze mapovat na aktuální sadu cílů a pravidla.

 Dojde pouze v případě, že se provádí analýzu pomocí příkazového řádku verze tohoto nástroje.
