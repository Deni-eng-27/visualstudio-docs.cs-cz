# <a name="ca0057"></a>CA0057
Porušení nelze mapovat na aktuální sadu cílů a pravidla

 K této chybě dochází, pokud FxCopCmd.exe je vyvolána ručně pomocí souboru projektu nebo / importovat sestavu XML, který má porušení zásad, které nelze mapovat na aktuální sadu cílů a pravidla.

 Nastane pouze v případě, že se provádí analýzu pomocí příkazového řádku verzi nástroje.
