# <a name="c6518"></a>C6518
Upozornění C6518: Neplatná Anotace: Vlastnost ' SAL_writableTo ' nesmí být zadána jako předběžná podmínka u vyrovnávacích pamětí, které nejsou zapisovatelné: ' \_Param @ no__t-1 (1) '

 Toto upozornění znamená, že existuje konflikt mezi hodnotou vlastnosti `SAL_writableTo` a vlastností s možností zápisu. To obvykle znamená, že vlastnost s možností zápisu nemá oprávnění k zápisu do poznámení s parametrem.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože anotace `_Out_` se zkompiluje, aby zahrnovala vlastnost `SAL_writableTo`, která nepovoluje přístup pro zápis:

```cpp
#include <sal.h>
void f(_Out_ const char* pc)
{
    //code that can't write to *pc ...
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
#include <sal.h>
void f(_Out_  char* pc)
{
    pc = "Hello World";
    //code ...
}
```
