# <a name="c28020"></a>C28020
Warning C28020: výraz Expression \<expr > není v tomto volání pravdivý.

 Toto upozornění je hlášeno, pokud \_splňuje\_ uvedený výraz není true. Často to indikuje nesprávný parametr.

 Pokud k tomu dojde v deklaraci funkce, poznámky označují nemožnou podmínku.
