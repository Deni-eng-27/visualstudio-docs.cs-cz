# <a name="c28272"></a>C28272
upozornění C28272: poznámky pro funkce, parametr při zkoumání není konzistentní s deklarace funkce

 Toto upozornění označuje chybu v poznámky, nikoli na kód, který je analyzován. Poznámky, které jsou uvedeny v definici funkce nejsou konzistentní s těmi, které jsou uvedeny v prohlášení. Dva poznámky by měla vyřešit tak, aby odpovídaly.
