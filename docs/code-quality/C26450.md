# <a name="arithmetic-overflow-operator-operation-causes-overflow-at-compile-time-use-a-wider-type-to-store-the-operands"></a>Aritmetické přetečení: '% – operátor %' operace způsobí přetečení v době kompilace. Pro uložení operandů použijte širší typ

 Toto upozornění signalizuje, že byl aritmetické operace prokazatelně míru ztrát v době kompilace. Můžete být uplatněna při operandy jsou všechny konstanty z doby kompilace.  V současné době zkontrolujeme operátor posunu vlevo, násobení, sčítání a odčítání operace takové přetečení.

Poznámka: C4307 je podobné kontrolu v kompilátoru C++ společnosti Microsoft.

## <a name="example-1"></a>Příklad 1

```cpp
int multiply()
{
    const int a = INT_MAX;
    const int b = 2;
    int c = a * b; // C26450 reported here
    return c;
}
```
 Chcete-li opravit toto upozornění, použijte následující kód.

```cpp
long long multiply()
{
    const int a = INT_MAX;
    const int b = 2;
    long long c = (long long)a * b; // OK
    return c;
}
```
