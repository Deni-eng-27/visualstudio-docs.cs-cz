# <a name="ca5370-use-xmlreader-for-validating-reader"></a>CA5370: Použít XmlReader pro ověřování čtečky

|Položka|Hodnota|
|-|-|
|RuleId|CA5370|
|Kategorie|Microsoft.Security|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Ověřování nedůvěryhodného vstupu XML s `XmlValidatingReader` instancí třídy bez `XmlReader` objektu může potenciálně vést k odepření služby, zpřístupnění informací a padělání požadavků na straně serveru. Tyto útoky jsou povoleny nedůvěryhodnou verzí DTD a zpracování schématu XML, což umožňuje zahrnutí bomby XML a škodlivých externích entit do XML. Pouze pomocí `XmlReader` je možné zakázat DTD. Vložené schéma XML je zpracováváno stejně jako `XmlReader` `ProhibitDtd` vlastnost a ve `ProcessInlineSchema` výchozím nastavení nastaveno na hodnotu `false` v .NET Framework počínaje verzí 4,0.

## <a name="rule-description"></a>Popis pravidla

Zpracování nedůvěryhodných schémat DTD a XML může povolit načítání nebezpečných externích odkazů. Toto nebezpečné načítání může být omezeno pomocí `XmlReader` zabezpečeného překladače nebo zpracování vloženého schématu DTD a XML. Toto pravidlo detekuje kód, který používá `XmlValidatingReader` třídu bez použití `XmlReader` parametru konstruktoru.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

- Použijte `XmlValidatingReader(XmlReader)` with `ProhibitDtd` a `ProcessInlineSchema` vlastností nastavte na `false` .
- Počínaje .NET Framework 2,0 `XmlValidatingReader` se považuje za zastaralou. Pomocí můžete vytvořit instanci aplikace pro ověřování <xref:System.Xml.XmlReader.Create%2A?displayProperty=nameWithType> .

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Toto upozornění můžete potenciálně potlačit, pokud `XmlValidatingReader` se vždy používá k ověření XML, které pochází z důvěryhodného zdroje, a proto nemůže být úmyslně poškozen.

## <a name="pseudo-code-examples"></a>Příklady kódu pseudo

### <a name="violation"></a>Selhání

Následující ukázka pseudo kódu znázorňuje vzor zjištěný tímto pravidlem.
Typ prvního parametru `XmlValidatingReader.XmlValidatingReader()` není `XmlReader` .

```csharp
using System;
using System.IO;
using System.Xml;
...
public void TestMethod(Stream xmlFragment, XmlNodeType fragType, XmlParserContext context)
{
    var obj = new XmlValidatingReader(xmlFragment, fragType, context);
}
```

### <a name="solution"></a>Řešení

```csharp
using System;
using System.Xml;
...
public void TestMethod(XmlReader xmlReader)
{
    var obj = new XmlValidatingReader(xmlReader);
}
```
