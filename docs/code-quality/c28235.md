# <a name="c28235"></a>C28235
Upozornění C28235: funkce nemůže být samostatná jako Poznámka.

 Toto upozornění indikuje, že došlo k pokusu o použití funkce, která nebyla deklarována jako anotace v kontextu poznámky. To zahrnuje použití primitivní operace (PrimOp) v samostatném kontextu. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
