# <a name="c28163"></a>C28163
Upozornění C28163: Funkce by nikdy neměla být volána v rámci bloku try/except

 Toto upozornění je hlášeno, je-li funkce typu, který by neměl být nikdy uzavřen v bloku `try/except`, který je nalezen v bloku `try/except`. Nástroj Analýza kódu nalezl alespoň jednu cestu, ve které byla funkce volána v rámci bloku `try/except`.
