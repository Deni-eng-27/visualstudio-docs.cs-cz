# <a name="c28196"></a>C28196
Upozornění C28196: Požadavek není splněn. (Výraz se nevyhodnotí jako true.)

 Toto upozornění znamená, že prototyp funkce pro analyzovanou funkci má `__notnull`, `__null` nebo `__drv_valueIs` na parametr `_Out_` nebo návratovou hodnotu, ale vrácená hodnota je nekonzistentní s touto poznámkou.
