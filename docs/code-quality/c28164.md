# <a name="c28164"></a>C28164
Upozornění C28164: Argument se předává funkci, která očekává ukazatel na objekt (ne ukazatel na ukazatel).

 Toto upozornění je hlášeno, je-li ve volání funkce, která očekává ukazatel na objekt, použit ukazatel na ukazatel.

 Funkce přebírá PVOID na této pozici. Obvykle to znamená, že se používá & p*xxx* , když je požadována p*xxx* .

 Některé *polymorfní funkce* (funkce, které mohou být vyhodnoceny a použity pro, hodnoty různých typů) jsou implementovány v jazyce C pomocí argumentu PVOID, který přijímá libovolný typ ukazatele. To však umožňuje programátorovi nakódovat ukazatel na ukazatel bez toho, aby došlo k chybě kompilátoru, a to i v případě, že tento typ není vhodný.

## <a name="example"></a>Příklad
 Následující příklad kódu vygeneruje toto upozornění:

```
PFAST_MUTEX pFm;
...
KeWaitForSingleObject(&pFm, UserRequest, UserMode, false, NULL);
```

 Následující příklad kódu zabraňuje upozornění:

```
PFAST_MUTEX pFm;
...
KeWaitForSingleObject(pFm, UserRequest, UserMode, false, NULL);
```
