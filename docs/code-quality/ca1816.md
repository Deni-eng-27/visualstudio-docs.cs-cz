# <a name="ca1816-call-gcsuppressfinalize-correctly"></a>CA1816: Volejte správně GC.SuppressFinalize

|||
|-|-|
|TypeName|CallGCSuppressFinalizeCorrectly|
|CheckId|CA1816|
|Kategorie|Microsoft. Použití|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>příčina

Porušení tohoto pravidla mohou způsobovat tyto příčiny:

- Metoda, která je implementací <xref:System.IDisposable.Dispose%2A?displayProperty=nameWithType> a nevolá <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>.

- Metoda, která není implementací <xref:System.IDisposable.Dispose%2A?displayProperty=nameWithType> a volání <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>.

- Metoda, která volá <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType> a předá něco jiného než [ThisC#()](/dotnet/csharp/language-reference/keywords/this) nebo [mne (Visual Basic)](/dotnet/visual-basic/programming-guide/program-structure/me-my-mybase-and-myclass#me).

## <a name="rule-description"></a>Popis pravidla

Metoda <xref:System.IDisposable.Dispose%2A?displayProperty=nameWithType> umožňuje uživatelům uvolnit prostředky kdykoli předtím, než se objekt stane dostupným pro uvolňování paměti. Pokud je zavolána metoda <xref:System.IDisposable.Dispose%2A?displayProperty=nameWithType>, uvolní prostředky objektu. To způsobuje nutnost finalizace. <xref:System.IDisposable.Dispose%2A?displayProperty=nameWithType> by měl volat <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>, aby systém uvolňování paměti nevolal finalizační metodu objektu.

Chcete-li zabránit odvozeným typům pomocí finalizační metody, aby bylo možné znovu implementovat <xref:System.IDisposable> a volat je, nezapečetěné typy bez finalizační metody by měly stále volat <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení tohoto pravidla:

- Pokud je metodou implementace <xref:System.IDisposable.Dispose%2A>, přidejte volání <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>.

- Pokud metoda není implementací <xref:System.IDisposable.Dispose%2A>, buď odeberte volání metody <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType> nebo ji přesuňte do implementace <xref:System.IDisposable.Dispose%2A> typu.

- Chcete-li předat [tuto hodnotuC#()](/dotnet/csharp/language-reference/keywords/this) nebo [mne (Visual Basic)](/dotnet/visual-basic/programming-guide/program-structure/me-my-mybase-and-myclass#me), změňte všechna volání na <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>.

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Potlačí upozornění z tohoto pravidla pouze v případě, že záměrně nepoužíváte <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType> k řízení doby života jiných objektů. Potlačí upozornění z tohoto pravidla, pokud implementace <xref:System.IDisposable.Dispose%2A> nevolá <xref:System.GC.SuppressFinalize%2A?displayProperty=nameWithType>. V takovém případě neúspěšné potlačení efektivity snižuje výkon a neposkytuje žádné výhody.
