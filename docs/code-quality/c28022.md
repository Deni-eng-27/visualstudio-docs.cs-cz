# <a name="c28022"></a>C28022
Warning C28022: třída Functions (ES) \<classlist1 > na této funkci se neshoduje s > třídy funkcí \<classlist2 na definici TypeDef použitou k jejímu definování.

 Toto upozornění se oznamuje v případě, že poznámky obsahují chybu. Definice typedef i funkce mají `_Function_class_` anotace, ale neodpovídají. Pokud se obě použijí, musí se shodovat.
