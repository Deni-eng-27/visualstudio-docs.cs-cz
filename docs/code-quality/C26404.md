# <a name="c26404--dontdeleteinvalid"></a>C26404 DONT_DELETE_INVALID
Jakmile vlastník ukazatel uvolní nebo přenáší jeho prostředků, získá do stavu "invalid". Odstranění takové ukazatel může vést k poškození okamžitou paměti z důvodu dvojité odstranit, nebo narušení přístupu při přístupu k prostředku z jiného vlastníka ukazatele.

## <a name="example-1-deleting-an-owner-after-transferring-its-value"></a>Příklad 1: Odstranění vlastníka po přenosu jeho hodnota
```cpp
gsl::owner<State*> validState = nullptr;
gsl::owner<State*> state = ReadState();
validState = state;
if (!IsValid(state))
    delete state;   // C26404
```

## <a name="example-2-deleting-an-uninitialized-owner"></a>Příklad 2: Odstranění Neinicializovaný vlastníka
```cpp
gsl::owner<Message*> message;
if (popLast)
    message = ReleaseMessage();
delete message; // C26404
```
