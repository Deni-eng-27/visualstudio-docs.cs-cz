# <a name="c6287"></a>C6287
Upozornění C6287: redundantní kód: levé a pravé dílčí výrazy jsou identické.

 Toto upozornění znamená, že ve výrazu byl zjištěn redundantní element.

 Je obtížné posuzovat závažnost tohoto problému bez přezkoumání kódu. Duplicitní test na vlastní je neškodný, ale důsledky odstranění druhého testu můžou být závažné. Kód by měl být zkontrolován, aby bylo zajištěno, že test nebyl vynechán.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 Následující kód ukazuje různé způsoby, jak toto upozornění opravit:

```
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
