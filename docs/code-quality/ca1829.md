# <a name="ca1829-use-lengthcount-property-instead-of-enumerablecount-method"></a>CA1829: použijte vlastnost length/Count namísto metody vyčíslitelné. Count.

|||
|-|-|
|TypeName|UsePropertyInsteadOfCountMethodWhenAvailableAnalyzer|
|CheckId|CA1829|
|Kategorie|Microsoft. Performance|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Metoda <xref:System.Linq.Enumerable.Count%2A> LINQ byla použita pro typ, který podporuje ekvivalentní, efektivnější `Length` nebo `Count` vlastnost.

## <a name="rule-description"></a>Popis pravidla

Toto pravidlo označuje volání <xref:System.Linq.Enumerable.Count%2A> metody LINQ na kolekce typů, které mají ekvivalentní, ale efektivnější `Length` nebo `Count` vlastnost pro načtení stejných dat. `Length`vlastnost `Count` nebo nevytvoří výčet kolekce, proto je efektivnější.

Toto pravidlo označuje <xref:System.Linq.Enumerable.Count%2A> volání následujících typů kolekcí s `Length` vlastností:

- <xref:System.Array?displayProperty=fullName>
- <xref:System.Collections.Immutable.ImmutableArray%601?displayProperty=fullName>

Toto pravidlo označuje <xref:System.Linq.Enumerable.Count%2A> volání následujících typů kolekcí s `Count` vlastností:

- <xref:System.Collections.ICollection?displayProperty=fullName>
- <xref:System.Collections.Generic.ICollection%601?displayProperty=fullName>
- <xref:System.Collections.Generic.IReadOnlyCollection%601?displayProperty=fullName>

Typy analyzovaných kolekcí mohou být v budoucnu rozšířeny, aby pokryly další případy.

> [!NOTE]
> Pravidlo CA1829 není k dispozici v analýze starší verze. Poprvé jsme zavedli v [analyzátorech FxCop](https://www.nuget.org/packages/Microsoft.CodeAnalysis.FxCopAnalyzers) verze 2.9.6.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení, nahraďte volání <xref:System.Linq.Enumerable.Count%2A> metody pomocí přístupu k `Length` vlastnosti `Count` nebo. Například následující dva fragmenty kódu ukazují porušení pravidla a jejich opravu:

```csharp
using System.Collections.Generic;
using System.Linq;

class C
{
    public int GetCount(int[] array)
        => array.Count();

    public int GetCount(ICollection<int> collection)
        => collection.Count();
}
```


```csharp
using System.Collections.Generic;

class C
{
    public int GetCount(int[] array)
        => array.Length;

    public int GetCount(ICollection<int> collection)
        => collection.Count;
}
```

> [!TIP]
> Pro toto pravidlo v aplikaci Visual Studio je k dispozici oprava kódu. Pokud ho chcete použít, umístěte kurzor na porušení a stiskněte **CTRL**+**.** (tečka). Pokud je k dispozici v seznamu zobrazených možností, vyberte **použít vlastnost length/Count namísto Count ()** .
>
