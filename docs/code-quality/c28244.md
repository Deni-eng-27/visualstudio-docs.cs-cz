# <a name="c28244"></a>C28244
Upozornění C28244: Poznámka pro funkci má neanalyzovaný parametr/externí anotaci.

 V tuto chvíli by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
