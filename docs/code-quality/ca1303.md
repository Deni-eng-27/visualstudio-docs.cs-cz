# <a name="ca1303-do-not-pass-literals-as-localized-parameters"></a>CA1303: Nepředávejte literály jako lokalizované parametry

|||
|-|-|
|TypeName|DoNotPassLiteralsAsLocalizedParameters|
|CheckId|CA1303|
|Kategorie|Microsoft.Globalizace|
|Narušující změny|Nenarušující|

## <a name="cause"></a>Příčina

Metoda předá literál řetězce jako parametr konstruktoru nebo metodě .NET a tento řetězec by měl být lokalizovatelný.

Toto upozornění je vyvolána při literál řetězec je předán jako hodnota parametr nebo vlastnost a jeden nebo více z následujících případů je true:

- Atribut <xref:System.ComponentModel.LocalizableAttribute> parametru nebo vlastnosti je nastaven na hodnotu true.

- Název parametru řetězce, který je předán metodě Console.Write nebo Console.WriteLine, je "hodnota" nebo "formát".

Ve výchozím nastavení toto pravidlo analyzuje celý základ kódu, ale toto je [konfigurovatelné](#excluded-type-names-with-derived-types).

Počínaje verzí 3.0.0 balíčku analyzátoru se heuristická pojmenování již nepoužívá, ale je [konfigurovatelná](#use-naming-heuristic).

## <a name="rule-description"></a>Popis pravidla

Řetězcové literály, které jsou vloženy do zdrojového kódu, je obtížné lokalizovat.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení tohoto pravidla, nahraďte literál řetězce <xref:System.Resources.ResourceManager> řetězcem načteným prostřednictvím instance třídy.

## <a name="when-to-suppress-warnings"></a>Kdy potlačit varování

Je bezpečné potlačit upozornění z tohoto pravidla, pokud knihovna kódu nebude lokalizována nebo pokud řetězec není vystaven koncovému uživateli nebo vývojáři používající knihovnu kódu.

Uživatelé mohou eliminovat šum proti metodám, které by neměly být předány lokalizované řetězce buď přejmenování parametr nebo vlastnost, nebo označením těchto položek jako podmíněné.

## <a name="configurability"></a>Konfigurovatelnost

Pokud používáte toto pravidlo z [analyzátorů FxCop](install-fxcop-analyzers.md) (a ne s analýzou starší verze), je toto pravidlo konfigurovatelné.

### <a name="excluded-type-names-with-derived-types"></a>Vyloučené názvy typů s odvozenými typy

Můžete nakonfigurovat, které typy, včetně odvozených typů, chcete vyloučit z analýzy. Chcete-li například určit, že pravidlo by nemělo být spuštěno u žádné metody v rámci typů s názvem `MyType` a odvozených typů, přidejte do souboru .editorconfig v projektu následující dvojici klíč-hodnota:

```ini
dotnet_code_quality.CA1303.excluded_type_names_with_derived_types = MyType
```

Povolené formáty názvů symbolů v hodnotě volby (oddělené |):

- Pouze název typu (zahrnuje všechny typy s názvem, bez ohledu na obsahující typ nebo obor názvů)
- Plně kvalifikované názvy ve [formátu ID dokumentace](https://github.com/dotnet/csharplang/blob/master/spec/documentation-comments.md#id-string-format) `T:` symbolu s volitelnou předponou.

Příklady:

| Hodnota možnosti | Souhrn |
| --- | --- |
|`dotnet_code_quality.CA1303.excluded_type_names_with_derived_types = MyType` | Odpovídá všem typům s názvem MyType a všem jeho odvozeným typům v kompilaci.
|`dotnet_code_quality.CA1303.excluded_type_names_with_derived_types = MyType1|MyType2` | Odpovídá všem typům s názvem MyType1 nebo MyType2 a všem jejich odvozeným typům v kompilaci.
|`dotnet_code_quality.CA1303.excluded_type_names_with_derived_types = M:NS.MyType` | Odpovídá určitému typu MyType s daným plně kvalifikovaným názvem a všemi odvozenými typy.
|`dotnet_code_quality.CA1303.excluded_type_names_with_derived_types = M:NS1.MyType1|M:NS2.MyType2` | Odpovídá určitým typům MyType1 a MyType2 s příslušnými plně kvalifikovanými názvy a všemi odvozenými typy.

Tuto možnost můžete nakonfigurovat pouze pro toto pravidlo, pro všechna pravidla nebo pro všechna pravidla v této kategorii (Globalizace). Další informace naleznete v [tématu Konfigurace analyzátorů FxCop](configure-fxcop-analyzers.md).

### <a name="use-naming-heuristic"></a>Použití pojmenování heuristické

Můžete nakonfigurovat, zda parametry nebo vlastnosti název obsahující "Text", "Zpráva" nebo "Titulek" spustí toto pravidlo.

```ini
dotnet_code_quality.CA1303.use_naming_heuristic = true
```
