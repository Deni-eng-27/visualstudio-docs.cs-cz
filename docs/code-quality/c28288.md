# <a name="c28288"></a>C28288
Upozornění C28288: Pro funkci. Chyba syntaxe v \_na\_poznámky () (neplatný název parametru)

 Nástroj Analýza kódu oznamuje toto upozornění, když je použita anotace `SAL_at` (`__drv_at`) a výraz parametru nelze interpretovat v aktuálním kontextu. To může zahrnovat použití nesprávně napsaného parametru nebo názvu člena nebo chybného pravopisu klíčových slov Return nebo this.
