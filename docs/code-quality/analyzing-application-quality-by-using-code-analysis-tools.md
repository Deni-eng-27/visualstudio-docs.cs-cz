### <a name="code-analysis-for-drivers"></a>Analýza kódu pro ovladače  
 Nástrojů pro analýzu kódu může pomoct zlepšit stabilitu a spolehlivost tohoto ovladače systematičtěji analýzou ovladač zdrojového kódu.  
  
 [Analýza kvality ovladačů pomocí nástrojů pro analýzu kódu](http://go.microsoft.com/fwlink/?LinkId=227618)  
 Analýzy kódu pro ovladače statické ověření kompilaci nástroj, který zjistí základní kódování chyby v programy C a C++ a zahrnuje specializovaná modul, který slouží ke zjištění chyby v kód ovladač režimu jádra (hlavně)... Statické ovladače ověřovatele (SDV) je nástroj statické ověření, který systematičtěji analyzuje zdrojový kód ovladače režimu jádra systému Windows. SDV Určuje, jestli ovladač správně komunikuje s jádrem operačního systému Windows.
