# <a name="c6312"></a>C6312
Upozornění C6312: možnou nekonečnou smyčku: použití konstanty EXCEPTION_CONTINUE_EXECUTION ve výrazu filtru výjimky try-except

 Toto upozornění označuje použití konstanty `EXCEPTION_CONTINUE_EXECUTION` (nebo jiné konstanty, která je vyhodnocena na hodnotu-1) ve výrazu filtru strukturované obslužné rutiny výjimky. Použití konstantní hodnoty `EXCEPTION_CONTINUE_EXECUTION` by mohlo vést k nekonečné smyčce. Například pokud byla výjimka vyvolána hardwarem, bude instrukce, která způsobila výjimku, restartována. Pokud je adresa, která způsobila výjimku, stále chybná, dojde k další výjimce, která bude zpracována stejným způsobem. To způsobuje nekonečnou smyčku.

 Explicitní volání `RaiseException` nebude přímo způsobovat nekonečnou smyčku, ale bude pokračovat v provádění kódu v chráněném bloku. To může být neočekávané a může vést k nekonečné smyčce, pokud se používá `RaiseException`, aby se předešlo přesměrování neplatného ukazatele.

 Obvykle by `EXCEPTION_CONTINUE_EXECUTION` měla být vrácena pouze funkcí volanou ve výrazu filtru, který má možnost opravit buď ukazatel, který způsobil výjimku, nebo základní paměť.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <excpt.h>
#include <stdio.h>
#include <windows.h>

void f (char *ptr)
{
  __try
  {
    // exception occurs if the caller passes null ptr
    // code...
   *ptr = '\0';
  }
  __except (EXCEPTION_CONTINUE_EXECUTION)
    // When EXCEPTION_CONTINUE_EXECUTION is used, the handler
    //  block of the structured exception handler is not executed.
  {
    puts("This block is never executed");
  }
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#include <excpt.h>
#include <stdio.h>
#include <windows.h>

void f (char *ptr)
{
  __try
  {
    // exception occurs if the caller passes null ptr
    // code...
    *ptr = '\0';
  }
  __except (GetExceptionCode()== EXCEPTION_ACCESS_VIOLATION ?
               EXCEPTION_EXECUTE_HANDLER : EXCEPTION_CONTINUE_SEARCH)
  {
    puts("Error Occurred");
  }
}
```
