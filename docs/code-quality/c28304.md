# <a name="c28304"></a>C28304
Upozornění C28304: u C++ parametru reference > < parameter_name byl v \<token > nalezen nesprávně umístěný operátor `_Notref_`.

 Operátor `_Notref_` by měl být použit pouze ve zvláštních případech týkajících C++ se parametrů odkazu a pouze v makrech poskytnutých systémem. Musí hned následovat operátor `_Deref_` nebo funkční Poznámka.
