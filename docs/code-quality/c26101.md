# <a name="c26101"></a>C26101
Upozornění C26101: selhání pro správné použití propojené operace pro proměnnou \<var >.

 Rozhraní API systému Windows nabízejí celou řadu vzájemně propojených operací. Anotace `_Interlocked_` určuje, že by měla být k proměnné přistupovat pouze prostřednictvím propojené operace. Upozornění C26101 se vydá v případě, že přístup není konzistentní s poznámkou `_Interlocked_`.

## <a name="example"></a>Příklad
 Následující příklad generuje upozornění C26101, protože došlo k porušení smlouvy `_Interlocked_`.

```

CRITICAL_SECTION cs;
typedef struct _DATA
{
    _Interlocked_ LONG data;
} DATA;

void Safe(DATA* p)
{
    InterlockedIncrement(&p->data); // OK
}

void Unsafe(DATA* p)
{
    p->data += 1; // Warning C26101
    EnterCriticalSection(&cs);
    LeaveCriticalSection(&cs);
}
```
