# <a name="c28024"></a>C28024
Upozornění C28024: ukazatel funkce, ke kterému se přiřazuje, je opatřen poznámkou se třídou funkce \<class >, která není obsažena ve třídách funkce \<classlist >.

 Toto upozornění je oznámeno, pokud byly obě funkce opatřeny poznámkami třídy funkce, ale třídy se neshodují.

 Toto upozornění se vydá, když má ukazatel na funkci anotaci `_Function_class_`, která určuje, že se k němu mají přiřadit jenom funkce konkrétní funkční třídy. U přiřazení nebo implicitního přiřazení ve volání funkce musí být zdroj a cíl stejné třídy Function, ale třídy funkcí se neshodují.
