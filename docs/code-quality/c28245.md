# <a name="c28245"></a>C28245
Upozornění C28245: Poznámka pro funkci má poznámku "This" na funkci bez členu.

 V souboru s rychlým modelem došlo k vnitřní chybě. Toto upozornění by nemělo být v typickém použití.
