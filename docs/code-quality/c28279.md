# <a name="c28279"></a>C28279
upozornění C28279: Pro symbol "begin" bylo nalezeno bez odpovídajícího "end"

 Poznámka jazyk podporuje begin a end (`{` a `}` v jazyce C) konstrukce a párování obdržel nevyvážené. Pokud se používají makra se můžete vyhnout této situaci.
