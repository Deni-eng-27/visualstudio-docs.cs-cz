# <a name="c28279"></a>C28279
Upozornění C28279: pro symbol byl nalezen prvek Begin bez odpovídajícího příkazu end.

 Jazyk anotace podporuje konstrukci Begin a end (`{` a `}` v C) a párování se vyrovnává. Tato situace se může vyhnout, pokud jsou použita makra.
