# <a name="c28280"></a>C28280
Upozornění C28280: Pro symbol bylo nalezeno "end" bez odpovídajícího "begin"

 Jazyk poznámek podporuje konstrukci Begin a end (`{` a `}` v C) a párování se vyrovnává. Tato situace se může vyhnout, pokud jsou použita makra.
