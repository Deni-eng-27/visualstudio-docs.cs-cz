# <a name="c6064"></a>C6064
Warning C6064: chybí argument typu Integer pro \<function > odpovídající specifikátoru převodu \<number >

 Toto upozornění označuje, že není k dispozici dostatek argumentů pro shodu s formátovacím řetězcem a jeden z chybějících argumentů je celé číslo. Tato vada může způsobit nesprávný výstup.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože při volání metody `sprintf` byl použit nesprávný počet argumentů a chybějící argument byl celé číslo:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello, World";

  sprintf(buff,"%s %d", string);
}
```

 Chcete-li toto upozornění opravit, zadejte chybějící argumenty, jak je znázorněno v následujícím kódu:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string = "Hello, World";

  sprintf(buff,"%s %d",string, strlen(string));
}
```

 Následující kód používá bezpečnou funkci manipulace s řetězci `sprintf_s` pro opravu tohoto upozornění:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello World";

  sprintf_s(buff,sizeof(buff),"%s %d", string, strlen(string));
}
```
