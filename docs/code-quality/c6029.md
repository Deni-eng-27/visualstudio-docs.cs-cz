# <a name="c6029"></a>C6029

Upozornění C6029: možné přetečení vyrovnávací paměti při volání metody \<function >: použití nezaškrtnuté hodnoty

Toto upozornění indikuje, že funkce, která přebírá vyrovnávací paměť a velikost, se předává Nekontrolovaná velikost. Data načtená z externího zdroje nebyla ověřena, aby bylo možné zjistit, zda je menší než velikost vyrovnávací paměti. Útočník může úmyslně určit mnohem větší než očekávanou hodnotu velikosti, což vede k přetečení vyrovnávací paměti.

Obecně platí, že kdykoli si přečtete data z nedůvěryhodného externího zdroje, Nezapomeňte ověřit její platnost. Je obvykle vhodné ověřit velikost, abyste se ujistili, že je v očekávaném rozsahu.

## <a name="example"></a>Příklad

Následující kód vygeneruje toto upozornění voláním funkce s poznámkou funkce [ReadFile](https://docs.microsoft.com/windows/desktop/api/fileapi/nf-fileapi-readfile) dvakrát. Po prvním volání vlastnost post atributu označí druhý parametr hodnota nedůvěryhodná. Proto předáním nedůvěryhodné hodnoty do druhého volání `ReadFile` vygeneruje toto upozornění, jak je znázorněno v následujícím kódu:

```cpp

#include "windows.h"

bool f(HANDLE hFile)
{
    char buff[MAX_PATH];

    DWORD cbLen;
    DWORD cbRead;

    // Read the number of byte to read (cbLen).
    if (!ReadFile (hFile, &cbLen, sizeof (cbLen), &cbRead, NULL))
    {
        return false;
    }
    // Read the bytes
    if (!ReadFile (hFile, buff, cbLen, &cbRead, NULL))  // warning 6029
    {
        return false;
    }

    return true;
}
```

Chcete-li toto upozornění opravit, ověřte velikost vyrovnávací paměti, jak je znázorněno v následujícím kódu:

```cpp
bool f(HANDLE hFile)
{
    char buff[MAX_PATH];

    DWORD cbLen;
    DWORD cbRead;

    // Read the number of byte to read (cbLen).
    if (!ReadFile (hFile, &cbLen, sizeof (cbLen), &cbRead, NULL))
    {
        return false;
    }
    // Ensure that there's enough space in the buffer to read that many bytes.
    if (cbLen > sizeof(buff))
    {
        return false;
    }
    // Read the bytes
    if (!ReadFile (hFile, buff, cbLen, &cbRead, NULL))  // warning 6029
    {
        return false;
    }

    return true;
}
```
