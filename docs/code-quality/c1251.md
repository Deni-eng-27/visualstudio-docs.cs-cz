# <a name="c1251"></a>C1251
Upozornění C1251: Nepovedlo se načíst modely.

 Nástroj Analýza kódu hlásí toto upozornění, pokud dojde k vnitřní chybě v souboru modelu, nikoli v analyzovaném kódu.
