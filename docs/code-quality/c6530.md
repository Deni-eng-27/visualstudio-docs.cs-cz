# <a name="c6530"></a>C6530
upozornění 6530: Nerozpoznaný formát řetězec styl \<name >  
  
> [!NOTE]
>  Toto upozornění se zobrazí pouze v kódu, který používá zastaralou verzi jazyka poznámky zdrojový kód (SAL). Doporučujeme vám, že je port kód a použít nejnovější verzi SAL. Další informace najdete v tématu [použití poznámek SAL k snížení závad kódu C/C++](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).  
  
 Toto upozornění znamená, že vlastnost FormatString používá jinou hodnotu než scanf nebo printf. Chcete-li toto upozornění, zkontrolujte kódu a použijte platnou hodnotu pro vlastnost stylu.  
  
## <a name="example"></a>Příklad  
 Následující kód vytvoří toto upozornění z důvodu překlepem v `Style` vlastnost:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printfd")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printfd")] char *px);  
  
```  
  
 Chcete-li toto upozornění, použijte platný styl, jak je znázorněno v následujícím kódu:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printf")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printf")] char *px);   
```
