# <a name="c6103"></a>C6103
Upozornění C6103: Vrácení \<variable > z neúspěšného volání funkce na řádku \<location >.

 Úspěšná cesta prostřednictvím funkce vrací proměnnou, která byla použita jako parametr `_Out_` do interního volání funkce, které selhalo.

 Problémem může být, že volaná funkce a volající funkce nejsou zcela anotovány. Volaná funkce může vyžadovat `_Always_`, `_Outptr_result_nullonfailure_` (`_COM_Outptr_` pro kód COM) nebo související anotaci a funkce volání může vyžadovat anotaci `_Success_`. Další možnou příčinou tohoto upozornění je, že Poznámka `_Success_` u volané funkce je nesprávná.
