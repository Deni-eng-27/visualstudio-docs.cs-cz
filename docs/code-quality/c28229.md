# <a name="c28229"></a>C28229
Upozornění C28229: Poznámka nepodporuje parametry

 Toto upozornění indikuje, že Poznámka byla použita s parametrem, pokud je Poznámka deklarována bez parametrů. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
