# <a name="c28229"></a>C28229
upozornění C28229: poznámky nepodporuje parametry  
  
 Toto upozornění označuje, že poznámky se používat s parametrem při Poznámka je deklarovaná bez parametrů. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v typické použití.
