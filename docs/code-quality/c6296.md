# <a name="c6296"></a>C6296
upozornění C6296: Špatně definovaná smyčka for: tělo smyčky se provede pouze jednou

 Toto upozornění indikuje, že smyčka for nemusí fungovat tak, jak je zamýšlená. Pokud je index nepodepsaný a smyčka se odpočítává od nuly, jeho tělo se spustí jenom jednou.

## <a name="example"></a>Příklad
 Následující kód generuje toto upozornění:

```cpp
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 Pro opravu tohoto upozornění použijte následující kód:

```cpp
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
