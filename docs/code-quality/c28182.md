# <a name="c28182"></a>C28182
Upozornění C28182: Přesměrování ukazatele NULL.

 **Další informace**: *\<pointer1 >* obsahuje stejnou hodnotu null jako *\<pointer2 >* *\<note >*

 Nástroj Analýza kódu oznamuje toto upozornění, když potvrdí, že ukazatel může mít hodnotu NULL. Pokud existují nepotvrzené instance, u kterých může dojít k chybě dříve v trasování, nástroj pro analýzu kódu přidá do zprávy upozornění číslo řádku první instance, abyste mohli změnit kód tak, aby se vyřešily všechny instance.

 *\<pointer2 >* je potvrzena jako potenciálně null. *\<pointer1 >* obsahuje stejnou hodnotu jako *pointer2* a právě se odkazuje na ni. Vzhledem k tomu, že tyto ukazatele mohou být v kódu na hodně různých místech, obě jsou hlášeny, abyste mohli určit, proč Nástroj pro analýzu kódu hlásí toto upozornění.

 Pokud existuje nepotvrzená dřívější instance podmínky, *\<note >* nahrazuje tímto textem: "Viz line *@no__t – 1number >* pro dřívější umístění, kde to může nastat."

## <a name="example"></a>Příklad
 Následující příklad ukazuje kód, který může způsobit, že nástroj pro analýzu kódu vygeneruje tuto zprávu upozornění. V tomto příkladu Nástroj Analýza kódu určuje, že `pNodeFree` v příkazu `if` má hodnotu NULL a cesta kódu k těle `if` je provedena. Vzhledem k tomu, že `nBlockSize` je potenciálně nula, není provedeno tělo příkazu `for` a `pNodeFree` zůstane beze změny. `pNodeFree` se pak přiřadí k `pNode` a `pNode` se použije, když by mohlo dojít k navýšení zpětného odkazování na hodnotu NULL.

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    pNode->pPrev = 0;
}
```

 Nástroj Analýza kódu hlásí následující upozornění:

```
:\sample\testfile.cpp(24) : warning C28182: Dereferencing NULL pointer. 'pNode' contains the same NULL value as 'pNodeFree' did.: Lines: 12, 14, 16, 23, 24
```

## <a name="example"></a>Příklad
 Jedním ze způsobů, jak opravit předchozí příklad, je zaškrtnout `pNode` pro nulu před tím, než se na něj odvolá, aby byl zpětný odkaz na hodnotu NULL. Následující kód ukazuje tuto opravu.

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    if(pNode != 0)
      pNode->pPrev = 0;
}
```
