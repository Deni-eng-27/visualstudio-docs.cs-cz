# <a name="ca2235-mark-all-non-serializable-fields"></a>CA2235: Označte všechna neserializovatelná pole

|Položka|Hodnota|
|-|-|
|CheckId|CA2235|
|Kategorie|Microsoft. Usage|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Neserializovatelný typ pole instance je deklarován v serializovatelném typu.

## <a name="rule-description"></a>Popis pravidla

Serializovatelný typ je jeden, který je označen <xref:System.SerializableAttribute?displayProperty=fullName> atributem. Při serializaci typu je <xref:System.Runtime.Serialization.SerializationException?displayProperty=fullName> vyvolána výjimka, pokud typ obsahuje pole instance typu, který není serializovatelný *a* neimplementuje <xref:System.Runtime.Serialization.ISerializable?displayProperty=fullName> rozhraní.

> [!TIP]
> CA2235 se neaktivují pro pole instance typů, která implementují, <xref:System.Runtime.Serialization.ISerializable> protože poskytují svou vlastní logiku serializace.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení tohoto pravidla, použijte <xref:System.NonSerializedAttribute?displayProperty=fullName> atribut pro pole, které není serializovatelný.

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Pouze potlačí upozornění z tohoto pravidla <xref:System.Runtime.Serialization.ISerializationSurrogate?displayProperty=fullName> , je-li deklarován typ, který umožňuje serializaci a deserializaci instancí pole.

## <a name="remarks"></a>Poznámky

CA2235 pravidla neanalyzuje typy, které implementují <xref:System.Runtime.Serialization.ISerializable> rozhraní (pokud nejsou označeny také <xref:System.SerializableAttribute> atributem). Důvodem je, že [CA2237 pravidla](../code-quality/ca2237.md) už doporučuje označení typů, které implementují <xref:System.Runtime.Serialization.ISerializable> rozhraní s <xref:System.SerializableAttribute> atributem.
