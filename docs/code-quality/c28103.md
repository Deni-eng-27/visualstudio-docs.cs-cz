# <a name="c28103"></a>C28103
Upozornění C28103: nevracení prostředků

 Zadaný objekt obsahuje prostředek, který nebyl uvolněn. Volaná funkce byla opatřena poznámkou `__drv_acquiresResource` nebo `__drv_acquiresResourceGlobal` a toto upozornění indikuje, že prostředek s názvem v anotaci nebyl uvolněn.

## <a name="example"></a>Příklad
 Následující příklad kódu vygeneruje toto upozornění:

```
res = KeSaveFloatingPointState(buffer);
```

 Následující příklad kódu předchází tomuto upozornění:

```
res = KeSaveFloatingPointState(buffer);
if (NT_SUCCESS(res))
{
    res = KeRestoreFloatingPointState(buffer);
}
```

 Pokud je toto upozornění oznámeno jako falešně pozitivní, nejpravděpodobnější příčinou je, že funkce, která uvolňuje prostředek, není opatřena poznámkou `__drv_releasesResource` nebo `__drv_releasesResourceGlobal`. Všimněte si, že pokud používáte funkce obálky pro systémové funkce, měly by mít funkce obálky stejné anotace jako systémové funkce. V současné době je mnoho systémových funkcí v souboru modelu označeno, takže poznámky nejsou viditelné v hlavičkových souborech.
