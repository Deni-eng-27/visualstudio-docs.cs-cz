# <a name="ca2249-consider-using-stringcontains-instead-of-stringindexof"></a>CA2249: Zvážit možnost místo string.IndexOf použít string.Contains

|Položka|Hodnota|
|-|-|
|RuleId|CA2249|
|Kategorie|Microsoft. Usage|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Toto pravidlo vyhledá volání, <xref:System.String.IndexOf%2A> kde výsledek slouží ke kontrole přítomnosti nebo nepřítomnosti podřetězce, a navrhne použití <xref:System.String.Contains%2A> namísto toho pro zlepšení čitelnosti.

## <a name="rule-description"></a>Popis pravidla

Když <xref:System.String.IndexOf%2A> se používá ke kontrole, zda je výsledek roven `-1` nebo větší nebo roven `0` , volání může být bezpečně nahrazeno <xref:System.String.Contains%2A> bez dopadu na výkon.

V závislosti na <xref:System.String.IndexOf%2A> použitém přetížení by navrhovaná oprava mohla získat `comparisonType` přidaný argument:

| Metody | Navrhovaná oprava |
|----|----|
| `String.IndexOf(char)` | `String.Contains(char)` |
| `String.IndexOf(string)` | `String.Contains(string, StringComparison.CurrentCulture)` |
| `String.IndexOf(char, StringComparison.Ordinal)` | `String.Contains(char)` |
| `String.IndexOf(string, StringComparison.Ordinal)` | `String.Contains(string)` |
| `String.IndexOf(char, NON StringComparison.Ordinal)`* | `String.Contains(char, NON StringComparison.Ordinal)`* |
| `String.IndexOf(string, NON StringComparison.Ordinal)`* | `String.Contains(string, NON StringComparison.Ordinal)`* |

*: Jakákoli `StringComparison` jiná hodnota výčtu než `StringComparison.Ordinal` :
  - <xref:System.StringComparison.CurrentCulture>
  - <xref:System.StringComparison.CurrentCultureIgnoreCase>
  - <xref:System.StringComparison.InvariantCulture>
  - <xref:System.StringComparison.InvariantCultureIgnoreCase>
  - <xref:System.StringComparison.OrdinalIgnoreCase>

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Porušení lze buď opravit ručně, nebo v některých případech pomocí rychlých akcí pro opravu kódu v aplikaci Visual Studio.

#### <a name="examples"></a>Příklady

Následující dva fragmenty kódu zobrazují všechna možná porušení pravidla v jazyce C# a jak je opravit:

```csharp
using System;

class MyClass
{
    void MyMethod()
    {
        string str = "My text";
        bool found;

        // No comparisonType in char overload, so no comparisonType added in resulting fix
        found = str.IndexOf('x') == -1;
        found = str.IndexOf('x') >= 0;

        // No comparisonType in string overload, adds StringComparison.CurrentCulture to resulting fix
        found = str.IndexOf("text") == -1;
        found = str.IndexOf("text") >= 0;

        // comparisonType equal to StringComparison.Ordinal, removes the argument
        found = str.IndexOf('x', StringComparison.Ordinal) == -1;
        found = str.IndexOf('x', StringComparison.Ordinal) >= 0;

        found = str.IndexOf("text", StringComparison.Ordinal) == -1;
        found = str.IndexOf("text", StringComparison.Ordinal) >= 0;

        // comparisonType different than StringComparison.Ordinal, preserves the argument
        found = str.IndexOf('x', StringComparison.OrdinalIgnoreCase) == -1;
        found = str.IndexOf('x', StringComparison.CurrentCulture) >= 0;

        found = str.IndexOf("text", StringComparison.InvariantCultureIgnoreCase) == -1;
        found = str.IndexOf("text", StringComparison.InvariantCulture) >= 0;

        // Suggestion message provided, but no automatic fix offered, must be fixed manually
        int index = str.IndexOf("text"); 
        if (index == -1)
        {
            Console.WriteLine("'text' Not found.");
        }
    }
}
```


```csharp
using System;

class MyClass

        found = str.Contains('x');

        found = !string.Contains("text", StringComparison.CurrentCulture);
        found = string.Contains("text", StringComparison.CurrentCulture);

        // comparisonType equal to StringComparison.Ordinal, removes the argument
        found = !string.Contains('x');
        found = string.Contains('x');

        found = !string.Contains("text");
        found = string.Contains("text");

        // comparisonType different than StringComparison.Ordinal, preserves the argument
        ;found = !string.Contains('x', StringComparison.OrdinalIgnoreCase)
        found = string.Contains('x', StringComparison.CurrentCulture);

        found = !string.Contains("text", StringComparison.InvariantCultureIgnoreCase);
        found = string.Contains("text", StringComparison.InvariantCulture);
```
