# <a name="c6315"></a>C6315
Upozornění C6315: Nesprávné pořadí operací: bitový operátor and má vyšší prioritu než bitový operátor OR. Přidat závorky k upřesnění záměru

 Toto upozornění signalizuje, že výraz v kontextu testu obsahuje operace bitového operátoru and (`&`) i bitového operátoru OR (`|`), ale způsobuje konstantu, protože bitová nebo operace nastane jako poslední. K upřesnění záměru by měly být přidány závorky.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 Chcete-li toto upozornění opravit, přidejte závorky, jak je znázorněno v následujícím kódu:

```
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
