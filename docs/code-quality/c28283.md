# <a name="c28283"></a>C28283
Upozornění C28283: U symbolu není zadaná specifikace velikosti ještě podporovaná.

 Upozornění indikuje, že Poznámka "Sentinel" specifikace velikosti obdržela jinou hodnotu než 0. V podstatě se volající pokusil říct, že řetězec je ukončen znakem jinou než binární nula.
