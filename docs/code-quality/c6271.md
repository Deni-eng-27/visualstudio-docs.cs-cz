# <a name="c6271"></a>C6271
Warning C6271: byl předán nadbytečný argument do \<function >: parametr \<number > není použit formátovacím řetězcem.

 Toto upozornění znamená, že další argumenty jsou k dispozici nad rámec hodnot určených formátovacím řetězcem. Sám o sobě Tato vada nebude mít žádný viditelný efekt, i když indikuje, že záměr programátora se neodrazí v kódu.

## <a name="example"></a>Příklad
 Následující vzorový kód vygeneruje toto upozornění:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 Chcete-li toto upozornění opravit, použijte následující vzorový kód:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 Následující vzorový kód volá funkci bezpečného zpracování řetězce `sprintf_s` pro opravu tohoto upozornění:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
