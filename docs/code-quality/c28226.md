# <a name="c28226"></a>C28226
Upozornění C28226: Poznámky nemůže být PrimOp (v aktuální deklarace)  
  
 Toto upozornění označuje, že je je deklarován jako PrimOp pojmenované poznámky a také bylo dříve deklarované jako normální poznámky. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v běžném provozu.
