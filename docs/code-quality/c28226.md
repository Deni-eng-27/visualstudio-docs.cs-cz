# <a name="c28226"></a>C28226
Upozornění C28226: anotace nemůže být také PrimOp (v aktuální deklaraci)

 Toto upozornění znamená, že pojmenovaná Poznámka je deklarována jako PrimOp a byla také dříve deklarována jako normální anotace. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává při běžném použití.
