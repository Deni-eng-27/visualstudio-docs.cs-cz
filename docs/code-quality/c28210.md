# <a name="c28210"></a>C28210
Upozornění 28210: Poznámky pro kontext `_On_failure_` nesmí být v explicitním kontextu pre.

 Poznámky `_On_failure_` musí být explicitně nebo implicitně označeny v kontextu `__post`, to znamená, že se mají použít po vrácení funkce.  Tuto hodnotu zajistěte pomocí `_drv_out`.
