# <a name="ca0056"></a>CA0056
Soubor projektu má nesprávnou verzi nástroje pro analýzu.

K tomuto upozornění dochází, je-li FxCopCmd. exe vyvolán ručně pomocí souboru projektu, který má nesprávnou verzi nástroje pro analýzu kódu.
