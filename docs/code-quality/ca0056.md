# <a name="ca0056"></a>CA0056
Soubor projektu má nesprávná verze nástroje pro analýzu.  
  
 Toto upozornění se zobrazí, pokud je FxCopCmd.exe je vyvolána ručně pomocí souboru projektu, který obsahuje verzi nástroje Analýza nesprávný kód.
