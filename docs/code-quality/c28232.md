# <a name="c28232"></a>C28232
upozornění C28232: _Pre\_, řetězec _Post\_, nebo _deref –\_ nevztahuje se na žádné poznámky  
  
 Toto upozornění znamená, že `_Pre_`, `_Post_`, nebo `_Deref_` operátor se zobrazí ve výrazu poznámky bez dalších funkční poznámky; modifikátor byl ignorován, ale označuje nesprávně programové poznámky.
