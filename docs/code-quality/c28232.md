# <a name="c28232"></a>C28232
Upozornění C28232: \_Pre @ no__t-1, \_Post @ no__t-3 nebo \_Deref @ no__t-5 se nepoužije na žádnou poznámku

 Toto upozornění signalizuje, že se operátor `_Pre_`, `_Post_` nebo `_Deref_` ve výrazu poznámky zobrazí bez následné poznámky. Modifikátor byl ignorován, ale označuje nesprávně kódované poznámky.
