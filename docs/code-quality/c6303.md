# <a name="c6303"></a>C6303
Upozornění C6303: Neshoda formátovacího řetězce: řetězec dlouhého znaku byl předán jako parametr \<number >, když je ve volání \<function > vyžadován řetězec znaků.

 Toto upozornění označuje, že řetězec formátu určuje, že je vyžadován řetězec znaků. Předává se ale řetězec s velkým znakem. Tato vada může způsobit selhání nebo poškození nějakého formuláře.

## <a name="example"></a>Příklad
 Následující vzorový kód vygeneruje toto upozornění:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 Chcete-li toto upozornění opravit, použijte `%ls`, jak je znázorněno v následujícím ukázkovém kódu:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 Následující vzorový kód používá funkci bezpečného zpracování řetězce `printf_s` pro opravu tohoto upozornění:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
