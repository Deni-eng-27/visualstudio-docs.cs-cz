# <a name="c28227"></a>C28227
Upozornění C28227: Poznámky nemůže být PrimOp (viz předchozí deklarace)

 Toto upozornění označuje, že je je deklarován jako obyčejnou poznámky pojmenované poznámky a také bylo dříve deklarované jako PrimOp. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v typické použití.
