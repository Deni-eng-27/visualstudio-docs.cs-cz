# <a name="c28227"></a>C28227
Upozornění C28227: anotace nemůže být také PrimOp (viz předchozí deklarace)

 Toto upozornění znamená, že pojmenovaná Poznámka je deklarována jako běžná Poznámka a byla také dříve deklarována jako PrimOp. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
