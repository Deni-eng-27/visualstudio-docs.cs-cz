# <a name="c6272"></a>C6272
Warning C6272: jako argument \<number byl předán jiný typ než float, pokud se při volání metody \<function > vyžaduje float, >

 Toto upozornění označuje, že formátovací řetězec Určuje, že je vyžadován typ float, například specifikace `%f` nebo `%g` pro `printf,`, ale je předávána hodnota, která není typu float nebo String. Tato vada by mohla vést k nesprávnému výstupu. v některých případech ale může dojít k chybě.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 Chcete-li toto upozornění opravit, použijte `%i` namísto specifikace `%f`, jak je znázorněno v následujícím kódu:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 Následující kód používá funkci bezpečného zpracování řetězců `sprintf_s` pro opravu tohoto upozornění:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
