# <a name="c6317"></a>C6317
Upozornění C6317: nesprávný operátor: logický not (!) není zaměnitelné s doplňkem (~)

 Toto upozornění signalizuje, že logický operátor NOT (`!`) se používá pro konstantu, která pravděpodobně bude znakem bitového označení. Výsledek logického typu not je Boolean; použití operátoru bitového operátoru and (`&`) na logickou hodnotu je nesprávné. Pomocí operátoru a doplňku (`~`) převraťte příznaky.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & !FLAGS) // warning
  {
    // code
  }
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & ~FLAGS)
  {
    // code
  }
}
```
