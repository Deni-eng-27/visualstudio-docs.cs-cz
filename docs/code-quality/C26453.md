# <a name="warning-c26253-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Upozornění C26253: Aritmetické přetečení: Levý posun záporného čísla se znaménkem je nedefinované chování

Toto upozornění signalizuje, že jsou ponechali jsme posunutí záporné podepsaný celočíselné hodnoty, který není dobrý nápad a aktivuje chování definované implementací.

## <a name="example"></a>Příklad

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 Chcete-li opravit toto upozornění, použijte následující kód:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
