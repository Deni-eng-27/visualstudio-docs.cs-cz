# <a name="c28309"></a>C28309
Upozornění 28309: < parameter_name >: operandy anotace musí být typy integer, Enum a Pointer. Operandy typu void C++ a přetížené operátory nejsou podporovány. Floaty jsou přibližné jako celá čísla. Typy: \<typelist >.

 Pokusili jste se použít void nebo funkci ve výrazu poznámky a analýza kódu ji nedokáže zpracovat.  K této chybě obvykle dochází, když je použita `operator==`, která je implementována jako funkce, ale mohou nastat i jiné případy. Projděte si typy v \<typelist > a podíváme se, co je špatně.
