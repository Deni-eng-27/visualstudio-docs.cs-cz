# <a name="c6214"></a>C6214

Upozornění C6214: přetypování mezi sémanticky odlišnými typy celých čísel: HRESULT na typ Boolean

Toto upozornění signalizuje, že `HRESULT` se přetypování na typ Boolean. Hodnota úspěchu (`S_OK`) `HRESULT` se rovná 0. 0 však označuje selhání pro typ Boolean. Přetypování `HRESULT` na typ Boolean a jejich použití v testovacím výrazu vrátí nesprávný výsledek. Někdy k tomuto upozornění dochází, pokud je `HRESULT` Uloženo v proměnné typu Boolean. Jakékoli porovnání, které používá logickou proměnnou k otestování `HRESULT` nebo selhání může vést k nesprávným výsledkům.

## <a name="example"></a>Příklad

Následující kód vygeneruje toto upozornění:

```cpp
#include <windows.h>

BOOL f( )
{
  HRESULT hr;
  LPMALLOC pMalloc;
  hr = CoGetMalloc(1, &pMalloc);
  if ((BOOL)hr) // warning 6214
  {
    // success code ...
    return TRUE;
  }
  else
  {
    // failure code ...
    return FALSE;
  }
}
```

Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
#include <windows.h>

BOOL f( )
{
  HRESULT hr;
  LPMALLOC pMalloc;

  hr = CoGetMalloc(1, &pMalloc);
  if (SUCCEEDED(hr))
  {
    // success code ...
    return TRUE;
  }
  else
  {
    // failure code ...
    return FALSE;
  }
}
```

Pro toto upozornění je typ `SCODE` ekvivalentem `HRESULT`.

K otestování hodnoty `HRESULT` je obvykle nutné použít makro `SUCCEEDED` nebo `FAILED`.
