# <a name="c6101"></a>C6101
Upozornění C6101: Vrácení neinicializované paměti

 Úspěšná cesta prostřednictvím funkce nenastavuje parametr s názvem `_Out_`. Tato zpráva je vygenerována na základě poznámek SAL, které naznačují, že funkce v otázce je vždy úspěšná. Funkce, která nevrací indikaci úspěšnosti nebo chyby, by měla nastavit všechny parametry `_Out_`, protože analyzátor předpokládá, že parametr `_Out_` je neinicializovaná data před voláním funkce a že funkce nastaví parametr tak, aby už není inicializovaný. Pokud funkce indikuje úspěch nebo neúspěch, parametr `_Out_` nemusí být nastaven v případě selhání a můžete zjistit a vyhnout se neinicializovanému umístění. V obou případech je cílem zabránit čtení neinicializovaného umístění. Pokud se funkce někdy nedotkne parametru `_Out_`, který se následně používá, musí být parametr inicializován před voláním funkce a musí být označen pomocí anotace `_Inout_` a v případě potřeby explicitní `_Pre_null_` nebo `_Pre_satisfies_()`. "Částečný úspěch" lze zpracovat pomocí anotace `_When_`. Další informace najdete v tématu [Použití poznámek SAL ke snížení vad CC++ /kódu](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).
