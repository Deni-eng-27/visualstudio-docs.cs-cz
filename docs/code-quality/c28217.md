# <a name="c28217"></a>C28217
Upozornění C28217: Pro funkci Počet parametrů anotace neodpovídá nalezeným na souboru

 Poznámky na aktuálním řádku a na řádku ve zprávě jsou nekonzistentní. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává při běžném použití.
