# <a name="ca1501-avoid-excessive-inheritance"></a>CA1501: Vyhněte se nadměrné dědičnosti

|Položka|Hodnota|
|-|-|
|CheckId|CA1501|
|Kategorie|Microsoft. udržovatelnost|
|Zásadní změna|Narušující|

## <a name="cause"></a>Příčina

Typ je více než čtyři úrovně hluboko v hierarchii dědičnosti.

Ve výchozím nastavení pravidlo vyloučí pouze všechny typy z `System` oboru názvů, ale tato možnost je [konfigurovatelná](#configurability).

## <a name="rule-description"></a>Popis pravidla

Hluboce vnořené hierarchie typů může být obtížné sledovat, pochopit a udržovat. Toto pravidlo omezuje analýzu na hierarchie ve stejném modulu.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení tohoto pravidla, odvoďte typ ze základního typu, který je méně hluboko v hierarchii dědičnosti, nebo odstraňte některé mezilehlé základní typy.

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Z tohoto pravidla je bezpečné potlačit upozornění. Údržba kódu však může být obtížnější. Všimněte si, že v závislosti na viditelnosti základních typů může řešení porušení tohoto pravidla vytvořit zásadní změny. Například odebrání veřejných základních typů je zásadní změna.

## <a name="configurability"></a>Konfigurovatelnost

### <a name="inheritance-excluded-type-or-namespace-names"></a>Nevyloučený typ dědičnosti nebo názvy oborů názvů

Můžete nakonfigurovat názvy typů nebo oborů názvů (oddělené |), aby se typ nebo obor názvů typu necount ve stromu hierarchie dědičnosti. Ve výchozím nastavení `System.*` jsou vyloučeny všechny typy z oboru názvů. Všimněte si, že bez ohledu na hodnotu, kterou nastavujete, se přidá tato výchozí hodnota.

| Hodnota možnosti | Souhrn |
| --- | --- |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType` | Porovná všechny typy s názvem ' MyType ' nebo jejichž nadřazený obor názvů obsahuje ' MyType ' a všechny typy z oboru názvů ' System ' |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType1\|MyType2` | Vyhledá všechny typy s názvem ' MyType1 ' nebo ' MyType2 ' nebo jejichž obor názvů obsahuje buď ' MyType1 ', ' MyType2 ' a všechny typy z oboru názvů ' System ' |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.MyType` | Odpovídá určitému typu MyType v oboru názvů NS a všem typům z oboru názvů System. |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS1.MyType1\|T:NS2.MyType2` | Odpovídá specifickým typům ' MyType1 ' a ' MyType2 ' s příslušnými plně kvalifikovanými názvy a všemi typy z oboru názvů ' System ' |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:NS` | Odpovídá všem typům z oboru názvů NS a všem typům z oboru názvů System. |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = My*` | Odpovídá všem typům, jejichž název začíná na "my" nebo jehož část obsahující části oboru názvů začíná na ' my ' a všechny typy z oboru názvů ' System ' |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.My*` | Odpovídá všem typům, jejichž název začíná na "my" v oboru názvů "NS" a všechny typy z oboru názvů "System". |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:My*` | Odpovídá všem typům, jejichž obor názvů začíná na my a všechny typy z oboru názvů System. |
