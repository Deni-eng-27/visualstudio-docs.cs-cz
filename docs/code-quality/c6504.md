# <a name="c6504"></a>C6504
Upozornění C6504: Neplatná Anotace: vlastnost lze použít pouze pro hodnoty typu ukazatel, ukazatel na člen nebo typ pole.

 Toto upozornění signalizuje použití vlastnosti u nekompatibilního datového typu. Další informace o datových typech podporovaných vlastnostmi najdete v tématu [vlastnosti poznámek](using-sal-annotations-to-reduce-c-cpp-code-defects.md).

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože vlastnost `_Null_` nelze použít pro referenční datový typ.

```cpp
#include<sal.h>

class Point
{
public:
    //  members
};

void f(_Pre_ _Null_ Point& pt)
{
    // code ...
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
#include<sal.h>

class Point
{
public:
    //  members
};

void f(_Pre_ _Null_  Point* pt)
{
    // code ...
}
```

 Chybný kód zobrazený dříve také generuje upozornění [C6516](../code-quality/c6516.md) , protože konflikty vlastností způsobily neplatnou anotaci.
