# <a name="c28225"></a>C28225
upozornění C28225: nebyla nalezena správný počet požadovaných parametrů v poznámky  
  
 Toto upozornění označuje, že pojmenované poznámky se používá s nesprávný počet parametrů. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v typické použití.
