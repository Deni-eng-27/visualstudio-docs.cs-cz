# <a name="c28219"></a>C28219
Warning C28219: pro anotaci parametru v poznámce je očekáván člen výčtu.

 Očekává se, že parametr pro poznámku bude členem pojmenovaného typu `enum` a byl zjištěn jiný symbol; Použijte člena typu `enum`. To obvykle označuje nesprávně kódované makro poznámky.
