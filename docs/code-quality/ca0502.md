# <a name="ca0502"></a>CA0502
Nepodporovaný jazyk

 K této chybě dochází, když se uživatel pokusí k analýze projektu ASP.NET, která obsahuje jeden nebo více stránek nebo třídy, které jsou napsané v jiném jazyce než v jazyce C# nebo Visual Basic. Chcete-li tuto chybu opravit, odeberte soubor z projektu nebo zákaz analýzy kódu.
