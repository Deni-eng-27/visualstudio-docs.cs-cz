# <a name="ca0502"></a>CA0502
Nepodporovaný jazyk  
  
 K této chybě dojde, když se uživatel pokusí o analýze projekt ASP.NET, který obsahuje jeden nebo více stránek nebo třídy, které jsou napsané v jiném jazyce než v C# nebo Visual Basic. Chcete-li tuto chybu opravit, odstraňte soubor z projektu nebo zákaz analýzy kódu.
