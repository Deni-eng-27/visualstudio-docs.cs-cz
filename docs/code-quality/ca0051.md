# <a name="ca0051"></a>CA0051
Nebyly vybrány žádné pravidla.

Tato chyba znamená, že nástroj pro analýzu spravovaného kódu je povolena, ale nebyly vybrány žádné pravidla. Chcete-li opravit tuto chybu, vyberte příslušná pravidla.
