# <a name="c28204"></a>C28204
Upozornění C28204: \<function >: Pouze jeden z těchto přetížení a druhý v \<filename > (\<line >) jsou opatřeny poznámkami pro \<paramname >: obojí nebo ani jeden z nich musí být opatřen poznámkami.

 Toto upozornění se oznamuje v případě, že poznámky obsahují chybu.
