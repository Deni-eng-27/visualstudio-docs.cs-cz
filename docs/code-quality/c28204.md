# <a name="c28204"></a>C28204
Warning C28204: \<function >: pro \<paramname > je k dispozici pouze jedno z těchto přetížení a druhý na @no__t 1filename > (\<line >): obě nebo ani jedna musí být opatřena poznámkami.

 Toto upozornění se oznamuje v případě, že poznámky obsahují chybu.
