# <a name="c1257"></a>C1257
Upozornění 1257: Inicializace modulu plug-in se nezdařila.

 Nástroj Analýza kódu hlásí toto upozornění, když v modulu plug-in dojde k vnitřní chybě, která není v analyzovaném kódu.
