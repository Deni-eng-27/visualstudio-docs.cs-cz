# <a name="c6306"></a>C6306
Warning C6306: nesprávné volání \<function >: Zvažte použití \<function >, který jako argument přijímá va_list.

 Toto upozornění signalizuje nesprávné volání funkce. Řada `printf` obsahuje několik funkcí, které přijímají seznam proměnných argumentů; Tyto funkce však nelze volat s argumentem `va_list`. K dispozici je odpovídající rodina funkcí `vprintf`, kterou lze použít pro taková volání. Volání nesprávné funkce Print způsobí nesprávný výstup.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
