# <a name="c28302"></a>C28302
Upozornění C28302: pro C++ parametr reference > < parameter_name byl pro \<annotation > nalezen další operátor `_Deref_`.

 Toto upozornění se oznamuje, když se pro parametr používá další úroveň `_Deref_`.

 POZNÁMKOU SAL2 nevyžaduje použití nadbytečné úrovně `_Deref_` při práci s parametry odkazu. Tato konkrétní Poznámka je nejednoznačná a je správně interpretována, ale měla by být opravena.

 To se často dá opravit pouhým odebráním starší anotace `__deref` a pomocí syntaxe POZNÁMKOU SAL2. Někdy může být nutné použít `_At_` pro odkazování na konkrétní objekt, který se má opatřit poznámkami.
