# <a name="how-to-view-managed-code-defects"></a>Postupy: Zobrazování defektů spravovaného kódu
V analýza kódu jsou uvedeny chyby, upozornění a zprávy v okně nástroje Analýza kódu. Analýza kódu okna se zobrazí automaticky při spuštění analýzy kódu na projekt nebo řešení.  
  
 V okně nástroje Analýza kódu můžete:  
  
-   **Zvolte název upozornění.**  
  
     Další informace o upozornění se zobrazí v okně nástroje Analýza kódu. Na řádku zdrojového souboru, která vyvolala upozornění se zobrazí a zvýrazněných v editoru zdrojového kódu.  
  
-   **Zvolte id upozornění.** ID upozornění předchází **certifikační Autority** a obsahovat čtyři číslice. Například CA1234.  
  
     Téma nápovědy knihovny MSDN pro toto upozornění se zobrazí. Téma nápovědy obsahuje podrobné informace o příčině upozornění a obsahuje příklady, které ukazují, jak problém upozornění dochází a je vyřešit.
