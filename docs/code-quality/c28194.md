# <a name="c28194"></a>C28194
Upozornění C28194: funkce byla deklarována jako aliasing hodnoty v proměnné a byla ukončena, aniž by tak učinila

 Toto upozornění znamená, že prototyp funkce pro analyzovanou funkci má anotaci `__drv_isAliased`, která označuje, že bude *alias* zadaného argumentu (tj. přiřadit hodnotu tak, aby se přestaly vracet z funkce). Funkce však nealiasuje argument podél cesty, která je označena poznámkou. Většina funkcí, které aliasují proměnnou, uloží svou hodnotu do globální struktury dat.
