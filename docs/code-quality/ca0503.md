# <a name="ca0503"></a>CA0503
Vlastnost je zastaralá. Nahrazující vlastnost

 CA0503 upozornění se vygeneruje, když vlastnost analýzy kódu nepoužívané obsahuje hodnotu v konfiguračním souboru sestavení nebo projekt. Pokud chcete vyřešit toto upozornění, pomocí zadané vlastnosti, která nahrazuje ji.
