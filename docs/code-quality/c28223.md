# <a name="c28223"></a>C28223
Upozornění C28223: Nebyl nalezen očekávaný Token nebo identifikátor pro poznámky, parametr  
  
 Toto upozornění označuje, že se očekává parametr, který se poznámky identifikátor a některé další symbol došlo. To obvykle znamená makru nesprávně programové poznámky. Použití klíčová slova jazyka C nebo C++ v této pozici způsobí, že k této chybě.
