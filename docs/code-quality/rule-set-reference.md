# <a name="code-analysis-rule-set-reference"></a>Referenční dokumentace sady pravidel nástroje Analýza kódu

Když konfigurujete analýzy kódu pro spravovaný kód projekty v sadě Visual Studio, můžete ze seznamu předdefinovaných *sad pravidel*. Můžete použít jednu z těchto sady předdefinovaných pravidel, nebo můžete [přizpůsobit sadu pravidel](../code-quality/how-to-create-a-custom-rule-set.md) podle svých požadavků projektu.

Témata v této části popisují předdefinované pravidlo sady a pravidel (nebo upozornění) obsahují.
