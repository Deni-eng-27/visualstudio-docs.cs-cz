# <a name="c28220"></a>C28220
Upozornění C28220: Očekáván celočíselný výraz pro okomentování parametru v poznámce

 Toto upozornění signalizuje, že parametr pro poznámku má být celočíselný výraz a byl zjištěn jiný typ. To obvykle označuje nesprávně kódované makro poznámky.
