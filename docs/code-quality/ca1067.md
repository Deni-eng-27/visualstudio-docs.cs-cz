# <a name="ca1067-override-equals-when-implementing-iequatable"></a>CA1067: přepište Equals při implementaci IEquatable

|||
|-|-|
|TypeName|EquatableAnalyzer|
|CheckId|CA1067|
|Kategorie|Microsoft. Design|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Typ implementuje <xref:System.IEquatable%601>, ale nepřepisuje <xref:System.Object.Equals%2A> metodu.

## <a name="rule-description"></a>Popis pravidla

Rozhraní implementující <xref:System.IEquatable%601> typ označuje, že podporuje porovnávání dvou instancí typu pro rovnost. Měli byste také přepsat implementace <xref:System.Object.Equals%2A> a <xref:System.Object.GetHashCode> metod základní třídy tak, aby jejich chování bylo konzistentní s implementací <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementace. Další podrobnosti najdete [tady](/dotnet/api/system.iequatable-1#notes-to-implementers).

Vaše <xref:System.Object.Equals%2A> implementace by měla vracet výsledky, které jsou <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> konzistentní s implementací.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení, <xref:System.Object.Equals%2A> přepište a implementujte ji vyvoláním <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementace. Například následující dva fragmenty kódu ukazují porušení pravidla a jejich opravu:

```csharp
using System;

public struct S : IEquatable<S>
{
    private readonly int _value;
    public S(int f)
    {
        _value = f;
    }

    public bool Equals(S other)
        => _value == other._value;
}
```

```csharp
using System;

public struct S : IEquatable<S>
{
    private readonly int _value;
    public S(int f)
    {
        _value = f;
    }

    public bool Equals(S other)
        => _value == other._value;

    public override bool Equals(object obj)
        => obj is S objS && Equals(objS);

    public override int GetHashCode()
        => _value.GetHashCode();
}
```

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Potlačí porušení tohoto pravidla.
