# <a name="c28273"></a>C28273
Warning C28273: pro funkci funkce jsou nekonzistentní s deklarací funkce.

 Toto upozornění označuje chybu v poznámkách, nikoli v analyzovaném kódu. Poznámky, které se zobrazují v definici funkce, jsou nekonzistentní s hodnotami uvedenými v deklaraci. Tyto dvě poznámky by se měly přeložit, aby odpovídaly.
