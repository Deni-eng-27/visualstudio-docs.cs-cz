# <a name="c6219"></a>C6219

Warning C6219: implicitní přetypování mezi sémanticky odlišnými typy celých čísel: porovnává HRESULT na 1 nebo TRUE. Zvažte místo toho použití makra [Succeeded](/windows/desktop/api/winerror/nf-winerror-succeeded) nebo [Failed](/windows/desktop/api/winerror/nf-winerror-failed) .

Toto upozornění indikuje, že `HRESULT` se porovnává s explicitní hodnotou, která není `HRESULT` a jedna (1). Toto porovnání pravděpodobně vede k nesprávným výsledkům, protože typická hodnota úspěšnosti `HRESULT` (`S_OK`) je 0. Pokud porovnáváte tuto hodnotu s typem Boolean, implicitně se převede na false.

## <a name="example"></a>Příklad

Následující kód vygeneruje toto upozornění, protože `CoGetMalloc` vrátí `HRESULT`, které pak bude porovnána s `TRUE`:

```cpp
#include <windows.h>

void f( )
{
  HRESULT hr;
  LPMALLOC pMalloc;
  hr = CoGetMalloc(1, &pMalloc);

  if (hr == TRUE)
  {
    // success code ...
  }
  else
  {
    // failure code
  }
}
```

Většinou je toto upozornění způsobeno kódem, který porovnává `HRESULT` s logickou hodnotou. Je lepší použít makra `SUCCEEDED` nebo `FAILED` k otestování hodnoty `HRESULT`. Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
#include <windows.h>

void f( )
{
  HRESULT hr;
  LPMALLOC pMalloc;
  hr = CoGetMalloc(1, &pMalloc);

  if (SUCCEEDED(hr))
  {
    // success code ...
  }
  else
  {
    // failure code
  }
}
```

Pro toto upozornění je typ `SCODE` považován za `HRESULT`.

Použití `malloc` a `free` (a souvisejících rozhraní API dynamické paměti) má mnoho nástrah jako příčinu nevracení paměti a výjimek. Chcete-li se vyhnout těmto druhům úniku a potížím s výjimkami, použijte ukazatele a C++ třídy kontejneru poskytované standardní knihovnou. Patří mezi ně [shared_ptr](/cpp/standard-library/shared-ptr-class), [unique_ptr](/cpp/standard-library/unique-ptr-class)a [Vector](/cpp/standard-library/vector). Další informace najdete v tématu [inteligentní ukazatele](/cpp/cpp/smart-pointers-modern-cpp) a [ C++ standardní knihovna](/cpp/standard-library/cpp-standard-library-reference).
