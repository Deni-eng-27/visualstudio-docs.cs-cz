# <a name="c28291"></a>C28291
Upozornění C28291: post null/NotNull na DEREF úrovni 0 nemá smysl pro funkci \<x > na \<číslo param >

 Nástroj Analýza kódu hlásí toto upozornění, je-li v případě, že je zadána podmínka post u nulového parametru s hodnotou null, nastavena vlastnost s hodnotou null nebo není null. K této chybě dochází, protože se hodnota nula na úroveň odkázat nemůže změnit.
