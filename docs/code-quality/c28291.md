upozornění C28291: Post hodnotu null nebo notnull v deref úroveň 0 je smysl pro funkci \<x > na param \<číslo >  
  
 Nástroj pro analýzu kódu sestavy, toto upozornění při post podmínku parametr úroveň nula dereference pomocí vlastnosti hodnotu null nebo jinou hodnotu než null. Tato chyba nastane, protože dereference hodnotu v úrovni nula nelze změnit.
