# <a name="c28209"></a>C28209
Upozornění C28209: deklarace pro symbol má konfliktní deklaraci.

 Toto upozornění označuje nesprávně vytvořenou deklaraci poznámky. Toto upozornění nikdy nesmí nastat při běžném použití.
