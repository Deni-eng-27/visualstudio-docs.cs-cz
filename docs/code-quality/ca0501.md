# <a name="ca0501"></a>CA0501
Nelze načíst výstup sestavy

 Tato situace nastane, když nelze zapisovat výstup sestavy z jednoho z následujících důvodů:

-   Disk je plný

-   Žádné oprávnění k zápisu do adresáře

-   Adresář neexistuje.
