# <a name="c6387"></a>C6387
Warning C6387: > \<argument může být \<value >: to nedodržuje specifikace pro funkci \<function >: Řádky: x, y

 Toto upozornění je vyvoláno, pokud je parametr funkce s poznámkou předán neočekávanou hodnotu. Například předání potenciálně hodnoty null parametru, který je označen pomocí anotace `_In_` vygeneruje toto upozornění.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože parametr null je předán do `f(char *)`:

```cpp

#include <sal.h>

_Post_ _Null_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh); // Warning C6387
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```cpp

#include <sal.h>

_Post_ _Notnull_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh);
}
```
