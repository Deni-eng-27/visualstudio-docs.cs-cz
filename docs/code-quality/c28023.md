# <a name="c28023"></a>C28023
Upozornění C28023: funkce, která má být přiřazena nebo předána, by měla obsahovat poznámku `_Function_class_` pro alespoň jednu ze tříd (ES) v: \<classlist >

 Toto upozornění je obvykle hlášeno, je-li použita pouze jedna třída funkce a zpětné volání příslušného typu není deklarováno.

 Toto upozornění je vystaveno, když je funkce na levé straně přiřazení (nebo implicitního přiřazení, pokud se jedná o volání funkce) opatřena příznakem, který označuje, že se jedná o typ funkce specifický pro ovladač, který používá anotaci `_Function_class_` nebo typedef, která obsahuje takové poznámky. Funkce na pravé straně přiřazení nemá anotaci `_Function_class_`. Funkce na pravé straně by měla být označena jako na stejném typu jako funkce na levé straně. To se obvykle nejlépe dokončí přidáním deklarace \<class1 > > \<funcname1 před aktuální první deklarací \<funcname2 >.
