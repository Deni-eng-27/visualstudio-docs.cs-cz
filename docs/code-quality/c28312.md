# <a name="c28312"></a>C28312
upozornění 28312: < annotation_name > poznámky na opakovaných deklaraci \<funkce > \<parametr > je obsolescent verzi SAL.  
  
 Poznámka je starý a by měly být aktualizovány na ekvivalent [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Toto upozornění není aktivováno, pokud bylo aktivováno předchozí upozornění na nekonzistentní poznámku a je pokud hlášeno pouze jednou za deklaraci.
