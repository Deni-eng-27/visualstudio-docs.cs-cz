# <a name="ca0504"></a>CA0504
Pravidlo adresář byl ignorován, protože neexistuje

 Upozornění CA504 se vygeneruje, když adresář zadaný v **CodeAnalysisRuleDirectories** vlastnost v konfiguračním souboru projektu nebo sestavení se nepovedlo najít. Chcete-li tento problém vyřešit, zadejte platnou cestu k adresáři.
