# <a name="ca0504"></a>CA0504
Directory pravidlo bylo ignorováno, protože neexistuje  
  
 Upozornění CA504 se vygeneruje, když adresář zadaný v **CodeAnalysisRuleDirectories** vlastnost v konfiguračním souboru projektu nebo sestavení nelze nalézt. Chcete-li vyřešit tento problém, zadejte platnou cestu k adresáři.
