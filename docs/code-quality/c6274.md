# <a name="c6274"></a>C6274
Upozornění C6274: jako parametr @no__t-> 0number byl předán jiný než znak, pokud je znak vyžadován při volání \<function >

 Toto upozornění označuje, že formátovací řetězec Určuje, že je vyžadován znak (například specifikace `%c` nebo `%C`), ale předává se neceločíselná hodnota, jako je například float, String nebo struct. Tato vada může způsobit nesprávný výstup.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 Následující kód používá bezpečnou funkci manipulace s řetězci `sprintf_s` pro opravu tohoto upozornění:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
