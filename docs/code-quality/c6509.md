# <a name="c6509"></a>C6509
Upozornění C6509: Neplatná Anotace: na hodnotu Return nelze odkazovat z předpodmínky.

 Toto upozornění signalizuje, že klíčové slovo `return` nelze v předběžných podmínkách použít. Klíčové slovo `return` slouží k ukončení provádění funkce a vrátí řízení volající funkci.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože v předběžných podmínkách je použit `return`:

```cpp
#include <sal.h>

int f (_In_reads_(return) char *pc)
{
    // code ...
    return 1;
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
#include <sal.h>

int f (_In_reads_(i) char *pc, int i)
{
    // code ...
    return 1;
}
```
