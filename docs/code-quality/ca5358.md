# <a name="ca5358-do-not-use-unsafe-cipher-modes"></a>CA5358: Nepoužívat nezabezpečené režimy šifrování

|Položka|Hodnota|
|-|-|
|RuleId|CA5358|
|Kategorie|Microsoft.Security|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Použijte jeden z následujících nebezpečných režimů šifrování, které nejsou schváleny:

- <xref:System.Security.Cryptography.CipherMode.ECB?displayProperty=fullName>
- <xref:System.Security.Cryptography.CipherMode.OFB?displayProperty=fullName>
- <xref:System.Security.Cryptography.CipherMode.CFB?displayProperty=fullName>

## <a name="rule-description"></a>Popis pravidla

Tyto režimy jsou zranitelné vůči útokům a mohou způsobovat expozici citlivých informací. Například použití `ECB` k zašifrování bloku prostého textu vždy vytvoří stejný šifrovaný text, takže může snadno zjistit, jestli jsou dvě šifrované zprávy identické. Použití schválených režimů se může vyhnout těmto zbytečným rizikům.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

- Používejte pouze schválené režimy ( <xref:System.Security.Cryptography.CipherMode.CBC?displayProperty=fullName> , <xref:System.Security.Cryptography.CipherMode.CTS?displayProperty=fullName> ).

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Upozornění z tohoto pravidla je bezpečné potlačit, pokud:
- Odborníci na kryptografii zkontrolovali a schválili použití režimu šifry.
- Odkaz se <xref:System.Security.Cryptography.CipherMode> nepoužívá pro kryptografickou operaci.

## <a name="pseudo-code-examples"></a>Příklady kódu pseudo

### <a name="assign-ecb-to-mode-property"></a>Přiřadit vlastnost režimu ECB

```csharp
using System.Security.Cryptography;

class ExampleClass {
    private static void ExampleMethod () {
        RijndaelManaged rijn = new RijndaelManaged
        {
            Mode = CipherMode.ECB
        };
    }
}
```

### <a name="using-the-value-ecb"></a>Použití hodnoty ECB

```csharp
using System;

class ExampleClass
{
    private static void ExampleMethod()
    {
        Console.WriteLine(CipherMode.ECB);
    }
}
```

### <a name="solution"></a>Řešení

```csharp
using System.Security.Cryptography;

class ExampleClass {
    private static void ExampleMethod () {
        RijndaelManaged rijn = new RijndaelManaged
        {
            Mode = CipherMode.CBC
        };

```
