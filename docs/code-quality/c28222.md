# <a name="c28222"></a>C28222
Upozornění 28222: \_Yes @ no__t-1, \_No @ no__t-3 nebo \_Maybe @ no__t-5 očekávat pro poznámku

 Toto upozornění znamená, že se očekává, že parametr u poznámky bude jeden ze symbolů `_Yes_`, `_No_` nebo `_Maybe_` a nějaký jiný symbol byl nalezen. To obvykle označuje nesprávně kódované makro poznámky.
