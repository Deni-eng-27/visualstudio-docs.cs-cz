# <a name="c28222"></a>C28222
upozornění 28222: _ano\_, _Ne\_, nebo _Maybe\_ očekávání pro poznámky  
  
 Toto upozornění označuje, že parametr, který se poznámky musí být mezi symboly `_Yes_`, `_No_`, nebo `_Maybe_`, a některé další symbol došlo. To obvykle znamená makru nesprávně programové poznámky.
