# <a name="c28224"></a>C28224
upozornění C28224: poznámky vyžaduje parametry  
  
 Toto upozornění označuje, že s názvem poznámky se použije bez parametrů a alespoň jeden parametr je povinný. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v běžném provozu.
