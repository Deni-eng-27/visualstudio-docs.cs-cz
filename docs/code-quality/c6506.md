# <a name="c6506"></a>C6506
upozornění C6506: Neplatný poznámky: \<name > vlastnost lze použít pouze na hodnoty ukazatele nebo pole typů  
  
 Toto upozornění označuje, že vlastnost se používá pro typ než ukazatel nebo pole typů. Přístup, Tainted a platný vlastnosti lze použít pro všechny datové typy. Další vlastnosti, jako je například ValidBytesConst, ValidElementsConst, ElementSize a NullTerminted podporovat ukazatele, ukazatel na členy a typy polí. Úplný seznam vlastností a podporované datové typy, najdete v části [poznámky vlastnosti](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Příklad  
 Následující kód vytvoří toto upozornění:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Chcete-li toto upozornění, použijte ukazatel nebo typ pole, jak je znázorněno v následujícím ukázkovém kódu:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
