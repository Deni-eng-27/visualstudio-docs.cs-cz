# <a name="ca0505"></a>CA0505
Nepoužívané vlastnost bude ignorována, protože není definována vlastnost nahrazující.  
  
 Upozornění CA505 se vygeneruje, když zadaný zastaralé i nahrazující vlastnosti analýzy kódu obsahovat hodnoty v projektu nebo sestavení konfiguračního souboru. Chcete-li tento problém vyřešit, odstraňte nepoužívané vlastnost.
