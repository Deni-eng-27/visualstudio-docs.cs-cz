# <a name="arithmetic-overflow-operator-operation-produces-a-negative-unsigned-result-at-compile-time"></a>Aritmetické přetečení: operace: % – operátor %' Vytvoří záporný výsledek bez znaménka v době kompilace

  Toto upozornění signalizuje, že operace odčítání vytvoří záporný výsledek, který se vyhodnotí v kontextu bez znaménka. To může způsobit nežádoucí přetečení.

## <a name="example"></a>Příklad

```cpp
unsigned int negativeunsigned()
{
    const unsigned int x = 1u - 2u; // C26454 reported here
    return x;
}
```
 Chcete-li opravit toto upozornění, použijte následující kód:

```cpp
unsigned int negativeunsigned()
{
    const unsigned int x = 4294967295; // OK
    return x;
}
```
