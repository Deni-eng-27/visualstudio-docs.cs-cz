# <a name="c28221"></a>C28221
upozornění C28221: řetězec pro parametr v anotace byl očekáván výraz  
  
 Toto upozornění označuje, že je parametr, který se poznámky očekáván řetězec a jiný typ došlo. To obvykle znamená makru nesprávně programové poznámky a neočekává se, že se zobrazí v běžném provozu.
