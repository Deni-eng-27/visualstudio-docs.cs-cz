# <a name="c28212"></a>C28212
upozornění C28212: ukazatele – výraz pro poznámky  
  
 Toto upozornění vyplývá, že parametr číslem pro poznámky (není funkce se poznámkou) se očekává ukazatel nebo typ pole, ale jiný typ došlo. Poznámka je třeba opravit.
