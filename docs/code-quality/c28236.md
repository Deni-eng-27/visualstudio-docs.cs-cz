# <a name="c28236"></a>C28236
upozornění C28236: anotace nelze použít ve výrazech

 Toto upozornění označuje, že byl proveden pokus o použití funkce deklarované jako poznámky v kontextu výrazu. To by neměl být možné, pokud se používá standardní makra pro poznámky; Toto upozornění se nepředpokládá, že se zobrazí v typické použití.
