# <a name="c28236"></a>C28236
Upozornění C28236: poznámku nelze použít ve výrazu.

 Toto upozornění indikuje, že došlo k pokusu o použití funkce deklarované jako anotace v kontextu výrazu. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
