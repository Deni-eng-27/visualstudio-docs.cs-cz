# <a name="ca0059"></a>CA0059
Přepínač příkazového řádku došlo k chybě.

 K této chybě dochází, pokud příkazový řádek, který vyžaduje chybí, například

 / out nebo není zadaný přepínač/Console. K této chybě dojde, pokud existuje neplatný přepínač, jako je například hodnota chybný časový limit.
