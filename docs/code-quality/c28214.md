# <a name="c28214"></a>C28214
Upozornění C28214: názvy parametrů atributu musí být P1... P9

 Toto upozornění znamená, že při vytváření deklarace poznámky jsou názvy parametrů omezené na P1... P9. Toto upozornění nikdy nesmí nastat při běžném použití.
