upozornění C28213: `_Use_decl_annotations_` poznámky musí být slouží k odkazování bez úprav, předběžné prohlášení. \<Proč >  
  
 Toto upozornění se použije v hlášení při odkazování na předběžné prohlášení bez požadované `_Use_decl_annotations_` poznámky.
