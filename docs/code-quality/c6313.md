# <a name="c6313"></a>C6313
Upozornění C6313: Nesprávný operátor: Příznak s nulovou hodnotou nelze testovat pomocí bitových operátorů and. Použijte test rovnosti pro kontrolu příznaků s nulovou hodnotou.

 Toto upozornění signalizuje, že jako argument operátoru bitového operátoru and (`&`) v kontextu testu byl poskytnutá konstantní hodnota nula. Výsledný výraz je konstantní a je vyhodnocen jako false; výsledek je jiný než zamýšlený.

 To je obvykle způsobeno použitím bitových operátorů a testováním příznaku, který má nulovou hodnotu. Pro otestování příznaků s nulovou hodnotou musí být proveden test rovnosti, například pomocí `==` nebo `!=`.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#define FLAG 0

void f(int Flags )
{
  if (Flags & FLAG)
  {
    // code
  }
}
```

 Chcete-li toto upozornění opravit, použijte následující kód:

```
#define FLAG 0

void f(int Flags )
{
  if (Flags == FLAG)
  {
    // code
  }
}
```
