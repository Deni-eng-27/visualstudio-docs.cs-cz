# <a name="c28208"></a>C28208
Warning C28208: funkce \<function > byla dříve definována s jiným seznamem parametrů v @no__t 1File > (\<line >). Některé analytické nástroje budou poskytovat nesprávné výsledky.

 Toto upozornění je hlášeno, pokud se známá definice funkce neshoduje s jiným výskytem.
