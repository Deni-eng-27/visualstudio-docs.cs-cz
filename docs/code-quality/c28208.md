# <a name="c28208"></a>C28208
Upozornění C28208: funkce \<funkcí > byla dříve definována s jiným seznamem parametrů na \<souboru > (\<řádek >). Některé analytické nástroje budou poskytovat nesprávné výsledky.

 Toto upozornění je hlášeno, pokud se známá definice funkce neshoduje s jiným výskytem.
