# <a name="c26135"></a>C26135
Warning C26135: chybějící anotace \<annotation > na funkci @no__t 1func >.

 Upozornění C26135 je vystaveno, když analyzátor odvodí, že funkce je zamykací Obálková funkce, která má zámek získat nebo uzamknout vedlejší dopad. Pokud kód není jako Obálková funkce, pak buď dojde k nevrácení zámku (Pokud se zámek získává), nebo když se uvolňuje nesprávně (Pokud se zámek uvolňuje).

## <a name="example"></a>Příklad
 Následující příklad generuje upozornění C26135, protože chybí odpovídající Poznámka vedlejšího efektu.

```

typedef struct _DATA
{
    CRITICAL_SECTION cs;
} DATA;

void MyEnter(DATA* p)
{
    // Warning C26135:
    // Missing side effect annotation _Acquires_lock_(&p->cs)
    EnterCriticalSection(&p->cs);
}

void MyLeave(DATA* p)
{
    // warning C26135:
    // Missing side effect annotation _Releases_lock_(&p->cs)
    LeaveCriticalSection(&p->cs);
}
```

## <a name="example"></a>Příklad
 Upozornění C26135 je také vystaveno v případě, že je detekována podmínka pro vedlejší zámky. Chcete-li použít poznámku s podmíněným efektem, použijte anotaci `_When_(ConditionExpr, LockAnnotation)`, kde `LockAnnotation` je buď `_Acquires_lock_`, nebo `_Releases_lock_`, a výraz predikátu `ConditionExpr` je logický podmíněný výraz. Vedlejší účinky jiných poznámek na stejnou funkci se vyskytnou pouze v případě, že je hodnota `ConditionExpr` vyhodnocena jako true. Vzhledem k tomu, že `ConditionExpr` slouží k předání podmínky zpět volajícímu, musí zahrnovat proměnné, které jsou rozpoznány v volajícím kontextu. Mezi ně patří parametry funkce, globální proměnné nebo členské proměnné třídy nebo návratová hodnota. Chcete-li zobrazit vrácenou hodnotu, použijte klíčové slovo Special v anotaci `return`, jak je znázorněno v následujícím příkladu.

```

 typedef struct _DATA
 {
     CRITICAL_SECTION cs;
     int state;
} DATA;

 _When_(return != 0, _Acquires_lock_(p->cs))
 int TryEnter(DATA* p)
 {
     if (p->state != 0)
     {
         EnterCriticalSection(&p->cs);
         return p->state;
     }

     return 0;
 }
```

 U sdílených/exkluzivních zámků, označovaných také jako zámky čtecího/zapisovacího zařízení, můžete vyjádřit uzamčené vedlejší účinky pomocí následujících poznámek:

- `_Acquires_shared_lock_(LockExpr)`

- `_Releases_shared_lock_(LockExpr)`

- `_Acquires_exclusive_lock_(LockExpr)`

- `_Releases_exclusive_lock_(LockExpr)`
