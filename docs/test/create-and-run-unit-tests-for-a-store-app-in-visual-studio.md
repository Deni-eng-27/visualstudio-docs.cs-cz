# <a name="create-and-run-unit-tests-for-a-uwp-app-in-visual-studio"></a>Vytváření a spouštění testování částí pro aplikace UWP v sadě Visual Studio
Testovací nástroje sady Visual Studio jednotky jsou navrženy pro podporu vývojáři a týmům, kteří začlenit testování v jejich postupy pro vývoj softwaru částí. Testování částí poskytnout vývojáři a testerům, sada efektivní, udržovatelný způsob hledání logické chyby v metody třídy v projektech Visual C#, Visual Basic a Visual C++.  
  
> [!NOTE]
>  Témata v této části popisují funkci Visual Studio 2012 Express pro Windows 8. Visual Studio Ultimate, VS Premium a VS Professional poskytnutí dodatečných funkcí pro testování jednotky.  
>   
>  -   V VS Ultimate, VS Premium a VS Professional můžete použít jakékoli třetí strany nebo otevřít zdroj částí unit test framework vytvořený adaptér rozšíření pro Microsoft testování Explorer. Můžete také analyzovat a zobrazit informace o pokrytí kódu pro testy s VS Ultimate a VS Premium.  
> -   V VS Ultimate můžete spustit testy po každé sestavení. Můžete také použít Microsoft Fakes, představuje izolace rozhraní pro spravovaný kód a testy zaměřit se na vlastní kód nahrazením testovacího kódu pro systém a funkce třetích stran.  
> -   **JavaScript**. Testování částí JavaScript není k dispozici z IDE nástroje Visual Studio Express pro Windows 8. Visual Studio Professional, Premium a Ultimate systémů testů jednotek JavaScript je dostupná jako rozšíření v Galerii Visual Studia. Můžete také otestovat kód jazyka JavaScript pomocí jedné samostatné třetích stran nebo otevřete systémů testů jednotek JavaScript zdroje.  
>   
>  Další informace najdete v tématu [si kód Test jednotky](../test/unit-test-your-code.md) v knihovně MSDN.
