Visual Studio 2019 je poslední verze ve programového testu UI bude k dispozici. Vezměte v úvahu [Xamarin.UITest](/appcenter/test-cloud/uitest/) pro iOS a Android testování aplikace s využitím NUnit rozhraní pro testování.
