# <a name="uninstall-visual-studio"></a>Odinstalace sady Visual Studio

Tato stránka vás provede odinstalováním sady Visual Studio, naší integrované sady nástrojů pro zvýšení produktivity pro vývojáře.

> [!NOTE]
> Toto téma platí pro Visual Studio v systému Windows. Visual Studio pro Mac najdete v [tématu Odinstalace Visual Studia pro Mac](/visualstudio/mac/uninstall).

::: moniker range="vs-2017"

1. Vyhledejte instalační program sady Visual Studio v počítači.

     Například v počítači se systémem Windows 10 Anniversary Update nebo novějším vyberte **Start** a přejděte na písmeno **V**, kde je uvedeno jako **Instalační služba sady Visual Studio**.

     ![Instalační program pro Visual Studio](media/locate-the-visual-studio-installer.png "Vyhledání Instalační služby sady Microsoft Visual Studio")

   > [!NOTE]
   > V některých počítačích může být instalační program sady Visual Studio uveden pod písmenem **"M"** jako **Instalační služba sady Microsoft Visual Studio**.<br/><br/> Instalační program sady Visual Studio najdete také v následujícím umístění:`C:\Program Files (x86)\Microsoft Visual Studio\Installer\vs_installer.exe`

1. V instalačním programu vyhledejte nainstalovanou edici sady Visual Studio. Dále zvolte **Další**a pak zvolte **Odinstalovat**.

     ![Odinstalace Visual Studia 2017](media/uninstall-visual-studio.png "Odinstalace Visual Studia 2017")

1. Klepnutím na **tlačítko OK** potvrďte svou volbu.

Pokud si to později rozmyslíte a budete chtít znovu nainstalovat Visual Studio 2017, spusťte znovu Instalační službu Visual Studia a pak na obrazovce výběru vyberte **Instalovat.**

## <a name="uninstall-visual-studio-installer"></a>Odinstalace Instalačního programu sady Visual Studio

Pokud chcete úplně odebrat všechny instalace Visual Studia 2017 a Instalační ho programu Visual Studia z počítače, odinstalujte je z aplikací & funkcí.

1. Ve Windows 10 zadejte **aplikace a funkce** do pole "Zadejte sem hledat".
1. Najít **Microsoft Visual Studio 2017** (nebo Visual Studio **2017).**
1. Zvolte **Odinstalovat**.
1. Potom vyhledejte **Instalační službu sady Microsoft Visual Studio**.
1. Zvolte **Odinstalovat**.

::: moniker-end

::: moniker range="vs-2019"

1. Vyhledejte instalační program sady Visual Studio v počítači.

     Například v počítači se systémem Windows 10 vyberte **start**a pak přejděte na písmeno **V**, kde je uvedeno jako **Instalační služba sady Visual Studio**.

     ![Otevření Instalační ho programu Visual Studio](media/vs-2019/vs-installer-windows-start.png "Otevření Instalační ho programu Visual Studio")

     > [!NOTE]
     > Instalační program sady Visual Studio najdete také v následujícím umístění:
     >
     > `C:\Program Files (x86)\Microsoft Visual Studio\Installer\vs_installer.exe`

    Před pokračováním bude pravděpodobně nutné aktualizovat instalační program. Pokud ano, postupujte podle pokynů.

1. V instalačním programu vyhledejte nainstalovanou edici sady Visual Studio. Dále zvolte **Další**a pak zvolte **Odinstalovat**.

     ![Odinstalace Visual Studia 2019](media/vs-2019/vs-installer-uninstall.png "Odinstalace Visual Studia 2019")

1. Klepnutím na **tlačítko OK** potvrďte svou volbu.

     ![Odinstalace potvrzení sady Visual Studio](media/vs-2019/uninstall-visualstudio-confirm.png "Potvrzení, že chcete odinstalovat Visual Studio 2019")

Pokud si to později rozmyslíte a budete chtít znovu nainstalovat Visual Studio 2019, spusťte znovu Instalační službu Visual Studia, zvolte kartu **Dostupné,** zvolte edici Visual Studia, kterou chcete nainstalovat, a pak vyberte **Instalovat**.

## <a name="uninstall-visual-studio-installer"></a>Odinstalace Instalačního programu sady Visual Studio

Chcete-li z počítače odebrat všechny instalace Sady Visual Studio 2019 a Instalační služby sady Visual Studio, odinstalujte ji z aplikace & funkce.

1. Ve Windows 10 zadejte **aplikace a funkce** do pole "Zadejte sem hledat".
1. Najít **Visual Studio 2019**.
1. Zvolte **Odinstalovat**.
1. Potom vyhledejte **Instalační službu sady Microsoft Visual Studio**.
1. Zvolte **Odinstalovat**.

::: moniker-end

## <a name="remove-all-files"></a>Odebrání všech souborů

Pokud dojde ke katastrofické chybě a nelze odinstalovat Visual Studio pomocí předchozích pokynů, je možnost "poslední možnost", kterou můžete zvážit použití místo. Další informace o úplném odebrání všech instalačních souborů sady Visual Studio a informací o produktu naleznete na stránce [Odebrání sady Visual Studio.](remove-visual-studio.md)
