# <a name="modify-visual-studio-2017-by-adding-or-removing-workloads-and-components"></a>Visual Studio 2017 upravit přidáním nebo odebráním úlohy a součásti
Ne jenom jsme provedli je snazší pro vás k přizpůsobení Visual Studio úlohám chcete dosáhnout, také jsme provedli ho usnadňují příliš přizpůsobit Visual Studio. Uděláte to tak, začněte nového instalačního programu Visual Studio a proveďte požadované změny.

Tady je způsob.  

## <a name="modify-workloads"></a>Upravit úlohy  
 Úlohy obsahují funkce, které potřebujete pro programovací jazyk nebo platformu, kterou používáte. Pomocí úloh upravit sadu Visual Studio tak, aby podporoval práci, kterou chcete provést, pokud chcete provést.  

>[!IMPORTANT]
>Pro instalaci, aktualizaci nebo upravit sadu Visual Studio, musíte se přihlásit pomocí účtu, který má oprávnění správce. Další informace najdete v tématu [oprávnění uživatele a Visual Studio](../ide/user-permissions-and-visual-studio.md).

1.  Vyhledání Instalační program Visual Studio v počítači.  

     Například v počítači se systémem Windows 10, vyberte **spustit**a potom přejděte k písmeno **V**, kde je uveden jako **instalační program Visual Studio**.  

     ![Instalační program sady Visual Studio](media/vs2017-locate-the-visual-studio-installer.PNG "vyhledejte instalační program Microsoft Visual Studio")

     >[!NOTE]
     V některých počítačích může instalační program Visual Studio uvedené v části písmeno **"M"** jako **Microsoft Visual Studio, instalační program**.<br/><br/> Alternativně můžete najít instalační program Visual Studio v následujícím umístění: `C:\Program Files (x86)\Microsoft Visual Studio\Installer\vs_installer.exe`

2.  Klikněte nebo klepněte sem a můžete spustit instalační program a potom zvolte **upravit**.  

     ![Spustit nebo upravit sadu Visual Studio](media/modify-visual-studio.png "upravit 2017 Visual Studio")

     Pokud máte aktualizace čekající na vyřízení, pak na tlačítko Upravit je v jiné. Klikněte na tlačítko **Další**a potom zvolte **upravit**.   

     ![Aktualizovat nebo upravit sadu Visual Studio](media/modify-or-update-visual-studio.png "aktualizace nebo upravit Visual Studio 2017")

3.  Z **úlohy** obrazovky vyberte nebo zrušte výběr úlohy, které chcete nainstalovat nebo odinstalovat.  

    ![Dialogové okno nastavení 2017 Visual Studio](media/vs2017-modify-workloads.PNG "vyberte úlohu v Visual Studio 2017")

4. Zvolte **upravit** znovu.  

5. Po instalaci nové úlohy a součásti, zvolte **spusťte**.

## <a name="modify-individual-components"></a>Upravit jednotlivé komponenty

Pokud nechcete použít funkci užitečný úlohy k přizpůsobení instalace Visual Studia, vyberte **jednotlivých součástí** z instalačního programu Visual Studio, vyberte vhodné a pak postupujte podle pokynů.
