## <a name="visual-studio-core-editor-included-with-visual-studio-team-explorer-2019"></a>Základní editor Visual Studio (zahrnutý v rámci sady Visual Studio Team Explorer 2019)

**ID:** Microsoft.VisualStudio.Workload.CoreEditor

**Popis:** příkazové prostředí sady Visual Studio core, včetně úprav kódu podle syntaxe, zdrojového kódu a správy pracovních položek.

--- | --- | --- | ---
Microsoft.VisualStudio.Component.CoreEditor | Visual Studio Core Editor | 16.1.28811.260 | Požadováno
