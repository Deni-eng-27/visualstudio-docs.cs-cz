# <a name="diagnostic-namespace"></a>diagnostic – obor názvů
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [diagnostických Namespace](https://docs.microsoft.com/visualstudio/profiling/diagnostic-namespace).  
  
`diagnostics` Obor názvů poskytuje funkce pro generování značek Vizualizéru souběžnosti.

```

```
