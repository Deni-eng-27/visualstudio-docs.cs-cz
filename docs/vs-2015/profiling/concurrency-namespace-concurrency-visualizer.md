# <a name="concurrency-namespace-concurrency-visualizer"></a>Obor názvů souběžnosti (Vizualizér souběžnosti)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

`Concurrency`Obor názvů poskytuje třídy a funkce, které vám umožní přístup k Concurrency Runtime, souběžně programovací rozhraní pro C++. Další informace najdete v tématu [Concurrency Runtime](https://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c).

```  
namespace Concurrency;  
```
