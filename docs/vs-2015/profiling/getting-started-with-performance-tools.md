# <a name="getting-started-with-performance-tools"></a>Začínáme s nástroji pro sledování výkonu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

V této části najdete přehled konceptů a úloh, které budete uplatňovat při shromažďování, zobrazování a analýze výkonu kódu pomocí nástrojů pro výkon sady Visual Studio.  
  
 Tyto nástroje poskytují několik způsobů, jak shromažďovat a analyzovat data o výkonu. V mnoha případech ale nejlepším způsobem, jak začít, je použít Průvodce výchozími nastaveními výkonu, které vám umožní shromáždit statistiku aplikace, které vám mohou Ukázat problémy s výkonem v kódu.  
  
- *Upozornění výkonu* vás upozorní na běžné problémy s kódováním. Upozornění se zobrazí v okně chyb sady Visual Studio. Můžete přejít z upozornění na zdrojový kód a podrobná témata nápovědy, která vám pomohou psát efektivnější kód.  
  
- Sestavy o výkonu poskytují zobrazení na různých úrovních struktury vaší aplikace od řádků zdrojového kódu až po procesy a data provádění programu z volání a volaných funkcí konkrétní funkce do stromu volání celé aplikace.
