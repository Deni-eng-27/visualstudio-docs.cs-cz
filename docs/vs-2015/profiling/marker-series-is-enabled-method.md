# <a name="markerseriesisenabled-method"></a>marker_series::is_enabled – metoda
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [marker_series::is_enabled – metoda](https://docs.microsoft.com/visualstudio/profiling/marker-series-is-enabled-method).  
  
Určuje, zda všechny relace má povoleno zprostředkovatele.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
bool is_enabled();  
bool is_enabled(  
   marker_importance _Importance,  
   int _Category  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `_Importance`  
 Úroveň důležitosti.  
  
 `_Category`  
 Kategorie.
