# <a name="the-designer-cannot-be-modified-while-debugging"></a>Návrháře nemůžete změnit během ladění.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [návrháře nelze změnit během ladění](https://docs.microsoft.com/visualstudio/data-tools/the-designer-cannot-be-modified-while-debugging).  
  
  
Tato zpráva se zobrazí, když je proveden pokus o upravovat položky na O/R Designer, když aplikace běží v režimu ladění. Když aplikace běží v režimu ladění, O/R Designer je jen pro čtení.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Klikněte na tlačítko **Zastavit ladění** na **ladění** nabídky.  
  
     Aplikace se zastaví ladění a je možné upravit položky v Návrháře relací objektů.
