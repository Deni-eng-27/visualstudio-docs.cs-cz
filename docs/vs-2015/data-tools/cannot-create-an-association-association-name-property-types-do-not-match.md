# <a name="cannot-create-an-association-ltassociation-namegt---property-types-do-not-match"></a>Nelze vytvořit přidružení &lt;název přidružení&gt; – typy vlastností se neshodují.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [nelze vytvořit přidružení &lt;název přidružení&gt; – typy vlastností se neshodují s](https://docs.microsoft.com/visualstudio/data-tools/cannot-create-an-association-association-name-property-types-do-not-match).  
  
  
Nelze vytvořit přidružení \<název přidružení > – typy vlastností se neshodují. Vlastnosti nemají odpovídající typy: \<názvy vlastností >.  
  
 Přidružení jsou definovány ve vybrané **vlastnosti přidružení** v **Editor asociace** dialogové okno. Vlastnosti na každé straně asociace musí být stejného datového typu.  
  
 Vlastnosti uvedené ve zprávě nemají stejné datové typy.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte zprávy a poznamenejte si vlastnosti v zprávy.  
  
2.  Klikněte na tlačítko **OK** zavřete dialogové okno.  
  
3.  Zkontrolujte **vlastnosti přidružení** a vyberte vlastnosti stejného datového typu.  
  
4.  Klikněte na tlačítko **OK**.
