# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>Co&#39;nového ve zdroji ovládací prvek modulu Plug-in API ve verzi 1.2
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [co&#39;s nové verze 1.2 zdrojového ovládacího prvku modulu Plug-in API](https://docs.microsoft.com/visualstudio/extensibility/internals/what-s-new-in-the-source-control-plug-in-api-version-1-2).  
  
Tato část se zaměřuje funkce správy pokročilé zdrojového kódu v rozhraní API modulu Plug-in zdroje ovládacího prvku verze 1.2. Následuje seznam nových funkcí ve verzi 1.2:  
  
## <a name="changes"></a>Změny  
 [Odstranění souborů ~SAK](../../extensibility/internals/elimination-of-tilde-sak-files.md)

 [Aplikace nastavení napříč různými připojeními projektů](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md)  
 Ukazuje, jak modul plug-in správy zdrojového kódu můžete používat dávkovou operaci provést stejnou operaci správy zdrojových kódů napříč více projekty nebo více kontexty připojení.  
  
 [Vytváření složek nadřazený kontejnerů pro řešení](../../extensibility/internals/creating-parent-container-folders-for-solutions.md)

 [Odebrání informací o správě zdrojového kódu ze souborů .Proj a .Sln](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md)
