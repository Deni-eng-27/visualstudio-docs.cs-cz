# <a name="extensibility-in-visual-studio"></a>Rozšiřitelnost v sadě Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio nabízí spoustu možností rozšiřitelnosti. Můžete vytvářet vlastní sady SDK, používat sady SDK pro Windows, Windows Phone a Azure (které jsou nainstalované jako součást sady Visual Studio) a samostatně roztáhnout sadu Visual Studio.

## <a name="extend-visual-studio"></a>Rozšiřování sady Visual Studio
 Sadu Visual Studio SDK můžete použít k rozšiřování pouze všech částí sady Visual Studio: příkazy, nabídky, okna, editory a projekty. Další informace najdete v tématu [Visual Studio SDK](../extensibility/visual-studio-sdk.md).

## <a name="create-your-own-sdks"></a>Vytváření vlastních sad SDK
 Přečtěte si, jak vytvořit, zabalit a nasadit vaše vlastní sady SDK pro platformy a rozšíření: [Vytvoření sady Software Development Kit](../extensibility/creating-a-software-development-kit.md).

## <a name="roslyn-extensibility"></a>Rozšiřitelnost Roslyn
 Zjistěte, jak používat Roslyn k rozšíření: [rozšiřitelnost .NET Compiler Platform ("Roslyn")](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).

## <a name="azure-sdk"></a>Azure SDK
 Zjistěte, jak povolit sadu Azure SDK: [Povolení sady Azure SDK](../extensibility/enabling-the-azure-sdk.md).
