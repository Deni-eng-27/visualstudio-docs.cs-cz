# <a name="enabling-the-azure-sdk"></a>Povolení sady Azure SDK
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Sada Azure SDK pro .NET vám pomůže vytvářet, konfigurovat, sestavovat, ladit, spouštět a zabalit škálovatelné webové aplikace a služby v Azure.  
  
 Sadu Azure SDK si můžete stáhnout z dialogového okna **Nový projekt** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Postup povolení sady Azure SDK  
  
1. V nabídce **soubor** vytvořte nový projekt. (Klikněte na **Nový** a potom na **projekt**.)  
  
2. Rozbalte uzel pro **Visual Basic** nebo **C#** a pak klikněte na **Cloud**.  
  
3. V prostředním podokně vyberte **získat sadu Azure SDK pro .NET**a pak klikněte na **OK**.  
  
     Postupujte podle pokynů v instalačním programu.  
  
    > [!NOTE]
    > Pokud chcete nainstalovat nástroje Azure, musíte mít Internetová informační služba 7,0 a nainstalované komponenty pro vývoj aplikací ASP.NET. Pokud tyto součásti nejsou nainstalovány, instalační program nástroje Azure vás vyzve k jejich instalaci.
