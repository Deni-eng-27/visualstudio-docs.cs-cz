# <a name="extending-user-settings-and-options"></a>Rozšíření uživatelských nastavení a možností
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

V aplikaci Visual Studio existují dva hlavní způsoby zachování uživatelských preferencí. Stránky **Nástroje/možnosti** umožňují uživatelům nastavit vlastní hodnoty pro mnoho různých typů nastavení, a to jak obecné, tak i pro konkrétní jazyk. Uživatelské nastavení umožňuje uživatelům zadat obecné vlastnosti konfigurace sady Visual Studio. Další informace o stránkách možností najdete v tématu [Možnosti a stránky možností](../extensibility/internals/options-and-options-pages.md). Další informace o uživatelských nastaveních najdete v tématu [Podpora uživatelských nastavení](../extensibility/internals/support-for-user-settings.md).
