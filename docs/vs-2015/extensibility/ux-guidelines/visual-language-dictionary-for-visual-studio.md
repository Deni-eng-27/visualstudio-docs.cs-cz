# <a name="visual-language-dictionary-for-visual-studio"></a>Slovník vizuálního jazyka pro Visual Studio
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Visual Studio používá společný vizuální jazyk k ilustraci konceptů, které jsou stavebními kameny svých Iconography a dalších snímků. Při navrhování nových ikon použijte tento slovník k identifikaci konceptů, které chcete předat vlastní imagi.

 **Obecné pokyny**

- Tyto vizuální koncepty se musí použít konzistentně s definicí slovníku. V opačném případě riskujete uživatele, kteří očekávají, že běžné prvky mají konzistentní význam.

- Vyberte jenom ty nejrelevantnější nebo důležité koncepty, které se mají použít ve svých ikonách, aby se v nich bylo zaostřené a snadné rozlišit.

- V každém obrázku použijte jenom jednu nebo dvě koncepty.

- Další strategie pro navrhování ikon najdete v tématu [design Icon](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
