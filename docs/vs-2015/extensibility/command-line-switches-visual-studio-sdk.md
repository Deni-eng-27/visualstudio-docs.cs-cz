# <a name="command-line-switches-visual-studio-sdk"></a>Přepínače příkazového řádku (Visual Studio SDK)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Devenv.exe spustitelný soubor, který spouští [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] integrované vývojové prostředí (IDE), podporuje přepínače příkazového řádku konkrétně pro vývoj VSPackage. Sady VSPackage mohou také mít své vlastní přepínače dostupné v příkazovém řádku devenv. Následující témata obsahují informace o použití přepínačů příkazového řádku v integrovaném vývojovém prostředí (IDE).
