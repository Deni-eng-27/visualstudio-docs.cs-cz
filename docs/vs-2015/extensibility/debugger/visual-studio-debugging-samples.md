# <a name="visual-studio-debugging-samples"></a>Ukázky ladění sady Visual Studio
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Dva ukázky ukazují, jak pomocí [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)]roztáhnout ladicí program sady Visual Studio.  
  
- Ukázka vyhodnocení výrazu ladicího programu pro spravované jazyky je součástí [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)]. V okně [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] klikněte na kartu **ukázky** , klikněte na **vše** nebo **C#** v seznamu jazyků a pak klikněte na **ladicí program**.  
  
- Ukázka ladicího modulu sady Visual Studio, pro nativní jazyky, je k dispozici v galerii kódu na webu MSDN. Další informace najdete na této stránce na webu společnosti Microsoft: [Ukázka ladicího stroje sady Visual Studio](https://go.microsoft.com/fwlink/?LinkId=150236).
