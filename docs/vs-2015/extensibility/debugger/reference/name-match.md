# <a name="namematch"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [NAME_MATCH](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/name-match).  
  
Vybere možnost případu pro odpovídající názvy.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>Členové  
 nmNone  
 Nejsou zadány žádné parametry.  
  
 nmCaseSensitive  
 Určuje, že jsou názvy lze porovnat malá a velká písmena.  
  
 nmCaseInsensitive  
 Značí, že názvy lze porovnat nejsou malá a velká písmena.
