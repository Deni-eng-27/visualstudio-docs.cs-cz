# <a name="idebugwindowscomputerport2getcomputerinfo"></a>IDebugWindowsComputerPort2::GetComputerInfo
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugWindowsComputerPort2::GetComputerInfo](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugwindowscomputerport2-getcomputerinfo).  
  
Načte informace o počítači, na kterém běží v ladicím programu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT GetComputerInfo(  
   COMPUTER_INFO * pInfo  
);  
```  
  
```csharp  
public int GetComputerInfo(  
   out COMPUTER_INFO[] pInfo  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pInfo`  
 [out] Odkaz na strukturu, která obsahuje informace o počítači.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
