# <a name="idebugengine3setallexceptions"></a>IDebugEngine3::SetAllExceptions
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugEngine3::SetAllExceptions](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugengine3-setallexceptions).  
  
Tato metoda nastaví stav všech nezpracovaných výjimek.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT SetAllExceptions(  
   EXCEPTION_STATE dwState  
);  
```  
  
```csharp  
int SetAllExceptions(  
   enum_EXCEPTION_STATE dwState  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `dwState`  
 [in] Jeden z [EXCEPTION_STATE](../../../extensibility/debugger/reference/exception-state.md) hodnoty.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
