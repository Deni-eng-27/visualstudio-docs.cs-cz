# <a name="idebugaliasgeticordebugvalue"></a>IDebugAlias::GetICorDebugValue
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Načte spravované rozhraní kódu, které představuje hodnotu přidruženou k tomuto aliasu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetICorDebugValue(  
   IUnknown** ppUnk  
);  
```  
  
```csharp  
int GetICorDebugValue(  
   out object ppUnk  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppUnk`  
 [out] `IUnknown` rozhraní, které představuje hodnotu přidruženou k tomuto aliasu. Toto rozhraní se může dotazovat na `ICorDebugValue` rozhraní.  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí S_OK; v opačném případě vrátí kód chyby.  
  
## <a name="remarks"></a>Poznámky  
 Tato metoda se vztahuje pouze na spravované hodnoty ( `ICorDebugValue` je rozhraní dostupné v [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] a je definováno v [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] sadě SDK v souboru cordebug. idl).
