# <a name="idebugnosymbolsevent2"></a>IDebugNoSymbolsEvent2
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Signály [!INCLUDE[vsprvs](../../../includes/vsprvs-md.md)] ladicího programu uživatelského rozhraní uživatele upozornit, že symbolů nebyl nalezen pro spuštěnou spustitelný soubor.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
IDebugNoSymbolsEvent2 : IUnknown  
```
