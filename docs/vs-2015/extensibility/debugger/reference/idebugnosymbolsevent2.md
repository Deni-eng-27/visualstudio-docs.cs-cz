# <a name="idebugnosymbolsevent2"></a>IDebugNoSymbolsEvent2
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Signalizuje [!INCLUDE[vsprvs](../../../includes/vsprvs-md.md)] uživatelské rozhraní ladicího programu upozorňující uživatele na to, že symboly pro spuštěný spustitelný soubor nebyly nalezeny.  
  
## <a name="syntax"></a>Syntax  
  
```  
IDebugNoSymbolsEvent2 : IUnknown  
```  
  
## <a name="notes-for-implementers"></a>Poznámky pro implementátory  
 Implementované moduly ladění a spotřebované v [!INCLUDE[vsprvs](../../../includes/vsprvs-md.md)] uživatelském rozhraní ladicího programu.
