# <a name="idebugthread2getprogram"></a>IDebugThread2::GetProgram
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugThread2::GetProgram](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugthread2-getprogram).  
  
Získá program, ve kterém je spuštěn podproces.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT GetProgram (   
   IDebugProgram2** ppProgram  
);  
```  
  
```csharp  
int GetProgram (   
   out IDebugProgram2 ppProgram  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppProgram`  
 [out] Vrátí [IDebugProgram2](../../../extensibility/debugger/reference/idebugprogram2.md) objekt, který představuje toto vlákno je spuštěn v programu.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
