# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugProgramHost2::GetHostId](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprogramhost2-gethostid).  
  
Získá identifikátor procesu procesu, který je hostitelem tohoto programu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT GetHostId(   
   AD_PROCESS_ID* pdwId  
);  
```  
  
```csharp  
int GetHostId(   
   AD_PROCESS_ID[] pdwId  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pdwId`  
 [out v] [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) struktura, která se vyplní informace o identifikátor procesu.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
