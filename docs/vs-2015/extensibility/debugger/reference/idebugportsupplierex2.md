# <a name="idebugportsupplierex2"></a>IDebugPortSupplierEx2

Poskytuje podporu pro dodavatele portu vyberte a interakci s jádra serveru.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
IDebugPortSupplierEx2 : IUnknown  
```
