# <a name="idebugdocumenttextevents2ondestroy"></a>IDebugDocumentTextEvents2::onDestroy
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugDocumentTextEvents2::onDestroy](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugdocumenttextevents2-ondestroy).  
  
Označuje, že došlo ke zničení celý dokument.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT onDestroy(   
   void   
);  
```  
  
```csharp  
int onDestroy();  
```  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
