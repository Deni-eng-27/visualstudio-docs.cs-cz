# <a name="idebugport2getportsupplier"></a>IDebugPort2::GetPortSupplier
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugPort2::GetPortSupplier](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugport2-getportsupplier).  
  
Získá dodavatele portu pro tento port.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT GetPortSupplier(   
   IDebugPortSupplier2** ppSupplier  
);  
```  
  
```csharp  
int GetPortSupplier(   
   out IDebugPortSupplier2 ppSupplier  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppSupplier`  
 [out] Vrátí [IDebugPortSupplier2](../../../extensibility/debugger/reference/idebugportsupplier2.md) objekt představuje dodavatele portu portu.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí kód chyby.
