# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDebugObject2::CreateAlias](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugobject2-createalias).  
  
Vytvoří jedinečné ID nebo alias pro tento objekt nebo vrátí existující alias.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT CreateAlias(  
   IDebugAlias** ppAlias  
);  
```  
  
```csharp  
int CreateAlias(  
   out IDebugAlias ppAlias  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppAlias`  
 [out] Nový (nebo existující) alias.  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí hodnotu S_OK; v opačném případě vrátí kód chyby.  
  
## <a name="remarks"></a>Poznámky  
 Alias je popisek, který představuje určitý objekt, pokud je objekt v paměti.
