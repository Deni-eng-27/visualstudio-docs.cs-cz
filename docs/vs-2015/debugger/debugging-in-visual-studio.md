# <a name="debugging-in-visual-studio"></a>Ladění v sadě Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]
Ladicí program sady Visual Studio pomáhá sledovat chování programu za běhu a hledat problémy. Ladicí program funguje se všemi programovacími jazyky sady Visual Studio a jejich přidruženými knihovnami. Pomocí ladicího programu můžete přerušit provádění programu za účelem prohlédnutí kódu, prozkoumávat a upravovat proměnné, zobrazit Registry, zobrazit pokyny vytvořené z vašeho zdrojového kódu a zobrazit paměťovou plochu, kterou používá vaše aplikace.

 Stručný úvod do ladění a diagnostiky v aplikaci Visual Studio naleznete v tématu [Debug, Profile and Diagnostics](https://www.visualstudio.com/features/debugging-and-diagnostics-vs).

- [Okna ladicího programu](../debugger/debugger-windows.md) Obsahuje seznam různých oken ladicího programu.

- [Základy ladicího programu](../debugger/debugger-basics.md) Popisuje základy používání ladicího programu sady Visual Studio. Témata obsahují základní informace o ladění, řízení spouštění, připojení ke spuštěnému programu, ladění za běhu, automatické spouštění ladicího programu, výpisy, zarážky, kontrolu vašeho programu, zpracování výjimek, úpravy a pokračování a používání výrazů v ladicím programu.

- [Další funkce ladění](../debugger/more-debugging-features.md) Popisuje různé funkce ladění.

- [Vzdálené ladění](../debugger/remote-debugging.md) Popisuje, jak nastavit a použít vzdálené ladění.

- [Ladění vícevláknových aplikací](../debugger/debug-multithreaded-applications-in-visual-studio.md) Popisuje, jak ladit vícevláknové aplikace.

- [IntelliTrace](../debugger/intellitrace.md) Popisuje funkci IntelliTrace.

- [Ladění aplikací](../debugger/debugging-applications.md) Popisuje, jak ladit různé druhy aplikací.

- [Ladění odkazu uživatelského rozhraní](../debugger/debugging-user-interface-reference.md) Poskytuje referenční informace o oknech a dialogových oknech, která používáte při ladění aplikace. Tato témata jsou k dispozici také po stisknutí klávesy F1 při zaměření na prvek uživatelského rozhraní ladění, jako je například okno **zarážky** .

- [Debug Interface Access SDK](../debugger/debug-interface-access/debug-interface-access-sdk.md) Popisuje Microsoft Debug Interface Access Software Development Kit (DIA SDK). DIA SDK poskytuje přístup k ladicím informacím uloženým v souborech databáze programu (PDB) generovaných nástroji Microsoft postcompiler Tools.
