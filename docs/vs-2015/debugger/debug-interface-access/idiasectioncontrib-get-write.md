# <a name="idiasectioncontribgetwrite"></a>IDiaSectionContrib::get_write
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [idiasectioncontrib::get_write –](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasectioncontrib-get-write).  
  
Získá příznak, který označuje, zda může být upraveno v části.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT get_write (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Vrátí `TRUE` části lze zapsat do; v opačném případě vrátí-li `FALSE`.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`. Vrátí `S_FALSE` -li tato vlastnost není podporována. V opačném případě vrátí kód chyby.
