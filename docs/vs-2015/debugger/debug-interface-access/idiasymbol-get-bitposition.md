# <a name="idiasymbolgetbitposition"></a>IDiaSymbol::get_bitPosition
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [idiasymbol::get_bitposition –](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-bitposition).  
  
Načte pozici bit umístění. Použít, pokud [locationtype – výčet](../../debugger/debug-interface-access/locationtype.md) je `LocIsBitField`.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT get_bitPosition (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Vrátí pozici bit umístění.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí `S_FALSE` nebo kód chyby.  
  
> [!NOTE]
>  Vrácená hodnota `S_FALSE` znamená, že vlastnost není k dispozici pro symbol.
