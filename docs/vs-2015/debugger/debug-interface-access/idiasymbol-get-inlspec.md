# <a name="idiasymbolget_inlspec"></a>IDiaSymbol::get_InlSpec
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Tato funkce načte příznak označující, zda byla funkce označena jako vložená (pomocí jednoho z [vložených __inline, \_ _forceinline](../../misc/inline-inline-forceinline.md) atributů).  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT get_inlSpec(  
   BOOL *pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 mimo Vrátí `TRUE` , zda byla funkce označena jako vložená; v opačném případě vrátí `FALSE` .  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí, `S_OK` jinak vrátí `S_FALSE` nebo kód chyby.  
  
> [!NOTE]
> Návratová hodnota `S_FALSE` znamená, že vlastnost není k dispozici pro symbol.
