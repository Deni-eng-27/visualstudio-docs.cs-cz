Nejnovější verzi tohoto tématu můžete najít v [idiasymbol::get_virtualbaseoffset –](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-virtualbaseoffset).  
  
Načte posun v tabulce virtuální funkce virtuální funkce.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT get_virtualBaseOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Vrátí posunutí v tabulce virtuální funkce virtuální funkce.  
  
## <a name="return-value"></a>Návratová hodnota

> [!NOTE]
>  Vrácená hodnota `S_FALSE` znamená, že vlastnost není k dispozici pro symbol.
