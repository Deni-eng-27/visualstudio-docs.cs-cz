# <a name="idiasymbolgetnumberofmodifiers"></a>IDiaSymbol::get_numberOfModifiers
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [IDiaSymbol::get_numberOfModifiers](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-numberofmodifiers).  
  
Získá počet parametrů, které se použijí k původnímu typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT get_numberOfModifiers(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Ukazatel `DWORD` , která určuje počet parametrů, které se použijí k původnímu typu.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud je úspěšná, vrátí `S_OK`; v opačném případě vrátí `S_FALSE` nebo kód chyby.
