# <a name="debugger-basics"></a>Základy ladicího programu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]
Ladicí program sady Visual Studio umožňuje sledovat chování programu za běhu a vyhledávat logické chyby. Ladicí program funguje se všemi programovacími jazyky sady Visual Studio a jejich přidruženými knihovnami. Pomocí ladicího programu můžete přerušit provádění na krokování kódu a vyhodnotit proměnné v aplikaci.
