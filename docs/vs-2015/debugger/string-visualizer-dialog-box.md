# <a name="string-visualizer-dialog-box"></a>Dialogové okno vizualizéru řetězce
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Toto dialogové okno je jedním ze standardních vizualizací, které jsou součástí ladicího programu sady Visual Studio. Kromě normálního textu může Vizualizér řetězců vyhodnotit řetězce HTML a XML, což vám umožní zobrazit výsledky v smysluplném formátu.
