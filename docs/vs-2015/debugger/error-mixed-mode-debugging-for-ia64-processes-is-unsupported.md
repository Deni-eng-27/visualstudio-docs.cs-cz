# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>Chyba: Ladění ve smíšeném režimu není pro procesy IA64 podporováno.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [Chyba: ladění ve smíšeném režimu pro procesy IA64 není podporován](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-for-ia64-processes-is-unsupported).  
  
[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] Ladicí program nepodporuje ladění smíšené nativního a spravovaného kódu v procesu s procesorem Itanium.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Sestavení 32bitové verze ladění vaší aplikace.
