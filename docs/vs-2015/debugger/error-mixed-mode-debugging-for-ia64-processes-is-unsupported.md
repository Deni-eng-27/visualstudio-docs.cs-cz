[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] Ladicí program nepodporuje ladění smíšené nativního a spravovaného kódu v procesu s procesorem Itanium.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Sestavení 32bitové verze ladění vaší aplikace.
