# <a name="error-unable-to-access-the-sql-server-debugging-interface"></a>Chyba: Nelze přistoupit k ladicímu prostředí serveru SQL Server.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [Chyba: Nelze přistoupit k ladicímu prostředí serveru SQL Server](https://docs.microsoft.com/visualstudio/debugger/error-unable-to-access-the-sql-server-debugging-interface).  
  
Tuto zprávu nastane při pokusu o ladění SQL na počítači, kde není povolené ladění SQL.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby
