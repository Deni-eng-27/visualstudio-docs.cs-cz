# <a name="visualizer-api-reference"></a>Referenční dokumentace rozhraní API vizualizéru
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Rozhraní API pro Vizualizér jsou k dispozici pro uživatele, kteří chtějí napsat Vizualizér pro ladicí program sady Visual Studio. Vizualizér je malá aplikace, která rozšiřuje funkčnost uživatelského rozhraní ladicího programu sady Visual Studio. Vizualizér může zobrazit (a volitelně také upravit) datový objekt konkrétního typu, pro který je Vizualizér navržený.
