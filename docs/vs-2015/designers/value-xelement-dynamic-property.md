# <a name="value-xelement-dynamic-property"></a>Value (dynamická vlastnost XElement)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Získá nebo nastaví obsah elementu.

## <a name="syntax"></a>Syntax

```
elem.Value
```

## <a name="property-valuereturn-value"></a>Hodnota vlastnosti / návratová hodnota
 <xref:System.String>Který představuje zřetězený obsah elementu.

## <a name="remarks"></a>Poznámky
 Tato vlastnost je ekvivalentní <xref:System.Xml.Linq.XElement.Value%2A> vlastnosti <xref:System.Xml.Linq.XElement?displayProperty=fullName> třídy, ale tato dynamická vlastnost také podporuje oznamování změn.
