# <a name="visual-studio-tools-for-apache-cordova"></a>Visual Studio Tools for Apache Cordova
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vytvářejte aplikace, které běží na zařízeních s iOS, Androidem a Windows, a sdílejte téměř 100% kódu. Sestavujte je s vašimi oblíbenými webovými technologiemi. Apache Cordova poskytuje nativní přístup k funkcím zařízení, jako je fotoaparát na telefonu, a co nejlépe je vše, je zdarma. Visual Studio přidá tyto doplňky, aby bylo ještě snazší sestavit aplikaci:  
  
- Snadná instalace a nastavení.  
  
- Pokročilý ladicí program, který vám může pomoct najít a opravit problémy.  
  
- Snadný přístup k emulátorům zařízení.  
  
- Návrhář pro správu modulů plug-in zařízení.  
  
- Projekt, který je zcela kompatibilní s rozhraním příkazového řádku.  
  
Pokud zvažujete tyto nástroje, ale nejste připraveni je vyzkoušet, navštivte stránku [funkce](https://www.visualstudio.com/explore/cordova-vs) a my vám pošleme další informace.  
  
Pokud jste připraveni začít, přečtěte si téma [vývoj pro mobilní zařízení pomocí JavaScriptu](/visualstudio/cross-platform/tools-for-cordova/?view=toolsforcordova-2017).
