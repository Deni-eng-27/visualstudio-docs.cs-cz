# <a name="refactoring-in-visual-studio"></a>Refaktoring v sadě Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Refaktoring je proces úpravy kódu, který usnadňuje údržbu, pochopení a rozšiřování, ale beze změny jeho chování. Mezi běžné *refaktoringy* patří:

1. rozdělení dlouhých funkcí na menší

2. Přejmenování proměnných, typů, funkcí nebo parametrů

3. Změna pořadí parametrů

4. odebrání nebo změna pořadí direktiv using

   Vzhledem k tomu, že logika refaktoringu je úzce spjata s syntaxí jazyka, každá služba jazyka v aplikaci Visual Studio poskytuje vlastní funkce refaktoringu. Další informace najdete na následujících stránkách:
