# <a name="customizing-the-editor"></a>Vlastní nastavení editoru
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Můžete přizpůsobit možnosti formátování, tabulátory, písma a barvy, IntelliSense a mnoho dalších funkcí editoru v obecném nebo konkrétním jazykovém editoru změnou možností. Další informace o těchto možnostech najdete v tématu [dialogové okno Možnosti textového editoru](../ide/reference/text-editor-options-dialog-box.md) nebo zadání `Editor` do **Rychlé spuštění**. Můžete také změnit způsob zobrazení oken editoru.

 [Postupy: Správa oken editoru](../ide/how-to-manage-editor-windows.md) Ukazuje, jak pracovat s okny editoru.
