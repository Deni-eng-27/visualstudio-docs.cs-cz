# <a name="how-to-copy-class-diagram-elements-to-a-microsoft-office-document-class-designer"></a>Postupy: Kopírování elementů diagramu tříd do dokumentu sady Microsoft Office (návrhář tříd)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Můžete kopírovat tvary z diagramu tříd .NET (soubor. CD) do jiných dokumentů. Buď získáte kopii tvaru nebo jeho podkladový kód na základě druhu dokumentu, kam ho vložíte. Chcete-li kopírovat obrazce z diagramů tříd UML v projektu modelování, přečtěte si téma [Export diagramů jako obrázků](../modeling/export-diagrams-as-images.md).

### <a name="to-copy-a-single-element"></a>Zkopírování jednoho elementu

- Klikněte pravým tlačítkem na tvar a vyberte **Kopírovat obrázek**.

### <a name="to-copy-several-elements"></a>Kopírování několika prvků

1. Vyberte obrazce na diagramu, který chcete kopírovat.

2. Klikněte pravým tlačítkem na výběr a vyberte **Kopírovat obrázek**.

### <a name="to-copy-all-the-elements-in-a-class-diagram"></a>Kopírování všech prvků v diagramu tříd

1. Klikněte pravým tlačítkem myši na plochu diagramu a zvolte **možnost Vybrat vše**. (Klávesnice: **CTRL + A**)

2. V nabídce **Upravit** vyberte **Kopírovat obrázek**.

   Můžete také zvolit **Kopírovat** místo **kopírování obrázku**. **Kopírovat** zkopíruje obrázek jako normální rastrový obrázek. Příkaz **Kopírovat obrázek** kopíruje obrázek v podobě vektorové image, která je pro většinu aplikací Office lepší.
