# <a name="how-to-create-starter-kits"></a>Postupy: Vytváření startovních sad
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Startovní sada obsahuje kód pro kompletní aplikaci a dokumentaci k úpravě nebo rozbalení aplikace. Vytvoření startovní sady je v podstatě stejné jako vytvoření normální šablony projektu. jediným rozdílem je, že Startovní sada obsahuje soubory dokumentace, které jsou nastavené tak, aby se otevíraly při vytvoření projektu založeného na startovní sadě.

## <a name="designing-and-developing-a-starter-kit"></a>Návrh a vývoj startovní sady
 Nejdřív musíte identifikovat typ startovní sady, kterou chcete vyvíjet, a definovat cílovou cílovou skupinu. Dále Navrhněte projekt a dokumentaci, aby splňovaly vaše cíle.

 Pokud vytváříte ukázkovou aplikaci nebo modul plug-in:

- Vytvoří projekt, který se sestaví bez chyb.

- Přidejte kód šablony pro implementaci dalších úloh (volitelné).

- Připravte dokumentaci.

  Pokud vytváříte školicí nástroj:

- Vytvoří projekt, který se sestaví bez chyb.

- Uspořádejte prostředky, jako jsou fragmenty kódu a šablony položek.

- Připravte dokumentaci.

## <a name="creating-a-template"></a>Vytvoření šablony
 Až dokončíte projekt a dokumentaci, budete připraveni vytvořit šablonu projektu pro Startovní sadu. Tento proces je přesně stejný jako vytvoření šablony projektu.

 Následující témata obsahují informace o vytváření šablon.

 [Postupy: vytváření šablon projektů](../ide/how-to-create-project-templates.md) Vysvětluje, jak použít průvodce **exportem šablony** k vytvoření šablony.

 [Postupy: Aktualizace existujících šablon](../ide/how-to-update-existing-templates.md) Popisuje, jak upravit exportovanou šablonu. Pomocí tohoto postupu můžete upravit soubor. vstemplate pro přizpůsobení startovní sady.
