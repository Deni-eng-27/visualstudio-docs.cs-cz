# <a name="visual-studio-notifications"></a>Oznámení sady Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio má systém oznámení, který vás upozorní na různé události:

1. K dispozici je aktualizace pro komponentu v sadě Visual Studio.

2. K dispozici je aktualizace sady Visual Studio.

3. Mohlo by se jednat o událost prostředí, o kterou byste měli zájem.

   Visual Studio vás upozorní, když je k dispozici aktualizace pro sadu Visual Studio, nebo pro některé součásti, které byly nainstalovány. Poskytuje také oznámení o určitých typech událostí, ke kterým dochází v prostředí sady Visual Studio. Ikona oznámení je příznak v pravém horním rohu záhlaví sady Visual Studio; Když je ikona červená, znamená to, že došlo k potížím s vaší licencí. Když je žlutá, je k dispozici aktualizace produktu Visual Studio, kterou si můžete nainstalovat. Když je "normální" nebo černé, zobrazí se zprávy, které jsou informativní nebo volitelné, například havárie, pro které byste mohli chtít najít alternativní řešení. Kliknutím na ikonu můžete otevřít **Centrum oznámení** a vybrat, na kterém se budou pracovat. Můžete zvolit, že se má jednorázová akce Zavřít na oznámení nebo vždycky ignorovat oznámení (najetím myší nad ním) a Visual Studio vás na tuto zprávu nikdy neukáže.

   Pokud se rozhodnete vždycky ignorovat oznámení, Visual Studio ho přestane zobrazovat. Seznam ignorovaných oznámení můžete obnovit v [dialogovém okně oznámení, prostředí, možnosti](../ide/reference/notifications-environment-options-dialog-box.md).
