# <a name="retain-topics-for-later-use"></a>Uchování témat pro pozdější použití
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Když vyhledáte užitečné téma, můžete k němu později získat přístup tak, že ho zkopírujete a vložíte do textového souboru, vytisknete ho nebo uložíte jako oblíbenou položku. Můžete například uchovávat obsah pro pozdější použití tak, že otevřete místní nabídku pro témata a zvolíte možnosti **Kopírovat**, **Tisk**, **Náhled**a **Přidat k oblíbeným položkám** .

## <a name="using-the-favorites-tab"></a>Použití karty oblíbené
 Na kartě **Oblíbené** můžete otevřít místní nabídku pro libovolnou položku a pak ji přejmenovat, odstranit nebo uspořádat položky do složek.
