# <a name="application-properties"></a>Vlastnosti aplikace
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Můžete nastavit obecné vlastnosti aplikace pro Visual Basic C#, a F# aplikace, jako je například typ aplikace, spouštěcí objekt a informace o sestavení, v podokně **aplikace** v **Návrháři projektu**.

 V podokně **aplikace** je možné provést následující úlohy:

- Zadat informace o sestavení

- Určení úvodní obrazovky pro aplikaci

- Zadat události na úrovni aplikace

- Změnit název sestavení

- Změna oboru názvů pro aplikaci

- Změnit typ aplikace

- Změna spouštěcího objektu aplikace

- Ikona určení aplikace

- Povolit vizuální styly systému Windows XP

- Určení chování vytváření instancí pro aplikaci

- Povolit vlastní ověřování uživatelů

- Určení chování při vypnutí pro aplikaci

  Úlohy, které jsou k dispozici pro vaši aplikaci, závisí na používaném vývojovém jazyce.
