# <a name="set-current-stack-frame-command"></a>Nastavit aktuální rámec zásobníku – příkaz
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [nastavit aktuální rámec zásobníku – příkaz](https://docs.microsoft.com/visualstudio/ide/reference/set-current-stack-frame-command).  
  
  
Umožňuje nastavit konkrétní rámec zásobníku.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## <a name="arguments"></a>Arguments  
 `index`  
 Požadováno. Vybere rámec zásobníku podle jejich indexu.  
  
## <a name="example"></a>Příklad  
  
```  
>Debug.SetCurrentStackFrame 1  
```
