# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
Spustí [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] a načte proměnné prostředí do **adresáře VC ++** dialogové okno.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>Příklad  
 Následující příklad spustí [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] a načte proměnné prostředí do **adresáře VC ++** dialogové okno.  
  
```  
Devenv.exe /useenv  
```
