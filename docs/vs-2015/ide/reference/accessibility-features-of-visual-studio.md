# <a name="accessibility-features-of-visual-studio"></a>Funkce pro usnadnění přístupu sady Visual Studio
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Kromě funkcí pro usnadnění přístupu a nástrojů v systému Windows je Visual Studio 2013 k dispozici pro osoby s postižením následující funkce:

- Tlačítko panelu nástrojů a rozšíření textu

- Možnosti velikosti textu v editorech

- Přizpůsobení barev v editorech

- Přizpůsobení klávesových zkratek

- Automatické dokončování pro metody a parametry

> [!IMPORTANT]
> Informace v této části se vztahují jenom na uživatele, kteří licencují produkty společnosti Microsoft v USA. Pokud jste tento produkt získali mimo USA, můžete použít kartu s informacemi o pobočkách, která byla součástí vašeho softwarového balíčku, nebo můžete navštívit [stránku Kontaktujte nás pro podpora Microsoftu](https://support.microsoft.com/contactus/) a kliknout na odkaz **najít Microsoft pobočky po celém světě** (u dolní části stránky). Tato stránka obsahuje seznam telefonních čísel a adres pro služby podpory společnosti Microsoft. Pokud chcete zjistit, jestli jsou typy produktů a služeb popsaných v této části dostupné ve vaší oblasti, obraťte se na svou pobočku. Informace o usnadnění jsou k dispozici i v dalších jazycích včetně japonštiny a francouzštiny.
