# <a name="installvstemplates-devenvexe"></a>/InstallVSTemplates (devenv.exe)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Registruje šablony projektů nebo položek, které jsou umístěny v *\<Visual cestě instalace sady studio >* \Common7\IDE\ProjectTemplates\ nebo *\<Visual instalační cestu >* \Common7\IDE\ItemTemplates\, aby mohly být k dispozici v dialogových oknech **Nový projekt** a **Přidat novou položku** .

> [!WARNING]
> Tento přepínač se podporuje jenom pro vývoj partnerů sady Visual Studio a není dostupný v edicích Express. Je nutné spustit devenv jako správce, aby bylo možné použít přepínače [/Setup (devenv. exe)](../../ide/reference/setup-devenv-exe.md) a [/installvstemplates (devenv. exe)](../../ide/reference/installvstemplates-devenv-exe.md) . Další informace najdete v tématu [uživatelská oprávnění](../../ide/user-permissions-and-visual-studio.md).

## <a name="syntax"></a>Syntaxe

```
devenv.exe /InstallVSTemplates
```
