# <a name="installvstemplates-devenvexe"></a>/InstallVSTemplates (devenv.exe)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Registruje šablony projektů nebo položek, které jsou umístěny v *\<Visual Studio installation path>* \Common7\IDE\ProjectTemplates\ nebo *\<Visual Studio installation path>* \Common7\IDE\ItemTemplates\, aby k nim bylo možné přistupovat prostřednictvím dialogových oken **Nový projekt** a **Přidat novou položku** .

> [!WARNING]
> Tento přepínač se podporuje jenom pro vývoj partnerů sady Visual Studio a není dostupný v edicích Express. Chcete-li použít přepínače [/Setup (devenv.exe)](../../ide/reference/setup-devenv-exe.md) a [/installvstemplates (devenv.exe)](../../ide/reference/installvstemplates-devenv-exe.md) , je nutné spustit devenv jako správce. Další informace najdete v tématu [uživatelská oprávnění](../../ide/user-permissions-and-visual-studio.md).

## <a name="syntax"></a>Syntax

```
devenv.exe /InstallVSTemplates
```
