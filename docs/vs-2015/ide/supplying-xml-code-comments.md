# <a name="supplying-xml-code-comments"></a>Zadávání komentářů ke kódu XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Můžete nastavit komentáře v souborech zdrojového kódu, které jsou k dispozici jako IntelliSense. Přidáte-li přepínač/doc do Visual Basic, C#nebo C++ příkazového řádku kompilátoru, je vygenerován soubor. XML, který slouží jako základ pro dokumentaci technologie IntelliSense.

 Další informace o dokumentu kódu v Visual Basic naleznete v tématu [How to: Create XML Documentation](https://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).

 Další informace o dokumentování kódu v nástroji C#naleznete v [dokumentaci k dokumentaci XML](https://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).

 Další informace o dokumentaci kódu v nástroji naleznete C++v tématu [Comments](https://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) and [XML Documentation](https://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
