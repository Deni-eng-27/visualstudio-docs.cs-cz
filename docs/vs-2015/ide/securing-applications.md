# <a name="securing-applications"></a>Zabezpečování aplikací
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

I když většina aplikací má běžné problémy se zabezpečením, každá doména aplikace má vlastní výzvy zabezpečení.

## <a name="general-security-considerations"></a>Obecné požadavky na zabezpečení
 Každý jazyk má vlastní hlediska zabezpečení a problémy.

 [Osvědčené postupy zabezpečení](https://msdn.microsoft.com/library/86acaccf-cdb4-4517-bd58-553618e3ec42) Poskytuje informace o funkcích zabezpečení a postupech, které jsou k dispozici při práci v vizuálu C++.

 [NENÍ v sestavách: zabezpečení aC# programování (a Visual Basic)](https://msdn.microsoft.com/227e2863-cf09-4c28-9611-bcd82be5e994) poskytuje informace o hlavních třech obavách zabezpečení pro C# Visual Basic a vývojáře: oprávnění, webové aplikace a nastavení sady Visual Studio.

## <a name="securing-mobile-applications"></a>Zabezpečení mobilních aplikací
 Vzhledem k rostoucímu množství mobilních zařízení se zabezpečení informací a dat v těchto zařízeních bude důležitější.

 [Požadavky na zabezpečení pro zařízení](https://msdn.microsoft.com/45fab484-8718-452e-8210-04fda3c6cb87) Popisuje několik faktorů ovlivňujících zásady zabezpečení pro zařízení.

 [Cíle zabezpečení pro prostředí .NET Compact Framework](https://msdn.microsoft.com/64ac2770-e2bc-40a3-abbf-56c8a2c0e364) Popisuje cíle pro prostředí .NET Compact Framework zabezpečení.

 [Návrh zabezpečeného mobilního formuláře stránky webu](https://msdn.microsoft.com/b69727c1-f81f-4221-a116-8f92f769365f) Popisuje plánování, implementaci a podporu zabezpečení v bezdrátových sítích a mobilních zařízeních.

## <a name="securing-web-applications"></a>Zabezpečení webových aplikací
 Špatně napsaná webová stránka může ohrozit integritu a zabezpečení celého serveru a potenciálně celou síť. Proto je nutné zkontrolovat otázky zabezpečení při plánování webové aplikace.

 [Architektura zabezpečení ASP.NET](https://msdn.microsoft.com/library/c34d6f4f-f64d-4697-bd32-02dd2ddf726f) V této části najdete přehled infrastruktury ASP.NET a vztahů subsystémů, které jsou v souvislosti se zabezpečením.

 [Zabezpečení webové aplikace ASP.NET](https://msdn.microsoft.com/library/658d0430-1644-4744-b52d-08b0d6fcacb8) Popisuje, jak řešit problémy s autorizací a ověřováním v ASP.NET.

 [Požadavky na zabezpečení pro jazyk JScript](https://msdn.microsoft.com/8572efc9-071a-472d-a1a4-f0a3b42644c1) Podrobně se nástrahy zabezpečení, ke kterým může dojít programátorům JScript.

 [Postupy: použití zabezpečení přenosu](https://msdn.microsoft.com/16210e41-5492-4cc8-9002-7366b1fc7297) Popisuje, jak používat zabezpečení přenosu při ověřování, když se připojíte ke službě WCF.

## <a name="securing-desktop-applications"></a>Zabezpečení desktopových aplikací
 Návrh zabezpečení pro aplikace klasické pracovní plochy je důležitým krokem při vývoji aplikací.

 [Model Windows Forms zabezpečení](https://msdn.microsoft.com/library/932d438a-5285-46d8-a958-8c93d0ad6cae) Poskytuje přehled implementace zabezpečení model Windows Forms.
