# <a name="c26101"></a>C26101
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C26101: selhání pro správné použití propojené operace pro proměnnou \<var> .  
  
 Rozhraní API systému Windows nabízejí celou řadu vzájemně propojených operací. Poznámka `_Interlocked_` Určuje, že proměnná by měla být k dispozici pouze prostřednictvím propojené operace. Upozornění C26101 je vystaveno v případě, že přístup není konzistentní s `_Interlocked_` poznámkou.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje upozornění C26101, protože došlo k porušení `_Interlocked_` kontraktu.  
  
```  
  
CRITICAL_SECTION cs;  
typedef struct _DATA   
{  
    _Interlocked_ LONG data;  
} DATA;  
  
void Safe(DATA* p)   
{  
    InterlockedIncrement(&p->data); // OK  
}  
  
void Unsafe(DATA* p)   
{  
    p->data += 1; // Warning C26101  
    EnterCriticalSection(&cs);  
    LeaveCriticalSection(&cs);  
}  
  
```
