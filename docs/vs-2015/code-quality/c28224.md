# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C28224: poznámka vyžaduje parametry  
  
 Toto upozornění signalizuje, že pojmenované poznámky se použije bez parametrů a alespoň jeden parametr je povinný. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se má zobrazit při běžném používání.
