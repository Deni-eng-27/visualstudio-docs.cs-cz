# <a name="c28196"></a>C28196
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28196](https://docs.microsoft.com/visualstudio/code-quality/c28196).  
  
upozornění C28196: požadavek není splněn. (Výraz není vyhodnocen na hodnotu true.)  
  
 Toto upozornění signalizuje, že obsahuje prototyp funkce pro analyzovanou funkci `__notnull`, `__null` nebo `__drv_valueIs` na `_Out_` parametr nebo návratovou hodnotu, ale hodnota vrácená není konzistentní s tuto poznámku.
