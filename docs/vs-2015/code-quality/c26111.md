# <a name="c26111"></a>C26111
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C26111: volající nedaří uvolnit zámek \<lock> před voláním funkce \<func> .  
  
 Poznámka `_Requires_lock_not_held_` ukládá předběžnou podmínku, že počet zámků pro zadaný zámek nemůže být při volání funkce větší než nula. Upozornění C26111 se vydá, když funkce neuvolní zámek před voláním jiné funkce.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje upozornění C26111, protože `_Requires_lock_not_held_` Předběžná podmínka je porušena voláním `DoNotLock` v rámci uzamčeného oddílu.  
  
```  
  
typedef struct _DATA   
{  
    CRITICAL_SECTION cs;  
    int d;  
} DATA;  
  
_Requires_lock_not_held_(p->cs)   
  
void DoNotLock(DATA* p)  
{  
    EnterCriticalSection(&p->cs);   
    p->d = 0;  
    LeaveCriticalSection(&p->cs);  
}  
  
void LockedFunction(DATA* p)   
{   
    EnterCriticalSection(&p->cs);   
    DoNotLock(p); // Warning C26111  
    LeaveCriticalSection(&p->cs);  
}  
  
```
