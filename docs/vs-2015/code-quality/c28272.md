# <a name="c28272"></a>C28272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28272](https://docs.microsoft.com/visualstudio/code-quality/c28272).  
  
upozornění C28272: poznámka pro funkci, parametr při zkoumání je nekonzistentní s deklarací funkce  
  
 Toto upozornění označuje chybu v poznámkách, není v kódu, který je analyzován. Poznámky povolí, v definici funkce nejsou konzistentní s těmi, která povolí, v deklaraci. Dva poznámky by měla vyřešit tak, aby odpovídaly.
