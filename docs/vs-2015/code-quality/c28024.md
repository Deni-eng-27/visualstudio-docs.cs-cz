# <a name="c28024"></a>C28024
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28024: ukazatel funkce, ke kterému se přiřazuje, je opatřen poznámkou se třídou Function \<class> , která není obsažena ve třídách Functions (ES) \<classlist> .  
  
 Toto upozornění je oznámeno, pokud byly obě funkce opatřeny poznámkami třídy funkce, ale třídy se neshodují.  
  
 Toto upozornění je vyvoláno, když ukazatel na funkci obsahuje `_Function_class_` poznámku, která určuje, že k ní by měly být přiřazeny pouze funkce konkrétní funkční třídy. U přiřazení nebo implicitního přiřazení ve volání funkce musí být zdroj a cíl stejné třídy Function, ale třídy funkcí se neshodují.
