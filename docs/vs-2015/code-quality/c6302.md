# <a name="c6302"></a>C6302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6302: Neshoda formátovacího řetězce: jako parametr \<číslo byl předán řetězec znaků, > je při volání funkce \<vyžadován řetězec s velkým počtem znaků >  
  
 Toto upozornění označuje, že řetězec formátu určuje, že je vyžadován řetězec s velkým znakem. Předává se však řetězec znaků. Tato vada může způsobit selhání nebo poškození nějakého formuláře.  
  
## <a name="example"></a>Příklad  
 Následující vzorový kód vygeneruje toto upozornění, protože řetězec znaků je předán `wprintf` funkci:  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf(L"%s", buff);  
}  
```  
  
 Následující vzorový kód používá `%hs` k určení jednobajtových znaků řetězce s funkcí `wprintf`:  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf(L"%hs", buff);  
}  
```  
  
 Následující vzorový kód používá funkci bezpečného zpracování řetězce `wprintf_s` k opravě tohoto upozornění:  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf_s(L"%hs", buff);  
}  
```
