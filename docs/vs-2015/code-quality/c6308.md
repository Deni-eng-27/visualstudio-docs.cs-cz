# <a name="c6308"></a>C6308
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6308: realokace může vracet nulový ukazatel: přiřazení nulového ukazatele na \<variable> , který se předává jako argument do realokace, způsobí, že původní blok paměti nebude vrácen.  
  
 Toto upozornění signalizuje nevracení paměti, které je výsledkem nesprávného použití funkce přealokace. Funkce opětovného přidělení haldy neuvolňují předanou vyrovnávací paměť, pokud je přerozdělení neúspěšné. Chcete-li chybu opravit, přiřaďte výsledek funkce realokace dočasnou a pak po úspěšném přerozdělení nahraďte původní ukazatel.  
  
## <a name="example"></a>Příklad  
 Následující vzorový kód vygeneruje toto upozornění:  
  
```cpp  
#include <malloc.h>  
#include <windows.h>  
  
void f( )  
{  
  char *x;  
  x = (char *) malloc(10);  
  if (x != NULL)  
  {  
    x = (char *) realloc(x, 512);  
    // code...  
    free(x);  
  }     
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```cpp  
#include <malloc.h>  
#include <windows.h>  
  
void f()  
{  
  char *x, *tmp;  
  
  x = (char *) malloc(10);  
  
  if (x != NULL)  
  {  
    tmp = (char *) realloc(x,512);  
    if (tmp != NULL)   
    {  
      x = tmp;  
    }  
    free(x);  
  }  
}  
```  
  
 Toto upozornění může způsobit šum, pokud existuje živý alias pro opětovné přidělení vyrovnávací paměti v době přiřazení výsledku funkce realokace.  
  
 Chcete-li se těmto druhům problémů vyhnout zcela, použijte mechanismy, které jsou k dispozici v knihovně C++ Standard Template Library (STL). Mezi ně patří [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)a [Vector](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace naleznete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [Standardní knihovna C++](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
