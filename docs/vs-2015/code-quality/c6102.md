# <a name="c6102"></a>C6102
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6102: použití \<variable> z nezdařeného volání funkce na řádku \<location> .  
  
 Toto upozornění je hlášeno místo [C6001](../code-quality/c6001.md) , pokud není nastavena proměnná, protože byla označena jako `_Out_` parametr u předchozího volání funkce, které selhalo.  
  
 Problémem může být, že předchozí volaná funkce není zcela anotována. Může vyžadovat `_Always_` , `_Outptr_result_nullonfailure_` ( `_COM_Outptr_` pro kód com) nebo související poznámku.
