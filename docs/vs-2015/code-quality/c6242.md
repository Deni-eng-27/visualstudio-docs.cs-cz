# <a name="c6242"></a>C6242
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6242: skok z tohoto bloku try vynutí místní unwind. Dojde k výraznému snížení výkonu.  
  
 Toto upozornění znamená, že příkaz skoku způsobí, že tok řízení opouští chráněný blok try-finally.  
  
 Ponechání chráněného bloku try-finally, který je mimo poslední příkaz, vyžaduje místní operaci unwind. Místní unwind obvykle vyžaduje přibližně 1000 instrukcí počítače; Proto je neškodný výkon.  
  
 Použijte `_leave` k ukončení chráněného bloku try-finally.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <malloc.h>  
void DoSomething(char *p); // function can throw exception  
  
int f( )  
{  
   char *ptr = 0;  
   __try   
   {  
      ptr = (char*) malloc(10);  
      if ( !ptr )   
      {  
         return 0;   //Warning: 6242  
      }  
      DoSomething( ptr );  
   }  
   __finally   
   {  
      free( ptr );  
   }  
   return 1;  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte, `__leave` jak je znázorněno v následujícím kódu:  
  
```  
#include <malloc.h>  
void DoSomething(char *p);   
int f()  
{  
   char *ptr = 0;  
   int retVal = 0;  
  
   __try   
   {  
      ptr = (char *) malloc(10);  
      if ( !ptr )  
      {  
         retVal = 0;  
         __leave;   //No warning  
      }  
      DoSomething( ptr );  
      retVal = 1;  
   }  
   __finally  
   {  
      free( ptr );  
   }  
  
   return retVal;  
}  
```  
  
 Použití typu \ a Free má mnoho nástrah z pohledu nevracení paměti a výjimek. Chcete-li se vyhnout těmto typům nevracení a potížím s výjimkami, použijte mechanismy, které jsou k dispozici v knihovně C++ Standard Template Library (STL). Mezi ně patří [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)a [Vector](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace naleznete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [Standardní knihovna C++](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
