Nejnovější verzi tohoto tématu můžete najít v [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
upozornění C28216: _Check_return\_ Poznámka se týká pouze po podmínky specifických parametrů funkcí.  
  
 `_Check_return_` Anotace byla použita v kontextu jiných než `__post`; může být nutné `__post` (nebo `__drv_out`) modifikátor, nebo může být umístěna nesprávně.
