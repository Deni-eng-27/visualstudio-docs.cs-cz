# <a name="c6306"></a>C6306
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6306: nesprávné volání \<function> : Zvažte použití, \<function> který přijímá va_list jako argument.  
  
 Toto upozornění signalizuje nesprávné volání funkce. `printf`Rodina obsahuje několik funkcí, které přijímají seznam proměnných argumentů. Tyto funkce však nelze volat pomocí `va_list` argumentu. K dispozici je odpovídající `vprintf` rodina funkcí, které lze použít pro taková volání. Volání nesprávné funkce Print způsobí nesprávný výstup.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <stdio.h>  
#include <stdarg.h>  
  
void f(int i, ...)  
{  
   va_list v;  
  
   va_start(v, i);  
   //code...  
   printf("%s", v); // warning 6306   
   va_end(v);  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
#include <stdio.h>  
#include <stdarg.h>  
  
void f(int i, ...)  
{  
   va_list v;  
  
   va_start(v, i);  
   //code...  
   vprintf_s("%d",v);  
   va_end(v);  
}  
```
