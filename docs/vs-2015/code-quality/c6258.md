# <a name="c6258"></a>C6258
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6258: použití TerminateThread nepovoluje vyčištění správného vlákna.  
  
 Toto upozornění signalizuje, že bylo zjištěno volání TerminateThread.  
  
 TerminateThread je nebezpečná funkce, která by měla být použita pouze v extrémních případech. Další informace o problémech spojených s voláním TerminateThread naleznete v tomto tématu v knihovně MSDN: [TerminateThread](https://go.microsoft.com/fwlink/?LinkId=150233).  
  
### <a name="to-properly-terminate-threads"></a>Pro správné ukončení vláken  
  
1. Vytvořte objekt události pomocí funkce `CreateEvent`.  
  
2. Vytvořte vlákna.  
  
3. Každé vlákno monitoruje stav události voláním funkce `WaitForSingleObject`.  
  
4. Každé vlákno ukončí své vlastní spuštění, když je událost nastavena na stav signalizace (`WaitForSingleObject` vrátí `WAIT_OBJECT_0`).  
  
   Další informace najdete v jednom z následujících témat:  
  
   [Ukončení vlákna](https://go.microsoft.com/fwlink/?LinkId=150234)  
  
   [WaitForSingleObject](https://go.microsoft.com/fwlink/?LinkId=150235)  
  
   [SetEvent](https://go.microsoft.com/fwlink/?LinkId=150232)
