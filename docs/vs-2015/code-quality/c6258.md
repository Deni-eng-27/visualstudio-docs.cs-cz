# <a name="c6258"></a>C6258
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6258: použití TerminateThread nepovoluje vyčištění správného vlákna.  
  
 Toto upozornění signalizuje, že bylo zjištěno volání TerminateThread.  
  
 TerminateThread je nebezpečná funkce, která by měla být použita pouze v extrémních případech. Další informace o problémech spojených s voláním TerminateThread naleznete v tomto tématu v knihovně MSDN: [TerminateThread](https://msdn.microsoft.com/library/ms686717(VS.85).aspx).  
  
### <a name="to-properly-terminate-threads"></a>Pro správné ukončení vláken  
  
1. Vytvořte objekt události pomocí `CreateEvent` funkce.  
  
2. Vytvořte vlákna.  
  
3. Každé vlákno monitoruje stav události voláním `WaitForSingleObject` funkce.  
  
4. Každé vlákno ukončí své vlastní spuštění, když je událost nastavena na signálný stav ( `WaitForSingleObject` vrátí `WAIT_OBJECT_0` ).  
  
   Další informace najdete v jednom z následujících témat:  
  
   [Ukončení vlákna](https://msdn.microsoft.com/library/ms686724(VS.85).aspx)  
  
   [WaitForSingleObject](https://msdn.microsoft.com/library/ms687032.aspx)  
  
   [SetEvent](https://msdn.microsoft.com/library/ms686211(VS.85).aspx)
