# <a name="c6384"></a>C6384
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6384: dělení sizeof ukazatele na jinou hodnotu  
  
 Toto upozornění znamená, že výpočet velikosti může být nesprávný. Chcete-li vypočítat počet prvků v poli, jedna někdy rozdělí velikost pole podle velikosti prvního prvku; Nicméně, pokud je pole ve skutečnosti ukazatel, výsledek je obvykle odlišný od zamýšleného.  
  
 Pokud je ukazatel parametrem funkce a velikost vyrovnávací paměti nebyla předána, není možné vypočítat maximální dostupnou vyrovnávací paměť. Pokud je ukazatel přiřazen místně, měla by být použita velikost použitá v přidělení.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <windows.h>  
#include <TCHAR.h>  
  
#define SIZE 15  
  
void f( )  
{  
  LPTSTR dest = new TCHAR[SIZE];  
  char src [SIZE] = "Hello, World!!";  
  if (dest)  
  {  
    _tcsncpy(dest, src, sizeof dest  / sizeof dest[0]);   
  }  
}  
```  
  
 Chcete-li toto upozornění opravit, předejte velikost vyrovnávací paměti, jak je znázorněno v následujícím kódu:  
  
```  
#include <windows.h>  
#include <TCHAR.h>  
  
#define SIZE 15  
  
void f( )  
{  
  LPTSTR dest = new TCHAR[SIZE];  
  char src [SIZE] = "Hello, World!!";  
  if (dest)  
  {  
    _tcsncpy(dest, src, SIZE);  
  }  
}  
```  
  
 Chcete-li toto upozornění opravit pomocí _tcsncpy_s funkce bezpečného řetězce, použijte následující kód:  
  
```  
void f( )  
{  
  LPTSTR dest = new TCHAR[SIZE];  
  char src [SIZE] = "Hello, World!!";  
  if (dest)  
  {  
   _tcsncpy_s(dest, SIZE, src, SIZE);  
  }  
}   
```  
  
 Všimněte si, že použití New a DELETE má mnoho nástrah z pohledu nevracení paměti a výjimek. Chcete-li se vyhnout těmto typům nevrácených chyb a potížím s výjimkami, použijte C++ mechanismy, které jsou k dispozici pomocí knihovny Standard Template Library (STL). Mezi ně patří [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)a [Vector](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace najdete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [ C++ standardní knihovna](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
