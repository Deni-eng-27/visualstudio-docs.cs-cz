# <a name="c28282"></a>C28282
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28282: řetězce formátu musí být v předběžných podmínkách  
  
 Toto upozornění indikuje, že `__drv_formatString` se najde Poznámka, která není v `_Pre_` `__drv_in` anotaci () (ve výchozím nastavení jsou parametry funkce `_Pre_` ). Ověřte, zda anotace použitá v explicitním bloku má `_Post_` `__drv_out` poznámku (). Pokud ano, odeberte anotaci z jakéhokoli nadřazeného bloku, který je vložil v `_Post_` kontextu.
