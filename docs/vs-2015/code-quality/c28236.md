# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
upozornění C28236: anotaci nelze použít ve výrazu  
  
 Toto upozornění signalizuje, že byl proveden pokus o použití funkce deklarovaná jako poznámka v kontextu výrazu. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se neočekává se zobrazí v typické použití.
