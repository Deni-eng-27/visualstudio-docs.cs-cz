# <a name="c28308"></a>C28308
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 28308: pozice argumentu seznamu formátu zadaná v poznámce je nesprávná.  
  
 Pozice argumentu formátovat seznam musí být buď název parametru, nebo celočíselný posun, který je v seznamu parametrů, nebo nula.  
  
 Druhý parametr pro `IsFormatString2` ( `where` ) může být v jednom ze dvou forem:  
  
- Název parametru, který je považován za první argument formátovacího řetězce.  
  
- Posun ( `n` ) relativní vzhledem k parametru formátovacího řetězce.  
  
  Ve druhém formuláři je prvním parametrem formátu řetězce parametr `n` -th za formátovacím řetězcem. Pokud `n` je nula, je jako parametr zadána tři tečky. Určení posunu nuly bez zadání tří teček jako prvního parametru formátovacího řetězce způsobí chybu.
