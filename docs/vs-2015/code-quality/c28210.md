# <a name="c28210"></a>C28210
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 28210: poznámky pro `_On_failure_` kontext nesmí být v explicitním předběžném kontextu  
  
 Poznámky `_On_failure_` musí být explicitně nebo implicitně označeny v `__post` kontextu, to znamená, že se mají použít po vrácení funkce.  Použijte `_drv_out` k zajištění tohoto.
