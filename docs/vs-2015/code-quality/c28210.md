upozornění 28210: poznámky pro `_On_failure_` kontextu nesmí být v explicitním předkontextu  
  
 Poznámky `_On_failure_` musí být explicitně nebo implicitně podle `__post` kontextu, to znamená, použije po vrácení funkce.  Použití `_drv_out` to zajistit.
