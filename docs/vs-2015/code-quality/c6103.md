# <a name="c6103"></a>C6103
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6103: návrat \<variable> z nezdařeného volání funkce na řádku \<location> .  
  
 Úspěšná cesta prostřednictvím funkce vrací proměnnou, která byla použita jako `_Out_` parametr pro vnitřní volání funkce, které selhalo.  
  
 Problémem může být, že volaná funkce a volající funkce nejsou zcela anotovány. Volaná funkce může vyžadovat `_Always_` , `_Outptr_result_nullonfailure_` ( `_COM_Outptr_` pro kód com) nebo související anotaci a volání funkce může vyžadovat `_Success_` anotaci. Další možnou příčinou tohoto upozornění je, že `_Success_` Poznámka u volané funkce je nesprávná.
