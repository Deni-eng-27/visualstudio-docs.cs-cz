# <a name="c28311"></a>C28311
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 28311: < annotation_name > poznámku na \<funkce > \<parametr > je zastarávající verze SAL.  
  
 Poznámka je starou verzí a měla by být aktualizována na ekvivalentní [poznámkou SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Toto upozornění není aktivováno, pokud bylo aktivováno předchozí upozornění na nekonzistentní poznámku a je pokud hlášeno pouze jednou za deklaraci. Zkontrolujte zbytek deklarace, zda neobsahuje zastaralou verzi SAL.
