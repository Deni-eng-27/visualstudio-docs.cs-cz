# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6995: nepovedlo se uložit soubor protokolu XML.  
  
 Toto upozornění znamená, že nástroj pro analýzu kódu nemůže vytvořit protokol vad, což je výstup analýzy kódu.  
  
 Tato chyba může znamenat chybu disku nebo indikuje, že nemáte oprávnění k vytvoření souboru v zadaném adresáři.
