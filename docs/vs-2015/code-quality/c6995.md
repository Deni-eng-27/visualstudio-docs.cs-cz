# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C6995: nepovedlo se uložit soubor protokolu XML  
  
 Toto upozornění signalizuje, že nástroj Code Analysis nelze vytvořit protokol chyb, který je výstup analýzy kódu.  
  
 Tato chyba může znamenat chybu disku nebo označení, že nemáte oprávnění k vytvoření souboru v zadaném adresáři.
