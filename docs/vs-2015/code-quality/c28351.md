# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28351: Poznámka \<annotation> popisuje, kde v podmínce nelze použít dynamickou hodnotu (proměnnou).  
  
 Toto upozornění je oznámeno při použití poznámky, kde je kontrolován typ C/C++.  
  
 Výraz ve výrazu `_When_` by měl být vyhodnocen jako konstanta. Rozhraní `_When_` je ignorováno.
