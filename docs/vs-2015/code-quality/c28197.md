# <a name="c28197"></a>C28197
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28197: možná nevrácení paměti  
  
 Toto upozornění je hlášeno pro paměť i nevracení prostředků, pokud je prostředek potenciálně aliasem jiného umístění.  
  
 *Ukazatel* ukazuje na přidělenou paměť nebo na jiný přidělený prostředek, který nebyl explicitně uvolněn. Toto upozornění je obvykle způsobeno nedostatečnými poznámkami k volané funkci, i když neodpovídající anotace volání funkce může také tuto situaci zvýšit.  
  
 Toto upozornění může být oznámeno funkcí ukončit, pokud vstupní argument obsahuje `__drv_freesMem` `__drv_aliasesMem` poznámku nebo. Toto upozornění obvykle označuje buď platnou netěsnou, nebo funkci volanou aktuální funkcí potřebuje další anotaci.  
  
 Konkrétně neexistence základního `_In_` a `_Out_` poznámky toto upozornění způsobí poměrně pravděpodobné, i když `__drv_aliasesMem` `__drv_freesMem` můžou být potřeba poznámky a. Falešně pozitivní je pravděpodobný výsledek chybějící `_In_` poznámky.  
  
 Funkce, které přijímají ukazatel a alias (proto zamezení úniku) by měly být opatřeny poznámkami `__drv_aliasesMem` . Vytvoříte-li funkci, která vloží objekt do globální struktury, nebo předá do systémové funkce, která to provede, měli byste přidat `__drv_aliasesMem` poznámku.  
  
 Funkce, které mají volnou paměť, by měly být opatřeny poznámkami `__drv_freesMem` . Hlavní funkce, které tuto poznámku již mají, mají volnou paměť.  
  
## <a name="example"></a>Příklad  
 Následující příklad kódu vygeneruje toto upozornění:  
  
```  
char *p = (char *)malloc(10);  
test(p);   // does not save a copy of p  
```  
  
 Následující příklad kódu předchází tomuto upozornění:  
  
```  
char *p = (char *)malloc(10);  
test(p);   // does not save a copy of p  
free(p);  
```
