upozornění C28215: typefix nelze použít pro parametr, který již poznámku typefix obsahuje  
  
 Použití `__typefix` se o chybu, anotace pro parametr, který už má tuto poznámku. `__typefix` Poznámky se používají pouze ve zvláštních případech několik a toto upozornění se má zobrazit při běžném používání.
