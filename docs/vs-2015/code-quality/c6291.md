# <a name="c6291"></a>C6291
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6291: bitová operace na logickém výsledku:! má vyšší prioritu než &#124;. Použijte &#124;&#124; nebo (! ( x &#124; y)) místo toho  
  
 `!`Operátor vrací logický výsledek a `|` operátor (bitový operátor OR) přebírá dva aritmetické argumenty. `!`Operátor má také vyšší prioritu než `|` .  
  
 Proto byla zjištěna jedna z následujících chyb:  
  
- Výraz je mis-parenthesised:  
  
   Vzhledem k tomu, že výsledek `!` je logická hodnota (nula nebo jedna), pokus o otestování, že dvě proměnné mají nastavené bity, bude pouze končit testováním, že na pravé straně je přítomen nejnižší bit: `((!x) | y) != (!(x | y))` when `x == 0` a `y == 1` .  
  
- `!`Operátor není správný a měl by být `~` místo toho:  
  
   `!`Operátor má logický výsledek, ale `~` operátor má aritmetický výsledek. Tyto operátory nejsou nikdy zaměnitelné, i když pracuje na logické hodnotě (nula nebo jedna): `((!x) | y) != ((~x) | y)` kdy `x == 1` a `y == 0` .  
  
- Binární operátor `|` je nesprávný a měl by být `||` :  
  
   I když `|` může být někdy vzájemně zaměnitelné `||` , není ekvivalentní, protože vynucuje vyhodnocování pravé strany výrazu. Některé vedlejší účinky v tomto typu výrazu mohou být terminál: `(!p | (*p == '\0'))` , kdy je nutné na tuto chvíli `p == NULL` odkázat, aby se vyhodnotila druhá polovina výrazu.  
  
  Toto upozornění není hlášeno, je-li `!` operátor na pravé straně `|` operátoru, protože tento případ je obvykle pouze poměrně neškodný případ nesprávného operátoru.  
  
  Je obtížné posuzovat závažnost tohoto problému bez přezkoumání kódu. Kód by měl být zkontrolován, aby se zajistilo, že k zamýšlenému testu dochází.  
  
  Toto upozornění vždy indikuje možnou nejasnost při použití operátoru nebo priority operátoru.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
void f(int x, int y )  
{  
  if (!x | y)  
  {  
    //code   
  }  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte jednu z metod, které jsou uvedeny v následujícím kódu:  
  
```  
void fC(int x, int y )  
{  
  /* When checking whether any bits are set in either x or y. */  
  if (!(x | y))  
  {  
    // code  
  }  
  /* When checking whether bits are set in either */  
  /* the complement of x or in y. */  
  if ((~x) | y)  
  {  
    // code  
  }  
}  
  
#include <windows.h>  
void f(int x, BOOL y )  
{  
  /* When y is a Boolean or Boolean result. */  
  if ((!x) || y)  
  {  
    // code  
  }  
}  
```
