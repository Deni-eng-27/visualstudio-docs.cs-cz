# <a name="c28243"></a>C28243
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28243: Poznámka pro funkci v parametru vyžaduje více odkazů, než je skutečný typ s poznámkami. umožňuje  
  
 Počet `__deref` operátorů v poznámce je větší než počet úrovní ukazatelů definovaných typem parametru. Opravte toto upozornění změnou čísla v poznámce nebo úrovní ukazatelů odkazovaného parametru.
