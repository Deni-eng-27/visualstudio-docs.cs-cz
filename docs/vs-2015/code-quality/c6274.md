# <a name="c6274"></a>C6274
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6274: jako parametr je předán jiný než znak, \<number> Pokud se při volání metody vyžaduje znak. \<function>  
  
 Toto upozornění označuje, že formátovací řetězec Určuje, že je vyžadován znak (například `%c` specifikace nebo), `%C` ale není předáno celé číslo, jako je například float, String nebo struct. Tato vada může způsobit nesprávný výstup.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str);  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str[0]);  
}  
```  
  
 Následující kód používá bezpečnou funkci manipulace s řetězci, `sprintf_s` aby bylo možné toto upozornění opravit:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf_s(buff,5,"%c", str[0]);  
}  
```
