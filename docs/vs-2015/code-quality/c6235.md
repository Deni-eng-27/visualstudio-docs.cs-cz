# <a name="c6235"></a>C6235
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6235: ( \<non-zero constant> &#124;&#124; \<expression> ) je vždycky nenulová konstanta.  
  
 Toto upozornění znamená, že na levé straně logického operátoru nebo operace, která se vyskytuje v kontextu testu, byla zjištěna nenulová konstantní hodnota, která je jiná než jedna. Pravá strana logického nebo operace není vyhodnocena, protože výsledný výraz je vždy vyhodnocen jako true. Tento postup se označuje jako "hodnocení krátkodobého okruhu".  
  
 Nenulová konstantní hodnota, která je jiná než jedna, naznačuje, že bitový operátor AND ( `&` ) mohly být zamýšlené. Toto upozornění se negeneruje pro Common idiom, pokud je nenulová konstanta 1, protože je použita pro selektivní povolování cest kódu, ale je vygenerována, pokud je nenulová konstanta vyhodnocena jako 1, například 1 + 0.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože `INPUT_TYPE` je 2:  
  
```  
#define INPUT_TYPE 2  
void f(int n)  
{  
   if(INPUT_TYPE || n) //warning 6235 issued  
   {  
      puts("Always gets here");  
   }  
   else  
   {  
      puts("Never gets here");  
   }  
}  
```  
  
 Následující kód používá operátor bitového `&` operátoru and () pro opravu tohoto upozornění:  
  
```  
#define INPUT_TYPE 2  
void f(int n)  
{  
   if((INPUT_TYPE & n) == 2)  
   {  
      puts("bitwise-AND comparison true");  
   }  
   else  
   {  
      puts("bitwise-AND comparison false");  
   }  
}  
```
