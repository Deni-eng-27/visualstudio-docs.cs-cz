# <a name="c6200"></a>C6200
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6200: index \<name> je mimo platný rozsah indexů \<min> \<max> pro velikost vyrovnávací paměti bez zásobníku. \<variable>  
  
 Toto upozornění označuje, že posun celého čísla do určeného pole překračuje maximální hranice tohoto pole. Tato vada může způsobit náhodné chování nebo selhání.  
  
 Jednou z běžných příčin této vady je použití velikosti pole jako indexu do pole. Vzhledem k tomu, že indexování pole C/C++ je založené na nule, je maximální právní index na pole menší než počet prvků pole.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože `for` smyčka překračuje rozsah indexu:  
  
```  
  
int buff[14]; // array of 0..13 elements  
void f()  
{  
   for (int i=0; i<=14;i++) // i exceeds the index  
   {  
     buff[i]= 0; // warning C6200   
     // code...  
   }  
}  
```  
  
 Chcete-li opravit obě upozornění, použijte správnou velikost pole, jak je znázorněno v následujícím kódu:  
  
```  
int buff[14]; // array of 0..13 elements  
void f()  
{  
   for ( int i=0; i < 14; i++) // loop stops when i < 14   
   {  
     buff[i]= 0; // initialize buffer  
    // code...  
   }  
}  
```
