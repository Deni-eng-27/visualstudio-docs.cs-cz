# <a name="c28288"></a>C28288

Upozornění C28288: pro funkci, Chyba syntaxe v \_ \_ poznámce () anotace (neplatný název parametru)  
  
 Nástroj Analýza kódu oznamuje toto upozornění při `SAL_at` `__drv_at` použití anotace () a výraz parametru nelze interpretovat v aktuálním kontextu. To může zahrnovat použití nesprávně napsaného parametru nebo názvu člena nebo chybného pravopisu klíčových slov Return nebo this.
