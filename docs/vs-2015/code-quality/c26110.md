# <a name="c26110"></a>C26110
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C26110: \<lock> před voláním funkce se volající nedaří udržet zámek \<func> .  
  
 Když je vyžadován zámek, nezapomeňte vyjasnit, zda by funkce nebo její volající získal zámek. Upozornění C26110 se vydá, když dojde k porušení `_Requires_lock_held_` poznámky.  
  
## <a name="example"></a>Příklad  
 V následujícím příkladu je vygenerováno upozornění C26110, protože Poznámka `_Requires_lock_held_` na funkci `LockRequired` uvádí, že volající `LockRequired` musí získat zámek před voláním `LockRequired` . Bez této poznámky `LockRequired` musí získat zámek před přístupem ke sdíleným datům chráněným zámkem.  
  
```  
  
typedef struct _DATA   
{  
    CRITICAL_SECTION cs;  
    int d;  
} DATA;  
  
_Requires_lock_held_(p->cs)  
  
void LockRequired(DATA* p)  
{  
    p->d = 0;  
}  
  
void LockNotHeld(DATA* p)   
{   
    LockRequired(p); // Warning C26110   
}  
  
```
