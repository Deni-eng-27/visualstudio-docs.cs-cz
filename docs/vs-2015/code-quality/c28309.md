# <a name="c28309"></a>C28309
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 28309: <parameter_name>: operandy anotace musí být typy integer, Enum a Pointer. Operandy void a přetížené operátory jazyka C++ nejsou podporovány. Floaty jsou přibližné jako celá čísla. Typy: \<typelist> .  
  
 Pokusili jste se použít void nebo funkci ve výrazu poznámky a analýza kódu ji nedokáže zpracovat.  K této chybě obvykle dochází `operator==` , je-li použita implementace, která je implementována jako funkce, ale mohou nastat i jiné případy. Projděte si typy v části, kde se dozvíte, \<typelist> co je špatně.
