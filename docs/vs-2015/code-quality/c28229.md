upozornění C28229: poznámka nepodporuje parametry  
  
 Toto upozornění signalizuje, že poznámka byla použita s parametrem při Poznámka je deklarována bez parametrů. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se neočekává se zobrazí v typické použití.
