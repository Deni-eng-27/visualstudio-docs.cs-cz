# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C28220: očekáván celočíselný výraz pro okomentování parametru v poznámce  
  
 Toto upozornění signalizuje, že parametr, který se má poznámka má být výraz celého čísla a jiný typ došlo. To obvykle znamená makru nesprávně programové poznámky.
