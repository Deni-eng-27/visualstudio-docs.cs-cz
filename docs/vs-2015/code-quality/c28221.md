# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28221: pro parametr v poznámce je očekáván řetězcový výraz.  
  
 Toto upozornění znamená, že parametr pro poznámku by měl být řetězec a nějaký jiný typ byl nalezen. To obvykle označuje nesprávně kódované makro poznámky a neočekává se, že bude vidět v normálním použití.
