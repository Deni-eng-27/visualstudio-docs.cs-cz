# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C28221: Řetězcový výraz očekávaný pro parametr v poznámce  
  
 Toto upozornění signalizuje, že parametr, který se má poznámka očekává se řetězec a jiný typ došlo. To obvykle znamená makru nesprávně programové poznámky a neočekává se projeví při běžném používání.
