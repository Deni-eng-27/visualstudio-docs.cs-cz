# <a name="c28273"></a>C28273
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28273](https://docs.microsoft.com/visualstudio/code-quality/c28273).  
  
upozornění C28273: pro funkci, nejsou konzistentní s deklarací funkce  
  
 Toto upozornění označuje chybu v poznámkách, není v kódu, který je analyzován. Poznámky povolí, v definici funkce nejsou konzistentní s těmi, která povolí, v deklaraci. Dva poznámky by měla vyřešit tak, aby odpovídaly.
