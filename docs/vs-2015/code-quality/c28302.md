# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28302: pro C++ parametr reference-< parameter_name > byl na \<poznámky > nalezen další operátor `_Deref_`.  
  
 Toto upozornění se oznamuje, když se pro parametr používá další úroveň `_Deref_`.  
  
 POZNÁMKOU SAL2 nevyžaduje použití nadbytečné úrovně `_Deref_` při práci s parametry odkazu. Tato konkrétní Poznámka je nejednoznačná a je správně interpretována, ale měla by být opravena.  
  
 To se často dá opravit pouhým odebráním starší poznámky `__deref` a pomocí syntaxe POZNÁMKOU SAL2. V některých případech může být nutné použít `_At_` k odkazování na konkrétní objekt, který se má opatřit poznámkami.
