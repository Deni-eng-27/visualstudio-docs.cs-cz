# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28302: pro referenční parametr jazyka C++ <parameter_name>, `_Deref_` byl nalezen další operátor \<annotation> .  
  
 Toto upozornění je hlášeno, je-li `_Deref_` v parametru použit další úroveň.  
  
 POZNÁMKOU SAL2 nevyžaduje použití další úrovně `_Deref_` při práci s parametry odkazu. Tato konkrétní Poznámka je nejednoznačná a je správně interpretována, ale měla by být opravena.  
  
 To se často dá opravit pouhým odebráním starší `__deref` poznámky a pomocí syntaxe poznámkou SAL2. V některých případech může být nutné použít `_At_` k odkazování na konkrétní objekt, který se má opatřit poznámkami.
