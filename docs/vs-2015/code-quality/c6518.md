# <a name="c6518"></a>C6518
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6518: Neplatná Anotace: vlastnost SAL_writableTo nesmí být zadána jako předběžná podmínka u vyrovnávacích pamětí, které nejsou zapisovatelné:*param \\ *(1).  
  
 Toto upozornění znamená, že mezi `SAL_writableTo` hodnotou vlastnosti a vlastností s možností zápisu existuje konflikt. To obvykle znamená, že vlastnost s možností zápisu nemá oprávnění k zápisu do poznámení s parametrem.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože `_Out_` Poznámka zkompiluje, aby zahrnovala `SAL_writableTo` vlastnost, která nepovoluje přístup pro zápis:  
  
```cpp  
#include <sal.h>  
void f(_Out_ const char* pc)  
{  
    //code that can't write to *pc ...  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```cpp  
#include <sal.h>  
void f(_Out_  char* pc)  
{  
    pc = "Hello World";  
    //code ...  
}  
```
