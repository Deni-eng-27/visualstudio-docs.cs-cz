# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28280: pro symbol byl nalezen ' end ' bez odpovídajícího ' begin  
  
 Jazyk anotace podporuje konstrukci Begin a end (`{` a `}` v jazyce C) a párování se nerovnováhy. Tato situace se může vyhnout, pokud jsou použita makra.
