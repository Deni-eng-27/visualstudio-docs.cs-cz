# <a name="ca0505"></a>CA0505

Vystaralá vlastnost bude ignorována, protože je definována vlastnost převádějící.

Upozornění CA505 je vygenerováno, pokud vlastnosti pro analýzu kódu, které jsou zadány jako zastaralé a převádějí, obsahují hodnoty v projektu nebo konfiguračním souboru sestavení. Chcete-li tento problém vyřešit, odeberte vystaralou vlastnost.
