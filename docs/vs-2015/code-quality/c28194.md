# <a name="c28194"></a>C28194
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28194: funkce byla deklarována jako aliasing hodnoty v proměnné a byla ukončena, aniž by tak učinila  
  
 Toto upozornění znamená, že prototyp funkce pro analyzovanou funkci má `__drv_isAliased` anotaci, která označuje, že bude *alias* zadaného argumentu (tj. přiřadit hodnotu tak, aby se přestaly vracet z funkce). Funkce však nealiasuje argument podél cesty, která je označena poznámkou. Většina funkcí, které aliasují proměnnou, uloží svou hodnotu do globální struktury dat.
