# <a name="c6270"></a>C6270
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6270: chybí argument typu float pro \<function> : přidejte argument typu float odpovídající specifikátoru převodu. \<number>  
  
 Toto upozornění označuje, že není k dispozici dostatek argumentů, aby odpovídaly formátovacímu řetězci; nejméně jeden z chybějících argumentů je číslo s plovoucí desetinnou čárkou. Tato vada může vést k chybám, a to i potenciálně nesprávnému výstupu.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
  char buff [25];  
  sprintf(buff,"%s %f","pi:");  
}  
  
```  
  
 Chcete-li toto upozornění opravit, předejte chybějící argument, jak je znázorněno v následujícím kódu:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
  char buff [25];  
  sprintf(buff,"%s %f","pi:",3.1415);   
}  
```  
  
 Následující vzorový kód používá funkci bezpečného zpracování řetězce, pomocí `sprintf_s` které lze toto upozornění opravit:  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f()  
{  
  char buff [25];  
  sprintf_s( buff, 25,"%s %f", "pi:",3.1415 );  
}  
```
