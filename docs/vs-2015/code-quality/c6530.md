# <a name="c6530"></a>C6530
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 6530: nerozpoznaný styl řetězce formátu \<name>  
  
> [!NOTE]
> K tomuto upozornění dochází pouze v kódu, který používá zastaralou verzi jazyka zdrojového kódu pro poznámky SAL. Doporučujeme, abyste svůj kód používali pro použití nejnovější verze SAL. Další informace najdete v tématu [Použití poznámek SAL ke snížení vad kódu C/C++](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).  
  
 Toto upozornění znamená, že vlastnost FormatString používá jinou hodnotu než scanf nebo printf. Chcete-li toto upozornění opravit, zkontrolujte kód a použijte platnou hodnotu vlastnosti Style.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění z důvodu překlepu ve `Style` vlastnosti:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printfd")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printfd")] char *px);  
  
```  
  
 Chcete-li toto upozornění opravit, použijte platný styl, jak je znázorněno v následujícím kódu:  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printf")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printf")] char *px);   
```
