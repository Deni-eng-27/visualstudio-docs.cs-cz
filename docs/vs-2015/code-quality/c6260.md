# <a name="c6260"></a>C6260
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6260: sizeof * sizeof je téměř vždy nesprávné, chtěli jste použít počet znaků nebo počet bajtů?  
  
 Toto upozornění znamená, že výsledky dvou `sizeof` operací byly vynásobeny dohromady. Operátor C/C++ `sizeof` vrátí počet bajtů úložiště, které objekt používá. Je obvykle nesprávné ho vynásobit jinou `sizeof` operací. obvykle se zajímá o počet bajtů v objektu nebo o počet prvků v poli (například počet znaků v poli).  
  
 K operátoru je přidruženo několik neintuitivního chování `sizeof` . Například v jazyce C, `sizeof ('\0') == 4,` protože znak je integrálního typu. V jazyce C++ je typ znakového literálu `char` , tedy `sizeof ('\0') == 1` . V jazyce C i C++ však platí následující:  
  
```  
sizeof ("\0") == 2.   
```  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <windows.h>  
  
void f( )  
{  
  int i;  
  i = sizeof (L"String") * sizeof (WCHAR);  
  // code ...  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
#include <windows.h>  
  
void f( )  
{  
  int i;  
  i= sizeof (L"String") / sizeof (WCHAR);  
  
  /* or to get bytes */  
  i = sizeof (L"String");  
  // code ...  
}  
```
