# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28225: nebyl nalezen správný počet požadovaných parametrů v poznámce  
  
 Toto upozornění znamená, že se pojmenovaná Poznámka používá s nesprávným počtem parametrů. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
