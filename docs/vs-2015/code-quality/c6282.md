# <a name="c6282"></a>C6282
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C6282: nesprávný operátor: přiřazení konstanty v logickém kontextu. Místo toho zvažte použití = =.  
  
 Toto upozornění indikuje, že v kontextu testu bylo zjištěno přiřazení konstanty k proměnné. Přiřazení konstanty k proměnné v kontextu testu je téměř vždy nesprávné. Chcete- `=` li `==` vyřešit toto upozornění, nahraďte parametr pomocí nebo odeberte přiřazení z kontextu testu.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
void f( int i )  
{  
   while (i = 5)  
   {  
   // code    
   }  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
void f( int i )  
{  
   while (i == 5)  
   {  
   // code    
   }  
}  
```
