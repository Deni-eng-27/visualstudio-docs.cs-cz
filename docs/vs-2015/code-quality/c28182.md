# <a name="c28182"></a>C28182
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28182: přesměrování ukazatele s hodnotou NULL  
  
 **Další informace**: *\<pointer1>* obsahuje stejnou hodnotu null jako u *\<pointer2>**\<note>*  
  
 Nástroj Analýza kódu oznamuje toto upozornění, když potvrdí, že ukazatel může mít hodnotu NULL. Pokud existují nepotvrzené instance, u kterých může dojít k chybě dříve v trasování, nástroj pro analýzu kódu přidá do zprávy upozornění číslo řádku první instance, abyste mohli změnit kód tak, aby se vyřešily všechny instance.  
  
 *\<pointer2>* byl potvrzen jako potenciálně NULL. *\<pointer1>* obsahuje stejnou hodnotu jako *pointer2* a je na ní právě odkázat. Vzhledem k tomu, že tyto ukazatele mohou být v kódu na hodně různých místech, obě jsou hlášeny, abyste mohli určit, proč Nástroj pro analýzu kódu hlásí toto upozornění.  
  
 Pokud existuje nepotvrzená dřívější instance podmínky, *\<note>* je nahrazena tímto textem: "viz řádek *\<number>* pro předchozí umístění, kde to může nastat".  
  
## <a name="example"></a>Příklad  
 Následující příklad ukazuje kód, který může způsobit, že nástroj pro analýzu kódu vygeneruje tuto zprávu upozornění. V tomto příkladu Nástroj pro analýzu kódu určuje, že `pNodeFree` v příkazu je null `if` , a cesta kódu k textu `if` je provedena. Protože je však `nBlockSize` potenciálně nula, tělo `for` příkazu není spuštěno a `pNodeFree` zůstane beze změny. `pNodeFree` je pak přiřazen k `pNode` a používá se, `pNode` Pokud by mohlo dojít k navýšení zpětného odkazování na hodnotu null.  
  
```  
typedef struct xlist {  
    struct xlist *pNext;  
    struct xlist *pPrev;  
} list;  
  
list *pNodeFree;  
list *masterList;  
int nBlockSize;  
  
void fun()  
{  
    if (pNodeFree == 0)  
    {  
        list *pNode = masterList;  
  
        for (int i = nBlockSize-1; i >= 0; i--, pNode--)  
        {  
            pNode->pNext = pNodeFree;  
            pNodeFree = pNode;  
        }  
    }  
  
    list* pNode = pNodeFree;  
    pNode->pPrev = 0;  
}  
  
```  
  
 Nástroj Analýza kódu hlásí následující upozornění:  
  
```  
:\sample\testfile.cpp(24) : warning C28182: Dereferencing NULL pointer. 'pNode' contains the same NULL value as 'pNodeFree' did.: Lines: 12, 14, 16, 23, 24  
```  
  
## <a name="example"></a>Příklad  
 Jedním ze způsobů, jak si předchozí příklad opravit, je vrácení nuly před jeho zrušením odkazování `pNode` tak, aby byl zpětný odkaz na hodnotu null odvedený. Následující kód ukazuje tuto opravu.  
  
```  
typedef struct xlist {  
    struct xlist *pNext;  
    struct xlist *pPrev;  
} list;  
  
list *pNodeFree;  
list *masterList;  
int nBlockSize;  
  
void fun()  
{  
    if (pNodeFree == 0)  
    {  
        list *pNode = masterList;  
  
        for (int i = nBlockSize-1; i >= 0; i--, pNode--)  
        {  
            pNode->pNext = pNodeFree;  
            pNodeFree = pNode;  
        }  
    }  
  
    list* pNode = pNodeFree;  
    if(pNode != 0)  
      pNode->pPrev = 0;  
}  
  
```
