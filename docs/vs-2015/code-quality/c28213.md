# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28213: Poznámka `_Use_decl_annotations_` musí být použita pro odkazování, bez úprav, předchozí deklarace. \<proč >  
  
 Toto upozornění je oznámeno při odkazování na předchozí deklaraci bez požadované anotace `_Use_decl_annotations_`.
