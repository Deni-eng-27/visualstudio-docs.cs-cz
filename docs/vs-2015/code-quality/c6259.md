# <a name="c6259"></a>C6259
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6259: označený kód je nedosažitelný: ( \<expression>  &  \<constant> ) ve výrazu Switch-expr nemůže být vyhodnocena jako\<case-label>  
  
 Toto upozornění indikuje nedosažitelný kód způsobený výsledkem porovnání bitového operátoru AND ( `&` ) ve výrazu Switch. Příkaz Case, který odpovídá konstantě ve výrazu Switch, je dosažitelný pouze; všechny ostatní příkazy Case nejsou dosažitelné.  
  
## <a name="example"></a>Příklad  
 Následující vzorový kód vygeneruje toto upozornění, protože výraz `switch``(rand() & 3)` nemůže vyhodnotit návěští Case ( `case` `4` ):  
  
```  
#include <stdlib.h>  
  
void f()  
{  
     switch (rand () & 3) {  
        case 3:  
            /* Reachable */  
            break;  
        case 4:  
            /* Not reachable */  
            break;  
        default:  
            break;  
    }  
}  
```  
  
 Chcete-li toto upozornění opravit, odeberte nedosažitelný kód nebo ověřte, zda je konstanta použitá v příkazu case správná. Následující kód odstraní nedosažitelný příkaz Case:  
  
```  
#include <stdlib.h>  
  
void f()  
{  
     switch (rand () & 3) {  
        case 3:  
            /* Reachable */  
            break;  
        default:  
            break;  
    }  
}  
```
