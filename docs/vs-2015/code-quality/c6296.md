# <a name="c6296"></a>C6296
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C6296](https://docs.microsoft.com/visualstudio/code-quality/c6296).  
  
upozornění C6296: špatně definovaná smyčka for: tělo provede jen jednou smyčky  
  
 Toto upozornění signalizuje, že smyčky for vycházející z nemusí fungovat očekávaným způsobem. Index je bez znaménka a smyčka odčítá dolů z nula, je jeho tělo spuštěn pouze jednou.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 Chcete-li opravit toto upozornění, použijte následující kód:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
