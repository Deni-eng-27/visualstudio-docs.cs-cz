# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28223: nebyl nalezen očekávaný token/identifikátor pro anotaci, parametr  
  
 Toto upozornění znamená, že parametr pro poznámku má být identifikátor a byl zjištěn jiný symbol. To obvykle označuje nesprávně kódované makro poznámky. Tato chyba způsobí použití klíčového slova C nebo C++ na této pozici.
