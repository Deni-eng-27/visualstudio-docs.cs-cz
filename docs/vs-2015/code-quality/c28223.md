Nejnovější verzi tohoto tématu můžete najít v [C28223](https://docs.microsoft.com/visualstudio/code-quality/c28223).  
  
Upozornění C28223: Nebyl nalezen očekávaný Token/identifikátor poznámky, parametr  
  
 Toto upozornění signalizuje, že parametr, který se má poznámka očekává se identifikátor a došlo k nějaké symbol. To obvykle znamená makru nesprávně programové poznámky. Použít klíčová slova jazyka C nebo C++ na této pozici způsobí tuto chybu.
