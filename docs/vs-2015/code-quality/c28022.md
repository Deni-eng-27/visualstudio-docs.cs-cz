# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28022: třídy funkcí \<classlist1> této funkce se neshodují s třídami Functions \<classlist2> na definici TypeDef použitou k jejímu definování.  
  
 Toto upozornění se oznamuje v případě, že poznámky obsahují chybu. Definice typedef i funkce mají tyto `_Function_class_` poznámky, ale neodpovídají. Pokud se obě použijí, musí se shodovat.
