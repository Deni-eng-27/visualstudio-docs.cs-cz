# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227).  
  
Upozornění C28227: Anotace nemůže být také PrimOp (viz předchozí deklarace)  
  
 Toto upozornění signalizuje, že je deklarované jako běžný Poznámka Poznámka s názvem a také se dřív deklaroval jako PrimOp. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se neočekává se zobrazí v typické použití.
