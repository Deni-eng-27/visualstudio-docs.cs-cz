# <a name="c28287"></a>C28287

Nejnovější verzi tohoto tématu můžete najít v [C28287](https://docs.microsoft.com/visualstudio/code-quality/c28287).  
  
upozornění C28287: pro funkci. Chyba syntaxe v poznámce _At\_poznámky () (nerozeznaný název parametru)  
  
 Nástroj pro analýzu kódu sestavy toto upozornění, když `SAL_at` (`__drv_at`) se používá poznámky a výraz parametru se nedá interpretovat v aktuálním kontextu. To může zahrnovat pomocí chybně parametr nebo název člena nebo chybně "return" nebo "Tento" klíčová slova.
