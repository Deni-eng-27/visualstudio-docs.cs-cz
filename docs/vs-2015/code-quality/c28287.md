# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28287: funkce, Chyba syntaxe v \_v poznámce\_() (nerozpoznaný název parametru)  
  
 Nástroj Analýza kódu oznamuje toto upozornění při použití anotace `SAL_at` (`__drv_at`) a výraz parametru nelze interpretovat v aktuálním kontextu. To může zahrnovat použití nesprávně napsaného parametru nebo názvu člena nebo chybného pravopisu klíčových slov Return nebo this.
