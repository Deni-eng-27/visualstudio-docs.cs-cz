# <a name="c6289"></a>C6289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C6289: nesprávný operátor: vzájemné vyloučení přes &#124;&#124; je vždy nenulová konstanta. Nechtěli jste místo toho použít && ?  
  
 Toto upozornění znamená, že v testovacím výrazu je proměnná testována proti dvěma různým konstantám a výsledek závisí na obou podmínkách true. Tato hodnota je vždy vyhodnocena jako true.  
  
 K tomuto problému obvykle dochází `||` v případě, že se používá místo `&&` , ale může to být způsobeno také použitím `!=` `==` zamýšleného umístění.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
void f(int x)  
{  
  if ((x != 1) || (x != 3))  
  {  
    // code   
  }  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
void f(int x)  
{  
  if ((x != 1) && (x != 3))  
  {  
    // code   
  }  
}  
  
/* or */  
void f(int x)  
{  
  if ((x == 1) || (x == 3))  
  {  
    // code   
  }  
}  
```
