# <a name="c6329"></a>C6329
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6329: návratová hodnota pro volání, která \<function> by neměla být porovnána s \<number>  
  
 Program porovnává číslo s návratovou hodnotou z volání `CreateFile` . Pokud je `CreateFile` to úspěšné, vrátí otevřený popisovač objektu. Pokud dojde k chybě, vrátí se `INVALID_HANDLE_VALUE` .  
  
## <a name="example"></a>Příklad  
 Tento kód může způsobit upozornění:  
  
```cpp  
if (CreateFile() == NULL)  
    {  
        return;  
    }  
```  
  
## <a name="example"></a>Příklad  
 Tento kód opravuje chybu:  
  
```cpp  
if (CreateFile() == INVALID_HANDLE_VALUE)  
    {  
        return;  
    }  
```
