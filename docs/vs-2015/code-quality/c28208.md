# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28208: funkce \<function> byla dříve definována s jiným seznamem parametrů v \<file> ( \<line> ). Některé analytické nástroje budou poskytovat nesprávné výsledky.  
  
 Toto upozornění je hlášeno, pokud se známá definice funkce neshoduje s jiným výskytem.
