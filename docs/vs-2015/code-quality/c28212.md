# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C28212: očekávaný výraz ukazatele pro poznámku  
  
 Toto upozornění signalizuje, že parametr číslované poznámku (ne funkce poznámkou) očekává se typ pole nebo ukazatel, ale došlo k nějakým typem. Poznámka je třeba opravit.
