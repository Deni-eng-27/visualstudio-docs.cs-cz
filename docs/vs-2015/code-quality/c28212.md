# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28212: očekával se výraz ukazatele pro poznámku.  
  
 Toto upozornění znamená, že číslovaný parametr poznámky (nikoli funkce, která je označena jako anotace) má být ukazatel nebo typ pole, ale byl zjištěn nějaký jiný typ. Poznámka musí být opravena.
