# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6506: Neplatná Anotace: \<name> vlastnost může být použita pouze na hodnoty typu ukazatel nebo pole.  
  
 Toto upozornění znamená, že vlastnost je použita pro jiný typ než typ ukazatele nebo pole. Pro všechny typy dat lze použít vlastnost Access, v ní a platné vlastnosti. Další vlastnosti, jako jsou ValidBytesConst, ValidElementsConst, ElementSize a NullTerminted, ukazatel na členy nebo typy polí. Úplný seznam vlastností a podporovaných datových typů najdete v tématu [vlastnosti poznámek](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte ukazatel nebo typ pole, jak je znázorněno v následujícím ukázkovém kódu:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
