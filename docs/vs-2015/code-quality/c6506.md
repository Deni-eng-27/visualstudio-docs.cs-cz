# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C6506: Neplatná anotace: \<name > vlastnost může být použita pouze u hodnot typu ukazatel nebo pole  
  
 Toto upozornění signalizuje, že vlastnost se používá u typu než ukazatel nebo pole. Přístup, Tainted a platné vlastnosti lze použít u všech datových typů. Další vlastnosti, jako je například validbytesconst by neměly, ValidElementsConst, ElementSize a NullTerminted podporují ukazatel, ukazatel na členy a typy, pole. Úplný seznam vlastností a podporované datové typy naleznete v tématu [vlastnosti poznámky](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Chcete-li opravit toto upozornění, použijte ukazatel nebo pole, jak je znázorněno v následujícím ukázkovém kódu:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
