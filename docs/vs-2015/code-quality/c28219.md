# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28219: pro anotaci parametru v poznámce je očekáván člen výčtu.  
  
 Očekává se, že parametr poznámky bude členem pojmenovaného `enum` typu a byl zjištěn nějaký jiný symbol; použijte člena tohoto `enum` typu. To obvykle označuje nesprávně kódované makro poznámky.
