# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28214](https://docs.microsoft.com/visualstudio/code-quality/c28214).  
  
upozornění C28214: názvy atributových parametrů musí být p1... p9  
  
 Toto upozornění signalizuje, že při konstrukci deklaraci anotace názvy parametrů jsou omezeny na p1... p9. Toto upozornění by nikdy neměly vzniknout při běžném používání.
