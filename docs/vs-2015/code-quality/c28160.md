# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C28160: chybová anotace  
  
 Toto upozornění je hlášeno při `__drv_error` anotace byla nalezena. Tato poznámka se používá k nastavení příznaku drželi osvědčených kódovacích postupů, které by měl být stanovena a je možné s `__drv_when` anotace označuje konkrétní kombinace parametrů.
