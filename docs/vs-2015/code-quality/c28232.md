# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28232: \_pre\_, \_post\_nebo \_DEREF\_ neplatí pro žádnou poznámku.  
  
 Toto upozornění znamená, že se operátor `_Pre_`, `_Post_`nebo `_Deref_` zobrazí ve výrazu poznámky bez následné poznámky k funkci. Modifikátor byl ignorován, ale označuje nesprávně kódované poznámky.
