# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28232: \_ před \_ , \_ po \_ nebo \_ za DEREF se \_ nepoužije žádná Poznámka  
  
 Toto upozornění označuje, že `_Pre_` `_Post_` operátor, nebo `_Deref_` se zobrazí ve výrazu poznámky bez následné funkční poznámky; modifikátor byl ignorován, ale označuje nesprávně kódované poznámky.
