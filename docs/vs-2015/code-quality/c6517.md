# <a name="c6517"></a>C6517
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6517: Neplatná Anotace: vlastnost SAL_readableTo nesmí být zadána u vyrovnávacích pamětí, které nejsou čitelné:*param \\ *(1).  
  
> [!NOTE]
> K tomuto upozornění dochází pouze v kódu, který používá zastaralou verzi jazyka zdrojového kódu pro poznámky SAL. Doporučujeme, abyste svůj kód používali pro použití nejnovější verze SAL. Další informace najdete v tématu [Použití poznámek SAL ke snížení vad kódu C/C++](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).  
  
 Toto upozornění znamená, že `SAL_readableTo` vlastnost nemá požadovaný přístup pro čtení. Tuto vlastnost nelze použít k zadání poznámky k parametru bez poskytnutí přístupu pro čtení.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože ve vyrovnávací paměti není uděleno oprávnění ke čtení:  
  
```cpp  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([Pre( ValidBytesConst=10 )][Pre( Deref=1, Access=Write )] char* buffer );  
  
```  
  
 Chcete-li toto upozornění opravit, udělte přístup pro čtení, jak je znázorněno v následujícím kódu:  
  
```cpp  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([Pre( ValidBytesConst=10 )][Pre( Deref=1, Access=Read)] char* buffer );  
```
