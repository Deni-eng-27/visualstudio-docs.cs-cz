# <a name="ca0501"></a>CA0501

Nelze načíst výstup sestavy.

K tomu dochází, pokud výstupní sestavu nelze zapsat z některého z následujících důvodů:

- Plný disk

- Žádná oprávnění k zápisu do adresáře

- Adresář neexistuje.
