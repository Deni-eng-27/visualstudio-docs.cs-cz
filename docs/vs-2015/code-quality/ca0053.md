# <a name="ca0053"></a>CA0053

Pravidlo sestavení nelze načíst.

To znamená, že pravidlo sestavení nelze načíst pomocí reflexe. Tato situace může nastat, pokud má uživatel vlastní pravidla, která nebyla správně sestavena, nebo nástroj pro analýzu nelze načíst sestavení z jiných důvodů, například přístupu k systému souborů byl odepřen.
