# <a name="c28163"></a>C28163
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28163: funkce by nikdy neměla být volána v rámci bloku try/except  
  
 Toto upozornění je hlášeno, je-li funkce typu, který by nikdy neměl být uzavřen v bloku, který `try/except`  je nalezen v `try/except` bloku. Nástroj Analýza kódu nalezl alespoň jednu cestu, ve které byla funkce volána v rámci `try/except` bloku.
