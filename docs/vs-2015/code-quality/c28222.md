# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění 28222: _ano\_, _Ne\_, nebo _Maybe\_ očekáván pro anotaci  
  
 Toto upozornění signalizuje, že se očekává parametr, který se má poznámka se symboly `_Yes_`, `_No_`, nebo `_Maybe_`, a některé další symbol došlo. To obvykle znamená makru nesprávně programové poznámky.
