# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění 28222: \_ \_ \_ \_ \_ \_ pro anotaci se očekává Ano, ne nebo možná.  
  
 Toto upozornění signalizuje, že parametr pro anotaci má být jeden ze symbolů `_Yes_` , `_No_` , nebo `_Maybe_` a byl zjištěn nějaký jiný symbol. To obvykle označuje nesprávně kódované makro poznámky.
