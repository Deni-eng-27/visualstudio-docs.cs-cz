# <a name="getting-started-with-ptvs-setting-up-visual-studio"></a>Začínáme s PTVS: Nastavení Visual Studia

[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Instalace PTVS a související knihovny je rychlá, pokud máte Visual Studio. Pokud nemáte sadu Visual Studio, můžete získat bezplatnou kopii profesionální verze kvality.

Tyto pokyny můžete sledovat ve velmi krátkém [videu YouTube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).

Postup vysoké úrovně je následující: nainstalujte Visual Studio, nainstalujte PTVS, nainstalujte Python a datové knihovny (Anaconda, zápoje) a pak nakonec Zkontrolujte instalaci.

První věc, kterou budete potřebovat, je Visual Studio. Pokud jste Open Source nebo samostatný vývojář, můžete použít Visual Studio [Community Edition](https://www.visualstudio.com/products/visual-studio-community-vs), který je zdarma a funkční pro odborníky. Můžete také použít edici Community v organizaci, pokud je určena pro vzdělávání učební, vědecký výzkum nebo přispívání na open source projekty. Studenti a startupy by se měli podívat na programy DreamSpark a BizSpark, abyste viděli, jestli máte nárok na bezplatný přístup, nebo se zeptejte zaměstnavatele na to, jestli máte předplatné MSDN.

Po instalaci sady Visual Studio budete muset [nainstalovat PTVS](https://archive.codeplex.com/?p=pytools). Toto je bezplatné samostatné rozšíření, které je plně podporované Microsoftem a které je vyvíjené otevřenými příspěvky z komunity.

Teď je potřeba [nainstalovat Python](https://www.python.org/download/). Python je udržován komunitou a jeho Domovská stránka je python.org. Pomocí analýzy Continuum získáte bezplatnou sadu s názvem Anaconda, která má Python, a spoustu užitečných knihoven (zejména pro vědy a zpracování dat) a Enthought vytvoří podobnou sadu s názvem zápoje. Stačí nainstalovat jenom jeden z těchto produktů. Pokud si nejste jisti, který z nich máte, začněte s [Anaconda](https://www.anaconda.com/products/individual), který vám poskytne nejaktuálnější verzi Pythonu a většinu obtížně instalovaných balíčků.

Spusťte Visual Studio a ujistěte se, že všechno funguje. V nabídce Zobrazit klikněte na položku ostatní okna. Zobrazí se položka s názvem prostředí Pythonu. V tomto okně se zobrazí všechny zjištěné Instalace Pythonu PTVS a všechny balíčky, které jste nainstalovali. Okno také řídí aktualizaci databáze pro zobrazení dokončení při úpravách kódu. Tento proces aktualizace nějakou dobu trvá, ale po dokončení PTVS může zobrazit užitečnější informace o balíčcích.

Pokud chcete používat IPython s PTVS, postupujte podle těchto [pokynů](https://archive.codeplex.com/?p=pytools).

Tyto pokyny můžete sledovat v krátkém [videu YouTube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).
