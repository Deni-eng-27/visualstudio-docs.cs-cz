# <a name="releasing-a-visual-studio-integration-product"></a>Vydání produktu pro integraci sady Visual Studio
Témata v této části ukazují, jak uvolnit integrační produkt sady Visual Studio sestavením a instalací VSPackage a pomocí testu integrace sady Visual Studio (VSIT).

## <a name="in-this-section"></a>V tomto oddílu
 [Instalace VSPackage](../misc/installing-vspackages.md) Popisuje, jak sestavovat a instalovat sady VSPackage a jak podporovat uživatele, kteří používají více verzí sady Visual Studio ve stejnou dobu.

 [Nástroj RegPkg](../extensibility/internals/regpkg-utility.md) Popisuje Nástroj RegPkg, který se používá k registraci VSPackage a příprava na nasazení.

 [Nastavení a uživatelská příručka k verzi Premier Partner Edition](https://msdn.microsoft.com/8ee4dad7-95d3-4f2d-a8d4-3ba9a80ecae2) Poskytuje partnerům VSIP s Distribuovatelný verzí integrovaného vývojového prostředí (IDE) sady Visual Studio, aby mohli přidat své vlastní vývojové nástroje a jazyky počítačů.
