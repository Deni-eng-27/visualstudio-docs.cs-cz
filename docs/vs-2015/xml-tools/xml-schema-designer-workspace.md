# <a name="xml-schema-designer-workspace"></a>Pracovní prostor Návrháře schémat XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Návrhář schématu XML (Návrhář XSD) je grafický nástroj, který vám pomůže prozkoumat schémata XML. Kromě [Průzkumníka schématu XML](../xml-tools/xml-schema-explorer.md), který umožňuje procházet a procházet strom schématu XML a provádět hledání, Návrhář XSD poskytuje tři zobrazení, která vám umožňují prozkoumat schéma XSD podrobněji. Zobrazení Start je bod spuštění pro návrháře XSD; v zobrazení Start můžete přejít k dalším zobrazením návrháře XSD a zobrazit podrobnosti o sadě schémat. Zobrazení grafu umožňuje zobrazit přehled sady schémat a vztahů mezi uzly schématu. Zobrazení modelu obsahu poskytuje grafické znázornění podrobností místních a globálních uzlů schématu, včetně jednoduchých a složitých typů, prvků, skupin, atributů a skupin atributů.

 Pokud chcete začít zkoumat uzly, které vás zajímají, musíte je přidat do pracovního prostoru. Pracovní prostor se sdílí mezi všemi zobrazeními.

## <a name="adding-nodes-to-the-workspace"></a>Přidání uzlů do pracovního prostoru
 Do pracovního prostoru můžete přidat uzly následujícími způsoby:

- V části Podrobnosti o sadě schémat v [zobrazení Start](../xml-tools/start-view.md)klikněte na odkaz **Přidat** vedle typu globální uzel.

- Přetáhněte globální uzly, uzly souborů a uzly oboru názvů z Průzkumníka schémat XML na libovolné tři zobrazení. Další informace naleznete v části "přetahování uzlů" v tématu [Průzkumník schémat XML](../xml-tools/xml-schema-explorer.md).

- Použijte kontextovou nabídku v Průzkumníku schémat XML. Další informace najdete v tématu [kontextové nabídky](../xml-tools/context-menus-xml-schema-explorer.md).

- Proveďte hledání v Průzkumníku XSD a klikněte na tlačítko **Přidat zvýrazněné uzly do pracovního prostoru** v podokně souhrnné výsledky. Další informace najdete v tématu [hledání sady schémat](../xml-tools/searching-the-schema-set.md).

## <a name="view-switching"></a>Přepínání zobrazení
 Chcete-li přepnout zobrazení, použijte jednu z následujících možností:

- Panel nástrojů Návrhář XSD

- Kontextové nabídky zobrazení modelu obsahu a zobrazení grafu

- Meze na stránce zobrazení Start nebo na meze v zobrazení modelu prázdného obsahu nebo zobrazení grafu.

- Klávesové zkratky: CTRL + 1 pro zobrazení Start, CTRL + 2 pro zobrazení grafu a CTRL + 3 pro zobrazení modelu obsahu.
