# <a name="msbuild-concepts"></a>Koncepty nástroje MSBuild
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

[!INCLUDE[vstecmsbuild](../includes/vstecmsbuild-md.md)] poskytuje základní schéma XML, které lze použít k určení, jak platforma sestavení sestaví software. Chcete-li určit komponenty v sestavení a jak mají být sestaveny, použijte tyto čtyři části nástroje MSBuild: vlastnosti, položky, úlohy a cíle.
