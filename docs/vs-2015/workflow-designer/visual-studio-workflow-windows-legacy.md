# <a name="visual-studio-workflow-windows-legacy"></a>Okna pracovních postupů v sadě Visual Studio (starší verze)
Starší verze [!INCLUDE[wfd2](../includes/wfd2-md.md)] poskytované pomocí [!INCLUDE[vs2010](../includes/vs2010-md.md)] lze použít k cílení na [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] nebo [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]. Starší verze návrháře je vizuální Návrhář, který můžete použít k vytvoření pracovního postupu. Aktivity můžete přetáhnout ze **sady nástrojů** na návrhovou plochu a pak je nakonfigurovat. Jako okno editoru [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] zabírá hlavní oblast okna, kterou používají jiné okna editoru [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]. Název pracovního postupu se zobrazí v záhlaví okna [!INCLUDE[vsprvs](../includes/vsprvs-md.md)].

 Většina starších oken návrháře by měla být obeznámena s zkušeným [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]m uživatelem. V následující tabulce jsou popsány primární okna.

|Okno|Popis|
|------------|-----------------|
|Starší verze pracovního postupu Návrhová plocha|Plocha vizuálního návrháře, která se používá k vytvoření pracovního postupu, který cílí na [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] nebo [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]. Umožňuje sestavovat aktivity vizuálně přetažením návrhářů aktivit ze **sady nástrojů** na návrhovou plochu.|
|Sada nástrojů|Poskytuje grafické reprezentace aktivit, které lze přetáhnout na návrhovou plochu.|
|Vlastnosti|Slouží ke konfiguraci vlastností aktivity pracovního postupu.|
|Průzkumník řešení|Zobrazí projekty pracovního postupu v hierarchickém stromovém zobrazení a umožňuje přístup k funkcím, které usnadňují správu projektů.|
|Ladit okna|Standardní okna ladění sady Visual Studio, jako jsou například **vlákna**, **zásobník volání**, **zarážka**, **místní nástroj**a **sledovací okna nástrojů** .|
