# <a name="state-machine-activity-designers"></a>Návrháři aktivit stavového stroje
[!INCLUDE[wfd1](../includes/wfd1-md.md)] obsahuje řadu návrhářů aktivity, které můžete použít při vytváření pracovních postupů. Tato část obsahuje návrháře aktivit poskytovaných systémem, který slouží k vytváření a správě pracovních postupů stavových počítačů. Následující témata popisují tyto návrháře a poskytují pokyny k jejich použití.

## <a name="in-this-section"></a>V tomto oddílu
 Stavový [stroj](../workflow-designer/statemachine-activity-designer.md) Obsahuje kolekci stavů a pracovních postupů modelů, které používají známé stavové paradigma počítače.

 [Stav](../workflow-designer/state-activity-designer.md) Představuje stav, ve kterém může být Stavový počítač v.

 [FinalState](../workflow-designer/finalstate-activity-designer.md) Představuje stav, který ukončuje instanci stavového počítače.

 [Přechod](../workflow-designer/transition-activity-designer.md) Představuje přechod mezi dvěma stavy.
