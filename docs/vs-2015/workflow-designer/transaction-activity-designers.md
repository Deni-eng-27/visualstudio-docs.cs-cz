# <a name="transaction-activity-designers"></a>Návrháři aktivit transakcí

## <a name="in-this-section"></a>V tomto oddílu

 [Aktivita CompensableActivity](../workflow-designer/compensableactivity-activity-designer.md) Slouží k vytvoření a konfiguraci aktivity <xref:System.Activities.Statements.CompensableActivity>.

 [Kompenzovat](../workflow-designer/compensate-activity-designer.md) Slouží k vytvoření a konfiguraci aktivity <xref:System.Activities.Statements.Compensate>.

 [Potvrdit](../workflow-designer/confirm-activity-designer.md) Slouží k vytvoření a konfiguraci aktivity <xref:System.Activities.Statements.Confirm>.

 [Objekt TransactionScope](../workflow-designer/transactionscope-activity-designer.md) Slouží k vytvoření a konfiguraci aktivity <xref:System.Activities.Statements.TransactionScope>.
