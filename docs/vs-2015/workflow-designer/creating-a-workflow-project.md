# <a name="creating-a-workflow-project"></a>Vytvoření projektu pracovního postupu
Pracovní postupy, [!INCLUDE[indigo1](../includes/indigo1-md.md)] služby pracovních postupů, vlastní aktivity a návrháře vlastních aktivit lze vytvořit pomocí [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] šablon projektu. Tato část popisuje, jak vytvořit knihovny a aplikace se šablonami projektů dostupnými v nástroji [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] .

 První čtyři témata níže popisují, jak používat šablony pracovního postupu, které se nacházejí v okně **Nový projekt** . K těmto šablonám lze přistupovat otevřením [!INCLUDE[vs2010](../includes/vs2010-md.md)] a výběrem **souboru**  ->  **Nový**  ->  **projekt...**, potom kategorie **pracovní postup** z části **Nainstalované šablony** na levé straně pod uzlem **Visual Basic** nebo **Visual C#** .

 Poslední téma popisuje, jak přidat aktivity pracovního postupu, návrháře a služby nebo jiné [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] položky do existujícího projektu pracovního postupu (nebo jiného).
