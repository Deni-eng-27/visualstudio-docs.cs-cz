# <a name="creating-a-workflow-project"></a>Vytvoření projektu pracovního postupu
Pracovní postupy, [!INCLUDE[indigo1](../includes/indigo1-md.md)] služby pracovního postupu, vlastní aktivity a návrháře vlastních aktivit lze vytvořit pomocí [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] šablon projektů. Tato část popisuje, jak vytvořit knihovny a aplikace pomocí šablon projektů, které jsou k dispozici v [!INCLUDE[vsprvs](../includes/vsprvs-md.md)].

 První čtyři témata níže popisují, jak používat šablony pracovního postupu, které se nacházejí v okně **Nový projekt** . K těmto šablonám je možné přistupovat otevřením [!INCLUDE[vs2010](../includes/vs2010-md.md)] a výběrem **souboru**  -> **Nový**  -> **projekt...** , pak kategorii **pracovního postupu** z části **Nainstalované šablony** vlevo v části **vizuál** Uzel Basic **nebo C# Visual**

 Poslední téma popisuje, jak přidat aktivity pracovního postupu, návrháře a služby nebo jiné [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] položky do existujícího projektu pracovního postupu (nebo jiného).
