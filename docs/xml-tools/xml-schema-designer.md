# <a name="xml-schema-designer-overview"></a>Přehled Návrhář schémat XML

Návrhář schématu XML (XSD Designer) je grafický nástroj, který vám pak umožní vizualizovat schéma, nastavte na různých úrovních, abstrakce. Tyto hlavní návrháře XSD součásti:

- [Průzkumník schémat XML](../xml-tools/xml-schema-explorer.md), což vám umožní procházet a procházet stromu XML schéma a provádět vyhledávání.

- [Zobrazení Start](../xml-tools/start-view.md), což je výchozím bodem pro návrháře XSD. Ze zobrazení spuštění můžete přejít na další zobrazení návrháře XSD a prohlédněte si podrobnosti o sadě schémat.

- [Zobrazení grafu](../xml-tools/graph-view.md), což vám umožní zobrazit základní informace o sadě schémat a vztahy mezi uzly schématu.

- [Zobrazení modelu obsahu](../xml-tools/content-model-view.md), která poskytuje grafické znázornění podrobnosti uzlů místní a globální schémat, včetně jednoduché a komplexní typy prvků, skupiny, atributy a skupiny atributů.
