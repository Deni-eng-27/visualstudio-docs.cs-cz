# <a name="error-unable-to-access-the-sql-server-debugging-interface"></a>Chyba: Nepodařilo se získat přístup k ladicímu prostředí SQL serveru.
Tuto zprávu nastane při pokusu o ladění SQL na počítači, kde není povolené ladění SQL.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zobrazit [nastavení ladění SQL](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2010/s4sszxst(v=vs.100)).
