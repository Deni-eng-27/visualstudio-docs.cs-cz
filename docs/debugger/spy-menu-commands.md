# <a name="spy-menu-commands"></a>Příkazy nabídky nástroje Spy
**Spy** nabídka obsahuje příkazy pro práci s nástroji Spy ++ Windows a zobrazení.

 **Windows** otevře nové zobrazení Windows.

 **Procesy** otevře nové zobrazení procesy.

 **Vlákna** otevře nové zobrazení vlákna.

 **Protokolování zpráv** otevře nové zobrazení zpráv a otevře [dialogové okno možností zpráv](../debugger/message-options-dialog-box.md) pro zobrazení.

 **Najít okno** zobrazí **FindWindow** dialogové okno Vyberte okno pomocí myši.

 **Ukončení** nástroje Spy ++ se zavře.
