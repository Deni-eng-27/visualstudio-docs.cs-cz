# <a name="spy-views"></a>Zobrazení nástroje Spy++
Nástroje Spy ++ zobrazuje zobrazení stromu systému windows, zprávy, procesy a vláken, které jsou k dispozici na vašem počítači.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Zobrazení oken](../debugger/windows-view.md)  
 Zobrazení stromu dostupné windows a ovládacích prvků.  
  
 [Zobrazení zpráv](../debugger/messages-view.md)  
 Zobrazí datový proud zpráv, které jsou přidružené k okno, proces nebo přístup z více vláken.  
  
 [Zobrazení procesů](../debugger/processes-view.md)  
 Zobrazení stromu active procesů.  
  
 [Zobrazení vláken](../debugger/threads-view.md)  
 Zobrazí stromové zobrazení aktivních podprocesů.
