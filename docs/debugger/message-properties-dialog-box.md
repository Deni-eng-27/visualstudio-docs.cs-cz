# <a name="message-properties-dialog-box"></a>Dialogové okno vlastností zpráv
Použijte toto dialogové okno pro další informace o konkrétní zprávu. Zobrazíte dialogovému oknu přesunout fokus [zobrazení zpráv](../debugger/messages-view.md) okna. Vyberte všechny zprávy uzel ve stromu a pak zvolte **vlastnosti** z **zobrazení** nabídky.

 **Obecné** je karta jenom karta zobrazí. K dispozici jsou následující nastavení:

 **Popisovač okna** jedinečné ID tohoto okna. Jsou opakovaně využívána popisovač okna; okno identifikují pouze po dobu platnosti tohoto okna. Klikněte na tuto hodnotu a zobrazte vlastnosti tohoto okna.

 **Úroveň vnoření** hloubka vnoření tuto zprávu, kde 0 je bez vnoření.

 **Zpráva** číslo, stav a název zprávy vybrané windows.

 **lResult** hodnotu *lResult* parametr, pokud existuje.

 **wParam** hodnotu *wParam* parametr, pokud existuje.

 **lParam** hodnotu *lParam* parametr, pokud existuje. Tato hodnota je dekódovat. Pokud je ukazatel na řetězec nebo strukturu.
