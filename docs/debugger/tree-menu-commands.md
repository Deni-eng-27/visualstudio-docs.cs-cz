# <a name="tree-menu-commands"></a>Příkazy nabídky stromu
**Stromu** nabídka obsahuje příkazy, které rozbalit nebo sbalit zobrazení.  
  
 **Rozbalit jednu úroveň**  
 Rozbalí aktuálně vybrané položky v zobrazení systému tím, že zobrazuje jednu úroveň pod ním.  
  
 **Rozbalit větev**  
 Rozbalí všechny úrovně pod aktuálně vybrané položky.  
  
 **Rozbalit vše**  
 Rozbalí a zobrazí všechny úrovně v zobrazení systému.  
  
 **Sbalit**  
 Sbalí všechny úrovně pod aktuálně vybrané položky.
