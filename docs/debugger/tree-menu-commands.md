# <a name="tree-menu-commands"></a>Příkazy nabídky stromu
**Stromu** nabídka obsahuje příkazy, které rozbalit nebo sbalit zobrazení.

 **Rozbalit jednu úroveň** rozbalí aktuálně vybrané položky v zobrazení systému tím, že zobrazuje jednu úroveň pod ním.

 **Rozbalit větev** rozbalí všechny úrovně pod aktuálně vybrané položky.

 **Rozbalit vše** Expands a zobrazí všechny úrovně v zobrazení systému.

 **Sbalit** sbalí všechny úrovně pod aktuálně vybrané položky.
