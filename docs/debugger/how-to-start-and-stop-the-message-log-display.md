# <a name="how-to-start-and-stop-the-message-log-display"></a>Postupy: Spuštění a zastavení displeje protokolu zpráv
Když [zobrazení zpráv](../debugger/messages-view.md) okno je aktivní, **spustit protokolování** nebo **zastavení protokolování** možnost se zobrazí na nástroje Spy ++ **zprávy** nabídce a odpovídající **spustit protokolování** nebo **zastavení protokolování** stane aktivní tlačítko panelu nástrojů.  
  
### <a name="to-start-the-message-log-display"></a>Spuštění zobrazení protokolu zpráv  
  
1.  Z **zprávy** nabídce zvolte **spustit protokolování**.  
  
### <a name="to-stop-the-message-log-display"></a>K zastavení displeje protokolu zpráv  
  
1.  Z **zprávy** nabídce zvolte **zastavení protokolování**.
