# <a name="select-code-type-dialog-box"></a>Dialogové okno Vybrat typ kódu
Dialogové okno otevřít, otevřete **připojit k procesu** dialogové okno a potom klikněte na **vyberte** tlačítko.  
  
 **Automaticky určit typ kódu k ladění**  
 Příslušná ladicího programu bude vybrána v závislosti na charakteru kód, který běží.  
  
 **Ladění tyto typy kódu:**  
 V seznamu vyberte typ kód, který chcete ladit.
