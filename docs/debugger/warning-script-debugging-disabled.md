# <a name="warning-script-debugging-disabled"></a>Upozornění: Ladění skriptů zakázáno
V aplikaci Internet Explorer je aktuálně zakázáno ladění skriptu  
  
 Toto upozornění se zobrazí při pokusu o ladění skriptu bez povolení ladění skriptů v aplikaci Internet Explorer. Z bezpečnostních důvodů aplikace Internet Explorer zakáže ladění skriptů ve výchozím nastavení.  
  
### <a name="to-enable-script-debugging-in-internet-explorer"></a>Chcete-li povolit ladění skriptů v aplikaci Internet Explorer  
  
1.  V Internet Exploreru **nástroje** nabídce zvolte **Možnosti Internetu**.  
  
2.  V **Možnosti Internetu** dialogové okno, klikněte **Upřesnit** kartě.  
  
3.  Na **Upřesnit** kartě, podívejte se **nastavení** pole **procházení** kategorie.  
  
4.  Zrušte **zakázat ladění skriptů (aplikace Internet Explorer)**.  
  
5.  Click **OK**.  
  
6.  Ukončete a restartujte aplikaci Internet Explorer.  
  
     Nové nastavení teď bude platit.
