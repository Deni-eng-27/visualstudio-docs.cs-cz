# <a name="idiasymbolgetupperboundid"></a>IDiaSymbol::get_upperBoundId
Načte identifikátor symbol horní mez dimenzi FORTRAN pole.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
HRESULT get_upperBoundId (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pRetVal`  
 [out] Vrátí ID symbol, který představuje horní mez dimenzi FORTRAN pole.  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí `S_OK`, jinak vrátí `S_FALSE` nebo chybový kód.  
  
> [!NOTE]
>  Vrácená hodnota `S_FALSE` znamená vlastnost není k dispozici pro symbol.  
  
## <a name="remarks"></a>Poznámky  
 Identifikátor je jedinečný hodnota vytvořené DIA SDK k označení všechny symboly jako jedinečný.
