# <a name="idiasymbolget_offset"></a>IDiaSymbol::get_offset
Načte posun umístění symbolu. Používá se, když je [výčet locationtype –](../../debugger/debug-interface-access/locationtype.md) `LocIsRegRel` nebo `LocIsBitField`.

## <a name="syntax"></a>Syntaxe

```C++
HRESULT get_offset ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>Parametry
 `pRetVal`

mimo Vrátí posun v bajtech umístění symbolu.

## <a name="return-value"></a>Návratová hodnota
 V případě úspěchu vrátí `S_OK`; v opačném případě vrátí `S_FALSE` nebo kód chyby.

> [!NOTE]
> Návratová hodnota `S_FALSE` znamená, že vlastnost není k dispozici pro symbol.

## <a name="remarks"></a>Poznámky
 Posun je z dříve stanoveného známého bodu. Například posun pro `LocIsBitField` typ umístění je obvykle od začátku obsahující třídy.
