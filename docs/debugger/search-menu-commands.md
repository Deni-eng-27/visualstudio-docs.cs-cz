# <a name="search-menu-commands"></a>Příkazy nabídky vyhledávání
**Vyhledávání** nabídka poskytuje příkazy pro vyhledání konkrétní windows.  
  
 **Najít nebo proces/vlákno/zpráv oken**  
 Otevře se **okno**/**proces**/**vláken**/**MessageSearch** dialogové okno, kde lze použít **FinderTool** nebo zadejte kritéria hledání.  
  
 **Najít další**  
 Vyhledá aktuální zobrazení pro další odpovídající okno, proces, vlákno nebo zprávy. Tuto položku nabídky (a související panelu nástrojů tlačítko) je k dispozici jenom v případě, že je platný vyhledávací výsledek, který není jedinečný.  
  
 **Najít předchozí**  
 Vyhledá aktuální zobrazení předchozího odpovídající okna, proces, vlákno nebo zprávy. Tuto položku nabídky (a související panelu nástrojů tlačítko) je k dispozici jenom v případě, že je platný vyhledávací výsledek, který není jedinečný.
