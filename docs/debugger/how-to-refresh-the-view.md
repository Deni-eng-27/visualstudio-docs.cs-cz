# <a name="how-to-refresh-the-view"></a>Postupy: Aktualizace zobrazení
Funkce Spy + + pořizuje "snímek" systémových tabulek a aktualizuje zobrazení na základě těchto informací. Je důležité pravidelně aktualizovat systémová zobrazení. Pokud máte otevřené zobrazení Spy + + a neaktualizovali jste zobrazení, neuvidíte tyto procesy, vlákna a Windows, které jsou následně vytvořeny. Můžete také zobrazit položky, které již neexistují. Příkaz **Refresh** je k dispozici pro všechna zobrazení kromě zobrazení zprávy.

### <a name="to-refresh-the-currently-active-view"></a>Aktualizace aktuálně aktivního zobrazení

- V nabídce **okno** vyberte **aktualizovat** nebo klikněte na tlačítko **aktualizovat** na panelu nástrojů.
