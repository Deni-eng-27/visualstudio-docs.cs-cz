Od verze Visual Studio 2017, Designer UML byly odebrány ze sady Visual Studio. Další informace najdete v tématu [tomto příspěvku na blogu](https://blogs.msdn.microsoft.com/visualstudioalm/2016/10/14/uml-designers-have-been-removed-layer-designer-now-supports-live-architectural-analysis/). 

Pokud stále potřebujete použít UML, můžete použít Visual Studio Enterprise 2015, což je vedle sebe kompatibilní s novější verzí sady Visual Studio.

Informace o použití UML v sadě Visual Studio 2015 najdete v části:

* [Analýza a modelování vaší architektury](https://msdn.microsoft.com/library/57b85fsc%28v=vs.140%29.aspx)
* [Vytváření modelů pro aplikaci](https://msdn.microsoft.com/library/dd409436%28v=vs.140%29.aspx)
* [Použití modelů ve vývojových procesech](https://msdn.microsoft.com/library/dd409423%28v=vs.140%29.aspx)
* [Ověřování systému během vývoje](https://msdn.microsoft.com/library/dd409448%28v=vs.140%29.aspx)
* [Rozšíření modelů a diagramů UML](https://msdn.microsoft.com/library/ee329484%28v=vs.140%29.aspx)
