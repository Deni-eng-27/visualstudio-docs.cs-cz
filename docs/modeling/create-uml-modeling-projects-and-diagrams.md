Od verze Visual Studio 2017, návrhářů UML byly odebrány ze sady Visual Studio. Další informace najdete v tomto [blogový příspěvek](https://blogs.msdn.microsoft.com/visualstudioalm/2016/10/14/uml-designers-have-been-removed-layer-designer-now-supports-live-architectural-analysis/).

Pokud budete stále muset použít UML, můžete použít Visual Studio 2015 Enterprise edition, který je vedle sebe kompatibilní s novějšími verzemi sady Visual Studio.
