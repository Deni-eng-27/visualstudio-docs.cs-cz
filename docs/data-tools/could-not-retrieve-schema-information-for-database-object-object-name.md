# <a name="could-not-retrieve-schema-information-for-database-object-object-name"></a>Nelze načíst informace o schématu pro databázový objekt. \<object name>

Tato zpráva se obvykle zobrazuje, když je objekt v **Průzkumník serveru** nebo **Průzkumník databáze** zkopírován do schránky, odstraněn z databáze a poté vložen do návrháře. Tato zpráva se zobrazí, protože objekt databáze již neexistuje.
