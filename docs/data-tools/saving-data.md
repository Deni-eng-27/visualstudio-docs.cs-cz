# <a name="saving-data"></a>Ukládání dat
Ukládání dat je, že proces uchování dat v datovém modelu aplikace vrátit zpět na původní úložiště dat, obvykle relační databáze jako SQL Server.  
  
 Aktualizace zdroje dat prostřednictvím datový model je obvykle ve dvou krocích. Prvním krokem je aktualizovat v datovém modelu se novými informacemi – nové záznamy, změnit záznamy nebo odstraněné záznamy. Druhým krokem je a uložte změny v modelu dat zpět do databáze.  
  
 Následující témata popisují koncepty a přidružené k ukládání dat úlohy.
