# <a name="one-or-more-selected-database-objects-return-a-schema-that-does-not-match-the-schema-of-the-target-class"></a>Minimálně jeden vybraný databázový objekt vrací schéma, které neodpovídá schématu cílové třídy.

Jeden nebo více vybraných databázových objektů vrací schéma, které neodpovídá schématu cílové třídy. Do návrháře se nic nepřidalo.

Při přetažení databázových objektů do existujících tříd entit musí data vrácená databázovým objektem odpovídat schématu cílové třídy entity. Ověřte, zda je vybrán správný databázový objekt a zda je cílena správná Třída entity.

## <a name="to-correct-this-error"></a>Oprava této chyby

1. Kliknutím na tlačítko **OK** zavřete dialogové okno.

2. Vyberte objekt databáze vracející data, která odpovídají schématu cílové třídy (třída, na kterou je objekt databáze umístěn v **Návrháři o/R**).
