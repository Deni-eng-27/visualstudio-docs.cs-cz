# <a name="the-designer-cannot-be-modified-while-debugging"></a>Návrháře nemůžete změnit během ladění.

Tato zpráva se zobrazí, když je proveden pokus o změnu položek v **Návrháři o/R** , když aplikace běží v režimu ladění. Je-li aplikace spuštěna v režimu ladění, je **Návrhář pro/R** jen pro čtení.

Chcete-li tuto chybu opravit, vyberte možnost **Zastavit ladění** v nabídce **ladění** . Aplikace zastaví ladění a můžete upravovat položky v **Návrháři o/R**.
