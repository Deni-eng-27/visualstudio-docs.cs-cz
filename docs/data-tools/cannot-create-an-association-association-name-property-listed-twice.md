# <a name="cannot-create-an-association-ltassociation-namegt---property-listed-twice"></a>Nelze vytvořit přidružení &lt;název přidružení&gt; -vlastnost uveden dvakrát.
Nelze vytvořit přidružení \<název přidružení >. Stejnou vlastnost je uveden více než jednou: \<název vlastnosti >.  
  
 Přidružení jsou definovány ve vybrané **vlastnosti přidružení** v **přidružení Editor** dialogové okno. Vlastnosti může být uvedený jenom jednou pro každou třídu v association.  
  
 Vlastnost ve zprávě se zobrazí více než jednou v k nadřazené nebo podřízené třídy **vlastnosti přidružení**.  
  
### <a name="to-resolve-this-condition"></a>Chcete-li vyřešit tuto podmínku  
  
-   Zkontrolujte zprávy a poznamenejte si vlastnosti zadaný ve zprávě.  
  
-   Klikněte na tlačítko **OK** se zavřít okno se zprávou.  
  
-   Zkontrolujte **vlastnosti přidružení** a odstraňte duplicitní položky.  
  
-   Click **OK**.
