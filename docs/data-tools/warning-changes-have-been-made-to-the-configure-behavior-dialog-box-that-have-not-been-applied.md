# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Upozornění. Do dialogového okna konfigurace chování, které nebyly použity byly provedeny změny
Upozornění. Do dialogového okna konfigurace chování, které nebyly použity byly provedeny změny. Opravdu chcete použít změny?  
  
 **Nakonfigurovat chování** dialogové okno umožňuje konfigurovat chování Insert, Update a Delete pro všechny třídy, které jsou k dispozici. Tato zpráva se zobrazí, když vyberete novou **třída** a **chování** kombinace a předchozí změny nebyla dosud nebyly použity.  
  
### <a name="to-apply-the-change-and-continue"></a>Použití změny a pokračovat  
  
-   Klikněte na tlačítko **Ano** na použití změny na vybrané **třída** a **chování**.  
  
### <a name="to-cancel-the-previous-change-and-continue"></a>Předchozí změny zrušit a pokračovat  
  
-   Klikněte na tlačítko **ne** zrušit změnu vybraný **třída** a **chování**.
