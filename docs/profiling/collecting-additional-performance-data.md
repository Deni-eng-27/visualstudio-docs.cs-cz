# <a name="collect-additional-performance-data"></a>Shromažďování dalších dat o výkonu

Kromě dat, která shromažďujete pomocí metody profilování, nástroje profilování sady Visual Studio umožňují zaznamenávat a zobrazovat další data.

## <a name="common-tasks"></a>Běžné úkoly

|Úkol|Související obsah|
|----------|---------------------|
|**Shromažďujte údaje o výkonu pro ADO.NET volání ve vaší aplikaci.** Přidejte data o synchronních voláních, která vaše aplikace provádí do databází.|- [Shromažďování dat interakce vrstvy](../profiling/collecting-tier-interaction-data.md)|
|**Shromážděte data čítače výkonu systému Windows.** Přidejte čítače výkonu systému jako profilovací značky do dat profilování. Značky lze použít k filtrování sestav.|- [Postup: Shromažďování dat čítačů systému Windows](../profiling/how-to-collect-windows-counter-data.md)|
|**Shromažďujte trasování událostí pro data systému Windows.** Shromažďujte data sledování událostí pro Systém Windows (ETW) a zaznamenávejte kromě dat profilování také události aplikace a systému.|- [Postup: Shromažďování dat sledování událostí pro Windows (ETW)](../profiling/how-to-collect-event-tracing-for-windows-etw-data.md)|
