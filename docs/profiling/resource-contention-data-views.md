# <a name="resource-contention-data-views"></a>Zobrazení dat kolizí prostředku
Tato část obsahuje referenční informace pro zobrazení a sestavy datových souborů profileru, které obsahují data profilování tvrzení podprocesu.

## <a name="in-this-section"></a>V tomto oddílu
- [Souhrnné zobrazení – data konfliktů profileru](../profiling/resource-contention-data-views.md)

 Zobrazí grafickou časovou osu dat profilování a zobrazí seznam funkcí a prostředků, které se účastnily nejvíce blokovacích událostí.

- [Zobrazení stromu volání](../profiling/call-tree-view-contention-data.md)

 Zobrazí hierarchický strom, který představuje cesty spuštění a data konfliktů prostředků funkcí v profilování spustit.

- [Zobrazení modulů](../profiling/modules-view-contention-data.md)

 Uspořádá data tvrzení vláken a prostředků podle modulu a zobrazí seznam funkcí, řádků zdrojového kódu a pokynů, které byly spuštěny při blokování událostí.

- [Zobrazení volajícího / volaných - data tvrzení](../profiling/caller-callee-view-contention-data.md)

 Zobrazí seznam dat tvrzení o vláknech a prostředcích pro vybranou funkci a funkce, které volaly a byly volány vybranou funkcí.

- [Zobrazení podrobností o prostředku](../profiling/resource-details-view-contention-data.md)

 Zobrazí grafickou časovou osu blokování událostí pro každý tvrdil zdroje a seznam zásobníku volání pro blokování událostí.

- [Zobrazení podrobností o vláknu](../profiling/thread-details-view-contention-data.md)

 Zobrazí grafickou časovou osu blokování událostí pro každé vlákno a zobrazí seznam zásobníku volání pro události blokování.

- [Zobrazení funkcí](../profiling/functions-view-contention-data.md)

 Zobrazí seznam dat tvrzení o vláknech a prostředcích podle funkce.

- [Zobrazení kolizí prostředku](../profiling/resource-contentions-view-contention-data.md)

 Zobrazí seznam dat konfliktů prostředků podle blokovaných prostředků.

- [Zobrazení řádků](../profiling/lines-view-contention-data.md)

 Zobrazí seznam dat tvrzení o prostředcích podle řádku zdrojového kódu.

- [Zobrazení ukazatelů na instrukce](../profiling/instruction-pointers-ips-view-contention-data.md)

 Zobrazí seznam konfliktů prostředků podle pokynů.

- [Zobrazení procesů](../profiling/process-view-contention-data.md)

 Zobrazí seznam konfliktů prostředků podle procesu a vlákna.

## <a name="reference"></a>Referenční informace
- [Zobrazení podrobností funkce](../profiling/function-details-view.md)

 Zobrazí grafický graf vztahu mezi vybranou funkcí a funkcemi, které volaly a byly volány vybranou funkcí.
