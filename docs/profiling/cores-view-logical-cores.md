# <a name="cores-view-logical-cores"></a>Jádra Zobrazení logických jader
Každý název kanálu představuje jedno z logických jader, které existují v profilovaném systému. Pojmenování se počítá od 0 do celkového počtu logických jader. Například čtyřjádrový systém se bude počítat od logického jádra 0 do logického jádra 3.
