# <a name="tier-interaction-views"></a>Zobrazení interakce vrstvy

Profilace interakce vrstvy poskytuje další informace o dobu provádění funkcí v víceúrovňových aplikací, které komunikují s databází prostřednictvím [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)]. Data jsou shromažďována jenom pro funkce synchronní volání. Můžete zahrnout dat interakce vrstev všechny metody profilování.

Zobrazení interakce vrstvy jsou k dispozici pouze v aplikaci Visual Studio Enterprise.
