# <a name="tier-interaction-views"></a>Zobrazení interakce vrstvy

Profilování interakce vrstev poskytuje další informace o spuštění s úspěšností funkcí ve víceúrovňových aplikací, které komunikují s databázemi prostřednictvím [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)]. Data se shromažďují pouze pro synchronní volání. Dat interakce vrstev můžete zahrnout všechny metody profilování.

Zobrazení interakce vrstev jsou k dispozici pouze v sadě Visual Studio Enterprise.
