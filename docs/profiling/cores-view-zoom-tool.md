# <a name="cores-view-zoom-tool"></a>Nástroj zvětšení zobrazení jader
Ovládací prvek Lupa je posuvník, který umožňuje přiblížení a oddálení časové osy, takže se můžete zaměřit na oblasti zvláštního zájmu. Tento ovládací prvek přiblíží střed aktuálního pohledu, takže před přiblížením zobrazení upravte jeho zobrazení.

## <a name="other-ways-to-zoom"></a>Další způsoby přiblížení

- Přiblížení: Tažením vyberte oblast v zobrazení časové osy. Když uvolníte tlačítko myši, zobrazení časové osy přiblíží vybraný rozsah.

- Přiblížení a oddálení: Najeďte myší na časovou osu a při pohybu kolečkem myši stiskněte **klávesu Ctrl.**
