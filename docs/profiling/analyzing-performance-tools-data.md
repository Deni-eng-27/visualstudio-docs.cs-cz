# <a name="analyze-performance-tools-data"></a>Analýza dat nástrojů pro měření výkonu
Sestavy výkonu [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] nástrojů profilování umožňují zobrazit a analyzovat problémy s výkonem aplikace. Tato část obsahuje přehled zpráv o výkonu a zobrazení, které lze použít ke kontrole dat o výkonu.

## <a name="common-tasks"></a>Běžné úkoly

|Úkol|Související obsah|
|----------|---------------------|
|**Pomocí pravidel výkonu můžete rychle identifikovat problémy:** Pravidla výkonu nástroje profilování identifikují běžné problémy a pomáhají vám snadno přejít na zdrojový kód, který problém obsahuje. Podrobná témata nápovědy mohou často navrhovat řešení.|-   [Použití pravidel výkonu k analýze dat](../profiling/using-performance-rules-to-analyze-data.md)|
|**Vysvětlení podrobností o zobrazení sestavy:** Profilování Nástroje sestavy zobrazení poskytují agregovaná data o výkonu pro procesy, vlákna, moduly a funkce profilování spustit. Data, která se zobrazí, závisí na metodě profilování, která byla použita ke shromažďování dat.|-   [Zobrazení sestavy výkonu](../profiling/performance-report-views.md)|
|**Konfigurace, řazení a filtrování zobrazení sestavy:** Můžete určit a seřadit datové sloupce tak, aby se zobrazovaly v sestavě, seřadit řádky sestavy a filtrovat data tak, aby zahrnovala pouze časový segment, který zadáte.|-   [Přizpůsobení zobrazení sestavy nástrojů výkonu](../profiling/customizing-performance-tools-report-views.md)|
