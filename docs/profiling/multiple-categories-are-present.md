# <a name="multiple-categories-are-present"></a>K dispozici je více kategorií
Je vybrán oddíl, který obsahuje více než jeden druh aktivity pro daný časový rozsah. Všimněte si, že jsou zobrazeny dvě barvy, jedna svisle skládaná na druhé. Tyto barvy představují pouze dvě nejčastější aktivity v daném časovém rozsahu. Frekvence těchto dvou činností jsou zobrazeny ve vzájemném vztahu jako třetiny nebo poloviny svislé ho. Chcete-li zobrazit další podrobnosti, zkuste tuto část zobrazení vlákna přiblížit.
