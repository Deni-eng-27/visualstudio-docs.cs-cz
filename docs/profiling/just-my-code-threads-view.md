# <a name="just-my-code-threads-view"></a>Pouze vlastní kód (Zobrazení vláken)
Pokud vyberete tuto možnost, budete filtrovat zásobník volání tak, aby zobrazoval pouze váš kód plus jednu úroveň volaných funkcí.

 Aktivací této možnosti můžete výrazně snížit složitost zásobníku volání a možná usnadnit diagnostiku konkrétního problému.

 V některých případech může výběr této možnosti odfiltrovat blokovací volání. Pokud požadujete úplné podrobnosti zásobníku volání, aby se toto určení, zrušte zaškrtnutí této možnosti vystavit zásobníku celé volání.
