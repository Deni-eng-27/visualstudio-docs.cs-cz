# <a name="overview-of-tools-for-measuring-performance-in-visual-studio"></a>Přehled nástrojů pro měření výkonu v sadě Visual Studio

Nástroje pro profilaci a diagnostiku vám pomohou diagnostikovat využití paměti a procesoru a další problémy na úrovni aplikace. Pomocí těchto nástrojů shromažďujete data (například hodnoty proměnných, volání funkcí a události) po určitou dobu, například čas, který spustíte aplikaci. Můžete zobrazit stav aplikace v různých bodech během provádění kódu nebo provést analýzu posmrtné analýzy výkonu aplikace.
