# <a name="zoom-control-utilization-view"></a>Ovládání lupy (zobrazení využití)
Ovládání lupy pomáhá přiblížit graf využití procesoru, abyste se mohli zaměřit na oblasti zvláštního zájmu. Tento ovládací prvek přiblíží střed pohledu. Proto byste měli před přiblížením vystředit oblast zájmu.

 Chcete-li vytvořit zvýrazněnou oblast, můžete přetáhnout ukazatel myši v grafu využití procesoru nebo grafu aktivity GPU. Po uvolnění tlačítka myši se zobrazení přiblíží k vybranému rozsahu.
