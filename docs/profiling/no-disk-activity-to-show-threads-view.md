# <a name="no-disk-activity-to-show-threads-view"></a>Žádná aktivita disku k zobrazení (zobrazení vláken)
Aktivitu disku pro aktuální časový rozsah v kanálech diskových vstupně-no su-disku nelze zobrazit. Při oddálení nebo posouvání může být zobrazena aktivita disku. Také se ujistěte, že žádné diskové kanály jsou skryté.
