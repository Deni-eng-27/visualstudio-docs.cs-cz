# <a name="how-to-set-the-current-session"></a>Postup: Nastavení aktuální relace

V **Průzkumníku výkonu**můžete otevřít více relací výkonu a poté vybrat jednu z viditelných relací výkonu, které se mají nastavit jako aktuální relace.

Po nastavení relace jako aktuální relace se klepnutím na **tlačítko Spustit** na panelu nástrojů **Průzkumníka výkonu** spustíprofilování aktuální relace.

## <a name="to-set-current-performance-session"></a>Nastavení aktuální relace výkonu

- V **Průzkumníku výkonu**klepněte pravým tlačítkem myši na relaci výkonu a potom klepněte na příkaz **Nastavit jako aktuální relaci**.

     Název relace se zobrazí tučně.
