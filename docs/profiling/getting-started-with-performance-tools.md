# <a name="getting-started-with-performance-tools"></a>Začínáme s nástroje pro sledování výkonu
Tato část obsahuje přehled o konceptech a úlohy, které se používají při shromažďování, zobrazit a analyzovat výkon kódu pomocí nástroje pro sledování výkonu sady Visual Studio.  
  
 Tyto nástroje můžete poskytnout několik způsobů shromažďování a analýza dat výkonu. Avšak v mnoha případech je nejlepší způsob, jak začít, použijte výchozí nastavení Průvodce výkonu, který slouží ke shromažďování statistik aplikace, která může ukazovat na problémy s výkonem v kódu.  
  
-   *Upozornění výkonu* oznámíme vám běžných problémů s kódování. Upozornění se zobrazí v okně chyb sady Visual Studio. Můžete přejít z upozornění na zdrojový kód a podrobné témata nápovědy, které vám pomůžou zápis efektivnější kódu.  
  
-   Sestavy pro zvýšení výkonu zadejte zobrazení na různé úrovně strukturu aplikace ze zdroje řádky kódu procesy a data spuštění programu ze volající a volaná funkce specifické funkce, která se stromu volání v celé aplikaci.
