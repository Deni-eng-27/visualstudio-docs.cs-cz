# <a name="performance-report-views"></a>Zobrazení sestav výkonu
V této části najdete referenční informace k sestavám a zobrazením [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] Nástroje pro profilaci. Zobrazení datových souborů profilování se zobrazují v [!INCLUDE[vs_current_short](../code-quality/includes/vs_current_short_md.md)] integrovaných environmentIDE vývoje. Sestavy jsou generovány nástrojem příkazového řádku [VSPerfReport](../profiling/vsperfreport.md) .

- [Zobrazení dat metody instrumentace](../profiling/instrumentation-method-data-views.md)
