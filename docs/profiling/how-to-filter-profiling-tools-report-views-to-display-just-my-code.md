# <a name="how-to-filter-profiling-tools-report-views-to-display-just-my-code"></a>Postup: Filtrování zobrazení sestavy nástrojů profilování pro zobrazení pouze můj kód

Visual Studio poskytuje filtrování pro sestavy výkonu tak, aby se zobrazily pouze kód, který jste implementovali. Tato funkce s názvem Jen můj kód je ve výchozím nastavení povolena. Je konfigurován a nakonfigurován pomocí možnosti **Jen můj kód** v dialogovém okně **Možnosti.**

## <a name="to-enable-just-my-code"></a>Povolení možnosti Pouze můj kód

1. V nabídce **Tools** (Nástroje) klikněte na **Options** (Možnosti).

2. V dialogovém okně **Možnosti** rozbalte **nástroje výkonu** a pak vyberte **Obecný** uzel.

3. V **části Obecné nastavení**zaškrtněte políčko **Povolit pouze můj kód.**

## <a name="to-disable-just-my-code"></a>Zakázání pouze můj kód

1. V nabídce **Tools** (Nástroje) klikněte na **Options** (Možnosti).

2. V dialogovém okně **Možnosti** rozbalte **nástroje výkonu** a pak vyberte **Obecný** uzel.

3. V **části Obecné nastavení**zrušte zaškrtnutí políčka **Povolit pouze můj kód.**
