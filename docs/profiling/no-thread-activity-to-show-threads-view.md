# <a name="no-thread-activity-to-show-threads-view"></a>Žádná aktivita vláken k zobrazení (Zobrazení vláken)
Tato oblast zobrazuje data o zobrazí vláken v současné době viditelný časový rozsah.  
  
 Pokud nejsou viditelné žádné informace, zkontrolujte následující nastavení:  
  
-   Úroveň zvětšení je vysoká? Zkuste přiblížení a oddálení mimo nebo posouvání a převeďte další aktivita vláken v rozsahu.  
  
-   Jsou příliš mnoho vláken skryté? Pokud ano, zkuste zobrazující všechna vlákna  
  
-   Pokud **pouze můj kód** je vybraná, můžete zobrazit jenom data o vašem kódu. Vymažte nastavení pro zjištění, zda je žádnou aktivitu systému přístup z více vláken.  
  
-   Ujistěte se, že snížení šumu je nastavená na nízké prahové hodnoty.
