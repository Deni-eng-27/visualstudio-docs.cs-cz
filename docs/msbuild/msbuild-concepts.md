# <a name="msbuild-concepts"></a>Koncepty MSBuild

MSBuild poskytuje základní schéma XML, které můžete použít k řízení, jak platforma sestavení vytváří software. Chcete-li určit součásti v sestavení a způsob jejich sestavení, použijte tyto čtyři části MSBuild: vlastnosti, položky, úkoly a cíle.
