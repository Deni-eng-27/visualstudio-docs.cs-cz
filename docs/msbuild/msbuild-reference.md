# <a name="msbuild-reference"></a>Odkaz na sestavení msbuild

MSBuild je systém sestavení pro visual studio. Následující odkazy vedou k tématům, která obsahují referenční informace MSBuild.

## <a name="in-this-section"></a>V tomto oddílu

- [Odkaz na schéma souboru projektu](../msbuild/msbuild-project-file-schema-reference.md)\
 Popisuje elementy XML, které tvoří formát souboru MSBuild.

- [Odkaz na úkol](../msbuild/msbuild-task-reference.md)\
 Popisuje některé typické úkoly, které jsou součástí MSBuild.

- [Podmínky](../msbuild/msbuild-conditions.md)\
 Popisuje podmínky, které jsou k dispozici v souborech MSBuild.

- [Podmíněné konstrukce](../msbuild/msbuild-conditional-constructs.md)\
 Popisuje způsob použití `Choose`prvků `When`, `Otherwise` a .

- [Vyhrazené a dobře známé vlastnosti msbuild](../msbuild/msbuild-reserved-and-well-known-properties.md)\
 Popisuje vyhrazené vlastnosti MSBuild.

- [Běžné vlastnosti projektu MSBuild](../msbuild/common-msbuild-project-properties.md)\
 Popisuje vlastnosti projektu, které jsou společné pro všechny typy projektů a také vlastnosti, které jsou často používány konkrétní typy projektů.

- [Běžné položky projektu MSBuild](../msbuild/common-msbuild-project-items.md)\

- [. Cíle na soubory](../msbuild/msbuild-dot-targets-files.md)\

- [Známá metadata položek](../msbuild/msbuild-well-known-item-metadata.md)\
 Zobrazí seznam metadat, která jsou vytvořena společně s každou položkou.

- [Soubory odpovědí](../msbuild/msbuild-response-files.md)\

- [WPF MSBuild odkaz](../msbuild/wpf-msbuild-reference.md)\

- [Speciální znaky k útěku](../msbuild/special-characters-to-escape.md)\
