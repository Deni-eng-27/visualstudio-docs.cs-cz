# <a name="formaturl-task"></a>FormatUrl – úloha

Převede adresu URL na správný formát adresy URL.

## <a name="parameters"></a>Parametry

 Následující tabulka popisuje parametry `FormatUrl` úkolu.

|Parametr|Popis|
|---------------|-----------------|
|`InputUrl`|Volitelný `String` parametr.<br /><br /> Určuje adresu URL, která má být formátována.|
|`OutputUrl`|Volitelný `String` výstupní parametr.<br /><br /> Určuje formátovanou adresu URL.|

## <a name="remarks"></a>Poznámky

 Kromě parametrů, které jsou uvedeny v tabulce, tato úloha <xref:Microsoft.Build.Tasks.TaskExtension> dědí parametry z <xref:Microsoft.Build.Utilities.Task> třídy, která sama dědí z třídy. Seznam těchto dalších parametrů a jejich popisy naleznete v tématu [TaskExtension base class](../msbuild/taskextension-base-class.md).
