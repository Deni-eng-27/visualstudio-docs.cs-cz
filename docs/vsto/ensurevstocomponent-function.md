```csharp
HRESULT EnsureVSTOComponent(
    IVSTProject *pProject
);
```

#### <a name="parameters"></a>Parametry

|Parametr|Popis|
|---------------|-----------------|
|*pProject*|Nepoužívejte.|

## <a name="return-value"></a>Vrácená hodnota
