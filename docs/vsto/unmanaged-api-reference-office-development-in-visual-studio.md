# <a name="unmanaged-api-reference-office-development-in-visual-studio"></a>Referenční dokumentace nespravovaného rozhraní API (vývoj pro Office v sadě Visual Studio)
  Od verze systému Microsoft Office 2007, pomocí aplikací Office [imanagedaddin – rozhraní](../vsto/imanagedaddin-interface.md) rozhraní provést volání do doplňku VSTO zavaděč komponentu, která je součástí [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)]. Tato součást se používá ke spouštění spravované doplňků VSTO. Můžete vytvořit vlastní zavaděč součást doplňku VSTO implementací toto rozhraní.  
  
> [!NOTE]  
>  Máte zájem o vývoji řešení, které rozšiřují Office prostředí napříč [více platforem](https://dev.office.com/add-in-availability)? Podívejte se na [Office Add in modelu](https://dev.office.com/docs/add-ins/overview/office-add-ins). Doplňky Office mají malé nároky ve srovnání s doplňků VSTO a řešení a můžete je vytvořit pomocí téměř jakoukoli webové programování technologie, jako je například HTML5, JavaScript, CSS3 a XML.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Imanagedaddin – rozhraní](../vsto/imanagedaddin-interface.md)  
 Rozhraní COM, které můžete implementovat, abyste zavedení a uvolnění spravované doplňků VSTO v aplikacích Office.
