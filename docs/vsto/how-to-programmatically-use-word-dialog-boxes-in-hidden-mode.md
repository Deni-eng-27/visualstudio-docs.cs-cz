Můžete to provést pomocí <xref:Microsoft.Office.Interop.Word.Dialog.Execute%2A> metodu <xref:Microsoft.Office.Interop.Word.Dialog> bez volání <xref:Microsoft.Office.Interop.Word.Dialog.Display%2A> metoda.
