# <a name="get-started-with-source-control-vspackages"></a>Začínáme s ovládacím prvkem zdrojového kódu VSPackages

Source-control VSPackage je pokročilejší alternativou k modulu plug-in správy zdrojového kódu. Další informace o modulech plug-in správy zdrojového kódu naleznete v [tématu Vytvoření modulu plug-in správy zdrojového kódu](../../extensibility/internals/creating-a-source-control-plug-in.md). Zdroj ovládacího prvku VSPackage umožňuje úplnou kontrolu nad modelu správy zdrojového kódu, funkce [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] a uživatelské rozhraní (UI) a integruje se do prostředí jako VSPackage.
