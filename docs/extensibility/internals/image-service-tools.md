# <a name="image-service-tools"></a>Nástroje služby image
Sada VS SDK obsahuje několik nástrojů, které pomáhají tvůrcům rozšíření používat službu image Visual Studio.

- Nástroj [Manifest from Resources](../../extensibility/internals/manifest-from-resources.md) pořídí seznam obrazových prostředků (.png nebo .xaml files) a vygeneruje soubor manifestu obrazu.

- Nástroj [Manifest to Code](../../extensibility/internals/manifest-to-code.md) pořídí soubor manifestu obrazu a vygeneruje soubor obálky, který se použije v souborech C++, C#, VB nebo .vsct.

- [Prohlížeč knihovny obrázků](../../extensibility/internals/image-library-viewer.md) může načítat, upravovat a vyhledávat manifesty obrázků.
