# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>Co&#39;nového ve zdroji ovládací prvek modulu Plug-in API ve verzi 1.2
Tato část se zaměřuje funkce správy pokročilé zdrojového kódu v rozhraní API modulu Plug-in zdroje ovládacího prvku verze 1.2. Následuje seznam nových funkcí ve verzi 1.2:

## <a name="changes"></a>Změny
- [Odstranění ~ SAK soubory](../../extensibility/internals/elimination-of-tilde-sak-files.md) vysvětluje, jak rozhraní API 1.2 zdrojový ovládací prvek modulu Plug-in komunikuje s dočasné soubory.

- [Aplikace nastavení napříč různými připojeními projektů](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md) ukazuje, jak plug-in správy zdrojových kódů dávkovou operaci používat ke spuštění stejnou operaci správy zdrojových kódů napříč více projekty nebo více kontexty připojení.

- [Vytváření složek nadřazený kontejnerů pro řešení](../../extensibility/internals/creating-parent-container-folders-for-solutions.md) vysvětluje, jak zdrojový ovládací prvek modulu Plug-in API verze 1.2 umožňuje uživateli a zadejte cílovou složku jeden kořenový zdrojového ovládacího prvku pro všechny webové projekty v řešení.

- [Volitelné porovnání z místní složky projektu do zdrojového ovládacího prvku Store](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md) vysvětluje, jak rozhraní API 1.2 zdrojový ovládací prvek modulu Plug-in porovnává verze souborů.

- [Odebrání informacích správy zdrojových kódů z. Proj a. Sln – soubory](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md) vysvětluje, jak zdrojový ovládací prvek modulu Plug-in API verze 1.2, jsou uloženy informace o projekty a řešení.
