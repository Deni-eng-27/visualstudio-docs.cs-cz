# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>Co&#39;s novinkou v rozhraní Plug-in API pro správu zdrojového kódu verze 1.2
Tato část zvýrazňuje pokročilé funkce správy zdrojového kódu v rozhraní PLUG-IN Source Control API verze 1.2. Následující seznam nových funkcí ve verzi 1.2:

## <a name="changes"></a>Změny
- [Odstranění souborů ~SAK](../../extensibility/internals/elimination-of-tilde-sak-files.md) Vysvětluje, jak rozhraní API 1.2 správy zdrojového kódu spolupracuje s dočasnými soubory.

- [Aplikace nastavení napříč více připojeními projektu](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md) Ukazuje, jak modul plug-in správy zdrojového kódu můžete použít dávkovou operaci ke spuštění stejné operace správy zdrojového kódu napříč více projektů nebo více kontextů připojení.

- [Vytváření nadřazených složek kontejnerů pro řešení](../../extensibility/internals/creating-parent-container-folders-for-solutions.md) Vysvětluje, jak rozhraní Plug-in plug-in správy zdrojového kódu verze 1.2 umožňuje uživateli zadat jeden cíl kořenového ovládacího prvku zdrojového kódu pro všechny webové projekty v rámci řešení.
