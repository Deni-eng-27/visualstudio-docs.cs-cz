# <a name="color-theme-tools"></a>Nástroje pro barevný motiv
Sada Visual Studio SDK poskytuje několik nástrojů, které vám pomohou vytvářet a spravovat barevné motivy.

- Nástroj [Editor barev VSIX](../../extensibility/internals/vsix-color-editor.md) může vytvářet a upravovat vlastní barvy pro Visual Studio.

- Nástroj [Překladač barev VSIX](../../extensibility/internals/vsix-color-compiler.md) převádí existující soubory *XML* motivu Visual Studio na soubory *.pkgdef* pro vaši příponu.
