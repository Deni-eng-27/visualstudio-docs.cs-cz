# <a name="list-of-available-services"></a>Seznam dostupných služeb

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]a sada Visual Studio SDK podporují následující služby. Některé balíčky nabízejí své vlastní služby, které zde nejsou uvedeny – například jazykové služby nemají jeden identifikátor GUID služby. Název jazyka je nutné použít k vyhledání identifikátoru GUID služby jazyka v registru.

Pomocí identifikátorů GUID služby uvedených v tomto seznamu nebo získaných z jiného zdroje (například jazykových služeb) získáte primární rozhraní nebo rozhraní zobrazená s každou službou.
