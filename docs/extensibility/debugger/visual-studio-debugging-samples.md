# <a name="visual-studio-debugging-samples"></a>Ukázky ladění sady Visual Studio
Dvě ukázky ukazují, jak rozšířit ladicí program sady Visual Studio pomocí [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)].

- Ukázka vyhodnocení ladicího programu pro spravované jazyky [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)]je zahrnuta v souboru . V [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] okně klikněte na kartu **Ukázky,** v seznamu jazyků klikněte na **Všechny** nebo **C#** a potom klikněte na **Debugger**.

- Ukázka modulu ladění sady Visual Studio pro nativní jazyky je k dispozici v galerii kódu MSDN. Další informace naleznete v [tématu Visual Studio ladicí modul ukázku](https://code.msdn.microsoft.com/Visual-Studio-Debug-Engine-c2e21c0e) na webu společnosti Microsoft.
