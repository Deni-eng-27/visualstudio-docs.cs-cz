# <a name="document-position"></a>Pozice dokumentu
Při [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ladění je *pozice dokumentu*:

- Poskytuje abstrakci pozice ve zdrojovém souboru, jak je známo ide. Pro většinu jazyků dnes lze pozici dokumentu považovat za pozici ve zdrojovém souboru.

- Popisuje pozici ve zdrojovém dokumentu ladicí modul.

- Je implementována rozhraním [IDebugDocumentPosition2.](../../extensibility/debugger/reference/idebugdocumentposition2.md)
