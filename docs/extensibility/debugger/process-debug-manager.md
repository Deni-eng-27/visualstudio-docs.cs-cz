# <a name="process-debug-manager"></a>Správce procesu ladění
Správce ladění procesu (PDM) je součástí sady Visual Studio, který spravuje programy a procesy, jejich zpřístupnění relaci ladění manager a moduly ladění.  
  
 PDM spravuje všechny procesy, které můžete ladit. Pokud chcete ladit, program musí být zaregistrované pomocí PDM. To se provádí v době, kdy program se spustí, port nebo modul ladění.
