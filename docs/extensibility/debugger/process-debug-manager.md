# <a name="process-debug-manager"></a>Správce ladění procesu
Správce ladění procesů (PDM) je součástí sady Visual Studio, která spravuje programy a procesy a zpřístupňuje je správci ladění relací a ladicím strojům.

 PDM spravuje všechny procesy, které lze ladit. Chcete-li ladit, musí být program registrován u PDM. Tato registrace se provádí v době spuštění programu, buď portem nebo ladicí mů laureát.
