# <a name="idebugobject2getalias"></a>IDebugObject2::GetAlias
Získá alias přidružený k tomuto objektu, pokud existuje.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetAlias(  
   IDebugAlias** ppAlias  
);  
```  
  
```csharp  
int GetAlias(  
   out IDebugAlias ppAlias  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppAlias`  
 [out] Vrátí [IDebugAlias](../../../extensibility/debugger/reference/idebugalias.md) objektu představující alias pro tento objekt; jinak vrátí hodnotu null.  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí S_OK; jinak vrátí kód chyby.  
  
## <a name="remarks"></a>Poznámky  
 Alias pro objekt se vytvoří pomocí volání [CreateAlias](../../../extensibility/debugger/reference/idebugobject2-createalias.md) metoda.
