# <a name="bp_location_code_file_line"></a>BP_LOCATION_CODE_FILE_LINE
Obsahuje data pro umístění zarážky na určitém řádku ve zdrojovém souboru kódu.

## <a name="syntax"></a>Syntaxe

```cpp
typedef struct _BP_LOCATION_CODE_FILE_LINE {
    BSTR                     bstrContext;
    IDebugDocumentPosition2* pDocPos;
} BP_LOCATION_CODE_FILE_LINE;
```

## <a name="members"></a>Členové
`bstrContext`\
Kontext zarážky, obvykle název metody nebo funkce, jak je vidět na zásobníku volání.

`pDocPos`\
[Objekt IDebugDocumentPosition2,](../../../extensibility/debugger/reference/idebugdocumentposition2.md) který představuje pozici dokumentu zarážky.

## <a name="remarks"></a>Poznámky
Tato struktura je členem [BP_LOCATION](../../../extensibility/debugger/reference/bp-location.md) struktury jako součást unie.

## <a name="requirements"></a>Požadavky
Záhlaví: msdbg.h

Obor názvů: Microsoft.VisualStudio.Debugger.Interop

Sestavení: Microsoft.VisualStudio.Debugger.Interop.dll
