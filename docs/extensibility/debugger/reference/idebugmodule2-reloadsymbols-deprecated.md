```cpp  
HRESULT ReloadSymbols(   
   LPCOLESTR pszUrlToSymbols,  
   BSTR*     pbstrDebugMessage  
);  
```  
  
```csharp  
int ReloadSymbols(   
   string     pszUrlToSymbols,  
   out string pbstrDebugMessage

```  
  
#### <a name="parameters"></a>Parametry  
 `pszUrlToSymbols`  
 [v] Cesta k úložišti symbol.
