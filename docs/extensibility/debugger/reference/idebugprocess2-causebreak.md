# <a name="idebugprocess2causebreak"></a>IDebugProcess2::CauseBreak
Požaduje, aby další program, který je spuštěn kód v tomto procesu zastavit a odeslat objekt události [IDebugBreakEvent2.](../../../extensibility/debugger/reference/idebugbreakevent2.md)

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT CauseBreak( 
   void
);
```

```csharp
int CauseBreak();
```

## <a name="return-value"></a>Návratová hodnota
 V případě `S_OK`úspěchu vrátí ; v opačném případě vrátí kód chyby.
