# <a name="name_match"></a>NAME_MATCH
Vybere možnost případu pro odpovídající názvy.

## <a name="syntax"></a>Syntaxe

```cpp
typedef enum { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
} NAME_MATCH;
```

```csharp
public enum NameMatchOptions { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
}
```

## <a name="fields"></a>Fields (Pole)
 `nmNone`\
 Nejsou zadány žádné možnosti.

 `nmCaseSensitive`\
 Označuje, že názvy, které mají být spárovány, rozlišují malá a velká písmena.

 `nmCaseInsensitive`\
 Označuje, že názvy, které mají být spárovány nejsou malá a velká písmena.
