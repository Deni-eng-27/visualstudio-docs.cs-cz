# <a name="program_destroy_flags"></a>PROGRAM_DESTROY_FLAGS
Vytvoří vyjmenovává platné hodnoty příznaků zničení programu.

## <a name="syntax"></a>Syntaxe

```cpp
enum enum_PPROGRAM_DESTROY_FLAGS
{
   PROGRAM_DESTROY_CONTINUE_DEBUGGING = 0x1
};
typedef DWORD PROGRAM_DESTROY_FLAGS;
```

```csharp
public enum enum_PPROGRAM_DESTROY_FLAGS
{
   PROGRAM_DESTROY_CONTINUE_DEBUGGING = 0x1
};
```

## <a name="fields"></a>Fields (Pole)
 `PROGRAM_DESTROY_CONTINUE_DEBUGGING`\
 Zničit program, ale pokračovat v ladění.

## <a name="remarks"></a>Poznámky
 Výčet je vrácena [GetFlags](../../../extensibility/debugger/reference/idebugprogramdestroyeventflags2-getflags.md) metoda.

 Sestavení: Microsoft.VisualStudio.Debugger.Interop.dll
