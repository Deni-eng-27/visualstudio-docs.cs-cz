# <a name="idebugstackframe2getcodecontext"></a>IDebugStackFrame2::GetCodeContext
Získá kontext kódu pro tento rámec zásobníku.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetCodeContext ( 
   IDebugCodeContext2** ppCodeCxt
);
```

```csharp
int GetCodeContext ( 
   out IDebugCodeContext2 ppCodeCxt
);
```

## <a name="parameters"></a>Parametry
`ppCodeCxt`\
[out] Vrátí objekt [IDebugCodeContext2,](../../../extensibility/debugger/reference/idebugcodecontext2.md) který představuje aktuální ukazatel instrukce v tomto rámci zásobníku.

## <a name="return-value"></a>Návratová hodnota
 V případě `S_OK`úspěchu vrátí ; v opačném případě vrátí kód chyby.
