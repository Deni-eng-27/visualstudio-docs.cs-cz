# <a name="stepkind"></a>STEPKIND
Určuje druh kroku pro krokování.

## <a name="syntax"></a>Syntaxe

```cpp
enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
typedef DWORD STEPKIND;
```

```csharp
public enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
```

## <a name="fields"></a>Fields (Pole)
 `STEP_INTO`\
 Kroky do funkce.

 `STEP_OVER`\
 Kroky přes funkci.

 `STEP_OUT`\
 Kroky z funkce.

 `STEP_BACKWARDS`\
 Kroky zpět do funkce.

## <a name="remarks"></a>Poznámky
 Předánjako argument [step](../../../extensibility/debugger/reference/idebugprocess3-step.md) metody.
