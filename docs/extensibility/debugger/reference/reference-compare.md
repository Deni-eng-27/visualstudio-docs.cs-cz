# <a name="referencecompare"></a>REFERENCE_COMPARE
Určuje typ porovnání pro odkazy.

## <a name="syntax"></a>Syntaxe

```cpp
enum enum_REFERENCE_COMPARE { 
   REF_COMPARE_EQUAL        = 0x0001,
   REF_COMPARE_LESS_THAN    = 0x0002,
   REF_COMPARE_GREATER_THAN = 0x0003
};
typedef DWORD REFERENCE_COMPARE;
```

```csharp
public enum enum_REFERENCE_COMPARE { 
   REF_COMPARE_EQUAL        = 0x0001,
   REF_COMPARE_LESS_THAN    = 0x0002,
   REF_COMPARE_GREATER_THAN = 0x0003
};
```

## <a name="members"></a>Členové
 REF_COMPARE_EQUAL určuje porovnávání rovnosti.

 Určuje REF_COMPARE_LESS_THAN méně – než porovnání.

 Určuje REF_COMPARE_GREATER_THAN větší-než porovnání.

## <a name="remarks"></a>Poznámky
 Předán jako argument [porovnání](../../../extensibility/debugger/reference/idebugreference2-compare.md) metody.

 Obor názvů: Microsoft.VisualStudio.Debugger.Interop
