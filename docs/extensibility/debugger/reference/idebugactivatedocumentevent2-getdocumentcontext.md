# <a name="idebugactivatedocumentevent2getdocumentcontext"></a>IDebugActivateDocumentEvent2::GetDocumentContext
Získá kontext dokumentu, který popisuje pozici v dokumentu, který má být aktivní ladicí balíček.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetDocumentContext ( 
   IDebugDocumentContext2** ppDocContext
);
```

```csharp
int GetDocumentContext ( 
   out IDebugDocumentContext2 ppDocContext
);
```

## <a name="parameters"></a>Parametry
`ppDocContext`\
[out] Vrátí objekt [IDebugDocumentContext2,](../../../extensibility/debugger/reference/idebugdocumentcontext2.md) který představuje pozici v dokumentu zdrojového souboru.

## <a name="remarks"></a>Poznámky
 Tato pozice může být použita například k zobrazení stříšky.

## <a name="return-value"></a>Návratová hodnota
 V případě `S_OK`úspěchu vrátí ; v opačném případě vrátí kód chyby.
