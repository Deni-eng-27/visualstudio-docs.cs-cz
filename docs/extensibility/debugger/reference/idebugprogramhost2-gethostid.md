# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
Získá identifikátor procesu hostujícího tento program.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetHostId( 
   AD_PROCESS_ID* pdwId
);
```

```csharp
int GetHostId( 
   AD_PROCESS_ID[] pdwId
);
```

## <a name="parameters"></a>Parametry
`pdwId`\
[dovnitř, ven] [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) struktury, která je vyplněna informacemi o identifikátoru procesu.

## <a name="return-value"></a>Návratová hodnota
 V případě `S_OK`úspěchu vrátí ; v opačném případě vrátí kód chyby.
