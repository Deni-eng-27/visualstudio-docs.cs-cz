# <a name="support-for-the-visual-studio-sdk"></a>Podpora sady Visual Studio SDK
Několik možností podpory jsou k dispozici pro zákazníky, kteří mají otázky o tom, jak rozšířit integrované vývojové prostředí sady Visual Studio (IDE).

## <a name="free-support"></a>Bezplatná podpora

### <a name="microsoft-newsgroups"></a>Diskusní skupiny společnosti Microsoft
 Členové vývojového týmu sady Visual Studio a další vývojáři, kteří rozšiřují sadu Visual Studio, se často účastní diskusní skupiny microsoft.public.vstudio.extensibility. Tato diskusní skupina je spravovaná diskusní skupina. Pokud jste univerzální předplatitel msdn, ujistěte se, že jste zaregistrovat bez spam alias, aby bylo možné pro Microsoft reagovat.

### <a name="msdn-forums"></a>Fóra na webu MSDN
 [Fóra MSDN](https://social.msdn.microsoft.com/Forums/en-US/home) poskytují další způsob, jak chatovat s kolegy a odborníky. Grafické uživatelské rozhraní (UI) fór je navržen tak, aby byly snadněji použitelné než diskusní skupiny. [Visual Studio rozšiřitelnost fórum](/azure/devops/integrate/index?view=azure-devops&viewFallbackFrom=vsts) je vhodné místo pro zahájení hledání odpovědi na otázky rozšiřitelnosti. Vždy hledat, než se zeptáte na otázku, protože někdo jiný může již požádal stejnou otázku.

### <a name="visual-studio-gallery"></a>Galerie Visual Studia
 [Visual Studio Marketplace](https://marketplace.visualstudio.com/) je místo pro rozšíření a ukázkový kód. Můžete najít nástroje, které zlepšují vaši produktivitu a uživatelské prostředí s Visual Studio, ukázkové aplikace, které ukazují, jak používat VSPackages a spravované rozšiřitelnosti (MEF) a další druhy vzorků.

### <a name="visual-studio-sdk-samples"></a>Ukázky sady Visual Studio SDK

Open source příklady rozšíření VS SDK najdete na GitHubu na [ukázkách Visual Studia](https://github.com/Microsoft/VSSDK-Extensibility-Samples). Toto úložiště GitHub obsahuje ukázky, které ilustrují různé rozšiřitelné funkce v sadě Visual Studio.

## <a name="included-support"></a>Zahrnutá podpora

### <a name="visual-studio-product-support"></a>Odborná podpora visual studia
 Určitá podpora je součástí [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] nákupu produktů. Další informace naleznete v tématu [Visual Studio Support](https://msdn.microsoft.com/vstudio/cc136615.aspx). Společnost Microsoft udržuje specializovaný tým podpory, který odpovídá na otázky týkající se [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] rozšiřitelnosti.

### <a name="msdn-subscription-support"></a>Podpora předplatného MSDN
 Balíčky předplatného MSDN mohou také obsahovat určitou podporu. Další informace naleznete v tématu [MSDN Subscriber Support](https://msdn.microsoft.com/subscriptions/aa718661.aspx).

## <a name="paid-support"></a>Placená podpora

### <a name="developer-tools-support"></a>Podpora nástrojů pro vývojáře

Podporu lze také zakoupit. Různé nabídky podpory jsou k dispozici v kategorii Nástroje pro vývojáře v [Centrech pro nápovědu a odbornou pomoc společnosti Microsoft](https://support.microsoft.com/supportforbusiness/productselection?fltadd=sps-business-1&sapId=4fd4947b-15ea-ce01-080f-97f2ca3c76e8).

Zákazníci, kteří [!INCLUDE[vsipsdk](../extensibility/includes/vsipsdk_md.md)] chtějí použít k vývoji složitých balíčků, mohou zvážit zakoupení smlouvy premier support se společností Microsoft. Další informace naleznete v tématu [Microsoft Premier Support](https://support.microsoft.com/premier). Pro malé společnosti může být [pokročilá podpora pro partnery](https://partner.microsoft.com/support/advanced-cloud-support) atraktivní alternativou k tradiční nabídce premier support.

> [!NOTE]
> Standardní podporu nelze použít ke konzultaci nebo k získání poradenství.

Dotazy týkající se nabídek podpory Premier ve spojení s [vsipsvs@microsoft.com](mailto:vsipsvs@microsoft.com)programem VSIP lze odeslat společnosti . Dotazy týkající se nového programu [advantg@microsoft.com](mailto:advantg@microsoft.com)Partner Advantage lze odeslat společnosti .

Informace o jednotlivých žádostech o podporu naleznete na [individuálních žádostech o podporu](https://support.microsoft.com/supportforbusiness/productselection).
