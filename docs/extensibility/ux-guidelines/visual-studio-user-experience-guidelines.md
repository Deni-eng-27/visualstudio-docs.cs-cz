# <a name="visual-studio-user-experience-guidelines"></a>Pravidla pro práci s uživatelským prostředím sady Visual Studio
Pokud navrhujete nové funkce pro Visual Studio, postupujte podle pokynů pro uživatelské prostředí sady Visual Studio. Tyto pokyny obsahují informace o běžných uživatelských modelech a vzorcích interakce, aby návrháři nových uživatelských rozhraní (UI) mohli vytvářet uživatelské prostředí, které je bezproblémové a konzistentní ve vývojovém prostředí.

Pokud vyvíjíte software pro produkty společnosti Microsoft, naučte se používat tyto zdroje:

- [Pokyny pro interakci s uživatelským prostředím systému Windows](/windows/win32/uxguide/guidelines) jsou základy chování a vzhledu plochy systému Windows.

- Tyto visual studio pokyny, které jsou specifické pro platformu pokyny, které nahrazují obecné windows pokyny, kdykoli je to vhodné pro naše prostředí.

- [Ovládací prvky a vzory pro aplikace UPW](/windows/uwp/design/controls-and-patterns) jsou odkazem pro vznikající vzory, které mohou být použity v některých případech v rámci sady Visual Studio.

Tyto pokyny poskytují informace o tom, co dělat při navrhování pro Visual Studio a v některých případech informace o tom, jak to udělat, je také zahrnuta. Podrobnější informace o implementaci a informace o konkrétních rozhraních API pro programování aplikací, která umožňují psát kód pro řešení, která se integrují do integrovaného vývojového prostředí sady Visual Studio (IDE), získáte pomocí [sady Visual Studio Software Development Kit (SDK).](../visual-studio-sdk.md)
