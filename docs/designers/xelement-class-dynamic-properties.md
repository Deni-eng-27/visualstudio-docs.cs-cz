# <a name="xelement-class-dynamic-properties"></a>Dynamické vlastnosti třídy XElement

Tato část popisuje dynamické vlastnosti daného <xref:System.Xml.Linq.XElement?displayProperty=fullName> třídy.

## <a name="in-this-section"></a>V tomto oddílu

|Téma|Popis|
|-----------|-----------------|
|[Atribut](../designers/attribute-xelement-dynamic-property.md)|Získá se používá k načtení atributu, který odpovídá zadané rozbalený název indexeru.|
|[Element](../designers/element-xelement-dynamic-property.md)|Získá se používá k načtení podřízený element, který odpovídá zadané rozbalený název indexeru.|
|[Elements](../designers/elements-xelement-dynamic-property.md)|Získá se používá k načtení podřízené prvky aktuálního elementu, které odpovídají zadaným rozbalený název indexeru.|
|[Potomci](../designers/descendants-xelement-dynamic-property.md)|Získá se používá k načtení všech podřízených prvků aktuálního elementu, které odpovídají zadaným rozbalený název indexeru.|
|[Hodnota](../designers/value-xelement-dynamic-property.md)|Získá nebo nastaví obsah elementu.|
|[Xml](../designers/xml-xelement-dynamic-property.md)|Získá neformátovaný reprezentaci XML pro prvek.|
