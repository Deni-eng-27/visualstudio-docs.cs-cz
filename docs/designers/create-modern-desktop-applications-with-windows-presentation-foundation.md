# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>Vytvoření moderních aplikací klasické pracovní plochy s Windows Presentation Foundation

Windows Presentation Foundation (WPF) v sadě Visual Studio poskytuje vývojářům jednotný programovací model pro vytváření moderních aplikací klasické pracovní plochy-obchodní v systému Windows.  

Pro podrobné návody začínat [Začínáme v grafickém subsystému WPF](../designers/getting-started-with-wpf.md) a sledovat až po jeho sekundární témata.

Porovnání dostupných nástrojů najdete v tématu [navrhování XAML v sadě Visual Studio a nástroj Blend for Visual Studio](../designers/designing-xaml-in-visual-studio.md).
