# <a name="how-to-add-a-new-item-to-a-workflow-project-legacy"></a>Postupy: Přidání nové položky do projektu pracovního postupu (zastaralé)
Po vytvoření projektu pracovního postupu pomocí starší verze [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] poskytované [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] , která se zaměřuje buď [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] nebo [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)], můžete přidat [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] položky a jiných známým rozhraním [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] položek do projektu.  
  
 Následující tabulka uvádí [!INCLUDE[wf2](../workflow-designer/includes/wf2_md.md)] položky, které můžete přidat do projektu pracovního postupu.  
  
|Položka|Popis|  
|----------|-----------------|  
|Aktivita|Aktivita, jejíž definici aktivity v kódu návrháře kódu souboru a uživatele v samostatném souboru kódu.|  
|Aktivity (s oddělení kódu)|Definici aktivity, vyjádřené jako pracovní postup kódu značek a uživatele v samostatném souboru kódu.|  
|Sekvenční pracovní postup (kód)|Sekvenční pracovní postup s definice pracovního postupu v kódu návrháře kódu souboru a uživatele v samostatném souboru kódu.|  
|Sekvenční pracovní postup (s oddělení kódu)|Sekvenční pracovní postup s definice pracovního postupu, vyjádřené jako pracovní postup kódu značek a uživatele v samostatném souboru kódu.|  
|Stav počítače pracovního postupu (kód)|Stav počítače pracovního postupu s definice pracovního postupu v kódu návrháře kódu souboru a uživatele v samostatném souboru kódu.|  
|Stav počítače pracovního postupu (s oddělení kódu)|Stav počítače pracovního postupu s definice pracovního postupu, vyjádřené jako pracovní postup kódu značek a uživatele v samostatném souboru kódu.|  
  
### <a name="to-add-a-new-item-to-a-workflow-project"></a>Chcete-li přidat novou položku do projektu pracovního postupu  
  
1.  Na **projektu** nabídky, klikněte na tlačítko **přidat novou položku**.  
  
     **Přidat novou položku** otevře se dialogové okno.  
  
2.  Vyberte položku.  
  
     V předchozí tabulce jsou uvedené dostupné možnosti Windows Workflow Foundation.  
  
3.  Klikněte na tlačítko **přidat** přidat položku do projektu workflow.
