# <a name="transaction-activity-designers"></a>Návrháři aktivit transakcí

Návrháři aktivity transakce slouží k vytváření a konfiguraci transakčních aktivit, jako je třeba kompenzovat a potvrzovat. Témata obsažená v této části a uvedená v následující tabulce obsahují pokyny k použití Návrhář postupu provádění a návrháře šablon.
