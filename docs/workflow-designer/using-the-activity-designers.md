# <a name="using-the-activity-designers"></a>Pomocí návrháře aktivit
Návrháři pracovních postupů Windows zahrnuje několik poskytované systémem aktivit, které můžete použít při vytváření aktivity a pracovní postupy. Tato část popisuje kategorie aktivit poskytované systémem a jejich návrhářů, které jsou k dispozici. Témata v jednotlivých aktivit a jejich odpovídajících designeru v každém z těchto kategorií popisují, co tyto aktivity se a poskytovat pokyny k jejich použití.
