# <a name="flowchart-activity-designers"></a>Vývojový diagram návrháře aktivit

Návrháři pracovních postupů Windows zahrnuje několik poskytované systémem aktivit, které můžete použít při vytváření vaše pracovní postupy. Tato část obsahuje poskytované systémem aktivity používat ke správě na vývojových diagramech v pracovním postupu. V následujících tématech popisují tyto aktivity a poskytují pokyny o tom, jak je používat.
