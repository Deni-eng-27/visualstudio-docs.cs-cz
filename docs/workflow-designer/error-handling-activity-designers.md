# <a name="error-handling-activity-designers"></a>Návrháře aktivit zpracování chyb

Návrháře aktivit zpracování chyb se používají ke správě zpracování chyb v Návrháři pracovních postupů Windows. Témata obsažené v této části poskytují pokyny o tom, jak pomocí aktivity v Návrháři pracovních postupů a návrháři šablon.
