# <a name="how-to-configure-workflow-project-properties-legacy"></a>Postupy: Konfigurace vlastností projektu pracovního postupu (zastaralé)
Následujícím postupem lze zobrazit nebo upravit vlastnosti, které jsou spojeny s projekty pracovního postupu.  
  
### <a name="to-view-or-modify-workflow-project-properties"></a>Chcete-li zobrazit nebo upravit vlastnosti projektu pracovního postupu  
  
1.  Pokud **Průzkumníku řešení** okno není viditelná, vyberte **Průzkumníku řešení** na **zobrazení** nabídky.  
  
2.  V **Průzkumníku**, vyberte projektu.  
  
3.  Pokud **vlastnosti** okno není viditelná, vyberte **vlastnosti – okno** na **zobrazení** nabídky.  
  
4.  V projektu **vlastnosti** okně zobrazovat a upravovat vlastnosti, které chcete změnit.
