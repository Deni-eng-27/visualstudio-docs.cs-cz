# <a name="visual-studio-workflow-windows-legacy"></a>Visual Studio pracovního postupu Windows (zastaralé)

Starší verze zadaná v Návrháři pracovních postupů pomocí sady Visual Studio 2010 lze cílit na rozhraní .NET Framework verze 3.5 nebo WinFX. Starší verzi návrháře je vizuálního návrháře, který můžete použít k vytvoření pracovního postupu. Můžete přetáhnout aktivity z **sada nástrojů** na návrh surface a že je nakonfigurovat. Jako okno editoru Visual Studio zabírá oblasti hlavní okno používá jiné oken editoru Visual Studio. Název pracovního postupu se zobrazí v záhlaví okna Visual Studio.

Většina starší verzi návrháře windows by měla být pro zkušené uživatele sady Visual Studio. Následující tabulka popisuje primární systému windows.

|Okno|Popis|
|------------|-----------------|
|Starší verze návrhové plochy|Visual návrháře prostor, který se používá k vytvoření pracovního postupu, který používá rozhraní .NET Framework verze 3.5 nebo WinFX. Poskytuje možnost vytvořit aktivity vizuálně pomocí návrháře aktivit přetažení z **sada nástrojů** na plochu návrháře.|
|Sada nástrojů|Poskytuje grafické reprezentace aktivity, které můžete přetáhnout na plochu návrháře.|
|Vlastnosti|Slouží ke konfiguraci vlastnosti aktivity pracovního postupu.|
|Průzkumník řešení|Projekty workflow se zobrazí v hierarchickém stromovém zobrazení a umožňuje přístup k funkcím, které pomáhají spravovat projekty.|
|Ladění windows|Standardní sady Visual Studio ladění windows, například **vláken**, **zásobníkem volání**, **zarážek**, **místní hodnoty – nástroj**a **Nástroj Sledování** systému windows.|
