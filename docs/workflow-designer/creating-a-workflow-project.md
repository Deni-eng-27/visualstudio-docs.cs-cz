# <a name="creating-a-workflow-project"></a>Vytvoření projektu pracovního postupu

Pracovní postupy, pracovní postup služby Windows Communication Foundation (WCF), vlastních aktivit a Návrháři vlastních aktivit můžete vytvořit pomocí šablony projektů Visual Studio. Tato část popisuje postup vytváření knihoven a aplikace se šablonami projektů, která je k dispozici v sadě Visual Studio.

První čtyři následující témata popisují způsob použití šablon pracovního postupu v nalezen **nový projekt** okno. Tyto šablony jsou přístupné otevřením Visual Studio 2010 a výběr **soubor** > **nový** > **projektu**, pak se  **Pracovní postup** kategorie z **nainstalovaných šablonách** na levé straně, v části buď **jazyka Visual Basic** nebo **Visual C#** uzlu.

Poslední téma popisuje, jak přidat aktivity pracovního postupu, Designer a služeb nebo jiných projektů sady Visual Studio položky do existující pracovního postupu (nebo jiných).
