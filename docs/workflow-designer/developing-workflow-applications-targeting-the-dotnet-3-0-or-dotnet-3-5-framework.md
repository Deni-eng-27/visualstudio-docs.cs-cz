# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>Vývoj aplikací pracovní postup zacílený na rozhraní .NET 3.0 nebo 3.5 rozhraní .NET Framework

Visual Studio 2010 poskytuje návrháře pracovních postupů Windows pro vytváření aplikací pro Windows Workflow Foundation (WF), které cílí na rozhraní .NET Framework 4. Návrháře využívá známé uživatelské rozhraní sady Visual Studio pro grafické vytváření a ladění aplikací pracovního postupu.

Visual Studio 2010 také poskytuje přístup k Návrhář starší verze, které je možné cílit na rozhraní .NET Framework verze 3.5 nebo WinFX. Tato část obsahuje pokyny k použití této starší verzi návrháře.
