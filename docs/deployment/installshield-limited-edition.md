# <a name="installshield-limited-edition"></a>InstallShield Limited Edition
Pomocí InstallShield Limited Edition, můžete vytvořit soubor instalace a distribuovat ji pro uživatele tak, aby si můžou nainstalovat desktopová aplikace nebo součást bez připojení k síti. InstallShield Limited Edition je zdarma pro uživatele edice Visual Studio Professional a Enterprise. Nahradí technologii Instalační služby systému Windows, která již nepodporuje sady Visual Studio. Jako alternativu můžete distribuovat aplikací a součástí s použitím technologie ClickOnce, který vyžaduje připojení k síti. V tématu [ClickOnce – zabezpečení a nasazení](../deployment/clickonce-security-and-deployment.md).  
  
> [!NOTE]
>  Můžete pokračovat pomocí Instalační služby systému Windows projektů vytvořených v dřívějších verzích sady Visual Studio instalací rozšíření projektů Visual Studio Instalační služby. V tématu [instalační program sady Visual Studio projekty rozšíření](http://blogs.msdn.com/b/visualstudio/archive/2014/04/17/visual-studio-installer-projects-extension.aspx).  
  
### <a name="to-enable-installshield-limited-edition"></a>Chcete-li povolit InstallShield Limited Edition  
  
1.  Na řádku nabídek zvolte **soubor**, **nový**, **projektu**.  
  
2.  V **nový projekt** dialogové okno, rozbalte seznam **jiné typy projektů** uzel a potom zvolte **instalace a nasazení** uzlu.  
  
3.  V seznamu šablony, vyberte **Povolit InstallShield Limited Edition**a potom zvolte **OK** tlačítko.  
  
4.  V okně prohlížeče, které se otevře, přečtěte si pokyny a potom zvolte **přejít na web stažení** odkaz.
