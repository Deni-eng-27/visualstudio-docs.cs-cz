## <a name="prerequisites"></a>Požadavky

::: moniker range=">=vs-2019"

* [Visual Studio 2019](https://visualstudio.microsoft.com/downloads) nainstalované s odpovídajícími úlohami pro váš jazyk:
  * ASP.NET: **ASP.NET a vývoj webových aplikací**
  * Node.js: **Vývoj souboru Node.js**
::: moniker-end
::: moniker range="vs-2017"
* [Visual Studio 2017](https://visualstudio.microsoft.com/vs/older-downloads/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=vs+2017+download) nainstalované s odpovídajícími úlohami pro váš jazyk:
  * ASP.NET: **ASP.NET a vývoj webových aplikací**
  * Node.js: **Vývoj souboru Node.js**
::: moniker-end

* Předplatné Azure. Pokud ještě nemáte předplatné, [zaregistrujte se zdarma](https://azure.microsoft.com/free/dotnet/), který zahrnuje 200 dolarů v kreditu po dobu 30 dnů a 12 měsíců populárních bezplatných služeb.

* Projekt ASP.NET, ASP.NET Core, .NET Core nebo Node.js. Pokud ještě projekt nemáte, vyberte možnost níže:
  * ASP.NET jádro: Postupujte podle [funkce Rychlý start: Pomocí sady Visual Studio vytvořte první ASP.NET základní webovou aplikaci](../../ide/quickstart-aspnet-core.md)nebo použijte Nový projekt **souboru** > **New Project**, vyberte **Visual C#** > **.NET Core**a pak vyberte ASP.NET základní **webovou aplikaci**. Po zobrazení výzvy vyberte šablonu **Webové aplikace (Model-View-Controller),** ujistěte se, že není **vybrána možnost Žádné ověřování,** a pak vyberte **OK**.
  * Node.js: Postupujte podle [rychlého startu: Pomocí sady Visual Studio vytvořte první aplikaci Node.js](../../ide/quickstart-nodejs.md)nebo použijte Nový projekt **souboru** > **New Project**, vyberte **JavaScript**a vyberte **webovou aplikaci Blank Node.js**.

* Před provedením kroků nasazení se ujistěte, že jste projekt vytvořili pomocí příkazu nabídky **Sestavit > sestavení řešení.**
