# <a name="visual-studio-tools-for-apache-cordova"></a>Visual Studio Tools pro Apache Cordova
Vytvářet aplikace, které běží na iOS, Android a Windows zařízení a sdílet téměř 100 % kódu. Sestavení je pomocí oblíbené webové technologie. Apache Cordova umožňuje nativní přístup k zařízení funkcí, jako funkce fotoaparátu u telefonního čísla a navíc je zdarma. Visual Studio přidává tyto funkce, aby bylo snazší i k sestavení aplikace:  
  
-   Snadná instalace a nastavení.  
  
-   Rozšířené ladicí program můžete najít a opravit problémy.  
  
-   Snadný přístup k emulátorů zařízení.  
  
-   Návrhář ke správě vašeho zařízení moduly plug-in.  
  
-   Projekt, který je plně kompatibilní s rozhraní příkazového řádku.  
  
 Pokud zvažujete těchto nástrojů, ale nejste připravení zkuste je ještě navštivte naše [funkce](https://www.visualstudio.com/explore/cordova-vs) stránky a my vás informovat o nich víc.  
  
 Pokud jste připravení začít, navštivte naše [webu](http://taco.visualstudio.com/en-us/docs/get-started-vs-tools-apache-cordova/).
