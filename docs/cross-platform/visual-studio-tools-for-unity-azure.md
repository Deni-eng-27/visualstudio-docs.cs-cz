# <a name="programming-with-unity-and-azure"></a>Programování s Unity a Azure

Azure poskytuje škálovatelné řešení k ukládání telemetrie a dalších herní dat v cloudu. S vydáním Unity 2017 Unity na experimentální podpory pro rozhraní .NET 4.6 díky integrace se službou Azure jednodušší než někdy tím, že pomocí sady Azure .NET SDK.

## <a name="experimental-azure-sdks"></a>Experimentální sady Azure SDK

> [!NOTE]
> Tyto sady SDK je nepodporovaná, ale poskytované pomoc zákazníkům vyzkoušení na Unity experimentální .NET 4.6 podporují.

Navštivte [Sandbox](/sandbox/) pokusit následující experimentální Azure SDK s Unity:

* [Úložiště Azure SDK for Unity](https://aka.ms/azstoragegamedev)
* [Centra událostí Azure SDK for Unity](https://aka.ms/azeventhubsgamedev)
* [Azure SDK mobilních aplikací pro Unity](https://aka.ms/azmobileappsgamedev)

## <a name="azure-sdk-sample"></a>Azure SDK – ukázka
