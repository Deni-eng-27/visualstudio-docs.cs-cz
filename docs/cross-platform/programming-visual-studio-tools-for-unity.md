# <a name="program-visual-studio-tools-for-unity"></a>Programování Visual Studio Tools for Unity
V této části najdete příklady pro použití visual studio nástroje pro rozhraní API Unity.

## <a name="examples"></a>Příklady
 Zde jsou některé příklady, které ukazují, jak můžete použít nástroje Visual Studio pro unity API.

### <a name="customize-project-files-created-by-vstu"></a>Přizpůsobení souborů projektu vytvořených službou VSTU
 Visual Studio Tools for Unity poskytuje volání ve stylu Unity během generování souboru projektu. Informace o tom, jak můžete upravit soubor projektu při jeho opětovném generování, naleznete [v tématu Příklad: Generování souboru projektu](../cross-platform/customize-project-files-created-by-vstu.md).

### <a name="share-the-unity-log-callback-with-vstu"></a>Sdílení zpětného volání protokolu Unity s VSTU
 Visual Studio Tools for Unity zaregistruje zpětné volání protokolu s Unity, aby bylo možné streamovat jeho konzole do sady Visual Studio. Pokud skripty editoru také zaregistrovat zpětné volání protokolu s Unity, zpětné volání VSTU může zasahovat do něj. Chcete-li se dozvědět, jak můžete sdílet volání protokolu Unity s VSTU, najdete [v tématu Příklad: Protokolovat zpětné volání](../cross-platform/share-the-unity-log-callback-with-vstu.md).
