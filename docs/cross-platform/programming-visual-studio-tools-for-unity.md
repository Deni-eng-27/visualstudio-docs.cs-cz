# <a name="programming-visual-studio-tools-for-unity"></a>Programování Visual Studio Tools for Unity
V této části najdete příklady pro používání sady Visual Studio Tools pro rozhraní API Unity.

## <a name="examples"></a>Příklady
 Zde jsou některé příklady, které ukazují, jak můžete sady Visual Studio Tools pro rozhraní API Unity.

### <a name="customize-project-files-created-by-vstu"></a>Přizpůsobení souborů projektu vytvořených nástrojem VSTU
 Visual Studio Tools for Unity poskytuje Unity stylu zpětné volání při generování souboru projektu. Informace o tom, jak můžete upravit soubor projektu vždy, když jsou obnoveny, najdete v části [příklad: generování souboru projektu](../cross-platform/customize-project-files-created-by-vstu.md).

### <a name="share-the-unity-log-callback-with-vstu"></a>Sdílení zpětného volání protokolu Unity s VSTU
 Visual Studio Tools for Unity registruje zpětné volání protokolu Unity mohli Streamovat její konzole k sadě Visual Studio. Pokud editor skriptů také zaregistrovat zpětné volání protokolu Unity, může zpětného volání VSTU narušovat ho. Informace o tom, jak můžete sdílet zpětné volání protokolu Unity s VSTU, najdete v části [příklad: zpětné volání protokolu](../cross-platform/share-the-unity-log-callback-with-vstu.md).
