# <a name="using-the-visual-studio-development-environment-for-c"></a>Použití vývojového prostředí sady Visual Studio pro jazyk C# #
Integrované vývojové prostředí (IDE) sady Visual Studio je kolekce nástroje pro vývoj, které jsou k dispozici prostřednictvím společného uživatelského rozhraní. Některé tyto nástroje jsou sdíleny s jinými [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] jazyky a jiné, jako je například kompilátor jazyka C# jsou jedinečné pro Visual C#. Toto téma obsahuje odkazy na nejdůležitější nástroje jazyka Visual C#.
