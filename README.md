# <a name="visual-studio-documentation"></a>Dokumentace sady Visual Studio

Vítáme vás! Toto repo obsahuje zdrojové soubory pro technickou dokumentaci sady Visual Studio. Témata jsou publikována na [docs.microsoft.com](https://docs.microsoft.com/visualstudio).

Tento repo byl přesunut na červen 23, 2017 z https://github.com/Microsoft/vsdocs. Sem je přesměrován provoz na staré adresy URL.

Dokumentace pro Visual Basic a Visual C# je umístěna v [dotnet docs repo](https://github.com/dotnet/docs/tree/master/docs)a dokumentace visual c++ je umístěna v [repo dokumentů jazyka C++](http://github.com/MicrosoftDocs/cpp-docs).

## <a name="contribute-to-the-documentation"></a>Přispějte do dokumentace

Vítáme vaše příspěvky, které nám pomáhají vylepšovat dokumenty sady Visual Studio. Všechny články v tomto úložišti používají značky s příchutí GitHubu.

Několik oblastí funkcí sady Visual Studio mají své vlastní složky v tomto repo, jako je například **ladicí program** pro témata o ladění, **ide** pro témata na Visual Studio interaktivní vývojové prostředí (IDE) a tak dále. Podsložka **/media** v každé složce obsahuje umělecké soubory pro témata.

Další informace o přispívání naleznete v [příručce Přispívání](CONTRIBUTING.md).
