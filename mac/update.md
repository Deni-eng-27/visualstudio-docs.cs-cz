# <a name="update-visual-studio-for-mac"></a>Aktualizace Visual Studia pro Mac

Visual Studio pro Mac distribuuje aktualizace pro rozhraní IDE a podporované architektury v pravidelných intervalech. Tyto aktualizace mohou být ve formě nových funkcí, vylepšení a oprav chyb.

Visual Studio for Mac poskytuje dva kanály pro získání těchto nejnovějších verzí:

* **Stabilní** - Poskytuje důkladně testované aktualizace. Tento kanál je doporučen pro nejlepší vývojové prostředí.
* **Náhled** – poskytuje včasný přístup k aktualizacím, které jsou kandidáty pro vydání ve stabilním kanálu. Tato vydání nemusí být spolehlivá pro každodenní použití.

## <a name="checking-for-updates"></a>Kontrola aktualizací

Pomocí pole **Aktualizace sady Visual Studio** můžete vyhledat nové aktualizace, změnit kanály a stáhnout a nainstalovat aktualizace.

Chcete-li otevřít **aktualizační službu sady Visual Studio**, přejděte do **> Vyhledat aktualizace v sadě Visual Studio**:

![Vyhledat možnost aktualizace](media/update-image1.png)

Zobrazí se pole aktualizačního programu:

![pole aktuaautomatická](media/update-image2.png)

## <a name="changing-the-updater-channel"></a>Změna kanálu aktualizačního programu

Chcete-li změnit kanál, vyberte jej z rozbalovacího seznamu kanálů a stiskněte **tlačítko Přepnout kanál**:

![Změnit rozevírací seznam Kanálu](media/update-image3.png)

## <a name="downloading-and-installing-updates"></a>Stahování a instalace aktualizací

Přepínání kanálů automaticky spustí proces stahování nových aktualizací.

Pokud jste vybrali možnost **Automaticky kontrolovat**, zobrazí se pole aktualizačního programu, když je visual studio pro Mac otevřené, abyste věděli, že jsou k dispozici nové aktualizace. Po zobrazení tohoto pole se do zařízení začne automaticky stahovat nové soubory ke stažení.

Chcete-li začít instalovat aktualizace, vyberte tlačítko **Restartovat a nainstalovat aktualizace:**

![Tlačítko pro zahájení instalace aktualizací](media/update-image4.png)

V závislosti na součástech, které je třeba nainstalovat, může být nutné přijmout další licence nebo zadat uživatelské jméno a heslo správce počítače.
