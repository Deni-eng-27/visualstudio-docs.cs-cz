# <a name="updating-visual-studio-for-mac"></a>Aktualizace sady Visual Studio pro Mac

Visual Studio pro Mac distribuuje aktualizace IDE a podporovaných architektur v pravidelných intervalech. Tyto aktualizace může být ve tvaru nové funkce, vylepšení a opravy chyb.

Visual Studio pro Mac poskytuje tři kanály získat tyto nejnovější verze:

* **Stabilní** -poskytuje důkladně neotestovali aktualizace. Tento kanál se doporučuje pro dosažení co nejlepších výsledků vývoj.
* **Beta** -poskytuje časná přístup k aktualizace, které jsou kandidáty pro použití v kanálu stabilní verze.
* **Alpha** -poskytuje časná přístup k aktualizace, které přidat nové funkce, ale nemusí být spolehlivé pro každodenní použití.

## <a name="checking-for-updates"></a>Kontrola aktualizací

Můžete použít **Visual Studio aktualizační** pole na kontrolu nových aktualizací, změnit kanálů a stáhnout a nainstalovat aktualizace.

Otevřete **Visual Studio aktualizační**, přejděte do **Visual Studio > Zkontrolovat aktualizace...** :

![Zkontrolujte možnosti aktualizace](media/update-image1.png)

Zobrazí se pole aktualizační:

![aktualizační metoda pole](media/update-image2.png)

## <a name="changing-the-updater-channel"></a>Změna aktualizační kanál

Chcete-li změnit je z tohoto kanálu rozevírací nabídky vyberte kanál a stiskněte klávesu **tlačítko přepínač kanál**:

![Změna kanál rozevíracího seznamu](media/update-image3.png)

## <a name="downloading-and-installing-updates"></a>Stahování a instalace aktualizací

Přepínání kanály automaticky spustí stahování proces nové aktualizace.

Pokud jste vybrali možnost **zkontrolujte automaticky**, pole aktualizační objeví se při Visual Studio pro Mac je otevřená a vám oznamuje, že jsou k dispozici nové aktualizace. Je také získáte možnost spustit, je stáhnout.

Chcete-li spustit instalaci aktualizací, vyberte **Restart a instalovat aktualizace** tlačítko:

![Tlačítko Spustit instalaci aktualizace](media/update-image4.png)

V závislosti na součásti, které je potřeba nainstalovat musíte přijmout další licence nebo zadejte uživatelské jméno správce vašeho počítače a heslo.
