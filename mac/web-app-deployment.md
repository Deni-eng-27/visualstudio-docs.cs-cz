# <a name="aspnet-core-web-app-deployment"></a>Nasazení webové aplikace ASP.NET Core

Důležitým krokem při vývoji webových aplikací je publikování (nasazení). Můžete publikovat ve vzdáleném prostředí, aby ostatní mohli používat aplikaci nebo pro účely testování. Visual Studio pro Mac nabízí dvě možnosti publikování aplikace. Publikování na Azure App Service umožňuje publikovat webovou aplikaci do Azure App Service. Možnost publikovat do složky umožňuje publikování aplikace do složky.

## <a name="publish-to-azure-app-service"></a>Publikování do Azure App Service

Pomocí Visual Studio pro Mac můžete publikovat webovou aplikaci přímo do Azure App Service. Chcete-li publikovat webovou aplikaci ASP.NET Core pro Azure App Service Přečtěte si článek věnované [publikování na Azure App Service](publish-app-svc.md) .

## <a name="publish-to-folder"></a>Publikování do složky

Webové aplikace ASP.NET Core můžete také publikovat do složky. Další informace o tom, jak můžete webovou aplikaci publikovat do složky, najdete v tématu věnovaném [publikování do složky](publish-folder.md).
