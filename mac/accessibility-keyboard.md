# <a name="keyboard-navigation-in-visual-studio-for-mac"></a>Navigace pomocí klávesnice ve Visual Studiu pro Mac

Visual Studio for Mac je přístupné uživatelům klávesnice, kteří používají nejběžnější klávesové zkratky macOS, a také mnoho, které jsou specifické pro Visual Studio for Mac. Tady se dozvíte o některých základních klávesových příkazech pro oba a o tom, jak nastavit klávesové zkratky Visual Studia for Mac.

## <a name="use-keyboard-navigation"></a>Použití navigace pomocí klávesnice

Podpora navigace pomocí klávesnice je integrovaná přímo do macOS. Chcete-li mít co nejkomplexnější prostředí, nastavte macOS tak, aby přesouvá fokus klávesnice mezi **všemi ovládacími prvky**. To umožňuje procházet všechny ovládací prvky v okně nebo dialogovém okně.

![Snímek obrazovky s možnostmi klávesnice s vybranými ovládacími prvky](media/accessibility-preferences-keyboard.png)

## <a name="customize-keyboard-shortcuts"></a>Přizpůsobení klávesových zkratek

Visual Studio pro Mac má řadu klávesových zkratek pro různé úkoly. Tyto zkratky můžete zobrazit a upravit následujícím způsobem:

1. Vyberte **předvolby sady Visual Studio (nabídka)** > **(&#8984;,)**.

1. Přejděte na**vazby klíčů** **prostředí** > .

1. Procházejte seznamem nebo vyhledejte zástupce nebo příkaz.

1. Vyberte příkaz.

1. Vyberte **vazbu klíče**a zadejte kombinaci kláves.

1. Vyberte **Použít**.

> [!TIP]
> Můžete si vybrat ze seznamu schémat vazby klíčů, aby co nejlépe vyhovovalvašim potřebám, pokud jste obeznámeni s jinými běžnými klávesovými zkratkami (například Visual Studio v systému Windows).
