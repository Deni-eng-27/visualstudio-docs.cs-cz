# <a name="task-comments"></a>Úlohy komentářů

Při psaní kódu, je standardní zvykem explicitně komentář ke kódu nedokončené nebo sporná nebo rychlé řešení s upozorněními. Tokeny signál výchozí poskytované sadě Visual Studio pro Mac jsou TODO, HACKERSKÝ, FIXME a UNDONE. Přizpůsobené tokeny lze definovat v části **Visual Studio > Předvolby... > prostředí > úlohy**, jak je znázorněno na následujícím obrázku:

 ![Předvolby seznamu úkolů](media/source-editor-image10.png)

Pokud chcete přidat nový komentář úloh, přidáte komentář, který obsahuje klíčové slovo úloh. Příklad:

```csharp
//TODO: Finish this for all properties.
```

Visual Studio pro Mac zdůrazňující tyto značky podle jejich zvýraznění v panelu pro seznam úkolů, které lze zobrazit přechodem na **zobrazení > dotyková zařízení > úloh**:

![Panel seznamu úloh](media/source-editor-image11.png)
