# <a name="installing-preview-releases"></a>Instalace Preview verze

Předtím, než je oficiálním vydání nové verze sady Visual Studio pro Mac, je vydána jako náhled. Ve verzi preview vám dává možnost vyzkoušet nové funkce a získat nejnovější opravy chyb, než jsou plně součástí produktu.

Verze Preview pro Visual Studio pro Mac se distribuují jako aktualizace, a nikoli prostřednictvím samostatný soubor ke stažení. Visual Studio pro Mac má tři aktualizační kanály, jak je popsáno v [aktualizace](update.md) článek: stabilní, Beta a alfa.

Většina verze preview bude k dispozici prostřednictvím i **Beta** a **Alpha** kanály, ale vždy zkontrolujte [poznámky k verzi Preview](/visualstudio/releasenotes/vs2017-mac-preview-relnotes) nejaktuálnější informace.

K instalaci preview sady Visual Studio pro Mac, použijte následující kroky:

1. Přejděte na **sady Visual Studio > Zkontrolovat aktualizace...** .
2. Vyberte v poli se seznamem kanálu aktualizace **Beta**.
3. Vyberte **přepínač kanál** tlačítko Přepnout vybraný kanál a začít stahovat nějaké nové aktualizace.
4. Vyberte **Restart a instalovat aktualizace** tlačítko Spustit instalaci aktualizace.

Další informace o aktualizaci v sadě Visual Studio pro Mac, najdete v článku [aktualizace](update.md) článku.
