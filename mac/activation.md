# <a name="activation"></a>Aktivace

Visual Studio pro Mac nabízí tři úrovně jiného předplatného můžete vybírat, všechny přístupné z jeden soubor ke stažení:

* Visual Studio Enterprise pro Mac
* Visual Studio Professional pro Mac
* Visual Studio Community pro Mac

Pokud chcete začít používat předplatné, přihlaste se k sadě Visual Studio pro Mac podle pokynů v [přihlášení k sadě Visual Studio pro Mac](~/signing-in.md) průvodce. To bude automaticky aktivovat licence na aplikaci a zobrazit ji v dialogovém okně účty:

![Zobrazit dialogové okno uživatelské licence](media/user-accounts-login.png)

## <a name="update-expired-licenses"></a>Aktualizovat vypršela platnost licence 

Pokud vypršela platnost předplatného, máte dvě možnosti:

1. Obnovte předplatné. Předplatné můžete procházením [https://www.visualstudio.com/vs/pricing/](https://www.visualstudio.com/vs/pricing/).

2. Pokud máte jiný účet s aktivní předplatné, můžete ho přidat do sady Visual Studio pro Mac podle popisu v [přidání víc uživatelských účtů](~/signing-in.md) tématu. Visual Studio pro Mac zjistí nejlépe dostupnou licenci ze všech účtů, které jste přidali. 

## <a name="product-key-usage"></a>Použití kódů Product key

V současné době neexistuje žádný způsob, jak využívat kód product key pro povolení Professional nebo Enterprise oprávnění pro Visual Studio for Mac. Pokud chcete používat Visual Studio Enterprise pro Mac nebo Visual Studio Professional pro počítače Mac musíte mít relevantní [předplatné](https://www.visualstudio.com/subscriptions/) a být [přihlášení](~/signing-in.md) k prostředí IDE.

## <a name="offline-activation"></a>Offline aktivaci

> [!NOTE]
> V současné době není žádné offline aktivaci pro sadu Visual Studio for Mac. Pokud chcete aktivovat předplatné, musíte být připojení k Internetu při přihlášení.

Pokud chcete aktivovat 2017 vaší Visual Studio pro Mac Professional nebo Enterprise předplatné, musíte být připojení k Internetu při přihlášení.  

**Poznámka:**: v současné době neexistuje žádné offline aktivace pro Visual Studio Professional pro Mac nebo Visual Studio Enterprise for Mac. 2017 Visual Studio pro Mac však lze použít v prostředích bez připojení k Internetu v souladu s licence.

Pokud jste zakoupili Visual Studio Enterprise 2017 pro systém Mac pro počítače bez připojení k Internetu, můžete pokračovat bez aktivace, ale Xamarin, nástroj Inspector profileru Xamarin, použijte sadu Visual Studio pro Mac a vložené sestavení funkce nebudou k dispozici.
