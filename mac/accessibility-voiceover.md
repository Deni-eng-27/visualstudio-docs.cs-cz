# <a name="using-voiceover-with-visual-studio-for-mac"></a>Použití VoiceOveru s Visual Studio pro Mac

Uživatelé, kteří vyžadují čtečku obrazovky, budou mít prospěch z macOS VoiceOveru. Visual Studio pro Mac je přístupné pomocí běžných příkazů VoiceOveru.

## <a name="how-to-enable-and-use-voiceover"></a>Postup: Povolení a používání VoiceOveru

Povolení nebo zakázání voiceoveru stiskněte **&#8984;F5**

VoiceOver příkazy se zobrazí v této příručce jako ** _Klíč_ VO,** čímž **VO** odkazuje na modifikátor nastavit v aplikaci **VoiceOver Utility.** Výchozí modifikátor je **.-**. Například v závislosti na modifikátoru VoiceOver bude **VO M** znamenat **-M**. Kurzorové klávesy budou **označovány**jako ← → ↑ ↓ .

## <a name="useful-voiceover-commands"></a>Užitečné příkazy VoiceOveru

Tady je seznam některých běžných příkazů VoiceOveru pro Visual Studio for Mac:

|Funkce|Popis|Zástupce|
|-------|-----------|--------|
|Navigace|Navigace ve všech prvcích uživatelského rozhraní|**VO→**, **VO←**|
|Navigace|Krok do nebo z prvků uživatelského rozhraní|**VO↓↓**, **VO?**|
|Interakci|Výběr ovládacího prvku zaostřené nebo s ním pracovat|**VO.**|
|Interakci|Interakce s panelem nabídek Visual Studia for Mac|**VO M**|

> [!TIP]
> Další informace o používání VoiceOveru a úplný seznam příkazů najdete v [průvodci VoiceOver EmStarted od společnosti Apple](https://support.apple.com/en-us/guide/voiceover-guide/welcome/web).
