# <a name="net-core-support"></a>Podpora platformy .NET Core

Následující tabulka popisuje verzích .NET Core nepodporuje stabilní verze a preview verze sady Visual Studio pro Mac:

Verze sady SDK .NET core  |Visual Studio for Mac 8.1 (stabilní verze)  |Visual Studio for Mac 8.2 (Preview)  |
|---------|---------|---------|
|v2.1.0 - v2.1.5xx    |✗|✗|
|v2.1.600 +     |✔︎|✔︎|
|v2.2.1 - v2.2.1xx|✗|✗|
|v2.2.200 + |✔︎| ✔︎ |
|verze 3.0 < ve verzi Preview 3 |✗|✗|
|verze 3.0 ve verzi Preview 3, 4, 5 |✔︎|✔︎ |
|verze 3.0 ve verzi Preview 6 |✗|✔︎ |

> [!IMPORTANT]
> Pokud jste dříve používali v2.2.1xx .NET Core pomocí sady Visual Studio pro Mac 8.0, bude nutné ručně aktualizovat na podporovanou verzi .NET Core, jak je uvedeno v předchozí tabulce. Doporučujeme, abyste buď [2.1.700](https://dotnet.microsoft.com/download/dotnet-core/2.1) nebo [2.2.300](https://dotnet.microsoft.com/download/dotnet-core/2.2)

* .NET core v2.1.700 nainstalovaný ve výchozím nastavení s instalačním programem.
* Stáhnout verzi .NET Core, najdete v tématu [dotnet stránky](https://dotnet.microsoft.com/download/dotnet-core).
* Informace o instalaci verze preview sady Visual Studio pro Mac, najdete v článku [nainstalovat si Preview verzi](https://docs.microsoft.com/visualstudio/mac/install-preview) průvodce.
