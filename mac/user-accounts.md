# <a name="user-accounts"></a>Uživatelské účty

Jakmile se mají stáhnout a nainstalovat Visual Studio pro Mac, musíte se přihlásit k aktivaci předplatného zajistí, že přístup všech funkcí sady Visual Studio for Mac.

Pomocí přihlášení, povolíte přístup ke všem službám, které jsou spojené s vaším uživatelským účtem, jakož i všechny účty, které jsou propojeny k tomuto účtu. Nastavení svého účtu přesouvat s vámi, bez ohledu na použijte sadu Visual Studio. Úrovní oprávnění můžete nakonfigurovat na konkrétní instanci sady Visual Studio pro jednotlivé uživatele.
