# <a name="user-accounts"></a>Uživatelské účty

Po stažení a instalaci Sady Visual Studio pro Mac se musíte přihlásit, abyste aktivovali předplatné, abyste získali přístup ke všem funkcím Visual Studia for Mac.

Přihlášením povolíte přístup ke všem službám přidruženým k vašemu uživatelskému účtu a ke všem účtům, které jsou s tímto účtem propojeny. Nastavení vašeho účtu se s vámi potuluje, ať používáte Visual Studio kamkoli. Úrovně oprávnění můžete nakonfigurovat pro konkrétní instanci sady Visual Studio pro jednotlivé uživatele.
