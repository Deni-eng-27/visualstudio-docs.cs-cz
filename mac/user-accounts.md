# <a name="user-accounts"></a>Uživatelské účty

Jakmile se mají stáhnout a nainstalovat Visual Studio pro Mac, musíte se přihlásit k aktivaci licenci a získat přístup k buď edice Community, Professional nebo Enterprise Xamarin Studio.

Pomocí přihlášení, povolíte přístup ke všem službám, které jsou spojené s vaším uživatelským účtem a také všechny účty, které jsou propojeny k tomuto účtu. Nastavení svého účtu přesouvat s vámi, bez ohledu na použijte sadu Visual Studio. Úrovní oprávnění můžete nakonfigurovat na konkrétní instanci sady Visual Studio pro jednotlivé uživatele.
