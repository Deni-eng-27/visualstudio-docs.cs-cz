# <a name="razor-support"></a>Podpora pro Razor

Visual Studio for Mac podporuje úpravy Razor, včetně Technologie IntelliSense a zvýraznění syntaxe v souborech *.cshtml.*

![úpravy holicí strojek ve visual studiu pro Mac](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Začínáme s Razor v Visual Studiu pro Mac

Existují dvě možnosti, které je třeba zvážit, když začínáte s Razor v Sadě Visual Studio pro Mac: Razor Pages v ASP.NET Core a ASP.NET Core MVC. Pro výukové programy a další informace o obou možnostech navštivte jeden z níže uvedených pokynů:

- [Začínáme s Razor Pages v ASP.NET Core na macOS s Visual Studio pro Mac](/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Začínáme s ASP.NET Core MVC a Visual Studio pro Mac](/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
