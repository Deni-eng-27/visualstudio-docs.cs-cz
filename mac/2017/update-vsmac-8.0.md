# <a name="update-visual-studio-2017-for-mac-version-7-to-visual-studio-2019-for-mac-version-80"></a>Aktualizace Visual Studia 2017 pro Mac (verze 7) na Visual Studio 2019 pro Mac (verze 8.0)

Visual Studio 2019 pro Mac přináší spoustu nových vylepšení: zbrusu nový editor C#, nové spuštění, podpora pro více instancí a řadu nových vylepšení pro vývojáře Xamarin, Unity a .NET Core. Další informace o všem, co je součástí Visual Studia for Mac 2019, najdete v následujících blogových příspěvcích:

- [Visual Studio 2019 pro Mac je teď k dispozici](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-is-now-available/)
- [Visual Studio 2019 pro Mac verze 8.1 je teď k dispozici (a preview pro 8.2)](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-version-8-1-is-now-available-and-a-preview-for-8-2/)

> [!IMPORTANT]
> Existuje známý problém s Visual Studio 2017 pro Mac **verze 7.7.3**, kde updater bude průběžně zobrazovat "Chyba načítání informací o aktualizaci". Pokud k tomu dojde, [stáhněte si Visual Studio pro Mac](https://visualstudio.microsoft.com/vs/mac/) a podle pokynů [nainstalujte](/visualstudio/mac/installation?view=vsmac-2019) Visual Studio 2019 pro Mac. Všimněte si, že to přepíše existující instance Visual Studia 2017 pro Mac.

Chcete-li získat nejnovější verzi, můžete buď použít [updater](/visualstudio/mac/update?view=vsmac-2017) procházením **sady Visual Studio > Vyhledat aktualizace...**:

![dialogové okno aktuajek](media/update-vsmac-updater.png)

Nebo, pokud dáváte přednost nebo máte problémy s aktualitou, [stáhněte si](https://visualstudio.microsoft.com/vs/mac/) zbrusu nový instalační program 2019 a rychle zprovozněte:

![instalační obrazovka](media/update-vsmac-installer.png)
