# <a name="install-a-preview-release"></a>Instalace verze Preview

> [!NOTE]
> Visual Studio 2019 pro Mac preview je [teď dostupné pro instalaci](/visualstudio/mac/installation/?view=vsmac-2019) a testování.

## <a name="install-an-update-for-visual-studio-for-mac"></a>Instalace aktualizace pro Visual Studio pro Mac

Před oficiálním vydáním nové verze Visual Studia for Mac je dostupná jako náhled. Verze preview vám dává možnost vyzkoušet si nové funkce a získat nejnovější opravy chyb předtím, než budou plně začleněny do produktu.

Verze preview do Visual Studia for Mac se distribuují jako aktualizace, nikoli prostřednictvím samostatného stahování. Visual Studio pro Mac má tři kanály updater, jak je popsáno v článku [aktualizace:](update.md) Stabilní, Beta a Alfa.

Většina verzí preview bude k dispozici prostřednictvím kanálů **Beta** i **Alpha,** ale vždy zkontrolujte, zda jsou [v poznámkách k verzi preview](/visualstudio/releasenotes/vs2017-mac-preview-relnotes) nejpřesnější informace.

Pokud chcete nainstalovat náhled Visual Studia pro Mac, postupujte takto:

1. Přejděte na **soubor Visual Studio > Vyhledat aktualizace**.
2. V poli Se seznamem Aktualizovat kanál vyberte **možnost Beta**.
3. Výběrem tlačítka **Přepnout kanál** přepněte na vybraný kanál a začněte stahovat nové aktualizace.
4. Chcete-li začít instalovat aktualizace, vyberte tlačítko **Restartovat a nainstalovat aktualizace.**

Další informace o aktualizaci ve Visual Studiu pro Mac najdete v článku [o aktualizaci.](update.md)
