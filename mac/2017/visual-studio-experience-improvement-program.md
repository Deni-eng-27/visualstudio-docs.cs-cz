# <a name="visual-studio-customer-experience-improvement-program"></a>Program Zlepšování softwaru a služeb na základě zkušeností uživatelů pro Visual Studio

Visual Studio Customer Experience Improvement Program (VSCEIP) je program určený k pomoci společnosti Microsoft zlepšit Visual Studio pro Mac v průběhu času. Tento program shromažďuje informace o hardwaru počítače a o tom, jak uživatelé používají aplikaci Visual Studio for Mac, aniž by uživatelé přerušili jejich úlohy v počítači. Shromažďované informace pomáhají společnosti Microsoft určit, které funkce je třeba zlepšit. Tento článek popisuje, jak se přihlásit nebo z VSCEIP.

Další informace o informacích shromážděných, zpracovávaných nebo přenášených programem VSCEIP naleznete v [Prohlášení společnosti Microsoft o zásadách ochrany osobních údajů](https://privacy.microsoft.com/privacystatement).

## <a name="choice-and-control-over-the-visual-studio-customer-experience-improvement-program"></a>Výběr a kontrola nad programem Zlepšování softwaru a služeb na základě zkušeností uživatelů sady Visual Studio

VSCEIP je ve výchozím nastavení zapnutý. Můžete jej vypnout nebo znovu zapnout podle následujících pokynů.

1. Otevřete Visual Studio pro Mac.

1. Na řádku nabídek zvolte **Předvolby > sady Visual Studio > jiná > zpětnou vazbu**.

1. Vyberte si, jak preferujete účast:
