# <a name="accessibility"></a>Přístupnost

Existuje celá řada nástrojů a funkcí usnadnění, které jsou integrovány přímo do macOS, které vám pomohou používat visual studio pro Mac.  Kromě těchto, Visual Studio pro Mac má také následující funkce, aby bylo přístupnější pro lidi s různými schopnostmi:

* Zvětšení textu editoru
* Zvětšení textu panelu
* Přizpůsobení barevného motivu editoru kódu
* Přizpůsobení klávesových zkratek
* Procházení pomocí klávesnice

> [!NOTE]
> Tato příručka odkazuje na Visual Studio pro Mac. V sadě Visual Studio ve Windows najdete [v tématu Funkce usnadnění v sadě Visual Studio](/visualstudio/ide/reference/accessibility-features-of-visual-studio).
