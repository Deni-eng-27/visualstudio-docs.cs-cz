# <a name="managing-app-resources"></a>Správa prostředků aplikace

Soubory prostředků aplikace, například bitové kopie, textové soubory a zvukových souborů jsou nedílnou součástí aplikace, které nejsou kompilovat s aplikací. Každou platformu, která je podporována sadou Visual Studio pro Mac zpracovává těchto typů souborů různými způsoby. Odkazovat na příručky níže Další informace o práci s prostředky aplikace v sadě Visual Studio for Mac.

## <a name="xamarinforms"></a>Xamarin.Forms

Spustí kód Xamarin.Forms ve více platformách - z nichž každá má svou vlastní systému souborů. To znamená, že čtení a zápis souborů je lze provést pomocí nativních rozhraní API souboru na každou platformu nebo přidáním soubory jako vložení zdrojů.

* [Práce s obrázky](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Práce se soubory]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
