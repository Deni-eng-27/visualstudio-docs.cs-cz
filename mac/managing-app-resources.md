# <a name="managing-app-resources"></a>Správa prostředků aplikace

Soubory prostředků aplikace, jako jsou obrázky, textové soubory a zvukové soubory, jsou nezbytné pro vaši aplikaci, ale nejsou zkompilovaná aplikace. Jednotlivé platformy podporované sadou Visual Studio pro Mac zpracovává tyto prostředky různými způsoby, jak je popsáno v následujících příručkách:

## <a name="xamarinforms"></a>Xamarin.Forms

Xamarin.Forms kód se spustí na více platformách – z nichž každý má svůj vlastní systém souborů a každý soubor systém určí čtení a zápis do souborů. V Xamarin.Forms umožňuje spravovat prostředky aplikace pomocí souboru nativní rozhraní API na každou platformu, nebo přidáním souborů jako vložené prostředky.

* [Práce s obrázky](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Práce se soubory]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
