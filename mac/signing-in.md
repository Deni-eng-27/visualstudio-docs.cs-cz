# <a name="signing-in-to-visual-studio-for-mac"></a>Přihlášení k sadě Visual Studio pro Mac

Při spuštění sady Visual Studio pro Mac poprvé, zobrazí se výzva k přihlášení pomocí účtu Microsoft. Používejte účet Microsoft, která je připojena k licenci, která chcete použít. Pokud nemáte účet Microsoft, přečtěte si téma [jak přihlásím účet](https://support.microsoft.com/instantanswers/d18cc497-d839-cf50-dea8-f99c95f2bd16/sign-up-for-a-microsoft-account). 

Pokud se rozhodnete není okamžitou přihlásit, nebudete moci používat zkušební kopie po dobu 30 dnů. Po 30 dnech se musí přihlásit k pokračovat v používání vaší kopie Visual Studio pro Mac 

## <a name="how-to-sign-in-to-visual-studio-for-mac"></a>Jak se přihlásit k sadě Visual Studio pro Mac

Před přihlášením k sadě Visual Studio pro Mac Ujistěte se, že jste připojeni k Internetu. Odběry může aktivovat jenom online.
 
Pro přihlášení k sadě Visual Studio pro Mac, postupujte takto:

1. Klikněte na přihlašovací odkaz na úvodní stránce k zobrazení dialogu účty a stiskněte klávesu **přihlásit** tlačítko:

    ![](media/signing-in-image12.png)

2. Zadejte své přihlašovací údaje Microsoft:

    ![](media/signing-in-image13.png)

    Všimněte si, že Xamarin přihlašovací údaje nelze použít pro přihlášení k sadě Visual Studio for Mac.

3. 1.   Jakmile budete mít úspěšně přihlášen, zobrazí se váš typ licence. Z tohoto dialogového okna můžete přidat další účty Microsoft, které byste mohli, že používáte:

    ![](media/signing-in-image14.png)

## <a name="adding-multiple-user-accounts"></a>Přidání víc uživatelských účtů

Visual Studio pro Mac podporuje přidávání více účtů ke svému účtu individuální nastavení. Tyto další účty vám umožní přístup k prostředkům, jako je Azure, z libovolného přidání účtu. 

Pokud chcete přidat další uživatelské účty, postupujte podle kroků v [jak se přihlásit k sadě Visual Studio pro Mac](#How_to_sign_in_to_Visual_Studio_for_Mac) části.
