# <a name="visual-studio-customer-experience-improvement-program"></a>Programu zlepšování zkušeností zákazníků v sadě Visual Studio

Visual Studio zákazníka prostředí zlepšování Program (VSCEIP) je program pomáhá společnosti Microsoft zlepšovat Visual Studio pro Mac v čase. Tento program shromažďuje informace o hardwaru počítače a jak lidé použijte sadu Visual Studio pro Mac, aniž by to ovlivnilo uživatele v jejich úlohy v počítači. Shromažďované informace pomáhají společnosti Microsoft určit funkce, které ke zlepšení. Tento dokument se zabývá aktivování nebo zrušení VSCEIP.

Další informace o informacích shromažďovaných, zpracovávaných a přenášených v rámci VSCEIP, najdete v článku [prohlášení o ochraně osobních údajů Microsoft](https://privacy.microsoft.com/privacystatement).  

## <a name="choice-and-control-over-the-visual-studio-customer-experience-improvement-program"></a>Možnost volby a kontrolu nad programu zlepšování zkušeností zákazníků Visual Studio

VSCEIP je zapnutá ve výchozím nastavení. Můžete ho vypnout, nebo zpět na znovu, postupujte podle pokynů níže.

1. Otevřete Visual Studio for Mac.

1. V řádku nabídek zvolte **Visual Studio > Předvolby > jiné > zpětné vazby**.

1. Vyberte zúčastníte:
