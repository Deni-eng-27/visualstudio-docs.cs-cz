# <a name="how-to-report-a-problem-in-visual-studio-for-mac"></a>Jak chcete nahlásit problém v sadě Visual Studio pro Mac

Pokud dojde k potížím s Visual Studio pro Mac, chceme vědět o něm. Tady je postup, chcete nahlásit problém, který chcete [komunity vývojářů](https://developercommunity.visualstudio.com/) tak, aby nám můžete diagnostikovat a opravit ho.

## <a name="how-to-report-a-problem"></a>Jak ohlásit problém

K ohlášení problému se sadou Visual Studio pro Mac, vyberte **Nápověda > ohlásit problém** z řádku nabídek:

![nahlásit problém odkaz](media/report-problem-image1.png)

Můžete také použít **nahlásit problém** odkaz umístěný na úvodní stránce.

Zobrazí se **nahlásit problém** webové stránky. Na této stránce můžete zobrazit všechny stávající problémy, Hlasujte o nich s podobnými problémy a odeslat řešení problémů hlášených jiní lidé:

![nahlásit problém webovou stránku](media/report-problem-image2.png)

Vyberte **nahlásit problém** tlačítko na webové stránce.

Pokud už jste nepřihlásili do komunity vývojářů aplikace, zobrazí se výzva, tak na následující stránce. Vyberte **přijmout** tlačítko Autorizovat komunity vývojářů v aplikaci pro svůj účet Microsoft:

![nahlásit problém webovou stránku](media/report-problem-image3.png)

Zadejte popisný název problému společně s jakékoli další informace o tomto problému, který máte pocit, že může být užitečné. Pokud je to možné, uveďte prosím kroky, jako to umožňuje nám to pochopit a opravit příslušný problém nejkratší možné době.

![nahlásit problém webovou stránku](media/report-problem-image4.png)

## <a name="provide-a-suggestion"></a>Poslat návrh

Použití **poslat návrh** odkaz na úvodní stránce směrovat na [Visual Studio for Mac navrhnout funkci stránky](https://developercommunity.visualstudio.com/content/idea/post.html?space=41). Odsud můžete přidat nové požadavky nebo hlasovat o existujících žádostí.
