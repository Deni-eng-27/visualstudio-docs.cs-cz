#   <a name="how-to-buy-a-subscription"></a>Jak koupit předplatné
Existují tři způsoby, jak zakoupit předplatné sady Visual Studio.  
- Cloudová předplatná, může být zakoupeno prostřednictvím Visual Studio Marketplace.
- Standardní odběry může být zakoupeno prostřednictvím Microsoft Store. 
- Organizace můžete zakoupit předplatné multilicence prostřednictvím prodejce.  

## <a name="buying-cloud-subscriptions-through-the-visual-studio-marketplace"></a>Kdybyste kupovali Cloudová předplatná prostřednictvím Visual Studio Marketplace
Můžete si koupit Visual Studio Professional a Visual Studio Enterprise [odběry](https://www.visualstudio.com/subscriptions/) z [Visual Studio Marketplace](https://marketplace.visualstudio.com). Další informace o naplno využívat výhody pro tyto odběry [porovnání funkcí](https://www.visualstudio.com/vs/compare/) IDE, a najít [ceny předplatné](https://www.visualstudio.com/vs/pricing/).

### <a name="before-you-start"></a>Než začnete

*   K vyúčtování vaše nákupy, budete potřebovat [předplatné](https://azure.microsoft.com/en-us/pricing/purchase-options/). Můžete [zaregistrovat](https://account.windowsazure.com/Subscriptions) před první nákupu nebo během první nákup ve Visual Studio Marketplace.
Další informace, které [předplatná Azure](https://docs.microsoft.com/en-us/vsts/marketplace/marketplace-billing-qa) můžete použít pro nákupy Visual Studio Marketplace. 

*   Zaregistrujte si předplatné Azure a kupte si Visual Studio Marketplace, budete potřebovat ["osobní" účet Microsoft](https://www.microsoft.com/account) nebo ["pracovní nebo školní účet"](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/) Pokud jste nákup pro vaši organizaci, a jejich Správa identit s Azure Active Directory (Azure AD).

## <a name="buying-standard-subscriptions-through-the-microsoft-store"></a>Kdybyste kupovali standardní odběry prostřednictvím Microsoft Store
Visual Studio standardní odběry jsou k dispozici prostřednictvím Microsoft Store.  Se koupit nebo obnovení předplatného:

1. Připojení k [Microsoft Store](https://www.microsoft.com).
2. Klikněte na **softwaru** kartě.
3. Vyberte **odborníky v oblasti softwaru**.

<img alt="Software Menu" src="_img/buy-vs-subscriptions/professional-software.png" style="border: 1px solid #CCCCCC" />

4. V **nakupovat kategorie** vyberte **Visual Studio**.

<img alt="Categories List" src="_img/buy-vs-subscriptions/shop-categories.png" style="border: 1px solid #CCCCCC" />

5. V **Visual Studio obchod** vyberte předplatné, které chcete koupit. (Visual Studio Professional 2017 je samostatnou licenci pro prostředí IDE a nezahrnuje předplatné.)

<img alt="Visual Studio Products" src="_img/buy-vs-subscriptions/shop-visual-studio.png" style="border: 1px solid #CCCCCC" />

6. Na stránce popis produktů budete mít možnost zakoupení nové předplatné nebo zakoupení nové.  Vyberte typ požadované předplatné a přidat do košíku. 

<img alt="Visual Studio Enterprise options" src="_img/buy-vs-subscriptions/enterprise-options.png" style="border: 1px solid #CCCCCC" />

7. V košíku, klikněte na tlačítko **najdete v článku věnovaném** tlačítko přesunout do **zkontrolujte a umístěte pořadí** stránky.  (Pravděpodobně muset přihlásit ke svému Account Microsoft pokračujte.) 

8. Vyberte typ platby a klikněte na tlačítko **vyberte**.

9. Pokud máte platebního nástroje u souboru, vyberte ho.  Chcete-li přidat platebního nástroje, jako je například platební kartu, zadejte podrobnosti pro vaším platebním nástrojem a klikněte na tlačítko **Uložit**.

10. Klikněte na tlačítko **odešlete objednávku** k dokončení nákupu.  

### <a name="activating-subscriptions-purchased-through-the-microsoft-store"></a>Aktivace předplatné zakoupené prostřednictvím Microsoft Store

Pokud jste zakoupili předplatné Visual Studio prostřednictvím Microsoft Store, kroky aktivace lze inicializovat z platebních potvrzovací stránku nebo potvrzení e-mailu. Poznámka: Pokud jste neobdrželi e-mailu, prosím ověřit e-mailovou adresu a zkontrolujte vaše složky nevyžádané pošty.

Aktivace předplatného: 

1. Klikněte na **kliknutím sem se přihlaste**. Budete přesměrováni na [https://my.visualstudio.com/subscriptions/activate](https://my.visualstudio.com/subscriptions/activate)

2. Zadejte e-mailovou adresu, které budete používat pro přístup k předplatnému Visual Studio a klikněte na tlačítko **pokračovat**.

3. V závislosti na e-mailovou adresu, může přesměrováni na buď **účet Microsoft (MSA)** přihlašovací stránce nebo **práci nebo ve škole** přihlašovací stránky účtu. 

4. Zadejte heslo nebo přihlášení požadavky.
5. Po úspěšném přihlášení **aktivovat vaše Visual předplatné Studio** se okno.
6. Zadejte **kód Product Key** informací dostupných ze platebních potvrzovací stránku nebo potvrzení e-mailu a klikněte na tlačítko **aktivovat**.

7. Vaše předplatné by teď měly být viditelné na [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions).
