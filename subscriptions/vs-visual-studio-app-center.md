# <a name="visual-studio-app-center-in-visual-studio-subscriptions"></a>Středisko pro aplikaci Visual Studio v sadě Visual Studio předplatných

## <a name="overview"></a>Přehled
Aplikace Center je zvláště řízení pro aplikace.

-  Automaticky sestavení aplikace, otestovat ji na skutečné zařízení a distribuovat ji pro beta testery
-  Shromážděte zprávy o chybách a analýzy chování uživatelů, včetně vlastních událostí.
-  Přidání nabízených oznámení do vaší aplikace v minutách

## <a name="activation-steps"></a>Postup aktivace

Začínáme s Visual Studio aplikace Center je snadné. 
1.  Přihlaste se k [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2. Najděte dlaždici Centrum aplikace Visual Studio a klikněte na **Začínáme**.

    ![Dlaždice aplikace Center VS](_img/vs-app-center/vs-app-center-tile.png)

3. Dále vytvořte účet.  Můžete použít existující účty pro Google, Facebook, GitHub nebo Microsoftu pro přihlášení nebo vytvořit nový účet zadáním uživatelského jména, e-mailu a hesla.  Klikněte na tlačítko **vytvořit účet** pokračujte. 

    ![Vytvoření účtu Center aplikace VS](_img/vs-app-center/vs-app-center-create-account.png)

4. To je všechno je k němu.  Stačí kliknout na **přidat nové organizace** nebo **přidat novou aplikaci** začít!

    ![Portál aplikace Center VS](_img/vs-app-center/vs-app-center-portal.png)

## <a name="faq"></a>Nejčastější dotazy
*Kontrola aktualizací*

### <a name="q--when-i-click-on-the-get-started-link-i-get-an-error-that-my-browser-is-unsupported--whats-wrong"></a>Otázka: po klepnutí na odkaz "Začínáme", zobrazuje se chyba, že v prohlížeči není podporována.  Co je? 
A: Center aplikace vyžaduje novější funkce, které nemusí být poskytované některé starší prohlížeče.  Pokud váš prohlížeč není podporovaný, postupujte podle těchto kroků:
1.  Otevřete jiný prohlížeč, jako je například Microsoft Edge nebo nejnovější verze nástroje Google Chrome, Firefox nebo Safari.  
2.  Připojení k sadě Visual Studio [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) a přihlaste se. 
3.  Nyní klikněte na **Začínáme** odkaz na dlaždici Centrum aplikace Visual Studio. 
4.  Teď by měla být přístup k portálu aplikace Center a vytvoření účtu.
