# <a name="introducing-visual-studio-subscriptions-with-github-enterprise"></a>Představujeme předplatná sady Visual Studio pomocí GitHub Enterprise  

Zákazníci, kteří mají smlouvy Enterprise (EA) se Microsoft mají nárok na zakoupení nové předplatné sady prostředků, který spojuje standardní předplatná sady Visual Studio a Githubu Enterprise. Je to snadné a ekonomičtější způsob pro předplatitele sady Visual Studio k získání Githubu Enterprise. 

- Inovace ve velkém měřítku pomocí sady Visual Studio a Githubu přenesením bezpečně open source kódem, community a osvědčených postupů do integrovaného vývojového prostředí a pracovních postupů.
- Nejlepší v třídě vývojové nástroje a služby pro vývojáře pomocí nejkomplexnější sada prostředků vytvářet, nasazovat a spravovat skvělé aplikace. 
- Spolupracovat, spravovat žádosti o přijetí změn na Githubu Enterprise a kontrola zdrojového kódu, kde byl soubor napsaný – v editoru. 

## <a name="whats-available"></a>Co je k dispozici? 

Pokud vaše organizace je zadání nové smlouvy EA nebo obnovení nějaký existující, jsou k dispozici následující možnosti:

- Visual Studio Enterprise pomocí GitHub Enterprise
- Visual Studio Professional pomocí GitHub Enterprise

Pokud vaše organizace má smlouvu EA, která není z důvodu obnovení ještě, si můžete koupit "Stupňované" SKU Githubu Enterprise přidat ke stávajícím předplatným sady Visual Studio Enterprise nebo Visual Studio Professional.  Potom když se tato možnost obnoví smlouvu, si můžete koupit sady.

Existujících skladových položek pro Visual Studio Enterprise, Visual Studio Professional a Githubu Enterprise bude i nadále k dispozici samostatně. 

Další informace o výhodách, možnostech a cenách najdete https://visualstudio.microsoft.com/subscriptions/visual-studio-github. 

## <a name="getting-your-subscriptions"></a>Načítání vašich odběrů

Visual Studio a Githubu Enterprise předplatným jsou přiřazeny ve dvou částech:
1. Předplatné sady Visual Studio (Enterprise nebo Professional)
2. Předplatné Githubu Enterprise

Pokud vaše organizace kupuje předplatná, jsou přiřazení správců.  Předplatná sady Visual Studio a Githubu Enterprise předplatná jsou přiřazeny samostatně.  

Pokud jsou přiřazeny sady Visual Studio + GitHub předplatného, dostanete e-mailu kterým vám dáme vědět, že správce předplatného sady Visual Studio má přiřazené předplatné pro vás.  Od tohoto okamžiku se budete moct podívat, jak to na https://my.visualstudio.com/subscriptions.  

Protože Githubu Enterprise předplatným jsou přiřazeny samostatně, můžete obdržet oznámení z jiného správce, v závislosti na tom, jak vaše společnost se rozhodla pro uspořádání svá předplatná.  Může také docházet k prodlevám mezi přiřazení předplatného sady Visual Studio a Githubu Enterprise předplatným, takže nemusíte mít obavy, pokud nejsou obě okamžitě k dispozici.  V procesu přiřazování předplatných Githubu je složitější a může trvat několik pracovních dnů, dokončete.

## <a name="frequently-asked-questions"></a>Nejčastější dotazy

### <a name="q--where-is-my-github-subscription"></a>DOTAZ:  Kde je Moje předplatné GitHub?

O:  Proces pro nastavení předplatného GitHub může trvat nějakou dobu.  Pokud jste už nějakou dobu čekáte a ještě něco slyšeli, můžete se obrátit na správce Githubu.  Je možné, že je stále na tom pracujeme správce, nebo se rozhodl není přiřazení předplatného Githubu pro vás. 

### <a name="q--how-do-i-reach-my-github-administrator"></a>DOTAZ:  Jak dosáhnu Můj správce Githubu

O:  Pokud používáte GitHub Enterprise serveru, můžete kliknout na **podporu** odkaz viditelné v dolní části klienta k připojení se na správce.

### <a name="q-do-i-get-the-full-visual-studio-subscription-professional-or-enterprise-or-do-i-just-get-the-ide-with-this-offering"></a>DOTAZ: Je možné dosáhnout úplné předplatné sady Visual Studio (Professional nebo Enterprise) nebo stačí získat integrované vývojové prostředí s touto nabídkou?

O:  Společně s předplatným Githubu Enterprise se zobrazí na plné předplatné sady Visual Studio.  Zobrazí všechny výhody, jako je stahování softwaru, cloudové služby, technickou podporu a partnerské výhody, které jsou součástí těchto předplatných, stejně jako kdybyste odběr byl zakoupen zvlášť.
