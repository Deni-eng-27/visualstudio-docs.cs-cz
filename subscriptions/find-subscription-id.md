# <a name="locating-your-subscription-id-for-visual-studio-subscriptions"></a>Vyhledání svoje ID předplatného pro předplatné sady Visual Studio

## <a name="what-is-a-subscription-id"></a>Co je ID předplatného?
"ID předplatného" je identifikátor GUID 32 číslice přidružený k odběru nebo program odběratele.  V některých situacích může požádejte tým podpory odběratele zajistit jejich ID odběratele zúžit zaměření potenciální problémy s konkrétní předplatné. 

## <a name="how-to-find-your-subscription-id"></a>Jak najít svoje ID předplatného
1. Přejděte na [ https://my.visualstudio.com/subscriptions ](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs).
2. Přihlaste se k předplatnému.  (Nápovědu [přihlášení](/visualstudio/subscriptions/signing-in).)
3. Jakmile jste přihlášení, zobrazí se seznam předplatná spojená s e-mailovou adresu, které jste použili k přihlášení.  Můžete zjistit:
    - Typ odběru nebo Program
    - ID předplatného 
    - Typ (prodejní, NFR atd.)
    - Platné datum.  (Vypršení platnosti)
    - Odkazy na obnovit, získání informací o podporu a v případě Visual Studio Dev Essentials, nechat program.
