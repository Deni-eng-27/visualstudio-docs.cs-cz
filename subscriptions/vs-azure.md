# <a name="using-microsoft-azure-in-visual-studio-subscriptions"></a>Použití v sadě Visual Studio předplatných Microsoft Azure
Jako odběratel Visual Studio můžete použít Microsoft Azure bez dalších poplatků.  S vaší [měsíčního kreditu Azure](https://azure.microsoft.com/pricing/member-offers/msdn-benefits-details/), je vaše osobní izolovaného prostoru pro vývojové a testovací Azure.  Můžete zřídit virtuální počítače, cloudové služby a dalším prostředkům služby Azure.  Částky Dal se liší podle úrovně předplatného. 

Aktivace z **nástroje** kategorie na stránce výhody [portál odběratele](https://my.visualstudio.com/benefits): 

1.  Kliknutím na odkaz "Aktivovat" v dolní části dlaždici výhody.   
![Azure dlaždice](_img\vs-azure\vs-azure-tile.png)

2.  Zadejte požadované informace k vytvoření vašeho předplatného Azure: ![Azure registrace](_img\vs-azure\vs-azure-sign-up-cropped.png)

 
3.  Pokud váš účet je připraven, budete mít možnost Vybrat plán podpory Azure (prodává samostatně) nebo začít pracovat s předplatným Azure.  
![Probíhá příprava Azure](_img\vs-azure\vs-azure-getting-ready-cropped.png)

4.  Načte řídicí panel Azure a na první použití, budete mít možnost prohlídka.  Prohlídku se skládá z řady tipy obrazovky, které bude odkazovat na tom, jak můžete provádět různé úlohy a hledání příslušných informací.  Dokončení nebo ukončování prohlídku se zobrazí řídicí panel Azure. 
5.  Záložky [portálu Azure](https://portal.azure.com) pro usnadnění přístupu v budoucnu.
![Řídicí panel Azure](_img\vs-azure\vs-azure-dashboard-cropped.png)
