# <a name="buy-visual-studio-professional-and-visual-studio-enterprise-cloud-subscriptions"></a>Zakoupení předplatných cloudů Visual Studio Professional a Visual Studio Enterprise
[Předplatná Visual Studio Professional a Visual Studio Enterprise](https://visualstudio.microsoft.com/subscriptions/) si můžete zakoupit na webu Visual Studio [Marketplace](https://marketplace.visualstudio.com). Tyto služby se nazývají **"cloudová předplatná"**.

[Porovnání výhod předplatného cloudu a cen](https://visualstudio.microsoft.com/vs/pricing/)

> [!NOTE]
> Microsoft už nenabízí roční předplatná Visual Studio Professional a roční předplatná Visual Studia Enterprise v předplatných cloudu. Stávající zákazníci se nezmění a možnost obnovit, zvýšit, snížit nebo zrušit jejich odběry. Noví zákazníci se [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) doporučuje přejít na prozkoumat různé možnosti nákupu Visual Studio. Stále můžete zakoupit nové měsíční cloudové předplatné.

## <a name="before-you-start"></a>Než začnete
Chcete-li fakturovat nákupy, potřebujete [předplatné Azure](https://azure.microsoft.com/pricing/purchase-options/). Můžete [se zaregistrovat](https://portal.azure.com) před prvním nákupem nebo během prvního nákupu na webu Visual Studio Marketplace.

## <a name="who-can-buy-visual-studio-cloud-subscriptions"></a>Kdo si může koupit předplatná cloudu sady Visual Studio?
Kdokoliv s [vlastníkem](https://docs.microsoft.com/azure/role-based-access-control/built-in-roles#owner), [správcem služeb nebo spolusprávcem](https://docs.microsoft.com/azure/billing/billing-add-change-azure-subscription-administrator#assign-a-subscription-administrator) přístupu k předplatnému Azure si může zakoupit cloudová předplatná.

## <a name="how-to-buy-cloud-subscriptions"></a>Jak koupit cloudová předplatná

1. Přihlaste se k [předplatným visual studia Marketplace >](https://marketplace.visualstudio.com/subscriptions).

2. Vyberte předplatné, které chcete koupit, například – Visual Studio Professional – měsíční předplatné.

3. Vyberte předplatné Azure, které se má použít k fakturaci. Nové předplatné Azure můžete vytvořit, pokud ho nemáte.
    > [!div class="mx-imgBorder"]
    > ![Vyberte předplatné Azure pro fakturaci.](_img/buy-vs-subscriptions/buy-vs-sub-Azure-sub.png)

4. Vyberte počet předplatných, která chcete koupit.
    > [!div class="mx-imgBorder"]
    > ![Výběr počtu předplatných](_img/buy-vs-subscriptions/buy-vs-sub-users.png)

    Pokud si nepřejete, aby si byla předplatná přiřazena, zrušte zaškrtnutí **zrušte zaškrtnutí možnosti Přiřadit mi toto předplatné**.

5. Potvrďte a dokončete nákup. Pokud jste si přiřadili předplatné, můžete navštívit [portál předplatitelů Sady Visual Studio](https://my.visualstudio.com) a začít stahovat software a používat další výhody pro předplatitele.

> [!div class="mx-imgBorder"]
> ![Dokončení nákupu bylo úspěšně dokončeno.](_img/buy-vs-subscriptions/buy-vs-sub-success.png)

> [!div class="mx-imgBorder"]
> ![Přejít na portál předplatitelů Visual Studia](_img/buy-vs-subscriptions/view-subscription-benefits-subscriptions-portal.png)

Pokud jste si zakoupili více než jedno předplatné, přejděte na [portál správy předplatných sady Visual Studio](https://manage.visualstudio.com) a přiřaďte předplatná ostatním.

> [!div class="mx-imgBorder"]
> ![Přiřazení předplatných ostatním](_img/buy-vs-subscriptions/buy-vs-sub-success-many.png)

> [!div class="mx-imgBorder"]
> ![Zobrazit předplatná](_img/buy-vs-subscriptions/assign-subscriptions.png)

## <a name="cancel-renewals-of-cloud-subscriptions-youve-purchased"></a><a name="manage-subscriptions"></a>Zrušení obnovení zakoupených cloudových předplatných

* Změna počtu zakoupených předplatných:

> [!div class="mx-imgBorder"]
> ![Změna předplatného](_img/buy-vs-subscriptions/manage-subscriptions.png)

* Chcete-li zrušit tato předplatná, snižte počet předplatných na nulu (0). Neobnoví se na začátku příštího fakturačního období (první z příštího měsíce pro měsíční předplatná cloudu; o 12 měsíců později pro roční předplatná cloudu).

## <a name="frequently-asked-questions"></a>Nejčastější dotazy

### <a name="for-new-customers"></a>Pro nové zákazníky:
#### <a name="q--i-was-considering-the-annual-cloud-subscription-and-now-i-cant-find-it-what-should-i-do"></a>Otázka: Zvažoval jsem roční předplatné Cloudu a teď ho nemůžu najít.Co bych měl/a dělat?
A: Prosím, [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) jděte na další možnosti nákupu.

#### <a name="q-what-purchasing-options-are-available-to-me"></a>Otázka: Jaké možnosti nákupu mám k dispozici?
A: Poskytujeme měsíční předplatné a standardní odběry v našich přímých prodejních kanálů. Další informace naleznete [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)v odhlašení .
Existuje také mnoho možností prostřednictvím našeho multilicenčního programu. Můžete zkoumat možnosti a [https://www.microsoft.com/en-us/licensing/how-to-buy/how-to-buy](https://www.microsoft.com/en-us/licensing/how-to-buy/how-to-buy)ceny na .

#### <a name="q-the-price-for-a-retail-subscription-is-higher-what-additional-benefits-am-i-receiving"></a>Otázka: Cena za maloobchodní předplatné je vyšší. Jaké další výhody dostávám?
A: Obdržíte trvalá licenční práva pro IDE sady Visual Studio, které bylo dodáno s předplatným. Když je čas na obnovení, platíte pouze cenu za záruku softwaru, která je výrazně nižší.

### <a name="for-existing-annual-cloud-subscribers"></a>Pro stávající roční předplatitele cloudu
#### <a name="q--how-will-this-change-impact-me"></a>Otázka: Jak mě tato změna ovlivní?
A: Tato změna nebude mít vliv na stávající předplatitele.

#### <a name="q--do-i-need-to-do-anything-as-a-result-of-these-changes"></a>Otázka: Musím v důsledku těchto změn něco udělat?
A: Ne.  Vaše předplatná zůstanou na svém místě a budete je moci spravovat stejně jako dnes.

#### <a name="q-what-if-i-want-to-increase-the-number-of-subscriptions-for-my-customers"></a>Otázka: Co když chci zvýšit počet předplatných pro své zákazníky?
A: Roční předplatné, které jsou zakoupeny během daného měsíce jsou účtovány okamžitě po celý rok a jsou platné pro celý rok. Z tohoto důvodu můžete změnit počet licencí pouze v aktuálním měsíci nákupu (budou platné po dobu jednoho roku a účtovány po dobu jednoho roku). Mimo měsíc nákupu již není možné zvýšit počet předplatných.

#### <a name="q-what-if-i-want-to-decrease-the-number-of-subscriptions-for-my-customers"></a>Otázka: Co když chci snížit počet předplatných pro své zákazníky?
A: Správce stále můžete snížit počet předplatných návštěvou https://manage.visualstudio.com a podle pokynů v tomto článku snížit počet předplatného. Systém *nebude* generovat poměrný kredit. Pro vrácení peněz budete muset kontaktovat fakturační tým Azure.

#### <a name="q-if-i-cancel-my-subscription-will-i-be-able-to-buy-another-annual-cloud-subscription-later"></a>Otázka: Pokud předplatné zruším, budu si moci později zakoupit další roční předplatné Cloudu?
A: I když nebudete moci koupit další předplatné Cloud Roční, existuje mnoho možností, jak zakoupit předplatné Sady Visual Studio.  Další informace [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)naleznete na adrese .

### <a name="for-cloud-solution-providers-csp"></a>Pro poskytovatele cloudových řešení (CSP)
#### <a name="q-how-will-this-change-impact-my-customers"></a>Otázka: Jak tato změna ovlivní mé zákazníky?
A: Vaši zákazníci nebudou moci zakoupit nové Visual Studio Cloud roční předplatné.

#### <a name="q-do-my-customers-need-to-do-anything-because-of-these-changes"></a>Otázka: Musí moji zákazníci kvůli těmto změnám něco udělat?
A: Stávající zákazníci nebudou ovlivněny. Noví zákazníci se budou muset rozhodnout, zda chtějí zakoupit měsíční předplatná sady Visual Studio nebo přejít na jiný kanál, jako je multilicence, a zakoupit předplatná sady Visual Studio.

#### <a name="q-what-if-i-want-to-increase-the-number-of-subscriptions-for-my-customers"></a>Otázka: Co když chci zvýšit počet předplatných pro své zákazníky?
A: Roční předplatné zakoupené během daného měsíce se účtuje okamžitě po celý rok a jsou platné po celý rok. Z tohoto důvodu může správce upravit pouze počet licencí v aktuálním měsíci nákupu (budou platné po dobu 1 roku a účtovány po dobu 1 roku). Mimo měsíc nákupu již není možné zvýšit počet předplatných.

#### <a name="q-what-if-i-want-to-decrease-the-number-of-subscriptions-for-my-customers"></a>Otázka: Co když chci snížit počet předplatných pro své zákazníky?
A: Správce stále můžete snížit počet předplatných návštěvou [https://manage.visualstudio.com](https://manage.visualstudio.com) a podle pokynů v tomto článku snížit počet předplatného. Systém **nebude** generovat poměrný kredit. Pro vrácení peněz budete muset kontaktovat fakturační tým Azure.

A: I když nebudou moci koupit další předplatné Cloud Roční, existuje mnoho možností, jak zakoupit předplatné Sady Visual Studio.  Další informace [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)naleznete na adrese .
