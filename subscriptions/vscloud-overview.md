# <a name="buying-visual-studio-cloud-subscriptions---overview"></a>Kdybyste kupovali Cloudová předplatná sady Visual Studio – přehled

S naše Cloudová předplatná můžete pronajímat Visual Studio, služby VSTS a výhody pro předplatitele, potřebujete bez dlouhodobé smlouvy. Fakturace je prováděna pomocí vašeho předplatného Microsoft Azure, podporu platebních platební karty, nákupy prostřednictvím smlouva Enterprise Agreement kontrakt *, a [poskytovatele Cloud Solution Provider](https://www.microsoft.com/solution-providers/search) partnery. Pokud nemáte předplatné Azure, ale buďte bez obav – je bezplatná nastavení a můžete to provést při provádění váš první nákup.

Porovnání cloudu [výhody předplatného a ceny](https://www.visualstudio.com/vs/pricing/).

## <a name="before-you-start"></a>Než začnete
Než si koupíte, postupujte podle těchto kroků můžete nastavit pro úspěšné transakce • k vyúčtování vaše nákupy, budete potřebovat [předplatné](https://azure.microsoft.com/pricing/purchase-options/).  Můžete [zaregistrovat](https://portal.azure.com/) před nákupu, nebo když váš první nákup do Visual Studio Marketplace.  

• Chcete-li se přihlásit do vašeho předplatného Azure a koupit z Visual Studio Marketplace, budete potřebovat [osobní účet Microsoft](https://www.microsoft.com/account) nebo [pracovní nebo školní účet](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/) při nákupu pro vaši organizaci a Správa identit s Azure Active Directory (Azure AD).
