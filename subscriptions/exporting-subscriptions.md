# <a name="exporting-subscription-information"></a>Export informace o předplatném

V sadě Visual Studio předplatných [portál pro správu](https://manage.visualstudio.com), můžete exportovat seznam uživatelů a podrobnosti o jejich přiřazení. Tyto informace obsahuje jejich název, e-mailovou adresu, alternativní e-mailovou adresu, úrovni předplatného, přiřazené datum, stav aktivace, datum vypršení platnosti, odkaz na pole, stahování povolené, země, jazyka, stav předplatného a GUID odběru. 

Tato funkce je užitečná pro několik scénářů. Například je možné snadno sledovat přiřazení a datum vypršení platnosti. Pokud jsou například přechod z sledování odběry pomocí zákaz pro identifikátory GUID, můžete pomocí vzorce SVYHLEDAT v aplikaci Microsoft Excel tuto sestavu tak, aby odpovídaly odběratele správně.  

Pokud chcete provést export, jednoduše vyberte **exportovat** kartě a soubor se stáhne do místního počítače. Soubor bude obsahovat název účtu Azure, který obsahuje předplatné uživatele, jakož i data exportu.
