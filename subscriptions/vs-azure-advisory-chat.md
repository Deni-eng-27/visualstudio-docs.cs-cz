# <a name="azure-advisory-chat"></a>Chat s poradcem pro Azure
Získejte nápovědu od Azure v reálném čase. Zahrnuje poradenství pro služby Azure PaaS.  Zvýhodnění chatu pro Azure Advisor je k dispozici v Visual Studio Enterprise Standard a ročních cloudových předplatných.

## <a name="activation-steps"></a>Kroky aktivace
Používání služby Azure Advisor chat je snadné.
1. Přihlaste se k webu [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).
2. V části Podpora Najděte dlaždici s poradenským chatem Azure a klikněte na **Chat Now (Spustit nyní**).
    > [!div class="mx-imgBorder"]
    > ![Dlaždice s poradním chatem pro Azure](_img/vs-azure-advisory/vs-azure-advisory-tile.png)

3. Zobrazí se oznámení o tom, že služba je momentálně nabízená jenom v angličtině, a to s výjimkou podpory pro účty, fakturace a problémy s předplatným.  Klikněte na **Pokračovat**.
    > [!div class="mx-imgBorder"]
    > ![Reklama chatu pro Azure Advisor](_img/vs-azure-advisory/vs-azure-advisory-disclaimer.png)

4. To je všechno, co je to vše.  Otevře se okno konverzace, ve kterém se zobrazí informace o tom, kde se nacházíte ve frontě.  Chat bude zodpovězený dalším dostupným odborníkem.
    > [!div class="mx-imgBorder"]
    > ![Chat s poradcem pro Azure](_img/vs-azure-advisory/vs-azure-advisory-chat.png)

## <a name="frequently-asked-questions"></a>Nejčastější dotazy

### <a name="q--what-is-the-difference-between-azure-advisory-chat-and-azure-advisor"></a>Č  Jaký je rozdíl mezi poradním chatem a Azure Advisorm Azure?
- O:  Poradenský chat Azure je chatovací služba v reálném čase, která pomáhá s dotazy ohledně Azure. [Azure Advisor](/azure/advisor/advisor-overview) je přizpůsobený cloudový poradce, který pomáhá dodržovat osvědčené postupy pro optimalizaci nasazení Azure. Analyzuje konfiguraci prostředků a telemetrii využití a potom doporučuje řešení, která vám pomůžou vylepšit efektivitu nákladů, výkon, vysokou dostupnost a zabezpečení vašich prostředků Azure.
