# <a name="visual-studio-team-services-vsts-benefits-for-visual-studio-subscribers"></a>Visual Studio Team Services (služby VSTS) výhody pro předplatitele sady Visual Studio

Jako aktivní předplatitele sady Visual Studio můžete použít celou řadu funkcí v služby VSTS, které jsou součástí vašeho předplatného jako výhody. Budete moct pomocí stejných funkcí v každé služby VSTS účtu, ve kterém jste členem, zda jste vytvořili účet, nebo byly přidány jiným uživatelem.

Při přihlašování do služby VSTS buď pomocí identity, která jste použili pro aktivaci předplatného sady Visual Studio, vaše [alternativní identity](vs-alternate-identity.md), Uvědomujeme si to automaticky.

[Zaregistrovat bezplatný účet služby VSTS](https://visualstudio.microsoft.com/team-services/)
