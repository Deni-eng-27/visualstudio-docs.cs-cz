# <a name="manage-visual-studio-subscriptions-offered-through-the-microsoft-partner-network-mpn"></a>Správa předplatných sady Visual Studio nabízených prostřednictvím sítě Microsoft Partner Network (MPN)
Partneři, kteří získají kompetenci nebo zakoupí balíček akcí v programu MPN, mohou používat předplatná sady Visual Studio. Tato předplatná jsou licencována na uživatele a jsou platná pouze v případě, že jsou přiřazena konkrétnímu pojmenovanému uživateli, který pracuje pro partnerskou společnost.

Podrobnosti najdete v tématu Předplatná sady Visual Studio nabízená partnerům v [síti Microsoft Partner Network (MPN).](program-mpn.md)

## <a name="microsoft-action-pack-subscribers"></a>Předplatitelé sady Microsoft Action Pack
Předplatitelka sady Microsoft Action Pack a předplatná sady Visual Studio
1. Správce programu se přihlásí do Centra členství partnera
2. Přidružení osob prostřednictvím **požadavků a majetku**
3. Vyhledání jednotlivců a **přiřazení rolí**
4. Vyberte první, druhý nebo třetí **technický kontakt** a **uložit**
5. Vybrat **požadavky & datových zdrojů**
6. Vybrat **oprávnění**
7. Vyberte **předplatná sady Visual Studio** (poznámka: Aktualizace předplatného sady Visual Studio čeká na vyřízení)
8. Vyberte požadovanou **osobu**
9. Klikněte na **Odeslat.**
10. Jednotlivec obdrží e-mail
11. Přiřazená osoba [https://my.visualstudio.com](https://my.visualstudio.com?wt.mc_id=o~msft~docs) bude mít přístup k přihlášení
12. Může se zobrazit stránka profilu sady Visual Studio, včetně:
    - Jméno
    - Příjmení
    - Kontaktní e-mail
13. Přihlášení je nyní dokončeno a účastník by měl mít přístup k výhodám

## <a name="competency-partners"></a>Partneři kompetencí
1. Vybrat **požadavky a majetek**
2. Vybrat **oprávnění**
3. Vyberte **předplatná MSDN** (Poznámka: to to bude brzy přeznačeno na "Předplatná sady Visual Studio")
4. Vyhledejte **požadovanou osobu**
5. Klikněte na **Odeslat.**
6. Jednotlivec obdrží e-mail
7. Přiřazená osoba [https://my.visualstudio.com](https://my.visualstudio.com?wt.mc_id=o~msft~docs) bude mít přístup k přihlášení
8. Může se zobrazit stránka profilu sady Visual Studio.
    - Jméno
    - Příjmení
    - Kontaktní e-mail
9. Přihlášení je nyní dokončeno a účastník by měl mít přístup k výhodám
