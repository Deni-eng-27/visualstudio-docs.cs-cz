# <a name="managing-subscriptions"></a>Správa předplatných

[Portál předplatitelů](https://my.visualstudio.com) sady Visual Studio umožňuje spravovat všechna vaše předplatná a členství v programech na jednom místě. Můžete provádět změny ve svém profilu, využívat všechny výhody, které jsou součástí předplatných, a rychle a snadno přepínat mezi předplatnými.

## <a name="managing-multiple-subscriptions"></a>Správa více předplatných

Portál předplatných sady Visual Studio sdružuje všechna vaše předplatná a programy sady Visual Studio v jediném vylepšeném prostředí portálu. Můžete mít například standardní předplatné, jako je Visual Studio Enterprise prostřednictvím práce a měsíční předplatné Visual Studio Professional nebo členství Visual Studio Dev Essentials, které používáte doma.

### <a name="which-subscription-am-i-using"></a>Jaké předplatné používám?

Na stránce Výhody se předplatné, které se používá, zřetelně zobrazí v modrém banneru. Je také zobrazen v pravém horním části stránky.
> [!div class="mx-imgBorder"]
> ![Aktivní předplatné](_img/manage-vs-subscriptions/current-subscription-cropped.png)

### <a name="to-switch-between-subscriptions-you-can"></a>Chcete-li přepínat mezi předplatnými, můžete:

- Klikněte na rozevírací seznam v pravém horním rohu a vyberte předplatné ze seznamu
- Klikněte na kartu Odběry a poté klikněte na požadované předplatné
  > [!div class="mx-imgBorder"]
  > ![Přepínání předplatných](_img/manage-vs-subscriptions/change-subscription-resized.png)

### <a name="if-your-subscription-is-not-visible"></a>Pokud vaše předplatné není viditelné...

- Může být její platnost vypršela. Na portálu odběratelů jsou zobrazena pouze aktivní předplatná.
- Může být spojena s jinou e-mailovou adresou. Pokud se například přihlásíte k předplatnému sady Visual Studio Professional pomocí pracovního a školního e-mailu a máte k vašemu účtu Microsoft přidružené členství Visual Studio Dev Essentials, zobrazí se pouze předplatné odpovídající e-mailové adrese, kterou jste použili k přihlášení k portálu odběratelů. Chcete-li zobrazit další předplatné, budete se muset odhlásit a pak se přihlásit pomocí jiné e-mailové adresy.
- Pokud vaše společnost kupuje předplatná prostřednictvím hromadné licence, je možné, že vám správce předplatné nepřiřadil nebo vám předplatné přeřadil jinému uživateli.

## <a name="renewing-my-subscriptions"></a>Obnovení předplatných

**Standardní předplatná**
- Microsoft Store: Pokud jste předplatné zakoupili v [microsoft storu](https://www.microsoft.com/store), můžete ho obnovit i tam.
- Hromadná licence: Pokud vaše společnost nakupuje předplatná prostřednictvím hromadné licence, měl by se správce obrátit na prodejce a obnovit vaše předplatná.

**Předplatná cloudu:**  Cloudová předplatná se nemusí obnovovat. Dokud bude vaše předplatné placeno, zůstane aktivní.

**Předplatná získaná prostřednictvím programů:**  Pokud předplatné získáte prostřednictvím programu, bude vaše předplatné vázáno na vaši účast v programu. Další informace získáte od kontaktů programu.

- [Představte si](https://imagine.microsoft.com/about)
- [Bizspark](https://bizspark.microsoft.com/About/Offers)
- [Microsoft Partner Network](https://partner.microsoft.com)
- [Certifikovaný trenér microsoftu](https://www.microsoft.com/learning/mct-certification.aspx)

## <a name="transferring-subscriptions"></a>Převod předplatných

Pokud bylo vaše předplatné zakoupeno prostřednictvím multilicenčního programu společnosti Microsoft a potřebujete ho znovu přiřadit někomu jinému v rámci organizace, bude to muset udělat správce za vás.
Maloobchodní předplatná lze přenést na jiného uživatele kontaktováním oddělení služeb zákazníkům. Bezplatné předplatné "Není k prodeji" nelze převést.

## <a name="managing-my-profile"></a>Správa profilu

Portál předplatných umožňuje vytvářet a spravovat profil, což usnadňuje správu předplatných. Můžete provést změny v informacích o identitě a můžete přidružit vaše předplatná s jinými přihlašovacími identitami z existujících účtů Azure nebo organizací Azure DevOps pomocí alternativních identit.

### <a name="changing-profile-information"></a>Změna informací o profilu

Kontaktní informace a předvolby můžete snadno změnit ve svém profilu.

1. Klikněte na své jméno v pravém horním rohu portálu předplatitelů.
2. Klepněte na tlačítko **Upravit profil.**
3. Na kartě **Informace** můžete změnit své jméno, e-mail, zemi nebo oblast, profilový obrázek a předvolby kontaktů. Na kartě **Předvolby** můžete vybrat preferovaný jazyk, vzorce data a času, časové pásmo a motiv uživatelského rozhraní.
4. Po dokončení změn klikněte na **Uložit**.
5. Kliknutím na tlačítko **Zpět** v prohlížeči se vrátíte na stránku Výhody.

### <a name="setting-communications-preferences"></a>Nastavení předvoleb komunikace
Pokud chcete dostávat pravidelné aktualizace o předplatném a výhodách, včetně měsíčního bulletinu předplatného sady Visual Studio, [aktualizujte předvolby](https://app.vsaex.visualstudio.com/me?workflowID=devprogram&tab=edit) zaškrtnutím políčka U předplatných sady Visual Studio v profilu sady Visual Studio. Tyto předvolby můžete kdykoli změnit a odhlásit se. 

   > [!div class="mx-imgBorder"]
   > ![Nastavení předvoleb](_img/manage-vs-subscriptions/change-prefs.png)
   
### <a name="linking-my-subscription-to-existing-azure-devops-services-or-azure-subscriptions"></a>Propojení předplatného se stávajícími službami Azure DevOps nebo předplatným Azure
Pokud se k předplatnému přihlásíte pomocí osobního účtu Microsoft (MSA) a máte předplatné, které zahrnuje služby Azure DevOps services nebo Microsoft Azure pomocí pracovního nebo školního účtu spravovaného službou Azure, můžete obě předplatná propojit. To se nazývá "alternativní účet" a umožňuje nadále používat MSA k přihlášení k předplatnému při přístupu k existující azure, Azure DevOps services nebo IDE prostředky, které jsou přidružené k vaší škole nebo pracovní účty.

I když oba účty používají stejnou e-mailovou adresu, musíte přidat svůj pracovní/školní účet jako alternativu, abyste využili své výhody v obou účtech.

Přidání alternativního účtu:

1. Přihlaste se na [portál uúčastnických služeb](https://my.visualstudio.com?wt.mc_id=o~msft~docs) pomocí svého účtu Microsoft (MSA).
2. Klikněte na kartu **Odběry.**
3. V části **Související odkazy**zvolte Přidat **alternativní účet**.
4. Zadejte e-mailovou adresu svého pracovního/školního účtu a klepněte na tlačítko **Přidat**.

Po přidání alternativního účtu jej můžete kdykoli aktualizovat nebo odebrat kliknutím na kartu **Odběry** a výběrem příslušné možnosti v části **Související odkazy.**
