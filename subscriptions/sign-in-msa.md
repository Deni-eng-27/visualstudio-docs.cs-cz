# <a name="signing-in-to-your-visual-studio-subscriptions-with-your-microsoft-account-msa"></a>Přihlášení k předplatnému sady Visual Studio pomocí účtu Microsoft (MSA)

Postup přihlášení k předplatnému Sady Visual Studio závisí na druhu účtu, který používáte.  Můžete například používat účet Microsoft (MSA) nebo e-mailovou adresu, kterou vám poskytl váš zaměstnavatel nebo škola.  Od ledna 2019 se teď můžete k přihlášení k některým předplatným přihlásit taky pomocí svého účtu GitHub. 

Tento článek vám poskytne postup pro přihlášení pomocí účtu Microsoft (MSA).

## <a name="signing-in-with-your-microsoft-account-msa"></a>Přihlášení pomocí účtu Microsoft (MSA)
1. Navštivte [https://my.visualstudio.com](https://my.visualstudio.com?wt.mc_id=o~msft~docs)prosím .
2. Zadejte e-mailovou adresu, kterou jste zadali při nastavování nebo nákupu předplatného sady Visual Studio.

   > [!NOTE]
   > Tato adresa je také identifikována v uvítacím e-mailu odběratele, který jste obdrželi při zakoupení předplatného nebo registraci do aplikace Visual Studio Dev Essentials. Pokud máte potíže s vyhledáním uvítacího e-mailu, zkontrolujte složky nevyžádané pošty.

3. Zadejte heslo.
4. Klikněte na **Přihlásit se**.
5. V tomto okamžiku by měla být zobrazena stránka "Výhody".

## <a name="frequently-asked-questions"></a>Nejčastější dotazy
### <a name="q--im-unable-to-sign-in-using-my-msa"></a>Otázka: Nemohu se přihlásit pomocí msa.  
Odpověď: Nejčastější příčinou problémů s přihlášením se pokoušíte přihlásit pomocí msa, který se liší od té, která je přidružená k předplatnému.  Pokud používáte několik různých e-mailových adres, je možné, že se pokoušíte přihlásit pomocí nesprávné.  Zkuste se přihlásit pomocí jiné adresy.  Pokud se to nepodaří, můžete se s žádostí o pomoc spojit s naším týmem [podpory pro předplatná.](https://visualstudio.microsoft.com/subscriptions/support/)  

### <a name="q--i-cant-see-my-subscription-where-is-it"></a>Otázka: Moje předplatné se mi nezobrazí. Kde to je?
A: Mnoho uživatelů má více než jedno předplatné.  Pokud se můžete přihlásit k portálu https://my.visualstudio.com předplatných na adrese, ale nemůžete zobrazit vaše předplatné, existují dvě běžné příčiny:
- Jste přihlášeni pomocí jiného účtu Microsoft.  Je možné, že máte více předplatných, jako je například předplatné Professional nebo Enterprise a členství Visual Studio Dev Essentials a jsou přidruženy k různým e-mailovým adresám. Chcete-li zobrazit ostatní odběry, odhlaste se a přihlaste se zpět pomocí jiného MSA.
- Máte více odběrů přidružených ke stejnému MSA.  Chcete-li zobrazit všechna předplatná přidružená k https://my.visualstudio.com/subscriptions vašemu MSA, navštivte a vyberte předplatné, které chcete použít. 

Pokud potíže přetrvávají, můžete požádat o pomoc náš tým [podpory pro předplatná.](https://visualstudio.microsoft.com/subscriptions/support/)
