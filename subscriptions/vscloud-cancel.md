# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Kdybyste kupovali sady Visual Studio Cloudová předplatná - zrušení předplatného

Pokud chcete zrušit cloudové předplatné sady Visual Studio, přihlaste se k sadě Visual Studio [portálu pro správu předplatného](https://manage.visualstudio.com/) pomocí účtu, který má přístup k předplatnému Azure používají k vytvoření původního nákupu.
1.  V levém horním rohu ověřte, zda jste vybrali příslušné předplatné Azure.
2.  V levém navigačním panelu vyberte **snížit** v rámci předplatného, které chcete zrušit.
3.  Zobrazí se okno pro úpravu počtu předplatná spojená s vaším účtem.
4.  Snižte počet odběrů požadované množství nebo nula (0) zrušit, a klikněte na**pokračovat**.
5.  Zkontrolujte si svoji objednávku, klikněte na tlačítko **potvrdit**a you'redone!
## <a name="faq"></a>Nejčastější dotazy:
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>Otázka: je možné můžu dál používat předplatné po I zrušit?
Odpověď: Ano, může vaše výhody pořád přístup k zbývající dobu trvání předplatného.
