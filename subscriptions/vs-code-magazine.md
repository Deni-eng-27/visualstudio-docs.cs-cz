# <a name="code-magazine-included-in-visual-studio-subscriptions"></a>Časopis CODE zahrnutý v předplatných sady Visual Studio

CODE Magazine je přední nezávislý časopis pro vývojáře softwaru, který pokrývá dev technologie na všech platformách.  Vybraná předplatná sady Visual Studio mají nárok na jednoroční předplatné tištěné nebo digitální edice časopisu.

## <a name="activation-steps"></a>Kroky aktivace
Proces nastavení předplatného časopisu CODE magazine je jednoduchý.  Postupujte podle následujících kroků:

1. Navštivte[https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
2. Vyhledejte dlaždici časopisu CODE v části Profesionální rozvoj nebo Vzdělávání na portálu a klikněte na **Získat kód**.
   > [!div class="mx-imgBorder"]
   > ![KÓD Časopis Dlaždice](_img/vs-code-magazine/vs-code-magazine-tile.png "DLAŽDICE ČASOPISU CODE")
3. Zobrazí se zpráva, že váš kód byl načten.  **Chcete-li pokračovat,** klepněte na tlačítko Aktivovat.  (Pokud nejste připraveni k aktivaci, to je v pořádku.  Vždy se můžete vrátit na dlaždici pro aktivaci.  Odkaz "Získat kód" bude číst "Aktivovat" po načtení kódu.)
   > [!div class="mx-imgBorder"]
   > ![Kód časopisu kód načten](_img/vs-code-magazine/vs-code-magazine-success.png "Kód byl úspěšně načten.")
4. Budete přesměrováni na vstupní stránku časopisu CODE. Vyberte svou zemi a zvolte, jak chcete časopis dostávat. 
   > [!NOTE]
   > Členové visual studio Dev Essentials mají přístup pouze k digitálnímu předplatnému. 
5. Na základě vašich výběrů se zobrazí poznámka s cenou předplatného spolu s částkou kupónu, který byl použit.
   > [!NOTE]
   > Pro předplatná pro tisk:
   > - Předplatitelům mimo Spojené státy mohou být účtovány přepravní poplatky. 
   > - Vyčkejte prosím několik týdnů na doručení první tištěné edice.
      > [!div class="mx-imgBorder"]
      > ![Podrobnosti o předplatném časopisu CODE](_img/vs-code-magazine/vs-code-magazine-details.png "Podrobnosti o předplatném a cena")
6. Dále budete muset zadat informace o odběrateli a adresu doručení.  (Dodací adresa je vyžadována i pro digitální odběry, protože podporuje plnění, výběr nabídek a procesy digitálního přístupu.)
7. Pokud žijete mimo Spojené státy a rozhodnete se získat předplatné tištěného časopisu, můžete být požádáni o poskytnutí fakturační adresy a platebních údajů na pokrytí nákladů na dopravu. 
8. Do newsletteru HTML se můžete přihlásit kliknutím na zaškrtávací políčko.
   > [!div class="mx-imgBorder"]
   > ![CODE Magazine informace o odběrateli](_img/vs-code-magazine/vs-code-magazine-subscriber-info.png "Informace o odběrateli a doručení adresy")
9. Chcete-li předplatné dokončit, klepněte na **tlačítko Objednat.**  
Po dokončení objednávky předplatného se zobrazí stránka s potvrzením, včetně odkazu, který můžete https://codemag.com/my/fulfillmentpoužít k zobrazení problémů, které jste obdrželi: .

## <a name="frequently-asked-questions"></a>Nejčastější dotazy
### <a name="q-if-the-subscription-is-free-why-am-i-being-asked-for-a-credit-card"></a>Otázka: Pokud je předplatné zdarma, proč jsem požádán o kreditní kartu?  
A: Je to nákladově-prohibitive posílat zdarma tištěné kopie na mezinárodní úrovni.  Předplatitelům, kteří se rozhodnou pro tiskovou edici časopisu CODE a mají bydliště mimo Spojené státy, bude účtován poplatek za dopravu. Chcete-li se vyhnout poplatkům za dopravu, zvolte digitální verzi časopisu. 

### <a name="q-why-do-i-need-to-provide-a-delivery-address-for-a-digital-subscription"></a>Otázka: Proč je nutné zadat doručovací adresu pro digitální předplatné?
A: CODE Magazine shromažďuje vaši adresu, bez ohledu na to, zda předplatné je pro tisk nebo digitální předplatné na podporu plnění, výběr nabídky a digitální přístup procesy.  Další informace o tom, co časopis CODE dělá s vašimi údaji, naleznete v jejich [Zásadách ochrany osobních údajů](https://www.codemag.com/Home/Privacy).
