# <a name="xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>Výhodou školení Xamarin univerzity odběry Visual Studio

Vytvořte nativní aplikace pro iOS a Android aplikace v jazyce C# s zaznamenaná kurzy a přednášek hosta.  
-  Školení Mobile univerzity Xamarin pro Visual Studio Dev Essentials poskytuje přístup k dvanáct kurzy a tři přednášek hosta. 
-  Školení Mobile univerzity Xamarin pro Visual Studio Professional a Visual Studio Enterprise předplatné zahrnuje stejné kurzy a lectures jako nabídka Essentials vývojáře Visual Studio, PLUS 18 další kurzy a šesti další přednášek.

## <a name="activation-steps"></a>Postup aktivace
1.  Přihlaste se k [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs). 
2.  Najděte dlaždici Xamarin univerzity školení v části Professional vývoj a klikněte na **získat kód** odkaz v levém dolním benefit dlaždice.   Obdržíte oznámení úspěšně načíst kód kupónů.  Klikněte na tlačítko **aktivovat** pokračovat.

    ![Xamarin univerzity školení Benefit dlaždice](_img\vs-xamarin\vs-xamarin-tile.png)

3.  Na stránce registrace dokončit vaše kontaktní údaje, klikněte na políčko souhlasu s podmínkami použití a ochrana osobních údajů příkaz a klikněte na **aktivovat výhody**. 

    ![Registrační Benefit školení univerzity Xamarin](_img\vs-xamarin\vs-xamarin-registration-resized.png)
4.  To je všechno je k němu!  Teď, když jste aktivován vaší Xamarin univerzity školení výhody, budete přesměrováni na školení Mobile univerzity Xamarin pro stránku Microsoft Visual Studio a budete připravení zahájit učení.  

    ![Xamarin univerzity školení Benefit domovské stránky](_img\vs-xamarin\vs-xamarin-home-resized.png)

## <a name="frequently-asked-questions"></a>Nejčastější dotazy
### <a name="q--what-topics-are-covered-in-the-classes-and-lectures"></a>Otázka: co jsou témata v třídy a přednášek?
Odpověď: najdete úplný seznam kurzů a přednášek obě nabídky na:  https://www.xamarin.com/university/mobile-training 

### <a name="q-im-already-a-xamarin-university-subscriber-does-this-affect-my-xamarin-university-mobile-training-benefit"></a>Otázka: jsem už Xamarin univerzity odběratele. To vliv Moje Xamarin univerzity Mobile školení benefit?
Odpověď: žádné existující Xamarin univerzity Odběratelé, kteří budou mít úplný přístup do roční předplatného jejich Xamarin univerzity, třeba za provozu kurzy, materiály a přednášek hosta tak, že protokolování in v university.xamarin.com.  
Kurzy zahrnuty jako výhoda v aplikaci Visual Studio Dev Essentials a Visual Studio odběry jsou oddělené od předplatného univerzity Xamarin a k němu přistupovat z Essentials vývojáře Visual Studio nebo Visual Studio portál, ne university.xamarin.com .  Při zaznamenávání kurzu Xamarin univerzity Mobile školení, zahrnuté v programu Visual Studio Dev Essentials a odběry Visual Studio jsou součástí předplatného roční Xamarin univerzity, tak dlouho, dokud program zůstaly dostupné členství nebo Visual Studio jsou platné předplatné. Bude nadále přístup z portálu Essentials vývojáře Visual Studio nebo Visual Studio, nezávisle na stav předplatného univerzity Xamarin.

### <a name="q--can-i-become-a-certified-xamarin-developer-with-the-xamarin-university-included-in-visual-studio-dev-essentials-what-about-through-visual-studio-subscriptions"></a>Otázka: je možné stát vývojář certifikované Xamarin s Xamarin univerzity součástí Visual Studio Dev Essentials? A co pomocí sady Visual Studio odběrů?
A: kurzy a materiálů, které jsou součástí Xamarin univerzity v Essentials vývojáře Visual Studio a Visual Studio nebo MSDN odběry Ne, nejsou platné pro certifikaci. Materiály, kurzy a záznamy, které jsou potřebné k vám Xamarin certifikační jsou zahrnuty pouze v plné předplatné univerzity Xamarin.
