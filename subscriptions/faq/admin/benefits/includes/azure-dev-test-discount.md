## <a name="how-do-my-developers-get-50-off-azure-devtest-workloads"></a>Jak naši vývojáři získají 50% slevu na úlohy Azure pro vývoj a testování?

Pokud má vaše organizace smlouvu Enterprise pro Azure, můžou si vývojáři vybrat mezi kredity Azure 50 – 150 USD a 50% slevou z ceny pro vývoj a testování. V rámci stejného předplatného nemůžou využít obojí. Podrobnosti o tom, jak svým vývojářům povolit cenové možnosti pro vývoj a testování, uvidíte ve videu [Jak předplatitelům umožnit využívat Azure na maximum](https://aka.ms/HelpingSubscriberswithAzure).
