# <a name="jsvalueref-typedef"></a>JsValueRef – Typedef
Odkaz na hodnotu JavaScript.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>Poznámky  
 Hodnotu JavaScript je jedním z následujících typů hodnot: nedefinované, Null, logická hodnota, řetězec, číslo nebo objekt.  
  
## <a name="requirements"></a>Požadavky  
 **Záhlaví:** jsrt.h
