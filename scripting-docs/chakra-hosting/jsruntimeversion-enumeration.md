# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion – výčet
Verze modulu runtime.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>Členové  
  
### <a name="values"></a>Hodnoty  
  
|Název|Popis|  
|----------|-----------------|  
|`JsRuntimeVersion10`|Vytvořte modul runtime s IE10 verzí.|  
|`JsRuntimeVersion11`|Vytvořte modul runtime s IE11 verzí.|  
|`JsRuntimeVersionEdge`|Vytvoření modulu runtime s nejvyšší verze na tomto počítači za běhu.|
