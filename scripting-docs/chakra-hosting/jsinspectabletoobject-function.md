# <a name="jsinspectabletoobject-function"></a>JsInspectableToObject – funkce
Vytvoří hodnotu JavaScript, která je projekci předaný v `IInspectable` ukazatel.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
STDAPI_(JsErrorCode) JsInspectableToObject(  
        _In_ IInspectable  *inspectable,  
        _Out_ JsValueRef *value  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `inspectable`  
 A `IInspectable` projekci.  
  
 `value`  
 JavaScript hodnota tedy projekci `IInspectable`.  
  
## <a name="return-value"></a>Návratová hodnota  
 Kód `JsNoError` v případě, že byla operace úspěšná, jinak kód chyby.  
  
## <a name="remarks"></a>Poznámky  
 Očekávaná hodnota můžete použít skript volání objektu WinRT. Hostitelé jsou zodpovědní za vynucování COM dělení na vlákna pravidla.  
  
 Vyžaduje aktivní kontext skriptu.  
  
 Toto rozhraní API je podporována pouze v režimu okraj.  
  
## <a name="requirements"></a>Požadavky  
 **Záhlaví:** jsrt.h
