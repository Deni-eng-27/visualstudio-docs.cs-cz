# <a name="script_e_recorded-error-code"></a>SCRIPT_E_RECORDED – kód chyby
Mezi skriptovacím modulem a hostitelem byla předána chyba. Hostitel musí předat kód chyby volajícímu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
