# <a name="ienumremotedebugapplications-interface"></a>IEnumRemoteDebugApplications – rozhraní
Vytvoří výčet objektů aplikace. Toto rozhraní je možné vytvořit výčet spouštění aplikací na počítači pro dialogové okno s "připojení k aplikaci".  
  
 Kromě metod zděděných z `IUnknown`, `IEnumRemoteDebugApplications` rozhraní poskytuje následující metody.  
  
## <a name="methods-in-vtable-order"></a>Metody v tabulce Vtable pořadí  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IEnumRemoteDebugApplications::Clone](../../winscript/reference/ienumremotedebugapplications-clone.md)|Vytvoří čítač, který obsahuje stejného stavu jako aktuální enumerátor.|  
|[IEnumRemoteDebugApplications::Next](../../winscript/reference/ienumremotedebugapplications-next.md)|Načte zadaný počet segmentů v sekvenci výčtu.|  
|[IEnumRemoteDebugApplications::Reset](../../winscript/reference/ienumremotedebugapplications-reset.md)|Návrat na začátek sekvence výčtu.|  
|[IEnumRemoteDebugApplications::Skip](../../winscript/reference/ienumremotedebugapplications-skip.md)|Vynechá zadaný počet segmentů v sekvenci výčtu.|
