# <a name="iactivescriptgarbagecollector-interface"></a>IActiveScriptGarbageCollector – rozhraní
Poskytuje způsob, jak spustit uvolňování paměti. Toto rozhraní by mělo být implementováno pomocí aktivních skriptovacích strojů, které chtějí vyčistit své prostředky.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Spustí uvolňování paměti.|
