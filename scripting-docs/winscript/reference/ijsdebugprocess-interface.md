# <a name="ijsdebugprocess-interface"></a>IJsDebugProcess – rozhraní
Poskytuje rutiny pro kontrolu a řízení cílových procesů.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
IJsDebugProcess : public IUnknown;  
```
