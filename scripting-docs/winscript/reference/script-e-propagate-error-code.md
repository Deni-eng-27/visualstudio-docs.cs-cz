# <a name="script_e_propagate-error-code"></a>SCRIPT_E_PROPAGATE – kód chyby
Chyba skriptu je šířena volajícímu, který může být v jiném vlákně. Hostitel by měl předat kód chyby volajícímu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
