# <a name="idebugasyncoperationcallback-interface"></a>IDebugAsyncOperationCallBack – rozhraní
Poskytuje stav události související s průběh `IDebugAsyncOperation` rozhraní hodnocení.  
  
## <a name="methods"></a>Metody  
 Kromě metod zděděných z `IUnknown`, `IDebugAsyncOperationCallBack` rozhraní poskytuje následující metody.  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Signalizuje, že výsledek je k dispozici z operace asynchronní ladění.|
