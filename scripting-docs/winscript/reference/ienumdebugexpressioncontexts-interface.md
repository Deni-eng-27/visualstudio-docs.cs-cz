# <a name="ienumdebugexpressioncontexts-interface"></a>IEnumDebugExpressionContexts – rozhraní
Vytvoří výčet kolekce `IDebugExpressionContexts` objekty.  
  
 Kromě metod zděděných z `IUnknown`, `IEnumDebugExpressionContexts` rozhraní poskytuje následující metody.  
  
## <a name="methods-in-vtable-order"></a>Metody v tabulce Vtable pořadí  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IEnumDebugExpressionContexts::Next](../../winscript/reference/ienumdebugexpressioncontexts-next.md)|Načte zadaný počet segmentů v pořadí výčtu.|  
|[IEnumDebugExpressionContexts::Skip](../../winscript/reference/ienumdebugexpressioncontexts-skip.md)|Vynechá zadaný počet segmentů v sekvenci výčtu.|  
|[IEnumDebugExpressionContexts::Reset](../../winscript/reference/ienumdebugexpressioncontexts-reset.md)|Návrat na začátek sekvence výčtu.|  
|[IEnumDebugExpressionContexts::Clone](../../winscript/reference/ienumdebugexpressioncontexts-clone.md)|Vytvoří čítač, který obsahuje stejného stavu jako aktuální enumerátor.|
