# <a name="expected-while"></a>Byl očekáván příkaz 'while'
Do `do...while` smyčky jste nezahrnuli podmínku `while`. Příkaz **do** musí mít odpovídající `while` test na konci bloku kódu.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jako uzavírací složenou závorku zahrňte příkaz `while` test.
