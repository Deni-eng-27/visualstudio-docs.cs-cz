# <a name="expected-"></a>Byl očekáván znak ']'
Provedené odkaz na element pole, ale neobsahuje pravou závorku. Libovolný výraz, který odkazuje na element pole musí obsahovat jak levé a pravé závorky.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte pravou závorku výrazu, který odkazuje na element pole.
