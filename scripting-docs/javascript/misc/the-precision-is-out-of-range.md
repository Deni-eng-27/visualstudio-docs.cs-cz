# <a name="the-precision-is-out-of-range"></a>Přesnost je mimo rozsah
Pokusili jste se neplatný argument předat do funkce **Number.prototype.toPrecision**. Argument **toPrecision** musí 1 až 21 (včetně).  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že argument `toPrecision` není příliš velký či příliš malý.
