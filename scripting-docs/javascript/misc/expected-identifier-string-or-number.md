# <a name="expected-identifier-string-or-number"></a>Byl očekáván identifikátor, řetězec nebo číslo
Použili jste nesprávnou syntaxi literálu pro deklaraci literálu objektu. Vlastnosti literálu objektu musí být identifikátor, řetězec nebo číslo. Literál objektu (označovaný také jako inicializátor objektu) obsahuje čárkami oddělený seznam dvojic vlastnost: hodnota, který je uzavřený v závorkách. Příklad:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že používáte správnou syntaxi literálů.
