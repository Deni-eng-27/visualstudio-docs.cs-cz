# <a name="string-expected"></a>Očekáván řetězec
Pokusili jste se vyvolat metodu **String. prototype. ToString** nebo **String. prototype. valueOf –** u objektu jiného typu než `String`. Objekt tohoto typu volání musí být typu `String`.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pro objekty typu `String` vyvolat pouze metody **String. prototype. ToString** nebo **String. prototype. valueOf –** .
