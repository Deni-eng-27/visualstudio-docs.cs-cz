# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>Použití klíčového slova pro identifikátor je neplatné
1048: použití klíčového slova pro identifikátor je neplatné.  
  
 Klíčové slovo nelze použít jako název proměnné, funkce nebo jiného identifikátoru definovaného uživatelem.  
  
 Další informace o klíčových slovech naleznete v tématu [vyhrazená slova jazyka JavaScript](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Lexical_grammar) a v [budoucnu rezervovaných slovech JavaScript](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Lexical_grammar).
