# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>Použití klíčového slova pro identifikátor je neplatné
1048: Použití klíčového slova pro identifikátor je neplatné  
  
 Klíčové slovo nelze použít jako název proměnné, funkce nebo jiný identifikátor definovaný uživatelem.  
  
 Další informace o klíčových slovech najdete v tématu [vyhrazená slova jazyka JavaScript](../../javascript/reference/javascript-reserved-words.md) a [JavaScript budoucí vyhrazená slova](../../javascript/reference/javascript-future-reserved-words.md).
