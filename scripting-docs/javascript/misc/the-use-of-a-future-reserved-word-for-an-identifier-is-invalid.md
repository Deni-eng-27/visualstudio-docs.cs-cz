# <a name="the-use-of-a-future-reserved-word-for-an-identifier-is-invalid"></a>Použití budoucího rezervovaného slova pro identifikátor je neplatné
1049: Použití budoucího rezervovaného slova pro identifikátor je neplatné  
  
 Budoucího rezervovaného slova nelze použít jako název proměnné, funkce nebo jiný identifikátor definovaný uživatelem.  
  
 Další informace o budoucích vyhrazených slov najdete v tématu [JavaScript budoucí vyhrazená slova](../../javascript/reference/javascript-future-reserved-words.md).
