# <a name="mathtan-function-javascript"></a>Math.tan – funkce (JavaScript)
Vrátí tangens čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `radians` argument je číselného výrazu, který obsahuje úhel v radiánech.  
  
 Vrácená hodnota je tangens číselné argument `radians`.
