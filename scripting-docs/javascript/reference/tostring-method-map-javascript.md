# <a name="tostring-method-map-javascript"></a>toString – metoda (Map) (JavaScript)
Vrátí řetězcovou reprezentaci mapy.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>Parametry  
 `mapObj`  
 Požadováno. A `Map` objektu.  
  
## <a name="property-valuereturn-value"></a>Hodnota vlastnosti / návratová hodnota  
 Řetězcová reprezentace mapy.
