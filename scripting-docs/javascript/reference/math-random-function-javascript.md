# <a name="mathrandom-function-javascript"></a>Math.random – funkce (JavaScript)
Vrátí pseudonáhodných číslo mezi 0 a 1.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Pseudonáhodná čísla generované je od 0 (včetně) 1 (kromě), to znamená, vrácený počet může být nula, ale bude vždy nižší než jednou. Generátoru náhodných čísel se nasadí automaticky při [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] prvním načtení.
