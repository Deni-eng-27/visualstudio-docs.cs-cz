# <a name="mathasin-function-javascript"></a>Math.asin – funkce (JavaScript)
Vrátí Arkus sinus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselného výrazu, pro kterou je potřeba Arkus sinus.  
  
 Vrácená hodnota je Arkus sinus `number` argument, v radiánech.
