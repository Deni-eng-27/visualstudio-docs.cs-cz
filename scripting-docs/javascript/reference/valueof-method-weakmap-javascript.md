# <a name="valueof-method-weakmap-javascript"></a>valueOf – metoda (WeakMap) (JavaScript)
Vrátí primitivní hodnotu zadaného objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
weakmapObj.valueOf()  
```  
  
#### <a name="parameters"></a>Parametry  
 `weakmapObj`  
 Požadováno. A `WeakMap` objektu.
