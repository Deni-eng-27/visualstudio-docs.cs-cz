# <a name="mathlog2-function-javascript"></a>Math.log2 – funkce (JavaScript)
Vrátí základní 2 logaritmus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud `x` je `NaN`, funkce vrátí hodnotu `NaN`. Pokud `x` je menší než nula (0), funkce vrátí hodnotu `NaN`.
