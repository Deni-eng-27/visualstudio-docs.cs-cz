# <a name="todatestring-method-date-javascript"></a>toDateString – metoda (Date) (JavaScript)
Vrátí data jako hodnotu řetězce.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované *objDate* je odkaz `Date` objektu.  
  
 `toDateString` Řetězcovou hodnotu obsahující datum v aktuální časové pásmo v praktické, vrátí metoda snadno přečíst formátu.
