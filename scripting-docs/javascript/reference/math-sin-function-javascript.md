# <a name="mathsin-function-javascript"></a>Math.sin – funkce (JavaScript)
Vrátí sinus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Poznámky  
 `radians` Argument je číselného výrazu, který obsahuje úhel v radiánech.  
  
 Vrácená hodnota je sinus číselný argument.
