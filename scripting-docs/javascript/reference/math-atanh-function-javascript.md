# <a name="mathatanh-function-javascript"></a>Math.atanh – funkce (JavaScript)
Vrátí inverzní hyperbolický tangens čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Poznámky  
 Pokud `x` je `NaN`, výsledkem je `NaN`.
