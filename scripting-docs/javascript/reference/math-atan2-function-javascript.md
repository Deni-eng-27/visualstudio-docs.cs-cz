# <a name="mathatan2-function-javascript"></a>Math.atan2 – funkce (JavaScript)
Vrací úhel (v radiánech) z osy X do bodu (*y*,*x*).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parametry  
 *x*  
 Požadováno. Číselný výraz představující kartézských souřadnice x.  
  
 *y*  
 Požadováno. Číselný výraz představující kartézských souřadnici y.  
  
## <a name="remarks"></a>Poznámky  
 Vrácená hodnota je mezi *-pí* a *pí*. Reprezentuje úhel zadaných (*y*,*x*) přejděte v radiánech.
