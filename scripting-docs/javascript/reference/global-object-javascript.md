# <a name="global-object-javascript"></a>Global – objekt (JavaScript)
Vnitřní objekt, jehož účelem je shromažďovat globální funkce a konstanty do jednoho objektu.  
  
## <a name="remarks"></a>Poznámky  
 `Global` Objekt se nikdy nepoužívá přímo a nedá se vytvořit pomocí `new` operátor. Vytvoří se při inicializaci skriptovací stroje, díky čemuž její funkce a konstanty k dispozici okamžitě.
