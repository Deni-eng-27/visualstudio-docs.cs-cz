# <a name="mathtrunc-function-javascript"></a>Math.trunc – funkce (JavaScript)
Vrátí celočíselnou část čísla odebrání všech míst za desetinnou čárkou.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselného výrazu, pro kterou je potřeba na zkrácený celé číslo.  
  
 Pokud `number` je `NaN`, vrátí `NaN`.
