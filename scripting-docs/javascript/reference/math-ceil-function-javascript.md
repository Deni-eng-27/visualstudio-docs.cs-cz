# <a name="mathceil-function-javascript"></a>Math.ceil – funkce (JavaScript)
Vrátí nejmenší celé číslo větší než nebo rovna hodnotě její číselnou argument.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselný výraz.  
  
 Vrácená hodnota je rovna nejmenší celé číslo větší než nebo rovna hodnotě její číselnou argument celočíselnou hodnotu.
