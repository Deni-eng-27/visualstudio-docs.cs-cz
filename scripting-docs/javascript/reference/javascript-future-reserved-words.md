# <a name="javascript-future-reserved-words"></a>JavaScript – vyhrazená slova pro budoucí využití
V jazyce JavaScript budoucí vyhrazená slova se nesmí používat jako identifikátory, i když mají žádný speciální význam v aktuální verzi.  
  
 Seznam rezervovaná slova v jazyce JavaScript naleznete v tématu [vyhrazená slova jazyka JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Budoucí vyhrazená slova  
 Tady jsou budoucí rezervovaná slova:  
  
1.  třída  
  
2.  enum  
  
3.  export  
  
4.  Rozšiřuje  
  
5.  import  
  
6.  Super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Budoucí rezervovaná slova v striktní režim  
 Níže jsou budoucí rezervovaná slova pouze v striktní režim. Další informace o striktní režim najdete v tématu [use strict – direktiva](../../javascript/reference/use-strict-directive.md).  
  
1.  implements  
  
2.  rozhraní  
  
3.  Balíček  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
