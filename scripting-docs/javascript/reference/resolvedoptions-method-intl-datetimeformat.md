# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions – metoda (Intl.DateTimeFormat)
Vrátí objekt obsahující vlastnosti a hodnoty objektu formátování data a času.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametry  
 `dateTimeFormatObj`  
 `DateTimeFormat` Objekt, který chcete načíst informace z.  
  
## <a name="remarks"></a>Poznámky  
 Vlastnosti vráceného objektu odpovídají počítané vlastnosti `DateTimeFormat` objektu.
