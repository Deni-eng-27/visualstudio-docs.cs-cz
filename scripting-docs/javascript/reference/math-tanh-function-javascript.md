# <a name="mathtanh-function-javascript"></a>Math.tanh – funkce (JavaScript)
Vrátí hyperbolický tangens čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Vrácená hodnota je stejná jako `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. Pokud `x` je `NaN`, výsledkem je `NaN`.
