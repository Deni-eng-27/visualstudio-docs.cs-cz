# <a name="mathexp-function-javascript"></a>Math.exp – funkce (JavaScript)
Vrátí číslo e (základ přirozeného logaritmu) na zadanou mocninu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselného výrazu představující sílu *e*.  
  
 Vrácená hodnota je číslo. Konstanta *e* je Eulerova number, přibližně rovna 2.71828 a `number` je zadaný argument.
