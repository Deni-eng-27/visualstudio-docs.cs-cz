# <a name="subtraction-assignment-operator---javascript"></a>Operátor přiřazení odčítání (-=) (JavaScript)
Odečte hodnotu výrazu z hodnoty proměnné a přiřadí výsledek proměnné.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parametry  
 *výsledek*  
 Všechny číselné proměnné.  
  
 *výraz*  
 Jakýkoli číselný výraz.  
  
## <a name="remarks"></a>Poznámky  
 Pomocí `-=` operátor je přesně stejný jako následujícím způsobem:  
  
```JavaScript  
result = result - expression  
```
