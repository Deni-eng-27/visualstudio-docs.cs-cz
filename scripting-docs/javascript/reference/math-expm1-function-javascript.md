# <a name="mathexpm1-function-javascript"></a>Math.expm1 – funkce (JavaScript)
Vrátí výsledek odečtením 1 z e (základ přirozeného logaritmu) na zadanou mocninu).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselného výrazu představující e exponentem.  
  
 Vrácená hodnota je číslo. Konstanta e se Eulerova na počet přibližně odpovídá 2.71828 a `number` je zadaný argument.
