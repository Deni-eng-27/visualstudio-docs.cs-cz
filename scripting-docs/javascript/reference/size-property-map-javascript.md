# <a name="size-property-map-javascript"></a>size – vlastnost (Map) (JavaScript)
Vrátí počet prvků v mapování.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>Parametry  
 `sizeVar`  
 Požadováno. Jakékoli číslo.  
  
 `mapObj`  
 Požadováno. Všechny `Map` objektu.
