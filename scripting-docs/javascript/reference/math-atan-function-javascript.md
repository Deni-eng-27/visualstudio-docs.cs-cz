# <a name="mathatan-function-javascript"></a>Math.atan – funkce (JavaScript)
Vrátí Arkus tangens čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselného výrazu, pro kterou je potřeba Arkus.  
  
 Vrácená hodnota je Arkus tangens `number` argument, v radiánech.
