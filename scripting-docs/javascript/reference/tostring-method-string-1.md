# <a name="tostring-method-string-1"></a>toString – metoda (String) 1
Vrátí řetězcovou reprezentaci řetězce.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Parametry  
 `string`  
 Požadováno. Pole představují jako řetězec.  
  
## <a name="return-value"></a>Návratová hodnota  
 Řetězcová reprezentace řetězce.  
  
## <a name="example"></a>Příklad  
 Následující příklad ukazuje použití **toString** metoda řetězcem.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
