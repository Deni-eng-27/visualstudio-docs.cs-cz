# <a name="javascript-reserved-words"></a>Vyhrazená slova jazyka JavaScript
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]má počet rezervovaných slov, které nelze použít jako identifikátory. Vyhrazená slova mají zvláštní význam [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] jazyka, jako jsou součástí na syntaxi jazyka. Pomocí vyhrazené slovo způsobí chybu kompilace při načítání vašeho skriptu.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]také obsahuje seznam budoucí rezervovaná slova. Tato slova nejsou aktuálně součástí [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] jazyka, i když jsou vyhrazené pro budoucí použití. Další informace o budoucí vyhrazená slova v jazyce JavaScript naleznete v tématu [budoucí vyhrazená slova jazyka JavaScript](../../javascript/reference/javascript-future-reserved-words.md).  
  
 Při výběru identifikátory je také důležité k zamezení slov, která jsou již názvy vnitřní [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] objekty nebo funkce, jako například `String` nebo `parseInt`.
