# <a name="mathpow-function-javascript"></a>Math.pow – funkce (JavaScript)
Vrací hodnotu výrazu základní prováděné na zadanou mocninu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Parametry  
 *základní*  
 Požadováno. Základní hodnota výrazu.  
  
 *exponent*  
 Požadováno. Exponentu hodnota výrazu.  
  
## <a name="example"></a>Příklad  
 V následujícím příkladu vrací číselného výrazu rovna baseexponent 1000.  
  
```JavaScript  
Math.pow(10,3);  
```
