# <a name="clear-method-weakmap-javascript"></a>clear – metoda (WeakMap) (JavaScript)
Odebere všechny elementy z `WeakMap` objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>Parametry  
 `weakmapObj`  
 Požadováno. `WeakMap` Objekt, který chcete vymazat.
