# <a name="bytelength-property-dataview"></a>byteLength – vlastnost (DataView)
Jen pro čtení. Délka tohoto zobrazení od začátku jeho ArrayBuffer v bajtech, jak je stanoveno v době konstrukce.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
var byteLength = dataView.byteLength;  
```  
  
## <a name="example"></a>Příklad  
 Následující příklad ukazuje, jak získat délku DataView z XMLHttpRequest.  
  
```JavaScript  
var req = new XMLHttpRequest();  
    req.open('GET', "http://www.example.com");  
    req.responseType = "arraybuffer";  
    req.send();  
  
    req.onreadystatechange = function () {  
        if (req.readyState === 4) {  
            var buffer = req.response;  
            var dataView = new DataView(buffer);  
            alert(dataView.byteLength);  
        }  
    }  
  
```
