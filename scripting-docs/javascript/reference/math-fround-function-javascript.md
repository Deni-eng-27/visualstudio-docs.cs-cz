# <a name="mathfround-function-javascript"></a>Math.fround – funkce (JavaScript)
Vrátí nejbližší jednoduchou přesností s plovoucí desetinnou čárkou formát čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>Poznámky  
 Pokud `number` je `NaN`, výsledkem je `NaN`.
