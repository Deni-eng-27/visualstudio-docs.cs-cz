# <a name="division-assignment-operator---javascript"></a>Operátor přiřazení dělení (-) (JavaScript)
Rozdělí hodnoty proměnné hodnotou výrazu a přiřadí výsledek proměnné.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>Parametry  
 `result`  
 Všechny číselné proměnné.  
  
 `expression`  
 Jakýkoli číselný výraz.  
  
## <a name="remarks"></a>Poznámky  
 Pomocí `/=` operátor je stejné jako zadání: `result = result / expression`.
