# <a name="mathlog1p-function-javascript"></a>Math.log1p – funkce (JavaScript)
Vrátí přirozený logaritmus 1 + číslo.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud `x` je `NaN`, funkce vrátí hodnotu `NaN`. Pokud `x` je menší než nula (0), funkce vrátí hodnotu `NaN`.
