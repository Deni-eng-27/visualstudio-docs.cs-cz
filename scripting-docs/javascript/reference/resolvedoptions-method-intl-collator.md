# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions – metoda (Intl.Collator)
Vrátí objekt obsahující vlastnosti a hodnoty kompletování objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametry  
 `collatorObj`  
 `Collator` Objekt, který chcete načíst informace z.  
  
## <a name="remarks"></a>Poznámky  
 Vlastnosti vráceného objektu odpovídají počítané vlastnosti `Collator` objektu.
