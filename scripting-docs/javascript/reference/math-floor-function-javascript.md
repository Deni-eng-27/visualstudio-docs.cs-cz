# <a name="mathfloor-function-javascript"></a>Math.floor – funkce (JavaScript)
Vrátí největší celé číslo menší než nebo rovna její číselnou argument.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované *číslo* argument je číselný výraz.  
  
 Vrácená hodnota je celočíselná hodnota, která je rovna největší celé číslo menší než nebo rovna hodnotě její číselnou argument.
