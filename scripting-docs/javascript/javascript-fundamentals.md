# <a name="javascript-fundamentals"></a>Základy jazyka JavaScript
[!INCLUDE[javascript](../javascript/includes/javascript-md.md)]je interpretovaný, na základě objektů skriptovací jazyk.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] Jazyk používá syntaxi podobná C a podporuje strukturovaná konstrukce, jako například `if...else`, `for`, a `do...while`. Složené závorky ({}) se používají pro vymezení blok příkazu. Jazyk podporuje různé typy dat, včetně `String`, `Number`, `Boolean`, `Object`, a `Array`. Zahrnuje podporu pro funkce Rozšířené datum, trigonometrické funkce a regulární výrazy.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)]prototypy používá místo třídy. Objekt můžete definovat vytvořením konstruktoru funkce.  
  
 [!INCLUDE[javascript](../javascript/includes/javascript-md.md)]je volného typu jazyk, což znamená, že jste nedeklarujte datové typy proměnných explicitně. V mnoha případech [!INCLUDE[javascript](../javascript/includes/javascript-md.md)] provádí převody automaticky, když jsou potřeba. Například když přidáte položku, která se skládá z textu (řetězec) číslo, číslo je převedeno na text.
