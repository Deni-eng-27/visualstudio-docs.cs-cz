# <a name="javascript-language-reference"></a>Referenční dokumentace jazyka JavaScript

JavaScript je skriptovací jazyk, který může být vložen do webových stránek a jiných aplikací.  
  
Implementace JavaScriptu od Microsoftu je kompatibilní se specifikací jazyka ECMAScript 5. Poskytuje také další funkce, které nejsou součástí norem ECMA.

> [!IMPORTANT]
> Společnost Microsoft se připojila k úsilí celé komunity, aby [MDN webovým dokumentům](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) webový odkaz na zastavení webu na jednom konci, počínaje přesměrováním všech 500 a odkazů na rozhraní JavaScript API společnosti Microsoft z [docs.Microsoft.com](https://docs.microsoft.com) na jejich MDN protějšky a pro účely historických a servisních přenosů odkazovali na odkazy na starší rozšíření js.
