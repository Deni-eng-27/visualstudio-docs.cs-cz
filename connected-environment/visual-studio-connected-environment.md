# <a name="vs-connected-environment-for-azure-container-service-aks-preview"></a>VS připojeném prostředí pro Azure Container Service (AKS) (Preview)
Prostředí připojení VS pomáhá vyvíjet s rychlostí na Kubernetes. Vytvořte prostředí na základě Kubernetes, která je plně spravovaná v Azure a optimalizována pro vývoj a interaktivně vyvíjet kontejnery v cloudu pomocí známých nástrojů, jako je VS Code, Visual Studio nebo příkazového řádku.

Tento přístup má několik výhod:

* Získá prostředí infrastruktury bez vývoj, který je typický pro produkční s úplným přístupem k prostředkům cloudu.
* Ladění kontejnerů Node.js a .NET Core přímo v Kubernetes s VS Code nebo Visual Studio. Všechny ostatní jazyky mohou být vytvořeny pomocí rozhraní příkazového řádku.
* Sdílet instanci Kubernetes v rámci vašeho vývojářského týmu, abyste ušetřili náklady a minimalizovat místního počítače nastavení pro nové členy týmu.
* Vývoj kódu v izolaci a do začátku do konce testování s ostatními součástmi bez replikace, nebo mocking až závislosti.
